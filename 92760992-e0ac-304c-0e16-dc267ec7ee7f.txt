　　在这种时候，谢雨晴想的根本不是吃醋，而是想要让她开心一下，毕竟她们经历的都是一样的痛苦。
<br/>　　电话里，谢雨晴没有透露具体内容，只说是有急事找她，让她赶紧来见自己。
<br/>　　“我现在……有一个很重要的会要主持，一会就开始了。”电话里，周静茹有些犹豫，“有什么急事，你电话里说。”
<br/>　　“很重要很重要的急事！我有个东西要给你看，必须你亲自过来！”谢雨晴突然有了种捉弄她的心理，毕竟自己已经是知情人，而且得知叶少阳没事，她也瞬间开心了起来。
<br/>　　“可是……”
<br/>　　“没有可是，你一定要来！好吧我去找你！给我个地址。”
<br/>　　周静茹说了会场的地址，谢雨晴立刻驱车赶了过去。二十分钟后，她在一家金碧辉煌的商务会所的大厅见到了周静茹。
<br/>　　周静茹又瘦了，神色很不好，穿着西装，脸上略施淡妆，有种强颜欢笑的感觉。
<br/>　　见到谢雨晴之后，她立刻迎上来，说道：“你有什么急事，赶紧说，我五分钟之后就要开会。”
<br/>　　“取消了。”
<br/>　　“什么？”
<br/>　　“我让你把会议取消了，延期。听了我说的事，你肯定没心情开会了。”
<br/>　　“那不可能，这个会十分重要，关系到我们家的一笔投资……”
<br/>　　谢雨晴笑道：“你这么有钱，怎么开会还亲自上阵？”
<br/>　　“当然了，重要的项目，当然只能自己上，不然对方也不会放心。你到底什么事啊，别耽误时间了！”两人太熟悉了，说什么也不用担心对方生气。
<br/>　　谢雨晴凝视着她，说道：“我有少阳的消息了！”
<br/>　　周静茹浑身颤抖了一下，态度猛然来了个一百八十度转弯，一把抓住谢雨晴的手，说道：“快说！”
<br/>　　谢雨晴把叶少阳写在灵符上的信拿出来给周静茹看。
<br/>　　周静茹坚持看完，几乎站不住，趴在谢雨晴怀里大哭起来。谢雨晴本来好多了，受到她的触动，也跟着一起哭。
<br/>　　这时候一个美女在大厅里找了过来，是周静茹的助理，会议快开始了，周静茹说出来见个人，半天却不回去，急匆匆地出来找，见到两人抱头痛哭，顿时就呆住了，等了好一会，周静茹趴在谢雨晴的怀里就是不起来，助理看了一下时间，只好小心翼翼地呼唤周静茹。
<br/>　　“周董，对不起，我不知道你们发生了什么，只是……会议要开始了，大家都落座了。”
<br/>　　“不开了不开了！”
<br/>　　果然如谢雨晴所言，周静茹现在满脑子都是叶少阳的事，哪里还有什么心情开会，说完之后觉得有点不妥，抬起头，泪眼婆娑地对助理说道：“你去主持，就说我生了急病，大不了我改天再接待。”
<br/>　　助理有点目瞪口呆，这可是一笔亿元项目，好不容易进入实际谈判的阶段，居然……
<br/>　　谢雨晴一摆手，说道：“你不是助理吗，遇到特殊情况，这是你发挥能力的机会啊，你去处理吧，你看她现在这个样子，就算去了也是把事情搞黄了，还是延期的好。”
<br/>　　助理一想也是，转身要走，看着两人的样子，忍不住劝道：“那个……我不知道发生了什么，但是，希望你们开心一点哦。”
<br/>　　谢雨晴对她挥了挥手，助理只好走了。
<br/>　　过了一会，周静茹缓和下来，已经不哭了，但还是趴在谢雨晴肩头不放开。
<br/>　　有几个人走过来，都对她们投来异样的目光，连服务生都装作无意地打量她们。
<br/>　　谢雨晴有点不好意思，拍了拍周静茹的肩膀，“喂喂，你先松开，这样别人看到还以为我们是拉拉在秀恩爱……”
<br/>　　周静茹听她这么说，只好坐起来，两个人又一起把信反复看了几遍，心情也逐渐好起来。
<br/>　　周静茹长出了一口气，喃喃道：“我就知道他会没事的，果然他没事……”
<br/>　　谢雨晴道：“是啊，只是他现在回不来而已。”
<br/>　　“那他是怎么样把这封信交给你的呢？”
<br/>　　谢雨晴把得到这封信的经过讲了一遍，周静茹听完，震惊不已。两人分析了一会情况，觉得应该把叶少阳身边的人都找来，大家一起分析，而且得到叶少阳的信息，自然要告诉他们。
<br/>　　周静茹立刻给张小蕊打电话，之后又联系老郭，二十分钟后，大家齐聚在老郭的棺材店里，看过叶少阳的信之后，大家都是百感交集。
<br/>　　“原来少阳去了1922年……”老郭怔怔发呆，“1922年，那是什么时节？”
<br/>　　“北洋政府时期，第一次直奉战争爆发，皖西掌控政权，当时中山先生在南方，徐图东山再起。”周静茹本身历史就不错，之前来的路上，又用手机搜了一下，更加确定了那段历史。
<br/>　　老郭听完，呆了半晌，说道：“那可真是个乱世啊。”
<br/>　　众人听了这话，心情顿时有点不好。
<br/>　　谢雨晴说道：“少阳是法师，什么样的乱世，跟他也没什么关系吧，他的目的是找到徐福，然后想办法回来。”
<br/>　　“世俗的力量还好，不会有什么问题，那个时代，据我所知，是法术界最后的黄金时代，妖魔横行，法术界也是人才辈出，少阳既然是法师，跟这些人未免不会产生关系……”老郭有些忧郁。
<br/>　　“郭大哥，你是怕，少阳在那个时代不好立足？”叶小萌说道。虽然老郭年纪大，但辈分上是叶少阳的外门师兄，自己又是叶少阳的堂妹，只能叫他哥，不能叫叔。
<br/>　　老郭脸皮抽搐了一下，看了她一眼说道：“别叫我哥，换个称谓。”
<br/>　　叶小萌道：“怎么了，我只能叫你哥啊，不然还能叫你什么，总不能叫你叔。”
<br/>　　老郭道：“不是这意思，你叫哥行，别叫郭大哥，咳咳，你又不是我蓉妹妹。”
<br/>　　“去你的，老不正经，美得你！”叶小萌破口大骂。
<br/>　　“敢占我姑姑便宜！”张小蕊一拳打在老郭肚子上，“而且还是在这个时候，你还有心情调戏妹子！”
<br/>　　张小蕊是练武出身，这一拳下去，老郭当场就惨叫起来，捂着肚子在柜台后面翻滚，缓了一会，委屈地说道：“我这不是得到了小师弟的消息，放心了吗，好心开玩笑缓解下气氛，你们还打我！”
