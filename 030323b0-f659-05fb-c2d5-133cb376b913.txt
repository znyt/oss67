　　苏莫抬眼看去，他的身前是一座山，一座有无数宝剑组成的剑山。
<br/>　　剑山占地方圆千丈，高达万丈，通体散发着惊心动魄的剑势，让人心神颤抖。
<br/>　　此时此刻，剑山之上，已经有了不少人。
<br/>　　公孙嘉，拓跋流明、韩生、古剑山等二十余人。
<br/>　　在苏莫到来之时，付元青已经到了，那云沐也几乎同时到了。
<br/>　　“秦大哥，若是蹬上这剑山之巅，便能到达最后一关！”
<br/>　　付元青来到了苏莫的身前，面色郑重，继续道：“不过，至今为止，从来没有人能登上剑山之巅。”
<br/>　　“嗯！”苏莫点了点头，他倒是想要看看，最后一关到底有什么。
<br/>　　如果到达了最后一关，能不能解开终极剑道的秘密。
<br/>　　“你叫什么名字？”
<br/>　　这时，一脸淡漠的云沐，转头看向苏莫，淡淡的问道。
<br/>　　“秦陨！”苏莫沉声说道，他不担心暴露身份。
<br/>　　“你很厉害！”云沐闻言点了点头，她在上一关收集道剑之时，遇到很多人均是没有道剑，而这些人的道剑均是被一个二品高级修为之人抢走了。
<br/>　　所以，她看到来到此地的苏莫，瞬间就明白了，苏莫就是那个抢夺道剑之人。
<br/>　　二品高级的修为，甚至抢夺了很多三品高级之人，这种实力的确很厉害。
<br/>　　“过奖了！”苏莫微微的摇了摇头。
<br/>　　云沐并未和苏莫多谈，简单的聊了一句之后，便身形一闪，落在了剑山最底部的长剑之上。
<br/>　　剑山，是由无数宝剑组成，每一柄宝剑，均是剑尖直指苍穹，有无形的锋芒冲天而起。
<br/>　　云沐落在剑山之上，立刻运转起了源力，守护己身，变得被剑山所伤。
<br/>　　随即，云沐抬起脚步，踩着一柄柄利剑，向上方山巅走去。
<br/>　　前方剑山的山腰之处，拓跋流明、古剑山等人，自然也看到了到来的苏莫和云沐，当时他们只是稍稍看了云沐一眼，便不在关注。
<br/>　　对于他们来说，登上山巅，此时重中之重。
<br/>　　传闻，进入最后一关，将有机会参悟道终极剑道的精髓，这是所有剑修的梦想。
<br/>　　苏莫没有急着蹬上剑山，他仔细的感受了一番，在这剑山之上，他能感受到一股危险的气息。
<br/>　　整个剑山，仿佛是一头沉睡的猛兽，一旦苏醒，能灭杀一切存在。
<br/>　　“秦大哥，我们上去吧！以你的天赋和实力，说不定真的能登上山巅！”付元青说道，此时此刻，他心中对于苏莫，并没有丝毫怨恨，反而还有一丝感激。
<br/>　　当然，更多的是钦佩，他从未见过苏莫这么妖孽的天才。
<br/>　　什么万古天骄，在苏莫面前，简直就是废骄，根本不值一提。
<br/>　　“这剑山不能飞上去？”苏莫向付元青问道，当然，他知道肯定不能，不然别人也不会一步步前进了。
<br/>　　“不能，一旦想要飞上去，剑山之中亏爆发恐怖绝伦的剑气，就算是神王都要被绞杀！”付元青摇了摇头。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，既然如此的恐怖，他自然不敢以身试险。
<br/>　　随即，他闪身落在了剑山最下方的边缘之上，双剑他在了两柄利剑的剑尖之上。
<br/>　　乍一登上剑山，苏莫便感觉到，脚下的长剑之中，涌入一股无匹的力量。
<br/>　　这股力量，无形无质，锋芒强大无比，堪比一名三品剑客的全力攻击。
<br/>　　苏莫不敢怠慢，体内玄力涌动，立刻动用源力护体，免得被这强大的锋芒所伤。
<br/>　　而付元青，更是谨慎，还未登上剑山，便用厚实的源力守护住了周身，才敢登上剑山。
<br/>　　苏莫没有理会付元青，抬起脚步，向剑山之上走去，没往上走一步，下方长剑的锋芒，便更强一分。
<br/>　　仅仅是走了十几步，剑山弥漫的锋芒，便增强了一倍左右。
<br/>　　付元青走的很慢，步步慎重，谨慎无比。
<br/>　　这时，剑山脚下，空间一阵波动，又有一人来到了这里。
<br/>　　这是一名身材矮壮的青年，面无表情，眸光极为犀利。
<br/>　　矮壮青年乍一到来，眸光有意无意的瞥了苏莫一眼，眼眸深处杀机一闪而过。
<br/>　　“小子，这一次定要让你死无葬身之地！”矮壮青年心中怒吼。
<br/>　　矮壮青年，自然便是千幻邪君所化，他没有急着出手，而是在等待机会。
<br/>　　千幻邪君面色平静，并未表现出什么，沉吟了片刻，便也蹬上了剑山。
<br/>　　来到此地的人不多，只有二十余人，目前的情况是，古剑山和韩生两人，稍微领先，距离剑山之巅只有三千丈。
<br/>　　而公孙嘉和另外几位三品高级的妖孽，稍微落后几百丈。
<br/>　　至于苏莫和云沐，因来的较晚，距离山巅还有九千余丈。
<br/>　　不过，正因为距离山巅较远，他们的速度，要比前面的人快不少。
<br/>　　“秦陨，你果然可怕！”
<br/>　　公孙嘉回头看了一眼苏莫，眸中精光闪烁，虽然现在苏莫的面容和形态，与当初完全不同，但是他现在已经认了出来。
<br/>　　因为，在公孙嘉看来，除了秦陨能有这么逆天，不可能再有其他人了。
<br/>　　而且，现在苏莫的修为，是二品高级，比以前提升了一级，完全在情理之中。
<br/>　　呼！
<br/>　　深深的吸了口气，公孙嘉没有过多关注苏莫，因为他已经举步维艰了。
<br/>　　他全身源力沸腾，在疯狂的抵御着脚下的剑之锋芒，已经撑不了多久了。
<br/>　　“打破极限，今日或许就是我突破的契机！”公孙嘉心中自喃，他的修为，停留在三品中级，时间已经不短了。
<br/>　　在剑山的强大压力之下，他正好借此突破境界，进入三品高级的地步。
<br/>　　“终极剑道的秘密！”
<br/>　　苏莫一边行走一边心中自语，他一定竭尽所能，登上山巅。
<br/>　　终极剑道，是一种无语伦比的道统，非常的重要。
<br/>　　每一步走出，他都感觉到，脚下的宝剑，传出无与伦比的锋芒。
<br/>　　这锋芒越来越强盛，让得他脚下的混沌之力，都不断的被磨灭。
<br/>　　苏莫的速度不快，但是相对于其他人而言，他的速度很快，比之云沐都所差不多。
<br/>　　静静是十几个呼吸的时间，他便向上心中了两千丈，可谓是紧跟在云沐的身后。
<br/>　　至于付元青，就很慢了，已经被千幻邪君超越。
