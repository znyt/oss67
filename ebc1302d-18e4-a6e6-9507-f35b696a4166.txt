　　修炼室外群情激愤，而修炼室内却是灵雾翻腾，滚滚涌动。
<br/>　　苏莫睁开了双眼，脸色一沉，眉头微皱了起来。
<br/>　　他自然听到了修炼室外的阵阵怒喝，也发现了一众巫族之人，全部都堵在了他的修炼室门前。
<br/>　　“不知所谓！”苏莫冷笑一声，随即再次闭上了双眼，继续修炼。
<br/>　　那些人想让他出去，他就偏不出去。
<br/>　　他丝毫无惧，反正修炼室内足够安全，外面根本打不开。
<br/>　　这些修炼室，建造的材料特殊，而且为了防止被人突然从外面打开，惊扰其内的人修炼导致走火入魔，均是有阵法防护。
<br/>　　除非苏莫在里面打开修炼室，不然的话，外面的人根本就进不来。
<br/>　　苏莫可不管外面那些人有多愤怒，只要他有足够的的灵气修炼，才不管那些巫族人的死活。
<br/>　　这些巫族人，可以说全部都是他的敌人，若不是实力不够，他会将这些人灭个精光。
<br/>　　当然，苏莫也不怕得罪这些人，因为在巫族之中，族规也是相当森严，严禁族人试下残杀，违者重重处罚。
<br/>　　轰隆隆！
<br/>　　苏莫再次沉浸在了修炼之中，体内墨色的玄力如同长江大河，奔流不息。
<br/>　　玄力中魔气滔天，距离转化为精纯的魔力已经不远了。
<br/>　　苏莫沉声的压力也是不小，这才是真正的魔功，远不是那种伪魔功可比。
<br/>　　苍穹世界的那些伪魔功，只是让玄力中蕴含魔气而已，根本不能称之为真正的魔功。
<br/>　　而这混天魔功才是真正的魔族魔功，修炼出来得是精纯的魔力。
<br/>　　魔气中蕴含这强烈的魔道意志，疯狂、暴戾，但苏莫固守心神，丝毫不受影响。
<br/>　　时间缓缓流逝，灵雾越发粘稠，苏莫身上的气息越发狂暴、浩大。
<br/>　　……
<br/>　　修炼室之外，众人喊了良久之后，修炼室中也没有任何动静，这让众人更加愤怒了。
<br/>　　“玛德，到底是哪个杂碎？”
<br/>　　“有胆搞鬼，就别躲在修炼室之中！”
<br/>　　“真是岂有此理，你一个人修炼，就让我们所有人无法修炼，简直是混蛋！”
<br/>　　众人纷纷怒骂出声，心中怒气冲天。
<br/>　　这修炼室内的人，若是一直不出来，在修炼室内待几个月，那他们岂不是几个月都不能用修炼室修炼了！
<br/>　　这也太坑爹了！
<br/>　　巫天御脸色阴沉如水，胸膛起伏，双拳捏的咯咯作响，恨不得一群轰碎修炼室。
<br/>　　“做缩头乌龟吗？我就不信你不出来！”冷笑一声，巫天御直接就地而坐，就在修炼室的门前等待了起来。
<br/>　　他就不信了，修炼室内的人会一直待在其内不出来，他倒要看看是何许人也，敢如此大胆。
<br/>　　“天御大哥，我陪你等！”巫长远也坐在了巫天御的身边，同样面色难看。
<br/>　　他倒是没有往苏莫身上想，虽然他没有发现苏莫，但只以为是离开了此地。
<br/>　　“草，老子倒要看看，里面到底是什么人？”
<br/>　　“此人若是出来，我第一个废了他！”
<br/>　　“我就不信他能在里面待一辈子！”
<br/>　　群情激愤的众人，没有人离开，均是和巫天御一样，纷纷决定继续等待。
<br/>　　他们倒要看看是何许人也，敢如此坑他们！
<br/>　　不仅没有人离开，此地反而是汇聚的人越来越多，很多刚刚进入地灵塔的人，见到此情此景不明所以。
<br/>　　询问之下，得知情况之后，也是心中震动。
<br/>　　不足一个时辰的时间，小小的第一层空间中，已经聚集了几万人，几乎人满为患。
<br/>　　而苏莫所在的修炼室，彻底被围的水泄不通，连只苍蝇都飞不出来了。
<br/>　　嘈杂的议论声此起彼伏，可谓是喧嚣震天，一片沸腾的场景。
<br/>　　“都让开！”
<br/>　　两个时辰之后，塔门之处响起朗喝之声，一名身穿褐色长袍的巫族老者大步走了进来。
<br/>　　“勋长老来了！大家都让开！”
<br/>　　有人急忙呼喊，随即，人群哗啦啦的分开了一条通道，直接通向苏莫所在的修炼室。
<br/>　　“怎么回事？”褐袍老者走到了修炼室门前，皱眉问道。
<br/>　　“勋长老，这座修炼室内，不知是哪位族人，也不知是用了什么方法，将龙脉中所有的灵气都吸了过去，让其它所有修炼室内都断了灵气，直接导致我们无法修炼！”
<br/>　　巫天御起身向褐袍老者抱拳，将事情简单说了一番。
<br/>　　这名褐袍长老名为巫勋，便是负责地灵塔事物的长老，职责便是维护地灵塔的日常运转。
<br/>　　“还有这种事？”巫勋长老闻言，立刻双眼一瞪，面现惊讶之色，他只是发现地灵塔内出问题了，才赶过来查看，事先并不知道是何事。
<br/>　　一座修炼室，居然将龙脉中的灵气都吸过去了，让得其它修炼室没有灵气？
<br/>　　这怎么可能？
<br/>　　他管理地灵塔也有一千多年了，可是从来没有遇到过这等状况。
<br/>　　仔细打量了一番眼前大门紧闭的修炼室，巫勋长老眉头紧皱，因为他发现的确是这样，龙脉中的灵气，正在疯狂的向修炼室内冲去。
<br/>　　这种灵气喷涌的速度，让他都感觉有些震撼，不知道的人还以为其内有个高阶武尊，正在吸纳灵气呢！
<br/>　　巫勋的脸色亦是阴沉了下来，这简直就是在捣乱嘛，如此疯狂的吸纳灵气，还让别人怎么修炼？
<br/>　　“勋长老，里面也不知道是谁？无论我们怎么喊，对方都不出来！”
<br/>　　“是啊！请勋长老打开修炼室的大门！”
<br/>　　“对啊，将这个混蛋揪出来！”
<br/>　　“此人真是太可恶了！”
<br/>　　众人纷纷开口，七嘴八舌，均是向巫勋长老表达自己的不满。
<br/>　　“嗯！”
<br/>　　巫勋闻言微微颔首，随即眼眸凝视着修炼室的大门，沉声喝道：“里面的小辈，请你立刻出来，否则，别怪老夫打扰你修炼了！”
<br/>　　巫勋没有直接打开了修炼室，毕竟，里面的人虽然有错，但若是正在修炼的紧要关头，他直接打开修炼室，难免会造成意外情况发生。
<br/>　　所以，他要先提醒一句，让里面的人有心理准备，或者让对方自行出来。
<br/>　　巫勋言罢，仅仅是两个呼吸的时间，修炼室内终于有了动静，仿若石质的修炼室大门缓缓的打开了。
<br/>　　一瞬间，所有人都睁大了眼眸，目光死死的盯着修炼室的大门，想要第一时间看清，修炼室内里面到底是何人！
