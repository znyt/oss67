　　苏莫通过楼梯，很快便来到了第二层的空间之中。
<br/>　　第二层的空间，与第一层无异，只不过更加的宽敞而已，仿佛有方圆数万里之巨。
<br/>　　半空之中，巫凌的投影已经在等他了。
<br/>　　嗖！
<br/>　　苏莫身形一纵，飞到了半空中，和对方相对而立。
<br/>　　“能闯到第二层，在我族可以称为真正的天才了！”巫凌的投影开口了，声音淡漠，没有丝毫的感情色彩。
<br/>　　苏莫闻言惊讶，难道这投影还有思维意识不成？
<br/>　　不过，他只是打量了一番，便否决了这个想法，巫凌的投影面无呆滞，双眼无神，根本就没有什么意识，就算是有的话，也只是有简单的战斗意识而已。
<br/>　　“斩！”
<br/>　　苏莫可没有兴趣和一个投影交谈，立刻试探性的斩出了一剑，剑光耀眼，魔威滚滚，直击巫凌的投影。
<br/>　　轰隆隆！
<br/>　　巫凌的投影再次出手了，同样是一指点出，击出了同样的一道指芒。
<br/>　　这一指明显的更为强大，磅礴的指劲形成了一根直径达十数丈的光柱，急速旋转，威势滔天。
<br/>　　光柱飞速迎向苏莫的剑气，磅礴的威势仿若一条巨大的怒龙，咆哮长空。
<br/>　　下一息，剑气和光柱狠狠的硬撼在了一起。
<br/>　　轰！
<br/>　　一声震天爆响，剑气和光柱纷纷炸裂，劲气席卷，浩浩荡荡。
<br/>　　“真是不凡！”苏莫有些惊讶，五成实力的投影，比三成实力的投影，强大的可不是一星半点。
<br/>　　但这还不足以阻挡他的脚步。
<br/>　　“死吧！”
<br/>　　苏莫厉喝一声，他全身魔气暴冲而起，形成滚滚狼烟，彻底展现出了最强的修为实力。
<br/>　　手中长剑化为了漆黑的魔剑，强大的剑芒暴涨百丈，剑威冲天。
<br/>　　咻咻咻！！
<br/>　　剑光爆闪，剑气飚射，铺天盖地的向巫凌投影汹涌而去。
<br/>　　这一次，苏莫甚至动用了郭剑青的长剑战魂，让得剑威更加强大，凌厉的锋锐之气，瞬间将他周身千里的空间全部粉碎。
<br/>　　这一击，是苏莫必杀的一击，区区五成实力的投影而已，瞬间便能斩杀。
<br/>　　面对着铺天盖地的剑气，巫凌的投影双掌齐出，连连拍击，顿时拍击出一道道巨大的掌印。
<br/>　　掌印如同一座座大山一般，飞驰而出，撼天动地。
<br/>　　大山般的掌印横亘在虚空之中，澎湃的劲力如同汪洋大海，深不可测。
<br/>　　轰轰轰！！
<br/>　　刹那之间，无数的剑气斩击在了一座座大山般的掌印之上，震天的巨响不断的响起。
<br/>　　一座座掌印顿时被撕裂，铺天盖地的剑气所向披靡，无可阻挡，瞬间便将巫凌的投影淹没。
<br/>　　少倾之后，剑气消散，一切都回归了平静，巫凌的投影已经消失不见。
<br/>　　苏莫微微的摇了摇头，这种投影虽说有巫凌五成的实力，但其实远远达不到五成。
<br/>　　因为这投影的力量虽强，但太过死板，根本不懂得变通，完全不可能像真正的武者那般，随机应变，发挥出真正的战斗力。
<br/>　　没有真正武者一般的战斗意识，剩下的只会是蛮力而已，不值一提。
<br/>　　而且，投影终究是投影，没有战魂的力量，实力再次大打折扣。
<br/>　　随即，前方再次的出现了一道古朴的楼梯，苏莫闪身飞上楼梯，向三楼走去。
<br/>　　此时的外界，几乎有九成九的巫族天才都被淘汰了，不是他们不够强，而是巫凌的投影太过强大了。
<br/>　　无数的巫族之人围拢在三层宝塔的四周，很多被淘汰而出的人均是面带不甘之色。
<br/>　　啊！
<br/>　　就在此时，一声惨叫声响起，一名高大的巫族青年口喷鲜血，从塔门处抛飞了出来，重重的砸在了地上。
<br/>　　“是巫同！”
<br/>　　“巫同居然被淘汰了！”
<br/>　　“怎么可能，以巫同的实力怎么可能会败？”
<br/>　　众人见到高大青年被淘汰，顿时一片哗然，因为这名叫巫同的青年乃是族中一名超级天才。
<br/>　　虽然其没有被排进少司候选榜，也也是仅次于少司候选榜上那些超级天才的人，如此天才人物，居然败了！
<br/>　　这让众人心惊不已，难道这三层宝塔，如此的难闯吗？
<br/>　　“巫同大哥，你怎么样？”立刻有几名天才上前，将巫同扶了起来，面带关切的问道。
<br/>　　巫同起身，摸了把嘴角的血迹，转头看到众人灼灼的目光，顿时脸色涨红。
<br/>　　身为族中的超级天才之一，连第一关的三层宝塔都没有闯过去，实在是让他颜面扫地。
<br/>　　“第三层的投影实力非常恐怖，而且战斗意识仿佛像是真正的巫凌大哥一般，我败的不冤！”深深的吸了口气，巫同沉声说道。
<br/>　　“什么？”众人闻言心中一惊，他们基本上都没有闯到第三层，自然不清楚第三层的情况。
<br/>　　原来第三层的投影这么强大，战斗意识和巫凌大哥无异，并不像第一层和第二层一般，投影没有什么意识，战斗起来非常僵硬。
<br/>　　这还了得！
<br/>　　如巫凌大哥一般的战斗意识，即便是同阶一战，且只有七成的实力，也恐怖到难以想象了。
<br/>　　如此的话，还有多少人能闯过三层宝塔，到达塔顶？
<br/>　　“这第一关的三层宝塔，能闯过去的人，绝对不会超过两百个！”巫同面色凝重的说道，不知道其说的是真实情况，还是在给自己的失败借口。
<br/>　　“一群废物！”
<br/>　　就在此时，一声淡漠的声音响起，只见塔顶之上的巫人王睁开了眼眸，扫视了一番众多被淘汰的人，眸中尽是不屑之色。
<br/>　　百万族中天才进入塔中，居然快被淘汰光了，这让巫人王非常的无语。
<br/>　　这些族中所谓的天才人物，简直就是一个笑话，居然还敢号称为天才，真是让他心中非常不屑。
<br/>　　“希望巫玛和巫幽兰能有点本事，陪我玩一玩，不然当真无趣！”巫人王暗叹一声，便再次闭上了眼眸。
<br/>　　在他心中，整个族中的天才之中，也就巫玛和巫幽兰两人能勉强和他争一争，其他人不过是走个过场而已
<br/>　　很多人都听到了巫人王侮辱性的言语，不少人均是脸色一沉，但却是敢怒不敢言。
<br/>　　没办法，人家有嚣张的资本，有藐视一切的底气。
