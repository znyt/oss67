　　当苏重恢复意识的时候，一股来自全身的痛疼，一波波猛烈冲击大脑。
<br/>　　“又是一副频死的身体。”苏重脸色不变，强烈的疼痛已不能让苏重动容。
<br/>　　几乎每一次降临，他都没有好下场。附体之人，无不是各种饿死冻死打死。即便不死，也是剩一口气那种，他都已经习惯了。
<br/>　　不急着睁开眼，苏重强大精神探入体内，一副身体三维解剖图，出现在脑海。
<br/>　　随着苏重不停穿越，他的灵魂早在不知不觉中发生变化。附身寻常身体，几乎瞬间就能驾驭，并突破身体障碍，做到清晰内视。
<br/>　　“肋骨断裂五根，大腿骨折断。膝盖碎裂，内脏多出轻微破损。这难道是被火车给撞了？”
<br/>　　就这伤势，怪不得前任直接一命呜呼。
<br/>　　刚想到身体前任主人，一股精神意识突兀冲击苏重灵魂。五光十色，各种声音图片纷至沓来，这是前任的意识碎片。
<br/>　　这种灵魂冲击杂乱无序，能瞬间冲垮灵魂防御！冗杂记忆肆虐，如果换了别人，第一时间就会记忆错乱。要么精神分裂，要么直接成傻子！
<br/>　　苏重却对这种危机视而不见，灵魂波动平稳非常。更加纯粹的神魂，让苏重保持清醒，甚至有时间去浏览前任记忆。
<br/>　　影武者联盟、忍者大师、地狱式训练、坠崖受伤……
<br/>　　好半晌，苏重陆续理清了杂乱记忆，同时也判断出这是什么世界。
<br/>　　“蝙蝠侠，而且是诺兰电影版蝙蝠侠。”苏重眉头微皱，这可不是个好消息。
<br/>　　诺兰版的蝙蝠侠更着重写实。把有些玄奇的蝙蝠侠真实化。这里没有什么超能力，只有高超的格斗技巧，各种合理又强悍的科技装备。
<br/>　　电影虽然好看，但对苏重就不太好了。
<br/>　　越接近真实世界，对超自然能力的压制就越大。没有超自然能力存在空间，苏重就只是一个普通人。遇到身体前任的那种伤势，他也会死！
<br/>　　“醒了就睁开眼，不要装睡。”一个冷冰冰的声音突兀响起。
<br/>　　苏重心中一紧，他竟然没发现房间内有人！
<br/>　　睁开眼看过去，一个身穿褐色武士服的黑人，静静的站在墙角阴暗处。
<br/>　　木质墙壁的颜色和对方衣服颜色一致，而且对方有意放缓呼吸，如果不是他出声，闭着眼睛的苏重根本就无法发现对方！
<br/>　　“影武者联盟果然不简单，真不愧是擅长潜伏暗杀的黑暗组织。”苏重立刻收起心中杂念。
<br/>　　自己刚到，就一头扎进了影武者联盟。这里的人，可各个都是冷血刽子手。一旦行差踏错，绝对会迎来杀身之祸！
<br/>　　“看来命是保住了，好好养伤吧。”壮硕黑人面无表情道，苏重却从对方眼中看到一丝冷然。
<br/>　　砰的一声门被关上，房间内再次回复寂静。
<br/>　　门外突然传来一阵小声交谈。
<br/>　　“他伤势怎么样？”一个陌生的声音问道。
<br/>　　“膝盖受伤，废了。”壮硕黑人的声音响起。
<br/>　　“看来是没用啦。”
<br/>　　“嗯，过两天就清理掉，这里的位置不能暴露。”壮硕黑人声音冰冷：“下面那些人是干什么吃的，怎么会送这么一个废物上来，就连基础训练都不过关！”
<br/>　　“嘿，咱们的训练能和下面的人一样吗？这家伙也倒霉，受伤死了不就完了，为什么还要活过来，还得再挨一刀……”
<br/>　　声音越来越远，渐渐不可闻。
<br/>　　两人声音非常小，本不应该被其他人听见。但苏重神魂强大，这让他五感敏锐。壮硕黑人和陌生人的对话，让他听了个一清二楚。苏重脸色顿时难看起来。
<br/>　　“果然不愧是影武者联盟，为保住基地位置，自己人竟也直接灭口！”苏重脸色冷然。
<br/>　　不管这个世界怎么样，当务之急是先保住小命！
<br/>　　“肋骨骨折，大腿折断，这都不是问题，只要时间足够，肯定能养好。最严重的问题是膝盖！”苏重勉强半坐起来，若有所思的打量膝盖。
<br/>　　此刻他们正在青藏高原，身处冰原山顶与世隔绝。骨折骨裂，这些伤势虽重，但依靠影武者联盟医术，这点儿伤根本不算事儿。
<br/>　　由于训练残酷，断手断脚是很常见的事情，这里的医生，治疗跌打损伤很在行。
<br/>　　但膝盖碎裂却完全不同，这里是高原冰山，地处偏僻设备有限。即便医术高超，也无法治疗骨骼粉碎性伤势。
<br/>　　而且即便能动手术治疗，也无法让膝盖骨完全愈合，这势必影响正常发力运动。所以壮硕黑人才会说，苏重的身体废了。
<br/>　　“必须治好膝盖！”
<br/>　　苏重倒是知道，这个世界科技极为发达，有能够代替关节的机械外骨骼。蝙蝠侠后期就曾膝盖受伤，装了一个外骨骼装置，不仅能正常行走，力量还更加大！但这里是在深山冰原之顶，去哪里找外骨骼。
<br/>　　“还得想其他办法。”闭上眼睛，苏重仔细感知膝盖伤势。
<br/>　　整个膝盖骨碎裂成十多块，只是经过了短暂的拼接，有的甚至还拼接不正！这种姿势愈合下去，百分百得废！
<br/>　　“必须重新拼接！”苏重忍着肋骨疼痛，半坐起来，快速拆掉膝盖处绷带。
<br/>　　也不管伤口血肉模糊，双手按在膝盖上，直接开始挤压挪动骨骼！刚刚止住的血液再次横流。
<br/>　　额头很快就冒出细密汗珠，苏重却脸色不变。始终闭着眼睛，完全依靠精神内视指引。
<br/>　　好一会儿，苏重才把大块骨骼拼接正确。但周围还散落着不少细小碎屑。怪不得壮硕黑人说苏重的腿废了。
<br/>　　影武者基地条件有限，膝盖碎裂骨屑无法完全取出，即便骨骼接的再好，他这条腿也得废！
<br/>　　睁开眼，看着血肉模糊的膝盖，床单已被血液染红。苏重一发狠，拔出放在床边的武士刀，朝着膝盖就下刀子。
<br/>　　皮肤割开，血液流的更快了。苏重不管不顾，眼神冷静如冰，武士刀弹入肉中，精准找到骨骼碎屑。刀尖跳动，数块血色骨头被挑出。
<br/>　　他动作迅速，又有内视精准扫描，只用五分钟，就把碎骨头完全挑出。
<br/>　　当啷一声把刀仍在地上，苏重脸色苍白如纸，眼前一阵发黑。
<br/>　　猛然一咬舌尖，苏重狠狠刺激身体，不让身体昏厥。然后拖着虚弱身躯，快速却又仔细的敷药包扎。
<br/>　　等到处理好膝盖伤势，他这才来得及长出一口气。如果不是他神魂强大，根本就扛不住身体的应激反应。说不定治疗到一半，就直接晕过去了。
<br/>　　好在撑过最艰难部分，剩下的治疗，已不需要这么惨烈。
<br/>　　“膝盖骨骼已经正好，下面就是骨骼愈合。”苏重眯着眼睛。
<br/>　　他刚才已经尝试过，这个世界确实没有超凡力量。道种根本无法凝聚，内力不能修炼，念气也不能聚集。
<br/>　　但是，气血却可以搬运！
<br/>　　苏重穿越数个世界，掌握秘法无数，其中就有催动气血，加快伤势恢复的妙法。
<br/>　　“单纯催动气血，速度并不理想。如果能结合念能力‘缠、绝’两字秘诀，锁住身体生机，必然能让愈合速度加快！”
<br/>　　想到就做，苏重强忍着身体的疲倦，集中全部精力，开始搬运气血治疗伤势。
<br/>　　时间紧迫，他必须在对方动手灭口前，恢复一部分伤势。让影武者联盟知道，自己并没有废，进而获得喘息之机！
