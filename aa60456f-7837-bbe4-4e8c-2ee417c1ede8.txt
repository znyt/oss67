　　白忧的情绪失控吓惨了一帮白家人，作为白家这一代长期以来唯一的嫡系，并且还是从小·便是少族长代理族长处理各种家族中事，白忧向来很擅长控制自身，无论是情绪、心态还是行为。
<br/>　　他们一个个都震惊原地，彻底将前方正在变化的道路抛之脑后，所有白家人都双目一眨不眨的盯着白忧，好似看到的是一个完全陌生的人。
<br/>　　白家人见过白忧冷漠将其他白家成员杀掉的画面，也见过白忧面无表情下达屠杀某个基地命令的画面，更见过白忧一拳砸烂了白家大宅的主宅，从满是尘埃之中走出却一脸淡然的样子。
<br/>　　白忧就是这样的人，无论在何种场合何种状况何种心情之下，也不会将自己内心最真实的一面暴露在外，他的情绪表达向来是通过残暴的行动。
<br/>　　所以此时在白忧情绪失控暴走之时，无论是大吼大叫还是他面上扭曲的表情，都让白家人有种从未认识过这个嫡系少爷的感觉。
<br/>　　罗杰也好不到哪去，在与白忧一样想通了某种可能后，便对自己将罗梵调走甚至死在狼牙手中的事万分后悔。
<br/>　　唯一能够让阴谷出口道路改变的只有火族人，所以当时罗梵说的话完全属实，那个火焰小球也正是来自于火族人之手。
<br/>　　但这一切都因为罗杰当时的一念之差，而忽略……
<br/>　　两人领悟到这一点的时间太晚，也对当下的情况没有任何改善作用，反而加剧了内心的恐惧，以及浪费时间的悔恨。
<br/>　　而白忧的情绪失控将众人的注意力都吸引过去，导致没有人关注到其他危险，从而引发了更加严重的后果。
<br/>　　当所有人都只注意到那移动的道路之时，忽然间——
<br/>　　轰！
<br/>　　一声好似什么东西爆裂的声音，伴随着一股焦味的飘出而响起。
<br/>　　众人都回过神，瞬间看向了声音传来的方向，紧接着他们便看到一名白家人被一团大火包裹着，身上各处地方都发出‘噗噗’的声音，火焰之中的黑色人影在晃动，但却越来越来细小，渐渐只能见到一具模糊的骨架。
<br/>　　白家人集体傻眼，完全无法相信眼前看到的一切，这个被巨大火焰一下子烧死，甚至血肉都尽毁只剩下骨骼的白家成员，是他们之中非常强大的一人，更是白家第五旁支的传人！
<br/>　　白家的族谱与其他三大神秘家族都不同，不仅嫡系旁支分的尤其清楚，各个旁支内更有着独立的生育模式，也就是说哪怕是白家的旁支成员，血脉浓度也不能低于某个数值，从而确保整个白家的血脉浓度不会与千年前相差太远。
<br/>　　而第五旁支的传人，自然便是这一旁支内血脉最浓的年轻一代，更是综合能力及战力都绝顶的天才。
<br/>　　很多白家人甚至觉得，白家的这几名出色的旁支传人，都比其他家族的嫡系强大。
<br/>　　可如此备受家族重视，在白家众人眼中的强大之人，竟然被火烧死，甚至还是分秒钟内就被烧到不复人形，这种冲击让他们感到很挫败，以及无尽恐惧。
<br/>　　如果是烧到他们身上，他们还活得成么？
<br/>　　这样的想法瞬间让白家人气势大减，原本信心十足冲进阴阳谷内探索的决心也付之东流，取而代之的便是低迷和退缩感。
<br/>　　人在感受到极大生命威胁的时候，往往下意识的反应便是强烈抵触，然后调头就跑，哪怕是高高在上的神秘家族成员也是一样。
<br/>　　当那名白家人被烧成一具漆黑焦炭的身躯倒下，发出‘啪’的声响，便像是某种信号，顿时让在场的所有罗家人和白家人‘哗’的一下四散而开，场面混乱的与末世初期的人类看见丧尸时没有任何区别。
<br/>　　此时哪还有什么队伍，一百多人看见道路就钻，也不管前方那片不断移动的出口迷宫会不会困住他们，于是在罗杰和白忧还未来得及出声管制的情况下，一百多人就当场走散。
<br/>　　一部分人迷失在了阴谷的出口迷宫里，一部分由于太过于惊慌害怕，像是无头苍蝇般的到处乱窜，还有一部分人在一开始就跑错了方向，反向往阴谷入口处跑，妄想能够再次回到那个沙滩然后原路返回。
<br/>　　白忧和罗杰是各自家族的嫡系，更是少族长，在被家族长期栽培到现在，最起码也有一定的应变能力。
<br/>　　两人在最初的惊慌失控过后，便很快冷静了下来，白忧反应更快一些，当即快速上前挡住了那条出口道路，将那些还想要冲进去的两族众人拦住。
<br/>　　罗杰慢了一拍，但也赶紧上前大声呵斥：“都乱跑什么！”
<br/>　　于是那还未走散的四十名两家族成员，终于不再处于暴动般的状态，虽然依旧无法战胜内心的恐惧，但还是渐渐聚集在两名嫡系的身旁，他们拥挤在一起根本不顾你是白家人还是罗家人，与刚进来时的高傲样子判若两人。
<br/>　　慌张大叫的声音也在此时渐渐硒鼓，周围除了彼此间由于过度紧张而粗重的呼吸，便再没有一丝声音。
<br/>　　罗杰手心里满是汗水，他不由的看向不远处，被秒杀的白家人骨骼依旧浸泡在通红的火焰里，哪怕已经没有东西可以烧，那火焰也根本没有熄灭。
<br/>　　白忧同样望向那一方向，只是他考虑的更多。
<br/>　　这团火焰是从哪里来的，为何有代表了火族的火焰在这，却没有任何火族人在他们的视野里？
<br/>　　而在这名白家人身上着火的刹那，他们为什么没有听到此人叫喊，只听到了一阵诡异至极的爆裂声？
<br/>　　种种问题在白忧的脑海一掠而过，随着思考的时间越长，他们这四十人便越发惊恐到不受控制。
<br/>　　依旧是什么都没有，若非不远处的火焰依旧还在，那个白家人的确是被烧成了漆黑的骨架，恐怕还会有人以为刚才所见都是幻觉。
<br/>　　“总不能一直在这里坐以待毙。”罗杰终于忍不住的开口。
<br/>　　白忧看向了身后，依旧完全变成了另一幅模样的出口道路：“出口已经变成了迷宫图形，那些已经进去的人恐怕……”
