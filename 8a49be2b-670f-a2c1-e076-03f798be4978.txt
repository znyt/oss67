　　白允儿眯起眼睛，心头不禁疑惑，这高度可不低，连她都无法看到这悬崖有多深。
<br/>　　死了？
<br/>　　白羽也在这时快速跑来，心惊肉跳的看着眼前的悬崖：“跳下去了？会摔死的吧？”
<br/>　　只是在他话音刚落之际，忽然一阵风声响起，呼呼的从悬崖底部传来，紧接着两人便看见一只巨大的飞行丧尸呼啸而上，从下方深不见底的悬崖冲出，进入了两人的视野。
<br/>　　而在这飞行丧尸的背上，那个跳崖的可怕对手就这么站在那，幽幽的望着远处的白允儿。
<br/>　　白羽心惊不已：“果然没死，但飞行丧尸怎么会去接住他？”
<br/>　　此时那飞行丧尸已经远离消失在高空天际，白允儿忽的将右手靠近鼻子下方，轻轻嗅了嗅。
<br/>　　见到这一幕的白羽一个激灵，目瞪口呆的望着白允儿：“小姐姐？那是血！”
<br/>　　“恩。”白允儿的神色有些古怪，盯着银色手套上的血渍目露探究之色：“异种的血，味道会变么？”
<br/>　　“你别吓我！”白羽惊悚无比。
<br/>　　他实在难以想象这些话会从白允儿的口中说出，这根本不像是一个身穿连衣裙的冷美人，而更像是丧尸或异种的行为。
<br/>　　白允儿却对白羽的反应漠不关心，只是对手上这血液的味道产生了疑虑，能操控飞行丧尸的必然是异种才对，而刚才那人无论是战力还是反应速度都无可挑剔，能与白允儿这等存在比肩的，恐怕也就只能是异种王了吧？
<br/>　　大惊之后的白羽也回神反应了过来，惊诧道：“你是说刚刚那个人是异种？”
<br/>　　白允儿顿了顿，道：“异种王才对。”
<br/>　　白羽瞳孔一缩：“异种王不是白忧的……他怎么这么强？难道我在银市见到的那个……”
<br/>　　白允儿嘴角勾了勾：“谁知道呢！”
<br/>　　四周一片废墟地带也已经结束了战斗，这个猎者联盟总部早已被毁于一旦，另一面的冷潭处到处都是尸体，在白允儿与此人缠斗的时候，那群丧尸就已经以可怕的速度攻占了这里，在此所剩不多的人类也全部难逃一死。
<br/>　　大批的丧尸群在结束战斗后，就好似退去的潮水，一下子消失在了这片山野中，这片地带回归了往日的宁静，若非一目了然的残破景象，根本难以想象这里发生过什么。
<br/>　　率领着丧尸群离开的几名异种快速的集合在他们来时的山头，此处依旧一片白雪皑皑，冷风中那只巨大的飞行丧尸静立在一侧，它的旁边是一块岩石，与白允儿缠斗的那人就依靠在那里，包扎着自己脖颈及面部的伤口。
<br/>　　“王！您受伤了！？”
<br/>　　异种们见到这一幕，纷纷大惊失色，全体都不可置信的望着雪地上的腥红色。
<br/>　　“不要叫我王，我是木叶。”木叶的声音很平淡，斯条慢理的一点点清理着伤口。
<br/>　　异种们回过神，看着木叶欲言又止，木叶的受伤实在让他们心惊不已，毕竟自从木叶统领了异种这一种族，并一次次暗中崛起的时候，就没有人见过木叶真正的本尊受过伤。
<br/>　　可今日·他们攻打猎者联盟，本该是一次轻而易举结束的战斗，甚至损失的丧尸数量都少于当初预计，却偏偏最不可能受伤的人受了伤。
<br/>　　看木叶这么长时间都未能彻底止住伤口流血的样子，可见受伤还不轻！
<br/>　　“去打听一下，白允儿的武器是什么。”木叶开口吩咐着，依旧平平淡淡。
<br/>　　“是。”一名异种立即离开，马不停蹄，显然能让木叶受伤的武器，已经令他们所有异种都警惕起来。
<br/>　　“那个白允儿真的这么强？”有异种好奇出声：“若是我到达十阶，和她一战的胜率能有几成？”
<br/>　　“零。”木叶回答，言简意赅。
<br/>　　四下一片死寂，所有异种都说不出话，被这一数字深深震撼，他们很清楚当下以这种平淡口吻回答的木叶，不是在开玩笑，这是他最认真的状态。
<br/>　　良久后，一名异种叹气，不禁疑惑道：“不是说白家最强的是白忧，那个白允儿不足为据，甚至很多白家人还不把她当回事么？”
<br/>　　“但是这个白允儿却能伤到王，这与我们拿到的情报相差太大了！”
<br/>　　“那白忧该更强？”
<br/>　　异种们都安静下来，气氛也逸散着一种压抑，他们看着木叶都有些畏惧的不敢说太多。
<br/>　　此时的木叶已经将伤口整理好，他的面上还带有一条长长的血线，正在缓慢愈合，这样的恢复速度在他这个等阶来说，实在是慢的不可思议，同时也更加突显出白允儿那双手套的特别。
<br/>　　木叶起身跳上飞行丧尸的背部，声音依然平淡：“去下一处。”
<br/>　　呼！
<br/>　　飞行丧尸一跃而起，异种们也四散而开，各自带着控制的丧尸群消失在这片山野。
<br/>　　——————
<br/>　　猎者联盟总部被毁于一旦的事还未发酵，另一则消息更先一步的爆炸在世界各处。
<br/>　　上京基地的两名老者接到了来自段氏基地的急报，内容有些诡异，让两名老人相互沉默了良久。
<br/>　　“段江伟说统计下来，这次收到波及的全部都是猎者联盟的猎者。”牧司令说着，眉头一直紧皱：“当下死亡人数高达三百之多，反观段氏基地的防守部队，人数却一个都没少，甚至很多迷失在边缘地带的人都找了回来，哪怕丧尸群就在附近，也没有对他们发起进攻。”
<br/>　　“只死了猎者联盟的人，这也太针对了吧？”洛冥眯着眼睛，思索道：“怎么看怎么像是有人刻意对猎者联盟下手，想要借此机会将其铲除。”
<br/>　　“神秘家族？”牧司令冷笑，道：“具体一点，神秘家族的白家。”
<br/>　　洛冥点头：“猎者联盟最大的克制对象就是神秘家族，对这个组织深恶痛绝的也是神秘家族，尤其是白家，那个常年不出关的白家族长，最希望看到的就是猎者联盟覆灭，还有某个大人物的身死。”
<br/>　　“其内部的辛秘我们调查这么多年，也只知一二。”牧司令摇摇头，道：“但若真是神秘家族在勾结丧尸瓦解人类内部联盟，我们这些老家伙也不能再隐藏底牌了！”
