　　延绵无际的蛮荒山脉，高.耸入云的蛮荒圣山，亘古长存。
<br/>　　一道挺拔的身影，脚踏虚空，瞬息千里，跨越长空，来到了蛮荒圣山之前。
<br/>　　身影便是苏莫。
<br/>　　身形落在圣山的通道口，苏莫长身而立，静静的等待了起来。
<br/>　　唰！
<br/>　　仅仅只是过了两个呼吸的时间，身影一闪，光头大汉来到了苏莫的身前。
<br/>　　随即，光头大汉仔细打量着苏莫，眼眸微微眯了下来。
<br/>　　“武皇境八重境界？”光头大汉面带惊疑之色，不过，他并不是惊疑苏莫修为提升的太快，反而是太慢了。
<br/>　　因为以前苏莫的修为提升速度，实在太快了。
<br/>　　按照他的猜测，这么几年过去了，苏莫现在应该有武皇境九重，乃至准帝之境了！
<br/>　　“怎么？象前辈还想再切磋切磋？”苏莫淡笑着问道。
<br/>　　“呃……！”光头大汉闻言一怔，随即干笑道：“算了，没心情！”
<br/>　　开玩笑！
<br/>　　苏莫武皇境五重修为之时，都能打败他，现在武皇境八重的修为，那还怎么打！
<br/>　　“跟我来吧！”
<br/>　　随即，光头大汉招呼苏莫一声，便向圣山内部走去。
<br/>　　苏莫跟随光头大汉，很快便来到了圣山内部，那几座巨石垒砌的宫殿之前。
<br/>　　此刻，其中一座宫殿之前，黑衣中年人正面色恭敬的站在殿门之前。
<br/>　　这座宫殿不属于光头大汉，亦不属于黑衣中年人，而是属于他们的大哥。
<br/>　　苏莫见此，顿时双眉一挑，难道这一次能见到蛮荒三妖中的老大？
<br/>　　“前辈！”来到黑衣中年人的身边，苏莫向对方微微抱拳。
<br/>　　“嗯！”黑衣中年人点头，但并未说什么，他的眼眸只是静静的望着眼前的宫殿。
<br/>　　光头大汉深吸口气，亦是面色郑重了起来。
<br/>　　苏莫见两人都在等待，便也无奈的等待了起来。
<br/>　　三人约莫等待了半刻钟的时间，苏莫清晰的感觉到，这座宫殿之中，有一股强大的气息，在缓缓的苏醒。
<br/>　　这股气息浩翰苍茫，仿若沉睡了无尽的岁月。
<br/>　　“八级妖兽？”苏莫立刻心中惊讶，这宫殿内的存在，分明是堪比武帝的八级妖兽！
<br/>　　原来蛮荒三妖中的老大，居然是武帝强者！
<br/>　　苏莫意外不已，原来苍穹大陆，居然还有武帝强者存在。
<br/>　　“进来吧！”片刻之后，宫殿中传出低沉沧桑的声音。
<br/>　　随即，黑衣中年人轻轻推开了宫殿的大门，带着光头大汉和苏莫走进了宫殿之中。
<br/>　　宫殿幽深，三人走到了宫殿深处，一间密室之前停了下来。
<br/>　　而后，黑衣中年人再次推开了密室的大门。
<br/>　　只见密室不大，约莫方圆五丈左右，昏暗无比，一名消瘦的身影正盘膝在地，背对着苏莫等人。
<br/>　　苏莫眸中精光一闪，此人给他的感觉深不可测，他也看不对方的修为。
<br/>　　暗暗催动幽冥魔瞳武魂，苏莫眸中绿光一闪，顿时看清了对方的修为，八级一重巅峰妖兽。
<br/>　　“大哥！”
<br/>　　“大哥！”
<br/>　　三人走进密室之中，黑衣中年人和光头大汉，立刻向消瘦的身影恭敬行礼。
<br/>　　苏莫倒是没有见礼，只是静静的打量着对方。
<br/>　　少顷，消瘦的身影身体自动旋转，其面目展露在了苏莫的身前。
<br/>　　这是一个面容苍老的老者，鹰钩鼻，眼眸狭长，嘴巴尖锐，非常的丑陋。
<br/>　　老者的目光，第一时间便落在了苏莫的身上，四目相对，两人都看出了对方眼中的惊讶。
<br/>　　苏莫心中微震，这是一个强者，绝对的超级强者，虽然对方和巫森同样是武帝境一重，但他有一种感觉，此人绝对比巫森强大十倍都不止。
<br/>　　而老者显然也看出了苏莫的不凡，眸中精光一闪，变得凌厉无比。
<br/>　　“大哥，我给你介绍，此子是苏莫，如今已经是苍穹世界人族第一强者！”
<br/>　　黑衣中年人为老者介绍，他最近几年也了解了一番苍穹世界的局势，知晓苏莫的地位和实力。
<br/>　　不过，黑衣中年人只是说苏莫是人族第一强者，并不包括妖族。
<br/>　　显然其不认为苏莫的实力，在苍穹世界无敌，因为还有他的大哥存在。
<br/>　　“苏莫，这是我们的大哥！”随即黑衣中年人又为苏莫介绍。
<br/>　　“前辈！”苏莫闻言，向消瘦老者微微抱拳。
<br/>　　“武皇境八重修为，人族第一强者！”
<br/>　　消瘦老者低声自语，这让他微微有些意外，武皇境八重修为，都能成为人族第一强者，现今的苍穹世界，已经如此没落了吗？
<br/>　　“二弟，时间快到了吗？”消瘦老者向黑衣中年人问道。
<br/>　　“快到了，就在这两年左右吧！”黑衣中年人点头。
<br/>　　消瘦老者闻言，眸中犀利的精芒一闪而过，仿佛他等这一天，已经等了很久很久了。
<br/>　　“大哥，苏莫的实力很强，你们两人联手，机会很大！”关头大汉开口道，声音洪亮无比。
<br/>　　消瘦老者点了点头，他也看出了苏莫的不凡，而且，二位义弟能将苏莫带来见他，便说明苏莫的实力，比其两人要强很多。
<br/>　　“坐吧！”老者一挥手，一阵微风卷过，原本密布灰尘的石板地面，变得明亮洁净。
<br/>　　随即，苏莫三人便在消瘦老者身前，盘膝坐了下来。
<br/>　　“小友，老夫秦不死，幸会！”消瘦老者微微抬拳，自我介绍。
<br/>　　苏莫闻言一怔，秦不死？居然还有这种名字？
<br/>　　不过，他还是同样抬拳回礼。
<br/>　　“小友，大劫将至，不知你有何想法？”秦不死开门见山的说道。
<br/>　　“杀！”苏莫淡淡的说道，对于对方说及大劫，他并没有什么意外。
<br/>　　毕竟，以对方的修为实力，大劫首当其冲。
<br/>　　“只有杀吗？”秦不死闻言，嘴角勾起了一缕笑意。
<br/>　　苏莫见此，略一沉吟，沉声道：“逃出去！”
<br/>　　“不错！”
<br/>　　秦不死闻言点了点头，道：“苍穹世界历代的巅峰强者，都想逃出去，逃出这个巫族圈养之地，看来你也不例外！”
<br/>　　“以前有人成功过吗？”苏莫问道，若是前人能走通，他们也就有希望了。
<br/>　　“不清楚，或许有吧！”
<br/>　　秦不死摇了摇头，叹气道：“老夫经历了三次大劫，深知此事的难度！”
<br/>　　“三次大劫？”苏莫闻言一怔，面上露出震惊之色，大劫千年一次，对方经历了三次，那对方活了多久了？
