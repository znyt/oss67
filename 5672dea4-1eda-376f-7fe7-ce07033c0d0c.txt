　　　　如今，就是离幽证明自己的好机会，这一战，他若胜了，那么就是功德圆满，从此蛇族就统治在他的手中。
<br/>　　“虽然你是蛇王了，但是我会证明我才应该是蛇族真正的王，这是生死战，我们是蛇，蛇与蛇的决斗很简单，输的一方就是成为赢方的食物”
<br/>　　蛇吃蛇，普通的食物链如此，成为妖灵也是如此。
<br/>　　吞下高贵蛇种，自身得以升华。
<br/>　　涂小安风轻云淡，神情自若，对离幽这话是完全的认同。
<br/>　　本来就是抱着吞了对方的想法才应战的。
<br/>　　说起来自从当初佘山涂小安跟变异吸血蛇一战，他就再也没有跟厉害的蛇种有过较量。
<br/>　　白家的那条蛇妖并不能计算在内，因为涂小安没有真正跟对方交锋过。
<br/>　　眼前这条超等帝皇烈阳蛇，品种无限高贵，应该算是涂小安出道以来，见过的最强蛇种了。
<br/>　　不对，他的三头邪蛇其实也好不逊色，只是那是涂小安自己培养的蛇种，而且已经具备了上古九头蛇的血脉。
<br/>　　涂小安也没有太自视甚高，毕竟对方等级在他之高，血统也不凡，而且敢如此明晃晃的跳出来挑战，肯定是有什么杀手锏。
<br/>　　但涂小安无所畏惧，他现在对自己的实力，更是信心爆棚，尤其是轻松就收拾了三大深山大妖之后。
<br/>　　他希望这条帝皇烈阳蛇可以越强越好，那样自己吞噬后，开启第三阶段性的进化，得到的好处就更大了。
<br/>　　涂小安看了离幽的黑金战甲一眼，光一眼他就知道离幽将自己蛇躯本体修炼到了极致，那蛇鳞片并不比涂小安的鳞片逊色几分。
<br/>　　为了表示尊重，涂小安全身金光灿灿了起来，体内开启了金元素防御属性。
<br/>　　“我让你先出手”涂小安是王，王就要有王的气度跟睥睨。
<br/>　　离幽冷哼一声：“那你可别后悔”
<br/>　　话音刚落，离幽也不矫情，朝着自己全身武装的黑金战甲一拍，这一拍，一道道黑金的蛇鳞片脱离了起来，他双手猛然前推，顿时形成了一道道极光，杀意弥漫，荡漾而出，如涟漪，似海浪，好似刀光剑影，每一片蛇鳞片绽放灿灿生辉，咻咻咻，朝着涂小安射去。
<br/>　　空气中立刻出现了无数道破空之音。
<br/>　　“这到有点意思”
<br/>　　涂小安没想到蛇鳞片还能这样被当成攻击的手段，这招偷学到了。
<br/>　　厉害的蛇，会将自己的身体开发到极致，普通的蛇只会注重毒牙，因为毒牙是它们杀敌的利器。
<br/>　　但是修炼蛇种，对于自己每个身体部位都非常的看重，尤其是蛇鳞片，代表了防御体系。
<br/>　　关键时候，甚至可以保命的。
<br/>　　这一道道鳞片极光射来，蕴含的威视非常的可怕，空间跟着扭曲起来。
<br/>　　所谓行家一出手，就知有没有，这算是大群伤的攻击手段了。
<br/>　　涂小安妖治的眼睛浑然一眯，系统中九道薄如蝉翼的飞刀倏然而出，意识力掌控，迎面斩去。
<br/>　　锵！锵！锵！
<br/>　　高空地带，出现了强烈的金属撞击上，火光迸发乍起，一道道黑金鳞片被飞刀一一斩落漂浮。

<br/>　　“你的飞刀，能挡得住我多少的鳞片！”
<br/>　　涂小安轻松道：“你有多少，我斩落多少”
<br/>　　只见满天寒光飞驰，不断有黑金鳞片随风散落。
<br/>　　落到了下方地面，落到一只鼠兽的身上，这只强壮体魄的鼠兽顿时口吐白沫了起来，一个咯吱声，便立刻的死于非命。
<br/>　　黑金鳞片带有剧毒，修炼到极致的蛇种，不单单毒牙有毒，身上每一道鳞片，都是剧毒无比，杀人于无形，见血封喉。
<br/>　　大股大股的黑金鳞片散落，让下方的鼠兽遭了大殃，只要被一道鳞片落上身，下一秒就口吐白沫的死去，死后尸体跟着就腐烂了，传出尸臭味。
<br/>　　离幽这也算间接的为蛇族杀鼠立了一大功劳。
<br/>　　“你这种攻击手段，对别人或许致命，但对你蛇王大人我来说，简直不值一提，换一种吧”涂小安意识力控制飞刀，然后随意的说出。
<br/>　　离幽借坡下驴，确实自己无数鳞片射出去，全部被斩落，在继续也毫无意义。
<br/>　　他全身帝皇之气愈演愈烈，阴风鼓卷，双眼猛然睁开，精光爆射，刺眼如穹天皓月，不可逼视。
<br/>　　嗡嗡嗡...
<br/>　　一阵筋肉颤动，拳脚破空之声骤然响起，瞬息之间，离幽直接拉近跟涂小安两者的距离，凭空打出了一拳。
<br/>　　“看我的蛇拳”离幽怒喝一声，有开山裂地之力。
<br/>　　这一拳打出，转眼变成了巨蟒张开了血盆大口，森冷的寒意如履薄冰，四周尽是腥臭之气。
<br/>　　这就是真的一张血盆大口啊。
<br/>　　“火拳”
<br/>　　涂小安干脆利落，没有那么多花里胡哨，蕴含火焰，极度刚猛的一拳迎接了过去。
<br/>　　轰...！

<br/>　　因为蛇鼠成了一窝，早就混战在了一起，难分彼此。
