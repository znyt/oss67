        能得到傅昼景的肯定，诺筱颖心里也跟着高兴不已。
<br/>
<br/>        不知不觉中，已经晚上九点了。
<br/>
<br/>        诺筱颖的肚子突然“咕咕”直叫，让她在傅昼景面前尴尬到脸红。
<br/>
<br/>        “你还没吃晚饭吗？”傅昼景微笑着问道。
<br/>
<br/>        诺筱颖腼腆地点了点头。
<br/>
<br/>        她是做了晚饭，但是是做给哥哥嫂嫂吃的。至于她，因为迫不及待地想要请教他问题，就一直在他家门外等着他回来。
<br/>
<br/>        她等了他许久的事情，她没说，他也不知道。
<br/>
<br/>        傅昼景抬起手来，揉了揉诺筱颖的头顶，笑着说：“傻丫头！我去给你做意大利面吧！”
<br/>
<br/>        “意大利面？！”诺筱颖好奇地眨了眨眼睛。
<br/>
<br/>        傅昼景笑了笑，盖上了笔套，放下了手中的图纸，起身离开了书房。
<br/>
<br/>        设计图上的疑点，她都已经问完了。
<br/>
<br/>        诺筱颖连忙收拾了一下图纸，跟着傅昼景一起去了厨房。
<br/>
<br/>        西餐，诺筱颖没吃过，就更别提什么意大利面了。
<br/>
<br/>        傅昼景煮面的时候，让诺筱颖去冰箱里拿圣女果出来洗。
<br/>
<br/>        诺筱颖非常勤快地照做。
<br/>
<br/>        在傅昼景面前，诺筱颖就像一个小妹妹一样，期待着大哥哥给她做晚餐。
<br/>
<br/>        其实，也只有在傅昼景这里，诺筱颖才能感受到被别人照顾着原来是如此温暖窝心。
<br/>
<br/>        毕竟，对诺筱颖来说，一直都是她在照顾着别人。
<br/>
<br/>        在家里，她照顾着老爸老妈的生活；在外，她照顾过苏漫雪的生活。
<br/>
<br/>        如今，她和哥哥嫂嫂住一块，又得照顾哥哥嫂嫂的生活。
<br/>
<br/>        这世上，除了外婆和老妈给她做的饭菜以外，傅昼景是第一个，说不定也是唯一一个为她下厨的男人。
<br/>
<br/>        不知道在厨房里忙活了多久，反正傅昼景将一盘意大利面端到诺筱颖的面前时，诺筱颖拿起筷子便开始狼吞虎咽。
<br/>
<br/>        傅昼景忍俊不禁，伸出手来，轻轻地夺走了诺筱颖手中的筷子，给她递上了叉子。
<br/>
<br/>        诺筱颖赧然一笑，接过叉子，不得不细嚼慢咽地吃。
<br/>
<br/>        傅昼景在诺筱颖的对面坐下，一手平放在餐桌上，一手半握拳头，优雅地抵着脑袋，看着正在吃意大利面的诺筱颖，嘴角情不自禁地微微上扬。
<br/>
<br/>        她与他身边的那些女人完全不同，个性倒是很像他的初恋女友，单纯、坦诚、直率、认真。
<br/>
<br/>        诺筱颖已经尽量在克制自己，吃饭的时候要淑女一些，但是，肚子实在是太饿了，又加之傅昼景做的意大利面也实在是太好吃了，所以，她还是忍不住大口大口地吃了起来。
<br/>
<br/>        傅昼景见诺筱颖将盘子里的食物吃得一干二净，又起身给她榨了一杯鲜橙汁，端到了她的面前。
<br/>
<br/>        诺筱颖说了声“谢谢”，然后端起杯子，准备喝橙汁的时候，傅昼景温和地提醒：“要慢点喝，不要呛到了。”
<br/>
<br/>        诺筱颖微笑着点了点头，举着杯子，慢慢地、缓缓地抿着橙汁喝，一点都不自在。
<br/>
<br/>        其实，诺筱颖看得出来，傅昼景出身非凡，教养很高。
<br/>
<br/>        这晚饭吃完了，诺筱颖很自觉地洗完碗，却被从健身房里出来的傅昼景告知，他家里其实有自动洗碗机。
<br/>
<br/>        傅昼景又将碗筷放入了自动洗碗机里，按下了启动键，又重新洗了一遍。
<br/>
<br/>        诺筱颖站在一旁，看得有些尴尬，双手不知道该往哪里放。
<br/>
<br/>        “那个……今晚打扰了，谢谢你。我就先回去了！”诺筱颖抬起手来，五指交叉戳动着，赧然微笑着说。
<br/>
<br/>        傅昼景点了点头，微笑着回答道：“嗯，晚安，我就不送了。”
<br/>
<br/>        “好！晚安，你早点休息！”诺筱颖抱起餐桌上的图纸，向傅昼景欠了欠身，然后转身离去。
<br/>
<br/>        傅昼景还是送她到了门口，然后看着她走进了电梯。
<br/>
<br/>        是个干劲十足的小丫头，希望，她以后真的能如她所愿，成为一名出色的室内设计师吧！
<br/>
<br/>        诺筱颖回到家中，发现餐桌上的碗筷都还未收拾，便放下手里的图纸，将碗筷收拾了拿去厨房里洗。
<br/>
<br/>        然后，她又把家里的卫生给搞了一遍，才去洗澡睡觉。
<br/>
<br/>        说真的，她受不了家里乱七八糟的样子，她爱干净，爱整洁，这样的家，住着才舒服。
<br/>
<br/>        有了傅昼景的提点，翌日，诺筱颖一回到办公室里，坐在电脑前，就开始修改自己的设计图。
<br/>
<br/>        “小梁，我电脑里的‘3DMAX’无法重装，你可不可以过来帮我看一下。”
<br/>
<br/>        当诺筱颖工作到一半的时候，坐在她对面格子间里的同事崔素妮忽然一脸委屈地向她求助。
<br/>
<br/>        在这个办公室里，诺筱颖向来有求必应，同事之间，互相帮忙是应该的。
<br/>
<br/>        诺筱颖抬起头来，看了崔素妮一眼，微笑着点了下头：“好，我马上就过来帮你看看。”
<br/>
<br/>        “谢谢你了啊！”崔素妮微微一笑，然后站起身来，接着说道，“我肚子有点儿痛，要一趟洗手间。我的电脑密码是201314，小梁，你直接过来帮我把软件重新装好便是。”
<br/>
<br/>        “嗯，好的！”诺筱颖保存了自己的设计图后，便起身去了崔素妮那儿。
<br/>
<br/>        这时候，崔素妮已经离开了办公室。
<br/>
<br/>        诺筱颖对电脑系统并不是太懂，所以，至于“3DMAX”为什么无法重装，她也是弄了好久，才把软件给重新装上去。
<br/>
<br/>        崔素妮出去了好一阵子，才回到办公室，看到诺筱颖帮她把软件重新装好了，还当众赞许了诺筱颖，说要请她喝下午茶。
<br/>
<br/>        诺筱颖委婉地谢绝了，还说，大家一起共事，相互帮助是应该的。
<br/>
<br/>        崔素妮在一旁不停地夸诺筱颖为人超级友善，大家都很喜欢她这样的同事。
<br/>
<br/>        能在办公室里与大家相处融洽，诺筱颖感到很欣慰。
<br/>
<br/>        诺筱颖回到自己的办公桌前，继续工作，等她的装修风格设计好了后，剩下的就是选材了。
<br/>
<br/>        因为是总裁办，所以，诺筱颖挑选的装修材料，都是上等优级的材料。

<br/>        短短的两个星期，诺筱颖将总裁办的装修图全部设计完毕，并且还做好了材料品牌规格型号的选择表，还有整体与局部的部分效果图，交到了孙副部长孙莉莉的手中。
