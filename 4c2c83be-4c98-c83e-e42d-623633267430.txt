　　第404章
<br/>　　双方都摆出了相当大的诚意，谈判进行得相当顺利，在第三天就把人接了回来。
<br/>　　同样，宋游渠一伙也交还给凉州城，不过，却唯独留下了薛问婷一人。为了确定下来这件事，双方费了很多口水，也因为这一点，才将时间拖了三天。
<br/>　　一辆马车拉着福德勒从凉州城回来，秦蛮亲自派人护送，把福德勒接回了凉州城。
<br/>　　“人在哪儿呢？”
<br/>　　程大雷和白元飞大步过来，却见到秦蛮脸色极其不好看。程大雷顿了顿，道：“怎么了？”
<br/>　　秦蛮欲言又止，白元飞已经迫不及待冲到马车前，伸手将车帘拉开，看到车内的福德勒时，忍不住发出一声惊呼。
<br/>　　秦蛮将他的手拦下来，把车帘盖住，低声冲程大雷说了一句：“一只眼睛瞎了。”
<br/>　　白元飞气得哇呀呀大叫，道：“我非杀了宋伯康不可。”
<br/>　　程大雷脸色也不是很好看，挥挥手道：“把人抬进去吧，请刘悲过来看看，目前人活着就好。”
<br/>　　福德勒到现在仍旧昏迷不醒，落在宋伯康手里，他很是受了一顿大刑。当然，这也并非不能理解，他在城主府大开杀戒，连杀十七人，这些人也有朋友亲戚，自然恨不得他死在大牢中。
<br/>　　这还是后来谈判开始，宋伯康特意吩咐不可让人死了。不然福德勒一条命就得交待在牢中。
<br/>　　如今幸运的是保住了一条命，身上的伤将养一阵子还是能痊愈，问题是一只眼睛瞎了，算是永生摆脱不了的残疾。
<br/>　　白元飞在凉州承过福德勒的情，这件事他一直搁在心底。福德勒山寨本无所谓什么朋友，但在白元飞心里，已经是将他当做了朋友。这次福德勒出事，白元飞比任何人都要愤怒，在房间内转来转去，誓要找宋伯康报仇。
<br/>　　而现在程大雷要思考的，则是如何将李行哉从凉州城捞出来。想要救他出来这件事并不容易，一切还需从长计议。
<br/>　　程大雷特意叫了徐神机、刘悲、吴用、和珅、阿喜几人聚在一起，共同商议这件事。因白元飞去过凉州城，了解城主府的情况，所以也把他叫了过来。
<br/>　　事情并不好办，想要在层层防守下救人出来，诸人都没有办法。
<br/>　　“大当家，干脆咱们杀到凉州城，把宋伯康杀了，抢出李行哉，事情不就解决了么。”白元飞还想着要杀宋伯康的事情。
<br/>　　程大雷白了他一眼，道：“好啦，别说废话了。凉州城是那么容易打的，况且，我们不能为了救一人，牺牲无数弟兄们的生命。”
<br/>　　白元飞闭口不言，但看上去，他并没有被程大雷的话说服。
<br/>　　“你去过凉州城，知道李行哉被关在哪里？”
<br/>　　“这个当然知道，他现在被困在城主府中，有一座单独的院子，外面防守很严。”白元飞。
<br/>　　“来，把城主府的地图画出来。”
<br/>　　“呃……大当家，你这可就有些强人所难了。我连笔都不会握，更不要说画图。”白元飞。
<br/>　　白元飞的本事不错，记忆力惊人。凡去过一次的地方，他都像刻在心底一样，永远不会忘记。但人各有所长，如果让他把一地的地图画出来，他的确做不到。
<br/>　　“大当家，不如从地下挖一条地道，直通城主府，将人从地道救出来。”阿喜道。
<br/>　　“这……有些太扯了吧，我们从城外挖掘地道，这么远的距离，谈何容易。”程大雷。
<br/>　　“其实并不困难，喜字队中有曾经盗墓的兄弟，善于开掘地道，我们现在人多，挖掘一条五里长的地道，也费不了太长时间。”
<br/>　　程大雷怔了怔，他恍然发现自己忽略了一件事，现在山寨人是多的，已经有了做大工程的能力。比如开掘地道，以前想都不敢想的事情，如今甚至谈不上困难。
<br/>　　“需要什么？”程大雷。
<br/>　　“需要分辨土质，看从哪里开掘容易。如果能有一份凉州城的地图就好了。”阿喜。
<br/>　　“找人把西泽叫过来。”程大雷挥挥手，吩咐下去。
<br/>　　冒险家西泽，能力就是绘图，恰好和白元飞的能力搭配起来。想办法让他们混入凉州城，描一份凉州城的地形图出来，这次救人要用，日后说不得还能有别的用处。
<br/>　　目前看来，也无其他法子，只有采纳阿喜的提议，从地下挖掘一条地道，直通城主府，救出李行哉。
<br/>　　办法是阿喜提出的，就由阿喜去执行，所需要的人手，由刘悲派给他就是。至于具体如何做，其实并不需要程大雷过分关心。
<br/>　　大家散去后，李婉儿来见程大雷。本来李行哉离开的消息，并没有告诉李婉儿，但现在事情已闹成这个样子，李婉儿也不可能不知道。
<br/>　　毕竟，李婉儿不是傻瓜。
<br/>　　“你放心啦，人肯定会没事的，我已经派人去救他了。就算待在凉州城，他也不会有什么事。”程大雷把茶推到李婉儿面前：“尝尝，从江南运过来的。”
<br/>　　“六哥的事情我不担心。”李婉儿摇摇头：“我担心的是长安城。”
<br/>　　程大雷一怔，这才恍然想起，凉州城有她的六哥，但长安城却有她的父皇和更多的兄弟姐妹。而一开始，连程大雷都觉得，林少羽打不下长安城。可随着时间发展，形势越来越复杂，最近长安城传来的诸多消息，都是对朝廷不利的。程大雷也忍不住觉得，或许，林少羽真的能打下长安。或许，上天也真的不再眷顾李家。
<br/>　　“程大雷，你实话与我说，长安还能不能守住？”
<br/>　　程大雷叹了口气道：“婉儿，你六哥临走前与我说，帝国明玉公主已经死在草原上。帝国未来如何，你和你无关，你现在要做的是，过好李婉儿的一生。”
<br/>　　李婉儿苦笑一声：“话是如此说，但你觉得可能么？”
<br/>　　程大雷再次沉默，半晌，从座位上站起来，道：“婉儿，天下大乱，近在眼前，福德勒已经瞎了一只眼，以后还有很多人死。你父皇、李行哉、包括你我，还有我们身边的每个人，都可能死在乱世中。”
<br/>　　
<br/>(本章完)
