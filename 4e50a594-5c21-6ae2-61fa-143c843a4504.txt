　　阁楼中。
<br/>　　苏莫和聂钱、韩盈相对而坐。
<br/>　　“苏兄，对不起，是我们连累了你！”聂钱面色沉重，向苏莫道歉。
<br/>　　其实上，廖云剑之事，是他和韩盈说了出来。
<br/>　　这倒不是他们迫害苏莫，而是面对太史轩和尤天韩的逼问，他们不得不说。
<br/>　　当然，他们就算不承认，也没有任何用处，因为太史轩已经认定，廖云剑死于苏莫之手。
<br/>　　不过，他们也没说是苏莫单独杀的廖云剑，而是他们三人联手所杀。
<br/>　　他们本来是想和苏莫共同承担责任，但却没想到，这反而成了太史轩和尤天韩威胁苏莫的把柄。
<br/>　　“苏兄，我们也不知道，那太史轩居然用此事胁迫你！”韩盈面带担忧的说道。
<br/>　　“两位不必自责，车到山前必有路，你们也无需担心！”苏莫摇了摇头说道。
<br/>　　“那苏兄，你准备怎么办？”聂乾和韩盈异口同声的问道，两人的命运都在苏莫的手中，自然想知道苏莫准备怎么做！
<br/>　　苏莫闻言，面露沉吟之色，他暂时也不知道要如何做，但太史轩想要他屈服，这绝不可能。
<br/>　　“聂兄，韩姑娘，你们放心吧！我不会让你们有事的！”
<br/>　　苏莫郑重的说道，实在不行，他就加入其他派系，将聂乾两人都带进去，到时候太史轩再想要对付他，也不会这么容易了。
<br/>　　聂乾和韩盈点了点头，但两人眼眸之中，还是留露出深深的担忧。
<br/>　　他们也担心苏莫强势反抗太史轩和尤天韩，最后导致三人全部被学府重罚，导致最后丢掉性命。
<br/>　　不过，两人也相信苏莫会做出合适的选择，不会拿自己的性命开玩笑。
<br/>　　“对了，刚才那尤天韩是什么人物？”苏莫想到了那名藏青色长袍的青年。
<br/>　　“尤天韩是太史世家派系的人，也是南苑弟子中第一高手！”聂乾说道。
<br/>　　“第一高手？这么强大？”苏莫闻言心中一惊，他虽然感觉到对方修为很高，但没想到居然达到了这等地步。
<br/>　　“嗯！此人武帝境九重巅峰境界，在整个南苑之中，也只有少数几人，能勉强与他抗衡而已。”聂乾点头说道。
<br/>　　苏莫默然了下来，原来这个尤天韩，居然是如此厉害的人物。
<br/>　　看来这太史世家派系，对他倒是格外的上心。
<br/>　　先前此人也说了，太史世家派系的圣子，都很看重他。
<br/>　　“聂兄、韩姑娘，你们先回去吧！让我仔细考虑考虑！”少倾之后，苏莫向两人挥了挥手。
<br/>　　“嗯！”两人点了点头，也没有多说什么，便告辞离去。
<br/>　　哎！
<br/>　　待聂乾两人离开之后，苏莫陷入了沉思之中，屈服于太史轩？
<br/>　　这绝对不可能！
<br/>　　他不可能屈服于任何人！
<br/>　　那只能加入其它派系了，亦或是前往内府火元峰，寻求火元圣师的帮助。
<br/>　　苏莫仔细的考虑了一番，相比于加入其它派系，还是在火元峰有火元圣师的庇护，更加的好一些。
<br/>　　因为这样的话，就不会陷入各大派系的争斗之中。
<br/>　　只不过，他以前拒绝了火元圣师，现在却是有些不好意思去找对方了。
<br/>　　而且，在这个关头去找对方，明显是寻求庇护，目的性太重了。
<br/>　　这不由得，让苏莫有些犯难了起来。
<br/>　　“鬼圣前辈！”
<br/>　　苏莫一时间拿不定不注意，想问问阎魔鬼圣的意见。
<br/>　　不过，呼喊了一声，对方却是没有任何回应。
<br/>　　“嗯？”
<br/>　　苏莫心中有些奇怪，意念立刻找到了战魂之中的阎魔鬼圣，传出意念之音：“鬼圣前辈，你怎么了？”
<br/>　　阎魔鬼圣的魔魂，正在他的战魂之中，并没有丝毫问题。
<br/>　　这时，阎魔鬼圣终于有反应了，声音幽幽的道：“苏莫，汝难道没有想过，为何突破到了武帝，却没有天劫降临？”
<br/>　　苏莫闻言一怔，这个问题他当然想过，他突破到武帝之后，没有天劫的感应，但也没怎么在意！
<br/>　　对于他来说，没有天劫倒是省了一些麻烦。
<br/>　　但现在阎魔鬼圣询问，难道其中还有什么缘由不成？
<br/>　　“这是为何？”苏莫沉声问道。
<br/>　　“吾思忖了很久，确实找到了一种原因！”阎魔鬼圣声音凝重的说道。
<br/>　　“什么原因？”苏莫好奇的问道。
<br/>　　“汝之生命，不在天道规则之中！”阎魔鬼圣说道。
<br/>　　“不在天道规则之中？”
<br/>　　苏莫闻言满心疑惑，不解的问道：“这是什么意思？”
<br/>　　“意思就是，汝不是天道规则之下诞生的生灵！”阎魔鬼圣声音低沉，其中隐隐有一丝悸动。
<br/>　　因为这个猜测，让他都感觉有些匪夷所思，这实在是太不可思议了！
<br/>　　但是，除了这个原因之外，他真的找不到其他的原因了。
<br/>　　没有任何原因，能解释没有天劫一事。
<br/>　　要知道，武者晋升武帝之境，天劫号称战魂劫，是武道一途之中，最大的劫难之一。
<br/>　　任何生灵，任何种族，实力达到这一地步，都会有天劫。
<br/>　　就算是只修肉身，不修玄力和战魂的巨灵族人，达到武帝之境，都有天劫降临。
<br/>　　而苏莫却没有，这是多么匪夷所思的事情！
<br/>　　“这……这不可能吧？”苏莫满脸诧异之色，天地万物，无不是在天道规则之下诞生，自己怎么可能不在天地规则之中？
<br/>　　这也太夸张了吧！
<br/>　　若是按照对方所言，自己这是超脱三界之外，不在五行之中了！
<br/>　　这和地球上神话传说中的那些远古诸神，一样的存在了！
<br/>　　这显然不可能，他以前可是有天劫，当初的武王劫不要太强了！
<br/>　　“这是唯一的可能！”阎魔鬼圣无比笃定的说道。
<br/>　　“那为何我以前有天劫？”苏莫沉声问道，他根本不信对方的话。
<br/>　　“汝以前有天劫？”阎魔鬼圣闻言一怔，若是如此的话，他的猜测就要被推翻了。
<br/>　　“当然，我当初的武王劫，非常的强大！”苏莫点头道。
<br/>　　听了苏莫的话，阎魔鬼圣陷入了沉默之中，难道他的猜测是错误的？
<br/>　　可是，只要是这个宇宙之中诞生的生灵，达到武帝的境界，怎么可能会没有天劫？
<br/>　　“汝是不是诞生在位面小世界之中？”良久之后，阎魔鬼圣仿似想到了什么，突然问道。
<br/>　　“不错！”苏莫点了点头，苍穹世界正是一个位面小世界！
<br/>　　“吾明白了！吾明白了！”阎魔鬼圣的自语声，不停的在苏莫脑海中响起。
<br/>　　“你明白什么了？”苏莫不解的问道。
<br/>　　不过，阎魔鬼圣却是没有再回答他，彻底的陷入了沉寂之中，再也没有开口。
<br/>　　这让苏莫无语至极，这老魔神神叨叨的，到底明白了什么？
