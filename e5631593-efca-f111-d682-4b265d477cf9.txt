　　无尽高空之上，茫茫云海之中，巫人王急速飞行，向着神武学府的方向飞去。
<br/>　　他准备去神武学府堵截苏莫，誓要将苏莫生擒活捉。
<br/>　　不错，是生擒活捉，而不是杀死。
<br/>　　因为苏莫的身上蕴含着关于血脉的秘密，他一定要得到这个秘密，从而也能拥有别人的血脉体质。
<br/>　　他之前已经用传讯符，将苏莫在古周星的消息，传给了距离最近的族中强者，并且将消息传递下去。
<br/>　　现在，古周星的的各大星空传送大阵，已经被严密的监视了起来。
<br/>　　不仅如此，也会有一部分的族中强者，飞出了古周星，进入了星空监视，防止苏莫从星空逃离。
<br/>　　当然，他带来的强者并不多，不可能将整个古周星监视住，只能尽量封锁古周星的逃走路线。
<br/>　　咻！
<br/>　　就在这时，后方响起了刺耳的破空声，一道流光从后方急速追来。
<br/>　　巫人王见此顿时皱眉，而后停了下来。
<br/>　　嗖！
<br/>　　瞬息之间，流光来到了巫人王的身前，化为了一名白衣青年。
<br/>　　“尤天韩，你为何跟踪我？”巫人王冷冷的问道，这到来的白衣青年正是尤天韩。
<br/>　　“我并不是跟踪你，而是要与你合作而已！“尤天韩微笑着说道。
<br/>　　“合作？”巫人王闻言，目露疑惑之色。
<br/>　　“不错，你应该是和苏莫有仇吧？我也和他有仇，说不定我们可以合作！“尤天韩点头说道。
<br/>　　“我为何要与你合作？”巫人王冷冷的说道，他并不信任尤天韩，对付苏莫也不需要尤天韩。
<br/>　　“仅凭你一个人，在古周星想杀苏莫，不仅不太可能成功，说不定连你自己都要陨落与此！”尤天韩说道。
<br/>　　“是吗？”巫人王不置可否，他可不止是一个人。
<br/>　　“当然，苏莫目前在神武学府炙手可热，加入了皇祖圣朝派系，若是皇祖圣朝派系圣子古天意知道你要杀他，你认为你能活命？”尤天韩说道。
<br/>　　巫人王闻言皱眉，虽然他不认识古天意，但也曾听说对方的名头。
<br/>　　而且，作为神武学府的圣子，都是武尊境修为，而且还不是普通的武尊境。
<br/>　　若是对方出手的话，的确是个大麻烦。
<br/>　　“如何合作？”巫人王问道。
<br/>　　“很简单，你不需要付出什么，我会为你提供苏莫的消息，若是有机会，你就出手击杀！”尤天韩笑着说道。
<br/>　　“可以！”巫人王没有过多考虑，便点头同意了。
<br/>　　他和尤天韩也没有仇怨，以前虽然有过一战，但不过是互相切磋而已。
<br/>　　既然对方愿意给他提供苏莫的消息，他自然不会拒绝。
<br/>　　“这是我族特制的传讯符，传讯距离极大，可以跨越古周星面积的五分之一！”
<br/>　　巫人王手掌一番，手中出现了一块黑色的传讯符，随即递给了尤天韩：“有苏莫的消息，你随时可以给我传讯！”
<br/>　　“嗯！”尤天韩收起传讯符，随即向巫人王抱了抱拳，道：“后会有期！”
<br/>　　言罢，尤天韩身形一转，便破空离去。
<br/>　　少倾，巫人王也再次急速离去。
<br/>　　两人飞行的方向，均是神武学府的方向，尤天韩是要回去，而巫人王自然不可能将希望全部压在对方身上，自己还是要行动的。
<br/>　　……
<br/>　　无尽虚空深处，苏莫被包裹在一股气流之中，急速而行。
<br/>　　他飞行的方向不是神武学府，而是途山镇。
<br/>　　现在，他彻底的暴露了，准备先躲进神武学府中，但回学府之前，必须先回途山镇看看。
<br/>　　一路无话，数日之后，苏莫回到了途山镇。
<br/>　　这一次回来，途山镇没有什么变化，苏莫神不知鬼不觉的进入了苍穹神宫所在府邸中。
<br/>　　一座阁楼之中，正在闭目修炼姜枫然，猛然睁开了眼眸，看着突然出现在面前的人。
<br/>　　“苏莫，你回来了？”姜枫然惊讶的说道。
<br/>　　“嗯，其他人都回来了吗？”苏莫问道，他指的是夕儿、战乱等人。
<br/>　　“龙腾和晓晓没有回来，其他人都回来了！”
<br/>　　姜枫然说道，由于上次苏莫离开之时有交代，他们这段时间，一直在召集所有人回来。
<br/>　　好在众人出去历练都没有走的太远，很快便召集回来了几位，但还差龙腾和翼晓晓两人。
<br/>　　“他们人呢？”苏莫疑惑的问道。
<br/>　　“不清楚，联系不上，也找不到他们！”姜枫然摇了摇头。
<br/>　　苏莫闻言顿时皱眉，这两人该不会一起离开了古周星吧！若是他不逃离此地，这倒是没有什么。
<br/>　　但若是他逃离古周星，两人没有回来的话，那就只能舍弃两人了。
<br/>　　“你去把其他几人找来！”苏莫对姜枫然道。
<br/>　　“好！”
<br/>　　姜枫然闻言，二话不说，便立刻离开了阁楼，前去召集其他人。
<br/>　　望着姜枫然离去的背影，苏莫暗暗点头，对方的修为已达武帝境二重巅峰，这种修炼速度的确是相当惊人。
<br/>　　现在的姜枫然，不仅是特殊体质，还拥有双生战魂，已经是古灵星河最顶尖的天才，比之以前的巫人王，都不差多少了。
<br/>　　而且，姜枫然和苏莫不同，不是太古武修，提升修为所需要的灵气，只是苏莫的几十分之一，所以修炼的速度，并不比苏莫慢多少。
<br/>　　没过多久，姜枫然回来了，夕儿、战乱、司空炎、闻人天都、清雅、彩芸等六人也全部来了。
<br/>　　“苏莫哥哥！”夕儿见到苏莫，顿时满脸的喜意。
<br/>　　“苏莫！”
<br/>　　“宫主！”
<br/>　　其他人也是纷纷开口，均是满脸的笑容。
<br/>　　苏莫同样面带笑容，与几人纷纷打招呼，随即他打量了众人一眼，一眼便看出了所有人的修为，几人的天赋不同，修为也不尽相同。
<br/>　　姜枫然、司空炎、闻人天都和战乱四人，都是武帝境二重的修为，而夕儿、彩芸和清雅三女，因为天赋略低，依旧是武帝境一重的修为。
<br/>　　不过，三女的修为，也达到了武帝境一重的巅峰。
<br/>　　随后，苏莫让众人在阁楼中落座，准备和众人开个小型会议。
<br/>　　“诸位，巫族之人现在已经知晓了我在古周星，我们安稳的日子，怕是过不下去了！”苏莫面色沉重的说道。
<br/>　　众人闻言一惊，巫族发现了？难道又要再次开始逃亡了不成？
