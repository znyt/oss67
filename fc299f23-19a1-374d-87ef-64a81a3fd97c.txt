<br/>    紫金山下，灯火阑珊，千灯齐明，一路摇曳。
<br/>    
<br/>        拍卖会还有两天，陈明带着四个小家伙，也是准备去紫金山逛逛，紫金山并不像燕山那般警备森严，倒是可以随意走动。
<br/>    
<br/>        陈明带着四个弟子到来，看门的守卫早就接到了消息，不敢有半点阻拦，直接放他们一行人进去，银翼闲的发慌，也是跟着过来凑凑热闹。
<br/>    
<br/>        虽然拍卖会还没有开始，但是现在的紫金山，可是相当的热闹，众多的妖族在紫金山上售卖着自己的东西。
<br/>    
<br/>        陈明扫了一眼，到处都有着紫金山的妖军巡逻，应该不会出什么危险，大手一挥，将自己以前的那个储物戒给了卓青瑶，其中放了四十万灵石，“青瑶，你带着师弟师妹去四处转转，看到有什么喜欢的东西，就买下来，里面有着四十万灵石。”
<br/>    
<br/>        卓青瑶接了过去，也没有兴趣拒绝，在她看来，他们四个拿师父的东西不是理所当然的吗？
<br/>    
<br/>        带着三个小家伙，去逛街去了。
<br/>    
<br/>        陈明也是乐得休闲，和银翼找了一个酒馆，叫上两个小菜，准备在这里喝上一会儿酒，难得有这么清闲的机会，对于逛街买东西，陈明可没有任何兴趣，他又不是主角，可以在地上随便买个啥都是宝贝。
<br/>    
<br/>        和银翼吃吃喝喝，银翼给陈明将酒倒上，陈明横了银翼一眼，“酒有什么好喝的，给我换果汁！”
<br/>    
<br/>        银翼疑惑了，“山主，果汁是啥？”
<br/>    
<br/>        陈明也是愣住了，原来这里没有果汁啊，“嗯，那就换果酒吧。”
<br/>    
<br/>        银翼给陈明倒了一杯果酒，陈明端起来尝了尝，酸酸甜甜的，味道还不错，就是酒味太淡了，不够烈。
<br/>    
<br/>        拿过银翼的酒壶喝了一口，好吧，跟啤酒差不多，这个酒有什么好喝的？
<br/>    
<br/>        从酒楼之外，走进一群穿着黑衣的家伙，都是完全化形的存在，在他们的最前方，是一个妙龄女子，黑色的紧身衣完美的勾勒出了女子的身段，脸上带着一张黑色面纱，但是那一双眼睛，却是灵动之极，如同一汪秋水。
<br/>    
<br/>        女子在陈明的对面坐下，女子身旁的黑衣侍从在女子的耳边耳语了几句，女子一双美眸落在陈明的身上，一身青色的道袍，手中一柄绿色的拂尘。
<br/>    
<br/>        银翼忽然奸笑着看向陈明，“山主，山主，有个女孩在看我！”
<br/>    
<br/>        陈明横了银翼一眼，看你就看你，这么嘚瑟干嘛！
<br/>    
<br/>        银翼继续道：“我能够感觉到，那个女孩看我的目光是那么深沉。”
<br/>    
<br/>        陈明扫了外面一眼，现在还没到春天啊。
<br/>    
<br/>        在陈明的身旁，一阵香风扑面，在他旁边的凳子上，坐下一个身穿黑衣的女孩，女孩看起来不过十七八岁的模样，一双秋水般的眸子落在陈明的身上，“这位道友，也是来参加拍卖会的？”
<br/>    
<br/>        陈明看了这女孩一眼，看身段确实很不错，那一双眼睛也很漂亮，“胸大果然无脑，不是来参加拍卖会的，来这干嘛？”
<br/>    
<br/>        女孩愣了愣，她还是第一次遇到这样和她说话，以前那些男人，谁不是巴结她，但是陈明，却是一种慵懒之极的神态在说话。女孩看向陈明身旁的银翼，正在看着自己，时不时擦了擦口水，这才是正确的表现啊。
<br/>    
<br/>        女孩尴尬地笑笑，“额，这位道友，还真是风趣！”
<br/>    
<br/>        陈明懒得理这女孩，像这样对他有所企图的女孩实在是太多了，他都已经懒得理了。
<br/>    
<br/>        女孩道：“相逢就是有缘，我敬道友一杯。”
<br/>    
<br/>        端起酒水，似乎是不小心，酒水落在了陈明的道袍上，女孩惊慌道：“抱歉，不小心弄脏了道友的道袍，我给道友擦一擦。”
<br/>    
<br/>        女孩低头，胸口雪白的一抹沟壑落入陈明的眼中，陈明伸手抓住女孩的手，淡淡道：“不用了，我这道袍，防水。”
<br/>    
<br/>        女孩看向陈明的道袍，果然，那些酒水瞬间便已经消失了，没有半点痕迹。
<br/>    
<br/>        女孩尴尬地笑笑，道：“道友这件道袍，还真是别出心裁。”
<br/>    
<br/>        陈明也懒得废话了，直接道：“你应该知道我是谁，所以，什么来意，直说吧。”
<br/>    
<br/>        女孩笑道：“道友说笑了，我们素未平生，怎么可能知道道友的身份？”
<br/>    
<br/>        陈明指了指银翼，“这个二货，肆无忌惮的释放自己神府境的威压，而我，只是有着入道境的气息，就算是我已经完全化形，但是怎么也比不上这个二货的气息强大，但是你一坐到这儿来，完全不理这个二货，直接和我说话，你说，这其中难道就没有什么问题吗？”
<br/>    
<br/>        女孩拱手道：“燕山之主真是慧眼如炬，小女子没有多余的要求，只求燕山之主可以带小女子进入拍卖会，在这次拍卖会之中，有一件宝物是小女子家传之宝，所以小女子想要将其买回来，但是却苦于没有资格进入拍卖会。若是燕山之主愿意带着小女子进入其中，小女子可以送出百万灵石报答山主。”
<br/>    
<br/>        陈明扫了一眼女孩的面纱，道：“我不喜欢跟一个身份不明的人说话，如果你可以取下这面纱和我在这紫金山逛上一圈，我可以考虑考虑。”
<br/>    
<br/>        女孩愣住了，“抱歉，这面纱若是取下，会有很多麻烦。有时候，生的貌美也是一种错。”
<br/>    
<br/>        陈明摆摆手，道：“那就这样吧。”
<br/>    
<br/>        女孩：.......
<br/>    
<br/>        正常的男人不是都会对自己的面容心生好奇吗？
<br/>    
<br/>        正在这时，酒楼外面忽然一阵骚乱，“听说了吗？赌坊哪儿快打起来了！”
<br/>    
<br/>        “啥，这儿的赌坊可是长河大妖王的儿子坐镇，谁敢去赌坊哪儿闹事？”
<br/>    
<br/>        “不知道啊，听说是两个少年，两个少女！”
<br/>    
<br/>        陈明听着这消息，估摸着是卓青瑶他们惹出麻烦了，陈明起身，“银翼，走了，青瑶他们出事了，我们过去看看。”
<br/>    
<br/>        陈明和银翼离开酒楼，那女孩也是跟着过来了，“山主，小女子虽然力有不逮，但是可以尽自己绵薄之力。”
<br/>    
<br/>        女孩抢先一步，走到陈明的身旁，和陈明并肩而行，银翼很是识趣地后退，让他们俩走在一起。
