<br/>　　“魏仑受伤了！”
<br/>　　众人见此，纷纷面色一变，面带惊讶之色。
<br/>　　就连褚彦，也不禁面色一凝，魏仑失败了吗？

<br/>　　少倾，魏仑落了下来，脚步刚刚踏在实地之上，口中更是鲜血涌动。

<br/>　　“魏仑，你不是那段惊天的对手？”褚彦阴沉着脸道。
<br/>　　他倒是小看了段惊天，居然能把魏仑击伤。
<br/>　　“褚彦哥，那家伙偷袭我！”魏仑深深的喘了两口气，脸色略显苍白的说道。

<br/>　　“偷袭你？”褚彦闻言，微微皱眉，这个段惊天如此阴狠吗？
<br/>　　众人亦是惊讶，这魏仑简直是偷鸡不成蚀把米啊。
<br/>　　“是啊！我刚拦住他，准备先教导一番，谁料他突然偷袭，才让我受伤的！”魏仑点头道。
<br/>　　“岂有此理！”褚彦怒了，一个蝼蚁般的人物，居然敢如此的猖狂。
<br/>　　褚彦眸中厉芒一闪，便准备亲自出手，下去教训苏莫。
<br/>　　“褚彦哥，让我出手吧！”这时，一名体态雄壮的青年大步走出。
<br/>　　此人身高两米，虎背熊腰，全身肌肉虬扎，甚是惊人。
<br/>　　此人，一看便知，是一位法体兼修的武者，力量强大。
<br/>　　最主要的是，此人是真神境的修为，修为是一品低级真神。
<br/>　　“鲁东，你的实力自然是能轻易镇压段惊天，但是，我担心你把人弄死了！”见出言的高大青年，褚彦微微摇头道。
<br/>　　这个鲁东，也是内宫弟子，但是，为人太过狂暴，下手没轻没重。
<br/>　　以前，对方就失手打死了两名同门，受到过重罚。
<br/>　　若是那段惊天被弄死了，不仅鲁东受罚，他也要受罚。
<br/>　　“褚彦哥，你放心吧！不一定不会把他弄死！”鲁东拍着胸脯保证道，这点分寸他还是会把握的。
<br/>　　“好吧，你注意点！”褚彦略一沉吟，便点了点头。
<br/>　　自己能不出手，自然最好。
<br/>　　不然的话，那段惊天事后向宫内告状，也有有些小麻烦。
<br/>　　“明白！”鲁东点头，随即立刻进入了熔岩池之中。
<br/>　　“哎！这个段惊天惨了！”
<br/>　　“碰到鲁东，不死也差不多了！”
<br/>　　“有意思，可惜不能亲眼看到！”
<br/>　　众人议论纷纷，均是面带调笑之色，对于这个段惊天，他们并无多少同情之心。
<br/>　　毕竟，他们都是土生土长的人王宫之人，而段惊天不过是个外人。
<br/>　　不远处，执事长老一直在注意着这一幕，不由得摇了摇头。
<br/>　　但愿那段惊天，不要被弄死了，不然的话，他只能向褚彦等人出手了。

<br/>　　苏莫不断的下潜，他已经进入了第四层之中，周身尽是绿色的岩浆。
<br/>　　越是向下，温度越是惊人，他体外的五行之力，被消磨的也越快。
<br/>　　不过，这远远没有到他的极限。
<br/>　　偶有火元精幻化的妖兽来袭，他能躲就躲，不能躲就全部击杀。

<br/>　　苏莫停了下来，到了第十一层，他的五行之力消耗的太快了。

<br/>　　就在此刻，一声暴喝，陡然响起，苏莫抬头望去，便见上方岩浆纷纷散开，一股强大的气势汹涌而来。
