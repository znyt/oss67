　　没过几日，苏莫便回到了苍穹神宫。
<br/>　　而后他立刻命人召唤安暖和天辰。
<br/>　　现如今，安暖被划分到了水月宫，成为了水月宫弟子，而天辰则是金煌宫弟子。
<br/>　　不多时，安暖和天辰两人，便先后来到了苏莫居住的宫殿中。
<br/>　　“拜见宫主！”
<br/>　　“拜见宫主！”
<br/>　　安暖和天辰，均是向抱拳苏莫行礼。
<br/>　　两人的目光望着苏莫，眼眸中蕴有无法言喻的感慨。
<br/>　　他们怎么也没有想到，以前的那位十一师弟，居然能成为苍穹世界无敌的强者。
<br/>　　而且，还是在短短两年多的时间里，就做到了这一步。
<br/>　　这真是超出了所有人的想像！
<br/>　　“两位免礼！坐吧！”苏莫挥了挥手，示意两人就座。
<br/>　　“是！宫主！”安暖和天辰恭敬有礼，抱拳道谢之后，便在宫殿中端坐了下来。
<br/>　　现如今，两人不再称呼苏莫为师弟，不是他们不认苏莫，而是时代变了。
<br/>　　苏莫望着这两位曾经的师兄师姐，心中暗叹，时隔这么久，随着修为实力的差距，和身份地位的变化，他们之间已经产生了一种无形的距离。
<br/>　　当年，他和天辰的关系极好，可以随意的开玩笑，但现在这种兄弟情义已经变了，变成了上下级的关系。
<br/>　　“二师姐、十师兄，此次让你们前来，是有一事相询。”苏莫沉声说道。
<br/>　　“宫主请说！”
<br/>　　“宫主，你尽管问吧！我们知无不言！”
<br/>　　安暖和天辰纷纷开口，两人心中猜测，苏莫肯定是要询问有关黄泉魔宗的事情。
<br/>　　苏莫沉吟一番，道：“你们二人在黄泉魔宗待了不短的时间，不知道了不了解搜魂术？”
<br/>　　“搜魂术？”两人闻言一怔，随即安暖摇了摇头，表示不了解，天辰则是陷入了沉思之中。
<br/>　　“十师兄，你了解？”苏莫见此立刻出声询问。
<br/>　　天辰沉默了片刻，道：“搜魂术，乃是黄泉魔宗的一种特殊的秘术，此术在黄泉魔宗，只有武皇境的高层，以及那些有贡献的武王境武者，才有机会修习。”
<br/>　　“那你呢？会吗？”苏莫立刻问道。
<br/>　　“不会！”天辰摇了摇头。
<br/>　　苏莫闻言有些无奈，看来还是要想办法擒拿那些分散在外的黄泉魔宗之人。
<br/>　　随即，苏莫又向天辰了解了一番收魂搜的情况，便挥退了两人。
<br/>　　而后，苏莫又招来了海明江，命其立刻派人在整个苍穹世界之内，捉拿黄泉魔宗的余孽。
<br/>　　海明江领命，立刻便下去准备了。
<br/>　　天涯海阁在外界的人手极多，虽然海明江现在不是阁主了，但也是苍穹神宫的长老人物，自然可以随意调动。
<br/>　　随着天涯海阁的动作，随后的几个月时间里，整个苍穹世界之内，动荡不已，大量的黄泉魔宗余孽被擒拿或斩杀。
<br/>　　那些有武王实力的黄泉魔宗余孽，纷纷被擒拿，而后带回苍穹神宫。
<br/>　　而武王境之下的人，则是被直接斩杀。
<br/>　　这一次，不仅是要抓人，也是一次大清洗，整个苍穹大陆五大洲，四方海域，无不波及。
<br/>　　其它的一些大势力，见苍穹神宫如此动作，都是噤若寒蝉，不敢有丝毫的异动。
<br/>　　另外，北海冰宫得罪苏莫的消息，也在苍穹世界传开了。
<br/>　　这让其它的一些大势力，立刻下令约束门下弟子，不能得罪任何苍穹神宫之人。
<br/>　　一时之间，偌大的苍穹世界，苍穹神宫威势一时无两，大有众生臣服之态。
<br/>　　而外界的风风雨雨，苏莫没有多管，一切交给海明江之后，他便过起了清闲日子。
<br/>　　每日他除了修炼，便是偶尔指导指导门下弟子。
<br/>　　夕儿又闭关了，得到了水月宫传承的她，修为和实力正经历高速提升的阶段。
<br/>　　苏莫目前的修为，已经达到了武皇境四重的巅峰，但他已经没有任何修炼资源了。
<br/>　　苍穹神宫的资源，即便有天涯海阁的供应，也只够苍穹神宫的正常运转，根本不能支撑他的修炼。
<br/>　　不然的话，他一动用资源，怕是就要将所有资源耗尽了。
<br/>　　不过，苏莫现在也不急躁，毕竟他已经无敌了，而距试炼者到来，还有几年的时间，足够他准备了。
<br/>　　当然，他也不可能只靠武魂吸纳天地间的灵气修炼，这样实在太慢了。
<br/>　　在苏莫清闲了一个月后，他等来了要等的人，北海冰宫宫主和三长老！
<br/>　　……
<br/>　　这一日，北海女皇和三长老到来，身形降临在了苍穹神宫的山门广场上。
<br/>　　北海女皇身材丰腴，肤白貌美，极为引人注目，但最引人注目的还是三长老。
<br/>　　只见三长老的背上，背着几根粗长的荆杖，前来负荆请罪。
<br/>　　这种荆杖，乃是北海冰宫惩罚门下弟子所用。
<br/>　　一时间，三长老的装束，吸引了大量的苍穹神宫弟子围观。
<br/>　　北海冰宫虽然距离中洲很远，但北海女皇来过苍穹神宫，基本上都认识她。
<br/>　　众人见此情景，再结合传闻中北海冰宫三长老得罪苏莫之事，哪里还不明白这是怎么回事！
<br/>　　“哈哈！这就是北海三长老吗？”
<br/>　　“好滑稽啊！堂堂武皇境九重强者，居然如此落魄！”
<br/>　　“嘿嘿，得罪了宫主，无论是什么身份、无论是什么修为，能活命就已经不错了！”
<br/>　　“不知宫主会怎样惩罚他？”
<br/>　　一众苍穹神宫弟子的嬉笑声、议论声四起，众人心中都充满了浓浓的傲气。
<br/>　　他们为身为苍穹神宫弟子而骄傲、为身为苍穹世界最强势力的弟子而骄傲，为他们无敌的宫主而骄傲。
<br/>　　北海女皇目无表情，不知其心中所想，而三长老听着周围弟子的嬉笑声，脸色黑如锅底，恨不得钻进地底下。
<br/>　　但是，即便再丢人，他也不敢不来，他若是不来，就算苏莫放过他，北海冰宫都不会放过他。
<br/>　　少倾，三长老深深的吸了口气，努力平复自己的心境，做到心如止水。
<br/>　　嗖！
<br/>　　破空声响起，苏莫得到门下弟子的禀报，很快便来飞临了山门广场。
<br/>　　“拜见宫主！”
<br/>　　“拜见宫主！”
<br/>　　“拜见宫主！”
<br/>　　霎时间，呼声震天，所有广场上的弟子，纷纷向苏莫行礼。
<br/>　　苏莫随意的挥了挥手，全场瞬间安静了下来。
