　　“聚元功并不难，它和太极玄清道有相似之处。”苏重站在众位师兄面前侃侃而谈。
<br/>　　功法是他自创，从原理本质到表面运行苏重一清二楚。
<br/>　　一说起来就一发不可收拾，滔滔不绝讲了半天才意犹未尽停下来。
<br/>　　然后就看到众位师兄一脸呆滞的看着苏重。
<br/>　　“众位师兄师姐，听懂了吗？”苏重问道。
<br/>　　“听……大概听懂了吧。”杜必书不自信道。
<br/>　　“二蛋师兄，你就告诉到底怎么练就行啦，其他的我不想知道。”田灵儿摇头果断道。
<br/>　　其他几人闻言立刻点头，表示他们也不想知道。
<br/>　　苏重翻了个白眼，自己这半天算是白讲了，眼前这群家伙一个也没听进去。
<br/>　　苏重换了种方法，就和当初宋大仁教他修炼玄清道一样。
<br/>　　苏重给聚元功设计好一套仪轨，先调整坐姿，然后调整呼吸，最后调整思绪，进而冥想符文，引动草木精气入体。
<br/>　　果然，苏重这么一改变策略，一众师兄师姐全都明白了。
<br/>　　苏重摇头，真是捡了芝麻丢了西瓜。要是明白聚元功原理，修炼起来就会百无禁忌速度飞快。
<br/>　　可真要按部就班一步步练习，速度可就会大大降低。
<br/>　　手把手教会众人，苏重就回到后山竹屋。
<br/>　　“聚元功已经练到先天，该考虑下一步如何修炼了。”
<br/>　　苏重闭上眼睛感知体内。丹田内布满白色聚元内力，氤氲如雾飘飘荡荡。盯着聚元内气，苏重思考下一步该何去何从。
<br/>　　想了半天却毫无头绪，无奈睁开眼，却突然发现不知何时，田不易已经站在院子里等着他了。
<br/>　　“师傅，您来了怎么不叫我。”苏重赶紧离开房间。
<br/>　　“我看你在修炼，就没打扰你。”田不易笑道。
<br/>　　他对自己这个小徒弟非常满意，尽管资质有限，但悟性惊人。没有因为无法修炼而产生丝毫负面情绪。和众位师兄师姐关系和睦，很快融入大竹峰。
<br/>　　今天更是毫不藏私，把自己辛苦创造的功法公开传授。他上午就隐在祠堂旁边偷听。苏重就连功法原理创造过程都和盘托出可见真诚。
<br/>　　只是自己那几个徒弟，一个个都是榆木脑袋，一点儿都没听进去！
<br/>　　田不易觉得，他该做些什么了。不能让自己这个小徒弟困在原地毫无方向。
<br/>　　“这是为师早些年历练，得到的一本秘法，威力不小。”田不易拿出一本泛黄书册递给苏重。
<br/>　　“庚金剑气？”苏重诧异，这是什么功法。
<br/>　　“这门功法实际上出自魔门。”田不易感慨：“不过我事后研究过，除了副作用比较大，其修炼路途还是比较正统的。”
<br/>　　和其他魔门功法，动辄需要他人精血魂魄，或者各种天奇毒相比，这本功法简直正派的不像话。不然也不会被田不易留下。
<br/>　　苏重迫不及待翻开书册，既然田不易把功法给他拿过来，必然是认为，这功法能解决他下一步的修炼问题。
<br/>　　苏重一头扎进剑气秘法，好半晌才抬起头。
<br/>　　他已经对庚金剑气有了初步了解。
<br/>　　“把飞剑融入自身法力，让法力具有锐利锋芒，释放出来威力惊人！只不过，这会损伤身体吧？”苏重了然道。
<br/>　　“不错，庚金剑气非常强横，只是太过强劲，好像有无数把细小飞剑在体内运转，每时每刻都在切割身体，对身体损伤极大！”田不易凝重到。
<br/>　　“师傅是看上了聚元功滋养身体的功效？”苏重明白了田不易想法。
<br/>　　“不错。想要修炼庚金剑气，要么身体始终比剑气强横，宛若剑鞘，能收摄锋芒。要么有修复身体损伤的灵丹妙药，能及时保护身体。聚元功同时具有这两个性质！如果你能让聚元功再上层楼，就可以任意运用庚金剑气！”田不易眼中神光一闪而过。
<br/>　　这本功法是他当年讨伐魔教时偶然所得。当时他可是在对方手底下吃过大亏的。不然也不会这么看重庚金剑气。
<br/>　　如果苏重能修炼成功，成就不一定就比修炼太极玄清道差。
<br/>　　“最重要的是，庚金剑气不需要纯粹灵气。你用聚元功牵引来的灵气，可以运行庚金剑气。”田不易已经考量了很久。
<br/>　　“多谢师傅费心！我已经知道该怎么做了。”苏重双眼发光。
<br/>　　“这一把断剑是我大竹峰收藏。来历已经不清楚，不过它有一个特性，就是非常坚韧。想必材料极其稀有，正适合你来修炼庚金剑气。”田不易递给苏重一把小臂长短黑色断剑道。
<br/>　　庚金剑气如果需要融合自身飞溅而起，苏重没有飞剑，如果用普通飞剑代替，效果一样不好。
<br/>　　这把短剑虽废，本质却极佳堪比九天神兵，用来修炼庚金剑气再好不过。
<br/>　　“谢师傅。”苏重接过断剑恭敬道。
<br/>　　可实际上，苏重并未打算用这把黑色断剑修炼。他刚才通读庚金剑气，却发现其实不用吸收外界金属，同样能练成庚金剑气！
<br/>　　送走田不易后，苏重立刻就开始钻研功法。
<br/>　　田不易到底是正统修士出身，只看到吸收金属修炼这一层。
<br/>　　这种修炼方法在他看来只能算是另辟蹊径，并不算邪魔外道。所以下意识的认为，这只不过是一种比较奇特的功法而已。
<br/>　　可苏重却发现，庚金剑气看似在吸收法宝飞剑内金属，实际上它吸收的确是另外一样东西。
<br/>　　“一种存在于天地间的凶戾气息！”苏重两眼发光。
<br/>　　修炼所用的法宝飞剑，只不过是一种媒介，借助它们可以吸引戾气聚集！在戾气冲刷下，法宝飞剑会变得腐朽不堪。就好似其中奇异金属被吸收掉一般！
<br/>　　再次把庚金剑气研读一遍，苏重确保没有理解错的地方，他立刻开始修炼。
<br/>　　闭上眼睛，按照功法所示观想符文，引动手中飞剑震动，汇聚天地间凶戾之气。
<br/>　　然后引起入体，苏重就感觉一丝冰凉刺骨却又暴躁不已的力量冲进经脉。
<br/>　　所过之处经脉遍体鳞伤。苏重立刻按照功法所示，停止吸引外界戾气，专心炼化吸收这一丝凶戾气息。
<br/>　　经过不断压缩运转，最终让其潜伏进丹田之内。
<br/>　　苏重睁开眼，感受着体内伤势，不由苦笑。这还只是一丝庚金剑气，如果以后吸收的更多。难保不被这剑气反噬。
<br/>　　聚元功确实能修复伤势滋养身体，但架不住庚金剑气太厉害！
<br/>　　必须想个法子，不能这么死练！苏重从来都不是个循规蹈矩之辈。
