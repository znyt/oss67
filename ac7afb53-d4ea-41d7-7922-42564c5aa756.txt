<br/>所有人都疑惑的看着这个从金广城人群中走出来的人，谁也不敢想象，竟然还有人敢面对一名圣魔导师的威慑出来硬抗的。<br/>　　<br/>　　“你终于舍得出来了！”欧米伽的语气中没有任何意外，他早就发现了隐藏在金广城一众人中的另一名圣级强者。<br/>　　<br/>　　看到紫色的斗气缓缓的覆盖住走出来的人，所有人的神色不由得一窒。<br/>　　<br/>　　圣战士！又一名圣级强者！所有人都下意识的后退了几步，万一这两人打了起来，他们可不想被波及到。<br/>　　<br/>　　李玄邈看到来人，眼睛眯了起来，眼神变得异常尖锐。这个人他认识，狂狮凯撒，两人之间还有一些渊源。<br/>　　<br/>　　“疯狮凯撒！没想到你竟然去了金广城。”李玄邈冷笑着说道。<br/>　　<br/>　　“我当是谁，原来是李玄邈大人啊！失敬失敬！”凯撒皮笑肉不笑的说道。<br/>　　<br/>　　年龄大的人听到对方竟然是疯狮凯撒，不由得倒吸一口冷气。疯狮并不是他的名字，而是大家送他的外号。因为他发起疯来十分可怕，六亲不认，而且力量惊人，和他实力强的人都很畏惧他。<br/>　　<br/>　　凯撒还是高级战士的时候曾经由于一点小事和一名黑铁战士闹了矛盾，结果被对方打的躺在床上一个月，等他伤好以后，失踪了两年，没有人知道他去了哪里。<br/>　　<br/>　　当他再次出现的时候，已经是一名黑铁战士了。他又去找了曾经那名黑铁战士报仇，结果对方已经成为了一名白银战士。不过让大家大跌眼镜的是，凯撒经过一番殊死搏斗，竟然以黑铁战士的实力，击杀了那名白银战士。这还不算，他竟然将白银战士一家全部杀死泄愤。<br/>　　<br/>　　后来只要是有人招惹到他，不管是谁，对方什么实力，他都会想尽各种办法将对方置于死地，然后屠杀对方的家小。他曾经以黑铁战士的实力，暗杀了一名黄金战士，从此以后更没有人敢招惹他了。<br/>　　<br/>　　他为人疯狂，做事情从来不顾后果，而且下手狠辣毫不留情。任何敢于冒犯他的人都会被他无情的杀死，就连家人都不放过，所以人们送他疯狮的称号。<br/>　　<br/>　　后来由于事情的闹的太大，让国王陛下知道了，国王陛下就责成李玄邈限期将他缉捕归案。<br/>　　<br/>　　结果双方大打出手，最后还是让他跑了，为此国王陛下还狠狠的责罚了李玄邈一番。后来听说他成为了一名圣战士，这件事情也就不了了之，帝国的通缉令也因此撤销了下来。<br/>　　<br/>　　根据卡罗帝国的法律，只要在被捕前，罪犯成为圣级强者，并且以后不再作恶，那么以前的种种罪行都可以一笔勾销，但是如果之后又有罪行，那么将会受到法律的严惩。<br/>　　<br/>　　疯狮凯撒因此又成为了一个神话，因为这条法律从建国时颁布到现在，一共也就赦免了不到五个人，其中就包括疯狮凯撒。<br/>　　<br/>　　自从他成为了圣战士以后，不但没有再听说他做过什么恶事，反而销声匿迹了。没想到竟然在这里见到了他，他竟然成为了金广城侯光祖的人。<br/>　　<br/>　　“凯撒，如果你敢再犯罪，我一定会把你抓捕归案！”李玄邈盯着凯撒，一字一顿的说道。<br/>　　<br/>　　“欢迎你随时带着证据来抓我，不过在你找到证据的时候，我一定会将你们红枫叶家族灭门销毁证据。”凯撒微笑着说道，丝毫没有把李玄邈放在眼里。铁血杀神在他眼中不过就是个笑话而已，毫无威慑力。<br/>　　<br/>　　李玄邈冷哼了一声，不再说话。这里是罗布森林，在神迹出现的一刻，这里面已经被默认为混乱之地了，不受任何法律约束。李玄邈现在心中清楚，如果不是有欧米伽威慑着，凯撒一定会毫不犹豫的对自己出手。<br/>　　<br/>　　“你们聊完了吗？”欧米伽很是不满的看了一眼凯撒，被当做空气让他很不爽，后果很严重。<br/>　　<br/>　　“不好意思，让你久等了！”虽然嘴上说着不好意思，但是凯撒丝毫没有抱歉的意思。<br/>　　<br/>　　“我的提议怎么样？你也拿到了不少好处，我们也不再追究，大家各退一步，海阔天空。”凯撒活动着身体，朝着欧米伽靠近着。<br/>　　<br/>　　欧米伽看到凯撒朝着自己靠近，丝毫不以为意，满不在乎的摇头否定着。<br/>　　<br/>　　通常情况下，圣魔导师实力是强，如果距离拉开，圣战士也许不一定是对手，但是距离近到一定程度，圣战士完全可以压制着圣魔导师，甚至可能击杀圣魔导师。但是欧米伽可不是普通的圣魔导师，他根本不在乎距离的远近。<br/>　　<br/>　　凯撒看到欧米伽对自己靠近毫无反应，心中暗喜。不过他并没有过于靠近，为了防止欧米伽起疑，他停在了欧米伽对面二十米的距离，这个距离对他来说和贴身没有区别。<br/>　　<br/>　　“你真的不准备接受我的建议？”凯撒眯起了眼睛，避免眼中的凶光闪露。<br/>
