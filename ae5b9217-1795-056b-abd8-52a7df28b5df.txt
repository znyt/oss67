　　“九域天才战。”
<br/>　　楚风眠默念一声。
<br/>　　之前在离开北邙宫的时候，这北邙学院的院长，告诉他的事。
<br/>　　这九域天才战，楚风眠也曾经听说过，算是从古至今，流传下来的一个传统，每过十年，举行的一次会战。
<br/>　　整个九域之中的天才，都会前去参加，而且这九域天才战，比起北武斗大会等，还要恐怖的多。
<br/>　　北域武斗大会，最多只有生死境的武者，前去参加。
<br/>　　而这九域天才战之中，甚至都不乏有些超越生死的人物，前去参加。
<br/>　　只要是年龄不过百岁的武者，都有资格，前去参加，对许多七大宗门的天才来说，他们足以是在四五十岁的时候，就已经是超越了生死，所以这九域天才战，才是无比残酷的战斗。
<br/>　　不过虽然残酷，但是却每一届之中都有着无数武者，乐此不疲的前去参加，因为这九域天才战，虽然残酷，但是经历一次次的生死之战。
<br/>　　只要是足以从中脱颖而出的武者，实力都将提升不止一个档次。
<br/>　　而且这九域天才战，所举行的地方，却正是一处古迹之中，远古角斗场。
<br/>　　在其中的一场场战斗，若是表现惊艳的话，还会被这远古角斗场，赐下奖励来。
<br/>　　远古角斗场的存在，已经是数十万年的时间，楚风眠在无数古籍之中，都曾经见过这个名字，但是传统依然不会变，真正的天才，便是会被赐下奖励。
<br/>　　这些奖励，乃是就连七大宗门等实力，都没有的宝物，所以这一点，也是足以吸引无数武者，前去参加的。
<br/>　　“居然这一届的九域天才战，就要开始了吗？”
<br/>　　楚风眠默念一声。
<br/>　　前世的他，也曾经去过这九域天才战，但是当时的他只能够是在一旁围观罢了，毕竟那个时候的楚风眠，没有任何的实力。
<br/>　　他早就想要参加一次九域天才战，看来这一次，倒是正好给楚风眠赶上了机会了。
<br/>　　楚风眠想要超越生死，谈何容易，他度过生死大劫，就比起寻常武者超越生死，都要困难的多。
<br/>　　虽然如今楚风眠距离超越生死，只差一步之遥，但是这一步，对楚风眠来说，却也是极为遥远，甚至是现在的楚风眠，都依然不知道这一步到底要如何走。
<br/>　　想要突破境界，最为需要的，便是在一场场的生死之战中磨练。
<br/>　　“去中域。”
<br/>　　楚风眠站起身来，已经下了决定，这九域天才战所举办的地方，远古角斗场，都是在中域之中。
<br/>　　在九域之中，便是有一句话，不入中域，无法成就强者之路。
<br/>　　中域，是整个九域之中强者最对的地方，也是最为繁华的地方，想要成就强者，都必须要前去中域，在无数天才之中脱颖而出，才行。
<br/>　　之前楚风眠实力不够，前去中域也只能是白白送死，不过现在，楚风眠却是自信满满了。
<br/>　　楚风眠现在的实力，已经是今非昔比了，足以是跟超越生死的人物平秋平坐，放在中域去，也绝不是弱者，更何况楚风眠还有着云羽天宫，剑皇殿，作为底牌。
<br/>　　遇到危险，大不了祭出剑皇殿来，也可以保住性命。
<br/>　　剑皇殿现在虽然吞噬了楚风眠大量的灵石，不过他的灵力恢复的依然不动，而且是用一点少一点，平时楚风眠也不敢轻易动用剑皇殿，毕竟这样一个大胃王，楚风眠可养不起。
<br/>　　楚风眠自己突破境界的灵石都不够用，又怎么够支撑剑皇殿的消耗，所以不到万不得已的时刻，楚风眠不会动用剑皇殿。
<br/>　　好在以现在楚风眠自己的实力，也不至于是被人随意拿捏。
<br/>　　“轰！”
<br/>　　楚风眠的身影，直接从悟剑峰之中飞了出去，他身形一动，猛然之间打破空间，从这北邙学院之中直接消失。
<br/>　　看到楚风眠离开。
<br/>　　在北邙学院的一处神秘空间之中，无数老祖，都坐在其中，这处神秘空间，也是北邙学院之中的一处圣地，只是无法跟北邙宫相比。
<br/>　　可是现在楚风眠将北邙宫带走，他们也只能够是来到这里，前来修炼。
<br/>　　“楚风眠离开了。”
<br/>　　“看样子，此子真的要去中州了。”
<br/>　　“哼，一个不知天高地厚的小子，他既然敢去中州，那好，那他就不用在想要回来北域之中来了。”
<br/>　　在神秘空间的最深处，坐在地上的一尊老者，夜皇，冰冷的开口道。
<br/>　　“此子就算是有着北邙宫的保护又如何？北邙宫，终究不能护他一辈子。”
<br/>　　坐在旁边的武皇，也是冰冷的开口道。
<br/>　　他们对楚风眠的仇恨，连绵不绝，无法洗刷，在他们的心中，几乎是每时每刻之间，都在研究如何，才可以轰杀楚风眠。
<br/>　　“居然敢夺走北邙宫，这小子肯定是跟圣君，有着些关系，不过北邙圣君，已经是一个死人了，北邙学院是属于我们的，这小子必须要死，绝不能留！”
<br/>　　又是一声阴狠的声音响起，这声音，似乎并非是楚风眠所见到过的，是一道陌生无比的声音。
<br/>　　“诸位，这楚风眠的确必须要死，不过在他死之前，我们还可以利用他做些事情。”
<br/>　　一声轻笑声响起，正是那北邙学院的院长，他的嘴角一直露出了几分若有若无的笑意，他的目光，一直是注视这楚风眠，看着楚风眠离开了北邙学院，前去中域。
<br/>　　“我已经布置好了计划，诸位放心，北邙学院肯定是属于我们的，那北邙圣君，不过是死人一个，当初他的后人，都被我们赶走了，一个区区小子，又能如何翻天？”
<br/>　　“不错，既然院长已经有了计划，那我们便等着吧……”
<br/>　　在神秘空间之中的声音，逐渐的沉寂起来。
<br/>　　与此同时，楚风眠已经是离开了北邙学院之中，直接前往中域。
<br/>　　他现在的实力，已经是今非昔比，就算是不借助传送阵，速度也是快到极点。
<br/>　　楚风眠脚下一动，便是在虚空之中踩出了一条通道来，这通道稳固无比，楚风眠走在其中，不受到任何一点空间风暴的侵蚀，健步如飞，一日千里。
