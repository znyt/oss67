　　所谓佛道之争，争夺的便是佛与道在洪荒之中的正统地位。
<br/>　　若是之前，西方立佛教，元始说不得还想阻止一二，但如今以他的修为，早已看破了，多一个佛教也没有被他放在心上。
<br/>　　大道之争，佛道二教争的其实并不只是正统地位，因为佛本是道，二圣还是从玄门道教分离出去的呢，有什么好争的，其实更多时候争的是洪荒气运。这点气运对他如今来说，已经作用不大了，他也不屑去破坏佛教成立之事。
<br/>　　不过，虽说气运对他无甚大用，但对于他门下弟子却有大用，元始眼珠一转，心中便有了想法。
<br/>　　西方二圣的佛教是必须立的，因为他们成圣之时，曾立誓向上天借功德，如今立下佛教，便是为了完成誓言，是故天道并虽然会降下一些考验，但并不大，以二圣的修为，能够轻易完成这些考验，成功立下佛教。
<br/>　　果然，天地间一股巨大威压降临，压向那虚空中的菩提巨树，想要将之压垮，但这并未坚持多久，便被二圣破去了，最终，二圣成功经受住天地考验，得愿立教。
<br/>　　“如是我闻。一切佛在舍卫国，袛树给孤独园，与大比丘众，千二百五十人俱，尔时世尊……”
<br/>　　“时长老须菩提……”
<br/>　　“……一切世间天人阿修罗，闻佛所说，皆大欢喜，信受奉行！”
<br/>　　一阵佛音传唱开来，以此经为佛教之本，接引声音宏大，传遍整个洪荒三界。
<br/>　　所有苍生、无尽修士、无数大神通者纷纷仔细聆听圣人讲道，接引此番讲佛家精义，其中蕴含着无尽大智慧，有着无数佛门妙法神通，乃是佛家立道之根基所在，玄妙无穷。
<br/>　　圣人讲道，自然是非同凡响。
<br/>　　但见西牛贺洲苍穹之上，霞光万道，佛光亿万，瑞彩千条，禅音无数，紫气千丈，在那无尽祥云笼罩之中，接引圣人端坐于虚空之中，身下自动化出大道金莲，口中滔滔不绝，宣讲佛道经典妙法，为天下苍生开启灵智，给他们内心深处埋下一颗佛种。
<br/>　　佛种埋下，就算他们现在不拜入佛教，但日后有机会，随时间的流逝，定然也会茁长成长，为众生开启一颗向佛之心。
<br/>　　接引圣人脑后一道道神光缭绕，化为一道道佛家神轮，有智慧之光，功德圣光，普渡之光、慈悲之光、寂灭之光……
<br/>　　光辉洒下，让无数修士颇有种立地顿悟的感觉，但就是差那么一点，让人心中痒痒，甚是不舍。
<br/>　　“佛道果然妙法无边，自有其独到之处，我当学之。”
<br/>　　“佛道之中蕴含大智慧，大超脱、大寂灭之法，若生能够学会，我突破在望啊。”
<br/>　　“这佛道修行果然另辟蹊径，神通之中更是专门克制魔修，我一定要学习。”
<br/>　　“我早已是大罗金仙圆满，但却始终不能突破准圣，这西方舍利大神通无需先天灵宝亦可斩尸，很适合我这种人……”
<br/>　　“……”
<br/>　　接引的声音宏大无边，传遍洪荒大地，许多修士，特别是散修或者贫穷的修士，都纷纷生出向佛之心，打算待佛教成立之后，便去拜入佛教。如此既能得到二位圣人的庇佑，又能学到大神通，何乐而不为？
<br/>　　元始摇了摇头，佛教能够在时间长河之中存在无数年，而道教却逐渐落寞于历史之中，这其中还是有一定道理的。佛门的蛊惑神通非常了不起，虽说独辟蹊径，但只要笑到最后，不管是什么，都没人会说其不是。
<br/>　　佛教初期的教义是众生平等，人人可以成佛做祖，而且还有许多其他好的教义，但随着时间的流逝，每个人领悟不同，亦或者被后人曲解，一代代传下去，佛教经义早已变了味，不似最初。
<br/>　　总之，前期的佛教教义虽有瑕疵，但也无甚大碍，但后期的佛教，元始便不敢恭维了。
<br/>　　佛教已立，气运之争就算你不去参加，也无法避免，这是东方玄门与西方佛教的争夺，元始虽不想如此，但为了东方，或者自己阐教，也要参与到其中。
<br/>　　“佛教，立！”
<br/>　　最后，随着一声大喝，响彻十方诸天，三界星空，宣告佛教的成立。
<br/>　　“自今以后，西方教已去，唯有大乘佛教，我为阿弥陀佛！”接引双手合十，大声道。
<br/>　　“我为准提佛母！”准提圣人亦是如此。
<br/>　　话音一落，天地立刻生出感应，梵音响彻洪荒三界，两者头顶长发自动脱落，身上道袍换上一身僧袍，脑后功德金轮护体，一身道家修为立刻换成佛家之力，给人一种慈悲之感。
<br/>　　“阿弥陀佛，我为佛教释迦牟尼佛，如来佛祖！”
<br/>　　这时，一尊全身金光大盛，禅音环绕，一手指天，一手指地，天上地下，唯我独尊的巨大金色身影从西方天地间缓缓升起，浩大无边，佛光普照诸天。
<br/>　　这正是接引圣人执尸转世之身，如今已经觉醒，修为一举恢复到准圣圆满，成佛做祖。
<br/>　　“嗡嗡嗡——”
<br/>　　虚空一颤，嗡嗡作响，随即，于西方灵山之上，显化出一片浩大的金色气运之海，云雾迷蒙，一条金色气运神龙在气运海中遨游穿梭，欢快吞吐着气运所化的云雾。
<br/>　　“佛道当立，以十二品功德金莲镇压气运！”
<br/>　　阿弥陀佛翻手间取出一座金莲，扔到上空的气运云海之中，金莲迅速变大，化为一株金色莲花，扎根于云海之中，莲叶摇曳，散发出莫名道韵，气运所化神龙绕着金莲欢快遨游。
<br/>　　“嗡——”
<br/>　　顿时，虚空一颤，滚滚道韵向外扩散开来，道音响彻之际，无数佛陀、菩萨、罗汉影像在上面一一闪现，有的已是实质，而有的却只是一道虚影，同时西牛贺洲的无尽气运向着气运云海汇聚而来，注入十二品金莲之中。
<br/>　　“滚——”
<br/>　　突然，一声大喝声从太华山顶传出，随后，诸神只见一片黑雾遮天，一条狰狞黑龙显现，朝着西方金龙一声长嘶，身旁，一朵十二品黑色莲花摇摇曳曳，散发着黑色灭世的气息。
<br/>　　那是十二品灭世黑莲！
<br/>　　顿时，太华山劫魔殿统治范围之内，气运回笼，不再向外流逝。
<br/>　　与此同时，万寿山五庄观上空，一本泛起土黄色的书本出现，发出莫名道韵，镇压自身气运不流失。
<br/>　　看到这一幕，阿弥陀佛与准提佛母心中苦涩不已，西牛贺洲本身便没有多少气运，如今已被占据了两处，他们佛教却是得不到多少气运，最多也只是与劫魔殿、万寿山形成三足鼎立局势。当然，目前看来，是劫魔殿最为强大，万寿山最弱。
<br/>　　见此，东方诸圣脸色多少有些好看了，其余众多大能亦脸色复杂地看向佛教。
<br/>　　“唧——”
<br/>　　突然，一道尖叫声响起，随即一道带着血气的黑色身影闪现，现于佛教气运之海中。那股气息中带着浓浓的邪恶与阴暗，更有一股嗜血之意，好似一头凶恶的远古凶兽。
<br/>　　那凶兽黝黑的双眼盯着那正在吞吐气运的十二品功德金莲，眼中闪过一阵兴奋。
