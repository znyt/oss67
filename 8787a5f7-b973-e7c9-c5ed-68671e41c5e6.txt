　　万道剑冢，距离不神剑山不是很远，约莫一个多时辰，苏莫便到了。
<br/>　　这里是一处荒芜的大地，方圆万里寸草不生，也没有城池和城镇。
<br/>　　只见大地之上，有着一个巨大的天坑，方圆十里，深不见底。
<br/>　　深坑之中，弥漫着浓浓的灰色雾气，不断的翻滚，但是却不会飞到深坑之外。
<br/>　　这灰色雾气，遮蔽了视线，挡住了神识，让人无法一窥其内真容。
<br/>　　苏莫之前询问过绫裳，万道剑冢之内，到底是什么情况。
<br/>　　对方告诉他，万道剑冢之内，是如同洞穴一般的地方，一座洞穴连着一座洞穴，没有尽头。
<br/>　　因为，从来没有谁，能走到尽头。
<br/>　　苏莫环视四方，周围没有半个人影，因为距离万道剑冢开启，还有约莫两个月的时间，暂时还没有人来。
<br/>　　随即，苏莫飞向不远处，身形落在了一片乱石林立之地，在两块大石之间盘坐了下来。
<br/>　　他并不着急，既然时间未到，那就等待就是。
<br/>　　随后，苏莫盘膝闭目开始了修炼。
<br/>　　他并未吸纳源石，因为修为已经遇到了瓶颈，需要一段时间的沉淀与巩固。
<br/>　　所以，他一边凝练修为，一边参悟吞噬法则。
<br/>　　时间在修炼之中，过的飞快，转眼之间，一个多月过去了。
<br/>　　万道剑冢的入口附近，开始不断的出现武者，并且越来越多。
<br/>　　这些武者，分散在万道剑冢入口附近，均是在等待。
<br/>　　苏莫睁开眼眸，查看了一番，这些人的修为都不低。
<br/>　　修为最低的，都和他相当，拥有二品高阶的修为，而修为高的人，大多都达到了三品，其中不乏一些三品高级修为之人。
<br/>　　“哈哈，古兄，没想到你也来了！”
<br/>　　“数十年不见，古兄的修为又有长进啊！”
<br/>　　“见过古兄！”
<br/>　　这时，朗喝声四起，只见一道高大挺拔的身影破空而来，停留在了万道剑冢上空。
<br/>　　很多人纷纷向那高大挺拔的身影打招呼，显得客气至极。
<br/>　　此人，面容刚毅，气质独特，不是别人，正是古剑山。
<br/>　　古剑山作为万古天骄，自然是众人交好的对象，一时间一群武者围拢而去。
<br/>　　古剑山也不算太孤傲，向众人一一点头回礼，随后便落在了万道剑冢所在的深坑边上，盘膝等待了起来。
<br/>　　不是的有人来到万道剑冢，均是年轻武者，每一个都是妖孽人物。
<br/>　　没过几日，万道剑冢附近，便已经聚集了一千余人。
<br/>　　苏莫长身而起，一步百丈，也来到了剑冢附近。
<br/>　　因为，他不想太早引起别人的注意，在人群之中更合适。
<br/>　　伫立在深坑边缘，苏莫环视众人，暗暗惊讶，每一个人貌似都非常不凡。
<br/>　　四面八方，依旧不时的有人前来，这些人基本上都是剑修，身上有无形的锋芒留露。
<br/>　　这时，只见天边再次有人到来，一群三人，如同疾电一般，破空而至。
<br/>　　这三人，有两人是三品高级的修为，另一人是三品中级的修为。
<br/>　　而那三品中级修为之人，不是别人，正是公孙嘉。
<br/>　　公孙嘉的到来，不免又引起了不少人的瞩目，相熟之人纷纷招呼了起来。
<br/>　　“诸位客气了！”
<br/>　　公孙嘉微微抱拳，向众人拱手，一番客套之后，便在周围寻找了起来。
<br/>　　他想要看看，苏莫有没有来此，不过，很快他就失望了，因为并未看到苏莫的身影。
<br/>　　“或许有改变了形态！”想到苏莫可以变化成许焱，连极空城城主这种高手都无法识破，公孙嘉猜测，苏莫可是用改变了形态。
<br/>　　苏莫微微瞥了公孙嘉一眼，便不再关注，他不可能去和对方相认。
<br/>　　此人能猜到他的真实身份，那么，天尘宇在找他的事情，肯定知道的人不少。
<br/>　　所以，他还是要小心点为妙。
<br/>　　“公孙嘉，这都几年了，你怎么还是那么不长进？”古剑山打量了公孙嘉一眼，面带揶揄的说道。
<br/>　　当初，他可是和公孙嘉同阶，现如今他已经踏入了三品高级，而对方还是三品中级。
<br/>　　“古剑山，追上你轻而易举！”公孙嘉轻笑一声说道，他的修为已经有了要突破的迹象，晋升三品高级已经不远了。
<br/>　　“哈哈，希望你能追的上！”古剑山大笑一声，他可不信对方能追上他，二十年之内，他就能晋升四品了。
<br/>　　咻咻咻！！
<br/>　　就在此刻，只听天际的尽头，响起一阵刺耳的破空声。
<br/>　　众人转头望去，便将一大群人，浩浩荡荡而来。
<br/>　　这群人，共有二十多人，个个气息强大，气势如虹。
<br/>　　“是终极神王殿的人！”
<br/>　　众人见此，立刻就知道了来人是谁，因为如此阵仗，必是终极神王殿之人。
<br/>　　万道剑冢的终极剑域，与终极神王殿有千丝万缕的关系，每一次万道剑冢开启，终极神王殿的妖孽，必然是最多的。
<br/>　　苏莫也目视这群人，随着这群人飞快临近，他很快就锁定了一人，慕容雷利。
<br/>　　慕容雷利，依旧是三品中级的修为，而其他终极神王殿的弟子，修为不一。
<br/>　　不仅有二品高级之人，甚至还有四品修为之人，气息无比的强大。
<br/>　　当然，修为高不等于战力也逆天，终极神王殿的万古天骄，只有慕容雷利一人。
<br/>　　这一刻，古剑山的眼眸之中，爆射出夺目的锋芒。
<br/>　　这一次，他来万道剑冢，不仅是为了磨炼自己的剑道，更为了击败终极神王殿的各大妖孽，包括慕容雷利。
<br/>　　眨眼之间，二十余名终极神王殿的弟子，来到了万道剑冢的上空，然后便停了下来。
<br/>　　一群人俯瞰下方众人，随即为首的一人开口了：“诸位，欢迎来到万道剑冢，作为东道主，在下韩生代表终极神王殿提醒诸位，在万道剑冢之中量力而为，莫要白白丢了性命！”
<br/>　　朗朗的声音传遍全场，开口之人，是一名白袍青年，面容古板而冷厉，修为高达四品低级。
<br/>　　来此的都是各界域顶级妖孽，甚至是万古天骄，终极神王殿不希望太多的人陨落，所以每次都会提醒一番。
<br/>　　“韩生，多谢你的提醒！”
<br/>　　“韩生，数十年不见，没想到你已经晋阶四品，恭喜了！”
<br/>　　“只希望你们终极神王殿，不要仗着人多，欺压我等！”
<br/>　　不少人开口了，有人感谢，有人赞誉，还有人冷嘲热讽。
<br/>　　毕竟，在万道剑冢之中，终极神王殿的弟子人多势众，很容易联手排斥他们。
<br/>　　对于这位韩生，很多人都不陌生，虽然不是万古天骄，但也是最顶级的妖孽，天赋和万古天骄也不差多少了，一身毁灭剑道强大无比。
<br/>　　面对各种不同的声音，韩生面无表情，没有理会，他只负责提醒一句。
<br/>　　若是真有人不自量力，死了也就死了，与他无关。
<br/>　　随即，一圈终极神王殿的弟子，纷纷降落而下。
<br/>　　“慕容兄，幸会！”
<br/>　　“慕容雷利，不知道你的雷电法则，修炼到了何种地步？”
<br/>　　作为万古天骄，慕容雷利显然比韩生更有人缘，刚一落下，便有不少人上前攀谈。
<br/>　　慕容雷利不想与不相干之人闲谈，客气了几句之后，便来到了公孙嘉的身前。
<br/>　　“公孙兄，上次之事，是我连累了你，还请莫怪！”
<br/>　　慕容雷利向公孙嘉抱拳，上次对方被擒之后，他就返回了终极剑域，不过后来也知道了对方有惊无险，被秦陨放了回来。
<br/>　　“无妨，并不是你的错！”公孙嘉摇了摇头，并未怪罪。
<br/>　　“公孙兄，下次若是能遇到秦陨，我定将此人打成死狗，带到你的面前，交给你惩罚！”慕容雷利沉声说道，想到那个秦陨，他心中就是一阵怒火。
<br/>　　不远处，苏莫自然听到了慕容雷利之言，顿时眼眸微眯，眸中寒光闪烁。
<br/>　　“这次，就是你的死期！”苏莫心中暗道，若是有机会，他定然杀了此人。
<br/>　　他还没有吞噬过万古天骄，此人，或许就是第一个。
