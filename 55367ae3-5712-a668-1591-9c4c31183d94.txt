        “好，我听你的就是！”傅昼景微笑着揉了揉童筱颖的脑袋，眼底满是宠溺。
<br/>
<br/>        童筱颖欣慰地点了点头，脑袋往傅昼景的胸膛上靠去。
<br/>
<br/>        不经意间，傅昼景又回想起昨晚他和季思妍之间所做的那件事情。
<br/>
<br/>        顿时，这让傅昼景的心里一阵阵发闷，如涸泽之鱼，连呼吸都变得异常艰难。
<br/>
<br/>        傅昼景的目光渐渐黯然失色，嘴角的弧度一点点收敛。
<br/>
<br/>        “筱颍，如果，我做了什么对不起你的事情，你一定要原谅我。或许，那些真的不是我的本意。”傅昼景努力扯了扯唇，心虚地从唇瓣间挤出这些话。
<br/>
<br/>        童筱颖忍俊不禁地说道：“嗯嗯嗯，我一定会原谅你！”
<br/>
<br/>        “筱颍，让我抱着你再睡一会儿。”傅昼景搂着童筱颖，在她的头顶吻了吻，缓缓闭上了眼睛。
<br/>
<br/>        两人相互搂着，相互依偎着，就像两条司峻筱颖的鱼。
<br/>
<br/>        每一次，在他最落魄的时候，陪在他身边的人，总是她。
<br/>
<br/>        小时候一样，长大了也一样。
<br/>
<br/>        从今以后，他和筱颍人生中的那本书该翻篇了。
<br/>
<br/>        临海机场。
<br/>
<br/>        一辆奢华至极且车牌号码格外独特的黑色越野车，停在了机场大门外前坪的马路边，引来百分百的回头率，更有路过的女生，忍不住举起手机偷拍起这辆车来。
<br/>
<br/>        而离这辆车不远处的一座米黄色大理石方柱后，倚着一个身着熨烫得挺直的黑色西裤和黑色衬衣的男人。
<br/>
<br/>        他戴着墨镜，棱角分明且俊美非凡的脸庞，犹如巧夺天工般精致。高挺的鼻梁下，薄唇紧抿，浑身透着一股生人勿近的冷酷。
<br/>
<br/>        前前后后有一些路过的女人，一个个露出惊艳的目光，朝男人看去。
<br/>
<br/>        她们全都被他身上散发出的寒气所震慑到，就算他长得再帅，也没有女人敢故意靠近他。
<br/>
<br/>        他一直站在那儿，明明格外引人瞩目，却仍旧入不了他心爱的那个女孩的心。
<br/>
<br/>        童筱颖和傅昼景拖着行李箱的身影，出现在了男人的墨镜上。
<br/>
<br/>        他们有说有笑，从他眼前路过。
<br/>
<br/>        傅夜沉摘下墨镜，握在手心里，目光追随着童筱颖的背影而去。
<br/>
<br/>        筱颖曾在“夜惑”里驻唱时，唱过这样一首歌，里面的歌词，让他此时此刻，莫名其妙地就记得一清二楚。
<br/>
<br/>        或许，是因为感同身受：
<br/>
<br/>        我知道，故事不会太曲折；
<br/>
<br/>        我总会，遇见一个什么人；
<br/>
<br/>        陪我过，没有了她的人生；
<br/>
<br/>        成家立业之类的，等等。
<br/>
<br/>        她做了，她觉得对的选择；
<br/>
<br/>        我只好，祝福她真的对了；
<br/>
<br/>        爱不到，我最想要爱的人；
<br/>
<br/>        谁还能要我，怎样呢？
<br/>
<br/>        我爱的人，不是我的爱人。
<br/>
<br/>        她心里每一寸，都属于另一个人。
<br/>
<br/>        她真幸福，幸福得真残忍。
<br/>
<br/>        让我又爱又恨，她的爱怎么那么深？
<br/>
<br/>        我爱的人，她已有了爱人。
<br/>
<br/>        从他们的眼神，说明了我不可能。
<br/>
<br/>        每当听见，她或他说我们，就像听见爱情永恒的嘲笑声……
<br/>
<br/>        筱颖真的走了，可他却不再想跟她说“再见”。
<br/>
<br/>        因为，从今以后，应该永远不会再见面了吧！
<br/>
<br/>        澳洲，悉尼。
<br/>
<br/>        “小卫，快看快看！那就是悉尼塔，对不对？好壮观！”童筱颖像个孩子一样，白净的脸上掩不住喜悦，拉着傅昼景的手，兴奋地东张西望。
<br/>
<br/>        第一次来到澳洲，童筱颖可说从下飞机的那一刻开始便惊呼连连。在这陌生又遥远的国度里，任何没看过的人和事物，对她来说都是新奇的。
<br/>
<br/>        “我们去悉尼塔里面逛逛！”傅昼景将童筱颖拉到自己跟前，宠溺地刮了刮她的鼻梁，微笑着说。
<br/>
<br/>        童筱颖先是点头，又猛地摇了摇头：“不行不行！我们得先去医院！”
<br/>
<br/>        “今天来了，我们就好好玩，先玩够了再说！”傅昼景一本正经地提议。
<br/>
<br/>        童筱颖仍旧摇头，拧着眉头，瘪瘪嘴，坚持道：“不要！先去医院做检查！健康比玩更重要啊！”
<br/>
<br/>        拗不过这丫头的倔强，傅昼景无可奈何地耸耸肩，只好跟她妥协道：“好好好，我们先去医院，然后再来玩。”
<br/>
<br/>        “嗯嗯！”童筱颖这才眉开眼笑地点点头。
<br/>
<br/>        可是在她表面上这坚强的笑容背后，隐藏着的却是一颗担忧的心。
<br/>
<br/>        童筱颖希望是自己的多虑。
<br/>
<br/>        到了医院后，一系列的检查做完，又在医院里等结果，找医生看结果。
<br/>
<br/>        童筱颖以为一切都会没事，而结果却跟她先前所料想的一样。
<br/>
<br/>        傅昼景的检查结果，并不乐观。
<br/>
<br/>        就像一道晴天霹雳，让两人刚刚还沉浸在快乐之中的气氛，瞬间被撕破，接肘而来的是一种炼狱与煎熬。
