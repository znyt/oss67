　　宫殿内，陷入了寂静之中，火元圣师眼帘低垂，沉默不语。
<br/>　　苏莫心中无奈，等了片刻之后，又问道：“前辈，现在巫族之人已经知道我在神武学府，我该怎么办？”
<br/>　　“你想怎么办？”火元圣师再次反问道。
<br/>　　苏莫闻言脸皮一抽，他询问对方，对方却老是询问他的想法。
<br/>　　深吸了口气，苏莫沉声道：“我是在想，若是神武学府都给予我庇护，那我就留在学府，若是无法给予我庇护，就只能先逃走了！”
<br/>　　一边说着，苏莫的眼眸还一边注意着火元圣师，因为神武学府是不允许弟子彻底脱离的，除非加入背后的十三个势力之一。
<br/>　　火元圣师闻听此言，沉吟了少倾，道：“你先在学府待着吧！只要在学府之内，就不会有事！”
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即略显迟疑的道：“那天冥星巫族的大祭司肯定不会放过我，若是亲自来此怎么办？”
<br/>　　“无妨，巫族主脉不插手的话，天冥星巫族支脉还不敢来神武学府撒野！”火元圣师摇头说道。
<br/>　　神武学府背后可是有十三个大势力，或许每一个势力都没有巫族天冥星支脉强大，但十三个势力加在一起，完全不是巫族天冥星支脉可以招惹的存在。
<br/>　　而且，就算不说背后的十三个大势力，就单单只是神武学府，都比巫族天冥星支脉强大的多。
<br/>　　神武学府存在了很久的岁月，历来都有很多弟子最后不愿意加入背后的十三个势力，从而一直留在神武学府。
<br/>　　就比如火元圣师，曾经是神武学府的弟子，后来留在学府做导师，一直到成就武圣，依旧没有离开学府。
<br/>　　像火元圣师这种人物，在神武学府之中，可是不在少数。
<br/>　　可以说，神武学府的实力，比之背后皇祖圣朝这种大势力，都要强大很多。
<br/>　　当然，神武学府和背后的十三个势力，关系千丝万缕，神武学府的众多强者，和十三个大势力中的很多人都是好友、师徒、甚至是师兄弟的关系。
<br/>　　神武学府的力量，从某种意义上讲，也算是十三个势力的力量。
<br/>　　当然了，天冥星巫族虽然不算多强，但巫族之中还有主脉，巫族主脉是整个古灵星河，最强大的几个势力之一，根本不是神武学府能抗衡的存在，因为巫族主脉有虚神境的大能。
<br/>　　“那就好！”
<br/>　　苏莫闻言松了口气，随即又问道：“那巫族主脉会插手吗？”
<br/>　　“不清楚！但想来此等事情，天冥星一脉会自行处理！”火元圣师摇头道。
<br/>　　这种事情一般情况下，支脉是不会请主脉出手的，毕竟对付苏莫一个小小的武帝境武者，天冥星支脉不太可能求助主脉。
<br/>　　除非苏莫有能力威胁到天冥星巫族支脉，他们才会请动主脉出手除掉苏莫。
<br/>　　苏莫闻言松了口气，这样看来他短时间能还是比较安全的，随即他向对方抱拳，道：“多谢前辈指点！”
<br/>　　“嗯，你也无需有什么压力，回去安心修炼吧！”火元圣师微微颌首，随即回了挥手道。
<br/>　　“晚辈告辞！”苏莫再次向对方行了一礼，随后缓缓退出了大殿。
<br/>　　苏莫走后，火元圣师叹了口气，随即再次闭上了眼眸，静静的打坐。
<br/>　　片刻之后，轻微的脚步声响起，一声白衣的木璃走进了大殿之中。
<br/>　　“导师！”木璃微微欠身，向火元圣师行礼。
<br/>　　“木璃啊！有何事？”火元圣师睁开眼眸，微笑着问道，对于木璃他还是比较满意的。
<br/>　　这个小丫头无论是修炼天赋还是炼器天赋，都非常不错，最主要的是勤奋好学，任劳任怨。
<br/>　　平常跑腿的事情，对方基本上都承包了。
<br/>　　“导师，刚才您和苏莫的谈话，我都听到了！”木璃略带歉意的说道。
<br/>　　“哦？你有什么想说的吗？”火元圣师笑道，他并没有责怪对对方。
<br/>　　“我想问您，您会庇护苏莫吗？”木璃沉声问道。
<br/>　　“当然会，如此天赋的人杰，老夫自然会护他周全！”火元圣师点了点头。
<br/>　　木璃闻言松了口气，随即面带惊疑的问道：“他真的像传闻中说的那样，拥有四生战魂和两种特殊体质？这应该不太可能吧？”
<br/>　　木璃有些不信，事实上任何人听到这个消息，都不会相信，实在是太过夸张了。
<br/>　　“不是！”火元圣师摇了摇头。
<br/>　　木璃闻言恍然，看来传闻并不属实，但火元圣师接下来的话，就让她彻底愣住了。
<br/>　　“他有三种特殊体质！”火元圣师声音淡淡，却如惊雷在木璃的耳边炸响。
<br/>　　木璃顿时石化了，面带不可思议之色，呆呆的望着火元圣师。
<br/>　　三种特殊体质？
<br/>　　这怎么可能？
<br/>　　一个人怎么可能拥有三种特殊体质？
<br/>　　“他是天才，真正的天才，盖压一个时代的天才！”火元圣师叹道，他刚才特意暗中查探了一番，才得知苏莫有三种特殊体质，以前并不知道。
<br/>　　不过，在他的感知之下，苏莫的三种特殊体质的血脉，好像已经逐渐融而为一，变成一种全新的血脉。
<br/>　　对于苏莫的天赋，火元圣师同样震惊，只不过没有表现出来而已。
<br/>　　他活了这么久的岁月，也没有听说过如此逆天的天赋。
<br/>　　如此绝世的天才，他自然不能眼睁睁的看着被巫族杀死，至于收徒，他也不做考虑了。
<br/>　　或许让苏莫自行成长，会更合适。
<br/>　　只从真圣榜的消息传出之后，学府中很多高层都被惊动了，都对于苏莫很上心，想要收为关门弟子。
<br/>　　但苏莫住在他火元峰上，其他人自然不好直接收徒。
<br/>　　前两日，府中还有两位武圣，前来询问苏莫之事，显露了收徒的想法。
<br/>　　对于此，火元圣师直接严明，苏莫的修炼由他指导，回绝了那两位武圣。
<br/>　　他不想让苏莫拜府中任何强者为师，苏莫这种妖孽，前途不可估量，不能成为任何人的筹码，不然会打破学府的平衡。
<br/>　　木璃神情怔怔，她不怀疑火元圣师的话，既然对方说苏莫有三种特殊体质，那就不会有错。
<br/>　　只是，四生战魂、三种特殊体质，这超出了她的认知之外。
<br/>　　苏莫若是真有这种天赋，那足以震动整个古灵星河。
<br/>　　“木璃丫头，你以后可以和他多多接触，此人极不简单，对你有利无害！“火元圣师嘱咐道。
<br/>　　“这……木璃只想跟着导师，安心学习炼器！”木璃闻言一怔，随即面带坚定的说道。
<br/>　　“呵呵！你心智坚韧，炼器一途的确需要这种韧性！”
<br/>　　火元圣师微微一笑，随后便询问了一番木璃炼器中遇到的问题，为对方讲解，稍稍加以指导。
