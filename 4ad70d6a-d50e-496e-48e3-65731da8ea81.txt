        &lt;h3&gt;第三卷第七十九章  西城之印象&lt;/h3&gt;
<br/>
<br/>        三兄弟的公司一直都是郝景文在打理，寒伯安有寒家的事情，江俞轩有江家的事情，他家老爹放手让他折腾，不但让他折腾还会帮着他将福建那边的区域打理得很好。
<br/>
<br/>        他独身一人，天南地北单飞客，今天这里，明天那里，俩个哥哥都是动动嘴皮子的人，他是实实在在干活的人。
<br/>
<br/>        公司的大小事务，人事管理，都是由他来管理。西城的从师傅已经到了北京好几次给他汇报情况，从师傅就是带着寒伯安考察市场的那个出租车司机。
<br/>
<br/>        这一次他到西城视察一下，其实也是因为有很久没有见到江俞轩了，刚好市场也视察了，江俞轩也见了。
<br/>
<br/>        从师傅是个聪明人，也是一个跑江湖的人，很懂得看酒下菜，跑出租车这么多年，什么人都见过，那些人好打交道他基本都能看得出来。
<br/>
<br/>        之前跑出租，他是最勤快，最活套的一个人，他深深的懂得，你敬我一尺我敬你一丈的道理，对客户那是服务周到，满脸堆笑。
<br/>
<br/>        认识陈子昂他觉得这姑娘有着深深的忧伤，笑容却又像冬天的暖阳，而且又是从it一条街上车，但是行李又不多，这种人要么就是老板，要么就是公司高层，她们不是去南方就是去北京，不过北京的穿着没有南方的潮。
<br/>
<br/>        试探着聊一聊天没有想到竟然遇着了贵人，陈子昂带着家人回西城游玩的那几天，他是不收费的，但是陈子昂一定要给，他就象征性的收了几百块钱。
<br/>
<br/>        公司三个老总，最主要的还是小文总，小文总别看年龄小，那一张嘴让你心里的想法无可遁形，死人能让他说活了。
<br/>
<br/>        他给你画的饼看起来又香又甜，让你充满着渴望，充满憧憬，充满着激情澎湃。从师傅开着他的a8开开心心的去机场接他的小文总。
<br/>
<br/>        现在每个月收入万儿八千的都是小数目，如今买上了新车，住上了新房，从师傅哼着“今天是个好日子。”在机场等郝景文。
<br/>
<br/>        郝景文第一次来西城，对西城充满了好奇，今天的天气不是很好，下着淅沥沥的小雨，西城有点雾蒙蒙的意思。
<br/>
<br/>        出差碰到下雨天其实是很麻烦的，每个人都喜欢艳阳天，如果碰到下雨天，看着雾蒙蒙的城市，心里多少还是有点郁闷的。
<br/>
<br/>        但是郝景文有他得力的手下-从师傅，从师傅拿着伞，早早的等在接机口，下着小雨其实是可以不用打伞的。
<br/>
<br/>        “郝总，欢迎到西城。”从师傅接过郝景文手中的行李箱，笑吟吟的将伞交到郝景文的手中。
<br/>
<br/>        “从大哥，好久不见。”从师傅比他们都大，三十多岁的人，所以郝景文客气的叫大哥。
<br/>
<br/>        “郝总能来西城视察，是西城分部的福气，郝总要多多给我们指导，我已经将各个分部的人都联系好了，他们都到西城了。”
<br/>
<br/>        “哈哈哈，你有心了。”郝景文并没有想到要召集大家一起，他们在每个地区都设有一级区域代理，下面的分部都有一级代理管理，他没有想到从师傅将大家都召集到了一起，很好，也可以和大家认识一下。
<br/>
<br/>        从师傅放好行李箱，将车门打开，等郝景文坐好，关上车门，整个动作一气呵成，好像他平时都是这么做的，没有做作，没有刻意，也没有逢迎之意。
<br/>
<br/>        “和从大哥认识这么久，都不知道从大哥的家庭情况。”郝景文笑着说。
<br/>
<br/>        “我啊，父亲和母亲都是教书的，老婆也是教书的，我小时候不好好读书，嘿嘿，后来就学了开车的手艺，跑了十几年的出租车，每个月几千块钱混日子，这不，碰上你们，也碰上了好政策，现在房也买了，车也买了。”
<br/>
<br/>        “从大哥的家庭教育一定很好。”看从师傅的行动就知道。
<br/>
<br/>        “嘿嘿，爸爸妈妈都是老师，也差不到哪里去，只是年少的时候叛逆，所以现在就这样了。不过啊，蛇有蛇路，鳖有鳖路，怎么样都是过一生。”从师傅的话里多少还是有点安慰自己的意思。
<br/>
<br/>        “你现在这种日子也是很好的了啊。”
<br/>
<br/>        “还不是各位老总给的机会。”
<br/>
<br/>        “你错了啊，从大哥，不是我们给你的机会，是社会发展的需要，是国家政策好，我们大家才有机会。”
<br/>
<br/>        “嗯嗯，郝总说得对。是政策好啊，我爸爸都没有想到我会这么出息，他老人家说，没想到你开个出租车还开个老总出来。”
<br/>
<br/>        “哈哈，社会形势大好，我们还是有用武之地，好好干，我们的事业也是很有前途的，不过啊，你一定要给大家强调服务性和及时性，客户就是我们的衣食父母，不要看着手中有两个钱了就嘚瑟，眼光一定要放长远。”
<br/>
<br/>        “郝总放心，我们会按照你的要求不定时的抽查和培训。培训都是分批次的，到达各个分部去培训的。还有，现在的app上不是也有了评价功能吗？我们会根据评价功能进行服务态度测评，服务态度好的配送员会给予适当的奖励。”
<br/>
<br/>        “你们这个倒是别具一格。”郝景文想到寒伯安曾经说过从师傅是个人才的话，他心里也有一点认同，这个世界不乏像从师傅这样的人，他们干一行爱一行，行行出状元，真是不错。
<br/>
<br/>        “郝总，今天先到酒店，你稍稍休息一下，晚上给你接风，明天让大家向你汇报情况。”从师傅将郝景文送到酒店，酒店不是五星级的酒店，但是在西城也是很有名气的。
<br/>
<br/>        从师傅没有定五星级的酒店其实是有原因的，因为有很多分部的经理会来西城，他将大家召集到一起既是让大家认识一下公司的老总，也是给郝景文表个态。
<br/>
<br/>        有些人都是地市级或者是县级的，因为提前没有给郝景文报告要召集大家一起来，这么多人，费用从公司支出怕不妥，所以大家的费用都是从自己的工资里面出的，太高档怕大家花费太多。
<br/>
<br/>        “这次大家来，能花费多钱？”郝景文听说大家来了西城，但是从师傅好像没有给财务申请这笔费用。
<br/>
<br/>        “我也是临时起意的，费用都是大家自己出的。让大家见见你，更有干劲。”
<br/>
<br/>        “既然是开会，哪能让大家自己掏费用这一说？是不是都住在一起？”
<br/>
<br/>        “是的，所以这个酒店可能比你平时住的档次要低一点。”
<br/>
<br/>        “已经很不错了，你等开完会将费用开好发票，交到公司统一结算。”郝景文吩咐。
<br/>
<br/>        “还有，江总也在西城，我等会联系一下，看看他晚上能不能来吃饭，后期江总在西城的日子很长，你多和他沟通。”
<br/>
<br/>        “好咧。那你先休息着，我去安排一下。”从师傅给郝景文的印象又加深了一层。
<br/>
<br/>        西城这个古老的城市在郝景文的心里亲切起来了。
