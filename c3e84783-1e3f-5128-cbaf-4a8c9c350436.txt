　　“我们都是道宗弟子，应该团结起来，你们觉得如何？”苏莫目光凝视着几人，沉声问道。
<br/>　　“这……苏莫，我们争不过天虚宗的弟子！”
<br/>　　“是啊，还是算了吧！再等等吧！”
<br/>　　另外两名武帝境九重的道宗弟子，纷纷开口说道，他们也参加过会武，知道苏莫的实力，但同样不认为他们能争的过天虚宗弟子，这根本就是明摆着的事情。
<br/>　　“你们呢？”苏莫又看向其他几人。
<br/>　　“我们争不过的！”
<br/>　　高大青年和另一名武帝境八重的青年，同时摇了摇头。
<br/>　　吕罡默不作声，显然依旧是坚持之前的态度，他们根本没有和天虚宗弟子争抢的实力。
<br/>　　随即，苏莫的目光，又落在了那名武尊境一重修为的银袍青年的身上，问道：“你呢？”
<br/>　　“苏莫，你是来搞笑的吧？”
<br/>　　银袍青年面无表情，冷笑一声，继续道：“你真以为自己在会武中得了第一，就天下无敌了？居然还想和天虚宗真传弟子争抢？”
<br/>　　银袍青年对于苏莫的想法，非常不屑，他感觉苏莫就是在搞笑，就凭他们几个人，怎么可能和天虚宗弟子争抢？
<br/>　　天虚宗的弟子，不仅有真传弟子在场，还人多势众，武尊境修为的人都是好几个，他们凭什么去争！
<br/>　　不是他看不起苏莫，只是事不可为。
<br/>　　苏莫闻言心中不禁暗叹一声，他也没有动气，这几人这种想法很正常，谁也不会做没有把握的事情。
<br/>　　现在明面上，他们的确是没有资格和天虚宗弟子竞争。
<br/>　　毕竟，他在道宗邀请别人挑战，展现了无限接近道宗真传弟子的实力之时，这些人都不在道宗，并不知晓。
<br/>　　“你们真的不愿意和我一起行动？”苏莫再次问了一句。
<br/>　　几名武帝境的弟子摇了摇头，那名武尊境的银袍青年则是默不作声，态度不言而喻。
<br/>　　“苏莫，我们继续等吧，你不要自找苦吃！”吕罡提醒道，他不希望苏莫去争抢，不然的话，就算不死也会脱一层皮。
<br/>　　苏莫轻叹一声，微微点了点头，他清楚了几人的态度，也不想再劝解了。
<br/>　　吕罡见苏莫点头，还以为苏莫听进了他的话，心中暗送一口气，他还真担心苏莫硬拉着他一起。
<br/>　　随即，几人都不再开口，吕罡等人纷纷就地而坐，一边打坐，一边静等。
<br/>　　苏莫面色沉了下来，既然没有人跟他一起合作，那他就只能独自行动了。
<br/>　　只不过，他独自行动，怕是难度极大，因为他也不太清楚，那天虚宗的真传弟子实力具体如何，想必要比秦长山厉害很多吧！
<br/>　　但是，他却不得不这么做，难道无休止的等下去？这简直就是在浪费时间。
<br/>　　无论如何，他都要一试，若是事不可为，他也有把握全身而退。
<br/>　　苏莫继续等待了起来，现在并没有任何的魔物，还需等待魔物再次出现。
<br/>　　这个等待的时间并不长，约莫十几个呼吸之后，又有魔物从魔渊中飞了出来。
<br/>　　嗷！
<br/>　　这只魔物体型如狼，背上足有十几根骨刺，乍一冲出魔渊，便仰天嚎叫，全身魔气滔滔。
<br/>　　这只魔物并没有多强，实力在武帝境八重的层次。
<br/>　　唰！
<br/>　　一名身穿黑衣的天虚宗弟子，身形一闪，便向魔物冲了上去。
<br/>　　眼见着这名天虚宗的弟子，将要接近魔物，正准备出手攻击，就在此时，异变陡生。
<br/>　　咻！
<br/>　　刺耳的破空声响起，一道犀利的剑光划破长空，速度快到极致，瞬间击中了魔物的脑袋。
<br/>　　嘭！
<br/>　　一声爆响，魔物的脑袋直接炸裂，黑色的魔血爆射八方。
<br/>　　然后，便见到一柄灵剑，挑着魔物的身体，向不远处飞去。
<br/>　　那名身穿黑衣的天虚宗弟子见此，顿时怔住了，完全没想到会发生这种情况。
<br/>　　其它所有看到这一幕的人，亦是神情一怔。
<br/>　　唰唰唰！！
<br/>　　仅接着，所有人都是齐刷刷的转头，向灵剑飞去之地望去，便见到一名青衣青年，正自顾自的将灵剑和魔物的尸体收了起来。
<br/>　　吕罡等几名道宗的弟子，均是瞪大了眼睛，不可思议的望着苏莫。
<br/>　　他们完全没有想到，苏莫真的出手了，还是在没有他们的帮助之下，独自动手了！
<br/>　　这让吕罡的人心中大惊，这苏莫莫非是傻了吗？居然真的和天虚宗弟子争抢？
<br/>　　难道他真的以为自己能与天虚宗的真传弟子抗衡，这也太自大了吧！
<br/>　　“苏莫，你居然敢抢夺？”
<br/>　　那名身穿黑衣的天虚宗弟子，目光凝视着苏莫，冷冷的喝道，他自然认识苏莫，因为也曾去参加会武。
<br/>　　不过，他虽然知道苏莫厉害，但此时却是毅然不惧，因为他们天虚宗弟子人多势众，还有真传弟子坐镇。
<br/>　　“猎杀魔物，能者得之！”苏莫淡淡的说道。
<br/>　　“能者得之？”
<br/>　　黑衣青年闻言冷笑，道：“难道你不知道，现在魔渊已经被我们天虚宗包场了？所有魔物都是我们的？”
<br/>　　“不好意思，从现在开始，这魔渊我苏莫——包场了！”
<br/>　　苏莫淡漠的说道，言罢，他身形一闪，向魔渊上空飞去，随即伫立在了魔渊的上方。
<br/>　　“找死！”黑衣青年闻言，顿时脸色铁青，你包场了？你一个人包个屁的场？
<br/>　　整个魔渊附近，很多人都是面露惊讶之色，这个道宗的苏莫，居然要和天虚宗弟子争抢，这也太大胆了吧！
<br/>　　这些人之中，各大势力的人都有，有的亲眼见识过苏莫的逆天，没有见识过的人，也听说过苏莫的名头。
<br/>　　但现在，苏莫的举动，还是让所有人惊讶，因为这简直就是不自量力。
<br/>　　吕罡等几名道宗的弟子见此，差点喷出一口老血，他们真不知道该哭还是该笑。
<br/>　　这苏莫居然如此白痴，还扬言包场了，这不是主动找虐的吗？
<br/>　　几人彻底无语了，他们没有同意和苏莫联手，以为苏莫放弃了，没想到其居然自己就动手了。
<br/>　　那一群二十多名天虚宗弟子，个个目光冰冷了下来，齐齐凝视着苏莫。
<br/>　　“不知死活的东西！”为首的那名黄袍青年，嘴角勾起一丝森冷的笑意。
