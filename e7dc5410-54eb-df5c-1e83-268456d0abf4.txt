        &lt;h3&gt;第三卷第二百二十九章  是经理好还是总裁助理好？&lt;/h3&gt;
<br/>
<br/>        是经理好还是总裁助理好？这个问题让李长卿很苦恼。
<br/>
<br/>        李少卿偷偷去了寒冰玉他们的遇袭现场，想办法得到了一点点遗留下来的残灰，当他准备敲门的时候，听到了寒伯安办公室里的谈话。
<br/>
<br/>        他站了一会，拿起电话，拨通了安男的电话：“安男姐姐，你在哪里？”
<br/>
<br/>        李少卿跟着寒伯安很久了，对于安男他从来都是叫“安男姐姐的。”
<br/>
<br/>        安男：“我在寒家老宅呢。”安男的语气有点无奈。
<br/>
<br/>        “呆了很久吗？”
<br/>
<br/>        “两三天了。”
<br/>
<br/>        “你一个人？”
<br/>
<br/>        “我这两天都在陪一个叫苓希的前女友讲你家寒总裁的过往。”
<br/>
<br/>        “你为什么要听，你回去啊。”
<br/>
<br/>        “我回不去啊，我被挽留了啊。”
<br/>
<br/>        “那你先照顾好自己，等着我们回来。一定要等着我们回来！”李少卿挂了电话，安男很聪明，应该知道他的意思。
<br/>
<br/>        “砰。”李少卿一脚踹开了寒伯安办公室的门。
<br/>
<br/>        里面俩个人都是一怔。
<br/>
<br/>        “寒伯安，你个小人，说是给我经理的位置，一直到现在都不给我，现在还将我当狗腿子用，让我去给你找死人骨灰，特么的，老子不干了。”李长卿将手中的东西扔到了寒伯安的办公桌上。
<br/>
<br/>        “疯子！”寒伯安冷冰冰的说。
<br/>
<br/>        “你特么的说谁是疯子，啊，我给你卖命了这么多年，你不但不感恩戴德，你还让我干一些不是人干的活，你........你。”李长卿那张帅气的脸都气歪了，他直接就朝寒伯安扑上去了。
<br/>
<br/>        “哎，哎，这位兄弟有话好说，有话好说。”沧岚雪眼看着俩个人打起来了，赶紧的出手阻拦。
<br/>
<br/>        他今天可是找寒伯安来谈判的，无论如何都得将寒家拿下来。
<br/>
<br/>        “你那根鸡毛？滚！”李长卿反手一拳就迎上了沧岚雪的眼睛，沧岚雪来不及躲闪，结结实实的挨了一拳，眼冒金星，双眼发黑。
<br/>
<br/>        但是他甩了甩头，睁开眼睛，一出手就将李长卿的胳膊给宁住了。
<br/>
<br/>        “寒总，这个人失心疯了，我替你教训教训他？”
<br/>
<br/>        “沧总见笑了，公司丑事，丑事啊，不劳沧总动手，放了他吧，你看我这还有一大堆破事情，我弟弟又下落不明，如果沧总真的有意和我合作的话，能不能改天？”
<br/>
<br/>        “寒总确定有意向和我合作？”
<br/>
<br/>        “好事不在忙上，你都愿意和我合作，我不可能将钱推出去，只要合理合规，一切都好说。”
<br/>
<br/>        “哈哈哈，我们会做得合理合规的，这个请寒总裁放心。”沧岚雪没有想到他挨的这一拳很值，寒伯安竟然松了口。
<br/>
<br/>        “那我就不打扰寒总裁了，我明天过来找你。”沧岚雪顶着熊猫眼出门了。
<br/>
<br/>        “长卿，你没事吧？”寒伯安看着李长卿龇牙咧嘴的摸着胳膊。
<br/>
<br/>        “特么的，胳膊都要断了，是个练家子。”
<br/>
<br/>        “怎么回事？”
<br/>
<br/>        “你的前女友苓希在寒家老宅，安男小姐姐已经被困了两天。”
<br/>
<br/>        “他们好计谋啊，让人查一下，苓家有没有参与。”
<br/>
<br/>        “我想都不用查，安男小姐姐可是被你阿姨叫过去的，整个事件就是一个坑。”
<br/>
<br/>        “从来都没有发现你像今天这么机智过。”
<br/>
<br/>        “废话，我从来都是这么机智的，只是我机智的光环在你的威压之下黯然无光罢了。”
<br/>
<br/>        “我现在准备回国，李总，你不再是我寒伯安的助理了。”
<br/>
<br/>        “哎，我终于脱离你的桎梏了，我终于是升级了，不过话说，我是李总了，那我还用不用听你的话？”
<br/>
<br/>        “你不用听我的话，你听我助理的话，我助理让你干什么你就干什么。”寒伯安看着李长卿。
<br/>
<br/>        “啊，我现在只听你一个人的，我去当经理之后了，我还要听别人的？你当我傻啊？”
<br/>
<br/>        “你以为你很聪明啊，寒家一人之下，多少人之上？你去当个经理你看有几个人搭理你，你是寒伯安总裁助理，你走哪里都吃香，要不要试试？”
<br/>
<br/>        “别，我还是不试了，赶快的，我要回国。”
<br/>
<br/>        “还是我回去吧，你这个经理在这里比较好，这里的事情你不能做主。”
<br/>
<br/>        “可是，我怕我们都走不出去。还有，二公子的事情......”
<br/>
<br/>        “你就在这里，把你的衣服脱下来给我。”
<br/>
<br/>        “连我的衣服都想要，还有没有良心。”李长卿嘀嘀咕咕的将衣服脱了下来。
<br/>
<br/>        .......
<br/>
<br/>        “特么的，骗子，一直说给我升职都不给我升，我不干了，哼！”李长卿骂骂咧咧的走了。
<br/>
<br/>        他出门挡了一辆出租车，七拐八拐的饶了很多圈最后朝机场方向疾驰而去。
<br/>
<br/>        在机场的路上，寒伯安接到了一个陌生的电话。
<br/>
<br/>        “哥，我是冰玉。”
<br/>
<br/>        “冰玉，你没事啊，可担心死我了。”
<br/>
<br/>        “险中求生存，还好反应快，要不然你就真的见不到你的弟弟了。”
<br/>
<br/>        “贫嘴，你现在在哪里？”
<br/>
<br/>        “我们现在联系一个朋友等着他们来接我们。你在哪里？”
<br/>
<br/>        “机场，我现在回国，你嫂子出了一点麻烦，这边也有一点麻烦事情，长卿在分公司，一系列的事件好像是一根线上的。”
<br/>
<br/>        “你注意安全，我这边你不用担心，关注新闻就是。”
<br/>
<br/>        李长卿坐在分公司总裁办公桌的椅子上，对于当经理好还是总裁助理好开始纠结不清。
<br/>
<br/>        如果是经理的话寒伯安会将他分配出去，在某个分部管理的是一部分人，总裁助理大家都会听他的，而他只听命于寒伯安一人。
<br/>
<br/>        他内线电话叫来了分部的经理：
<br/>
<br/>        “李助。”王增毕恭毕敬的对着他，平时都没有注意到王增的态度，今天看起来很不一样。
<br/>
<br/>        “你去汉森哪里在跑一趟，就说是我说的，看看今天上午的事情处理的如何了。”
<br/>
<br/>        “好，我这就去。”
<br/>
<br/>        王增对他的吩咐没有任何异议也没有任何不满。
<br/>
<br/>        “咦，你就不质疑我一下吗？”
<br/>
<br/>        “我什么要质疑你？”王增感觉很奇怪，你李长卿是总裁助理，你说的话等于是总裁说的话。
<br/>
<br/>        “我说什么你就听什么？”
<br/>
<br/>        “是啊，你是总裁助理，基本等同于总裁代言人，我们当然要听你的啊。”
<br/>
<br/>        “好吧，你赶紧去忙去吧，明天那个什么沧岚雪要是来了，就给我推了，就说总裁昨晚把腰闪了，躺在床上休息。”
<br/>
<br/>        “是。”
<br/>
<br/>        王增转身离去，留下李长卿惬意的坐在办公室后面：“李长卿，你个憨憨，总裁助理多好啊，你还想升个什么职？”
<br/>
<br/>        “嘿嘿，不过如果能当总裁就更好了。不过寒伯安太聪明了，他想什么他都知道，天啦鲁，如果寒伯安知道他想当总裁，他李长卿就无路可走了。这种想法还是不要有了，什么人什么命啊！“
