　　无数雷电劈下。
<br/>　　猎妖师们，包括李元、王羽等一众先天武者都面露死灰。
<br/>　　想起那些在雷电下化成青烟，被瞬间秒杀的猎妖师，他们感到无力和恐慌。
<br/>　　但面临死亡，任何人都会拼死一搏。
<br/>　　他们也不例外，浑身真气澎湃而出，架起一道真气屏障，试图抵挡。
<br/>　　轰轰轰....
<br/>　　雷电劈落，李元等先天武者体内一阵气血翻滚。
<br/>　　后天修为的猎妖师，则猛地吐出一口鲜血。
<br/>　　竟然没有人死亡。
<br/>　　“大家勿怕。”经验最老，见识最广的李元想起了什么，当即高声道，“这头孽畜仍旧在狂血玄君的禁制之下，一身修为被压制，没那么容易杀死我们的。”
<br/>　　天星雷鳄确实还被压住住修为，所以才要借助这里所有武者的精元去冲破禁制。
<br/>　　之前，16条走廊内，猎妖师们分散着，天星雷鳄只需释放数十道雷电就能将他们逐一击杀。
<br/>　　可现在，这里有数百位猎妖师，联手应敌。
<br/>　　天星雷鳄需要操控数百道雷电，每道雷电的威力自然减少了。
<br/>　　而且，能有命逃出长廊的，每一个都是猎妖师中的精英，实力更强。
<br/>　　上方宝座上，天星雷鳄的眼眸猛地一冷。
<br/>　　“愚蠢而卑微的人类，不过是垂死挣扎罢了。”天星雷鳄冷笑一声。
<br/>　　“在我的天雷地网之下，你们还妄图抵挡？乖乖地叫出你们的精血和修为吧，哈哈哈哈。”
<br/>　　李元等人脸色凝重，喝道，“孽畜，休要张狂，今日是谁死还不一定呢。”
<br/>　　他们口上叫嚣着，其实根本没有把握。
<br/>　　每一道雷电落下，都让他们一阵难受。
<br/>　　后天境的武者就更不用说了，一个个口吐鲜血，脸色苍白。
<br/>　　他们就算能暂时抵挡，但也坚持不了多长时间。
<br/>　　天星雷鳄冷冷看着，眼中尽是嗜血的光芒，“不出半个时辰，你们全都会在我的雷电下化为粉末。”
<br/>　　天星雷鳄一脸戏谑之色地坐在宝座上，静静地看着密布雷电下的众人，仿佛在看一群蝼蚁在苦苦挣扎。
<br/>　　如同在观赏着杀戮盛宴。
<br/>　　.....
<br/>　　宫殿下层，萧逸通过光幕，看到铁达大叔等人暂时没事，微微松了口气。
<br/>　　没敢耽搁，立刻坐于蒲团上修炼，
<br/>　　冰鸾剑和控火兽两大武魂齐开。
<br/>　　紫色武魂，绿色武魂，加上蒲团的增幅，周围浓郁的灵气排山倒海般朝他体内涌去。
<br/>　　灵气转化为真气，真气快速地修补着破损的主脉。
<br/>　　同时，分出一丝心神，炼制恢复经脉的丹药。
<br/>　　在第一个房间得到的天材地宝，以及这里得到的天材地宝中，有不少适合炼制治疗主脉的丹药。
<br/>　　待得丹药成形，立马吞入腹中。
<br/>　　真气快速温养，加上丹药，体内破损的主脉快速恢复着。
<br/>　　萧逸有条不紊的做着一切，但速度丝毫不慢，想起书籍上记载的事情，他的脸色变得无比凝重。
<br/>　　之前凭空出现的那本书籍，果然是记载这个洞府一切事情的。
<br/>　　当年，强横一时的狂血玄君，根本不是因为大限来临而死，而是因为封印这头天星雷鳄。
<br/>　　天星雷鳄，生而拥有破玄境九重的修为，借着天地雷电，涂炭一方，毁尽生灵。
<br/>　　一切生灵的生机，灵气、真气等力量，都是它提升修为的来源。
<br/>　　当年的雷鸣荒泽就是最好的例子。
<br/>　　雷电之下，万物不生，宛如死寂之地。
<br/>　　雷鸣荒泽，只是它的开始。
<br/>　　当雷鸣荒泽被悉数毁灭，它得到了所有的生机，修为大涨后，便会寻找下一个目的地。
<br/>　　一个又一个地方，毁灭一切后，它就会离开，只留下一片死地。
<br/>　　毫无疑问，距离雷鸣荒泽最近的，是破玄城。
<br/>　　狂血玄君因此而出手。
<br/>　　虽然他是破玄境中的最强者，更曾击杀过一位地元境。
<br/>　　但，天星雷鳄拥有远古雷兽的一丝血脉，操控雷电，肉体强横无比，实力恐怖至极。
<br/>　　便是狂血玄君，也奈何不了它。
<br/>　　最终他只能以一生修为封印了它。
<br/>　　更在死前留下这个洞府，囚禁天星雷鳄和雷鸣荒鳄。
<br/>　　狂血玄君也是无奈，他自知已经得罪了天星雷鳄，一旦这头妖兽突破地元，将是他的死期。
<br/>　　与其这样，倒不如和天星雷鳄同归于尽。
<br/>　　那一个个房间，根本不是放宝贝的地方，而是用来囚禁雷鸣荒鳄的。
<br/>　　那些天材地宝，也是用来增幅洞府的禁制力量。
<br/>　　整个洞府，每一角落，每一寸土地，都被强大的灵气屏障封锁着。
<br/>　　这也是为什么当初那个大坑会自动复原，还有之前那房间的屏障如此强大，最后还是得靠冰鸾剑才能破开。
<br/>　　那些屏障，困着一头头先天级别，乃至洞玄境的雷鸣荒鳄。
<br/>　　当然了，在灵气屏障之下，这些妖兽的修为都大降。
<br/>　　任何生灵进入洞府，哪怕修为再高，只要高不过狂血玄君，便都会被压制在先天九重之下。
<br/>　　无论是人类武者抑或妖兽。
<br/>　　这份压制，主要是针对天星雷鳄。
<br/>　　所以说，天星雷鳄如今不过是先天九重的实力，而不是强大的破玄九重。
<br/>　　只不过，天星雷鳄操控雷电的手段太过强横，这才让宫殿内的所有猎妖师都不是对手。
<br/>　　这有点像之前萧逸使用烈焰手套对付先天级别的荒鳄。
<br/>　　萧逸可以不停释放噬火百刃来击杀荒鳄；同样，天星雷鳄也是靠不停释放雷电来对付数量庞大的猎妖师。
<br/>　　言归正传。
<br/>　　这时，萧逸体内的八条主脉已经全部恢复，重新变回后天八重的修为。
<br/>　　而且，他还借着这里的庞大灵气，在主脉全部被修复之时，一举冲破了第九条主脉，成为后天九重。
<br/>　　萧逸握了握有力的拳头，沉声道，“修为终于全部恢复了，还提升了一重，接下来是炼制金脉丹了，铁达大叔你们一定要撑住。”
<br/>　　说罢，萧逸快速开始炼丹。
<br/>　　培源朱果，以及三级妖兽的内丹，还有其他材料，他已经凑齐了。
<br/>　　.....
<br/>　　宫殿上方，猎妖师们还在苦苦支撑。
<br/>　　看样子，还能再支撑一段时间。
<br/>　　每到雷电劈落，都会让后天境的武者猛地吐血，接近重伤。
<br/>　　可每到这时，就会有数道充满生机的力量打到他们身上，让他们缓解伤势。
<br/>　　这里面有十个药师，其中有三个还是先天武者。
<br/>　　每当有猎妖师支撑不住时，他们便会释放出武魂力量，为其疗伤。
<br/>　　但，令人惊讶的是，一般的药师，顶多为数人同时疗伤。
<br/>　　那三个三天层次的武者，还是黄色武魂的，也仅仅能同时治疗十数人。
<br/>　　反倒是最弱的浅沫，长生藤武魂一出，挥挥手就让数十人恢复伤势。
<br/>　　也不知她的武魂到底是何等品阶的。
<br/>　　不少武者都惊讶的看向她。
<br/>　　她最弱，一直都躲在铁达大叔的真气屏障下，不时释放武魂力量为其他人疗伤。
<br/>　　这时，上方宝座的天星雷鳄，显然也注意到了浅沫。
<br/>　　“哼，该死的人类，浪费我的时间。”天星雷鳄身影一闪，嗖的一声，凭空出现在浅沫面前。
<br/>　　“阿。”浅沫惊呼一声。
<br/>　　“孽畜，你想做什么？”铁达大叔和林飞等人顿时一惊。
<br/>　　但他们已经在苦苦支撑雷电的攻击了，根本无法伸出援手。
<br/>　　“人类小女娃，你知不知道你打扰了本妖王的兴致？”天星雷鳄面露杀意，轻轻地伸出了手指。
<br/>　　“休想伤浅沫。”铁达大叔一刀劈出。
<br/>　　“滚。”天星雷鳄不屑地吐出一个音符，手指一弹，铁达大叔的真气屏障瞬间被破，铁达大叔直接被轰飞。
<br/>　　“噗。”铁达大叔猛地吐出一口腥血，被弹飞了数十米，瞬间重伤，甚至连再次架起真气屏障的力气都没有。
<br/>　　“铁达。”幸好，他被弹飞的方向，正是王羽的方向。
<br/>　　王羽连忙将他拉入了自己的真气屏障，否则，铁达大叔必定被雷电劈成粉末。
<br/>　　天星雷鳄正眼都不瞧铁达大叔一眼，而是伸出手指，缓缓地点向浅沫。
<br/>　　“小女娃，最好不要反抗，让我点一下，痛苦只有一瞬间，别逼我折磨你。”天星雷鳄狞笑着。
<br/>　　林飞等人痛苦地闭上了眼睛，仿佛已经看到浅沫被雷电劈成齑粉的画面。
<br/>(本章完)
