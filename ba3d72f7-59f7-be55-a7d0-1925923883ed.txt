        &lt;h3&gt;第三卷第三百三十五章  怀疑&lt;/h3&gt;
<br/>
<br/>        陈子昂拿了车钥匙，并没有叫陈天天和她一起去警局，而是叫上了冯乔，俩个人一同去警局。
<br/>
<br/>        “乔哥，你说夏颜去你家找二嫂了？”陈子昂问冯乔。
<br/>
<br/>        “是啊，这个女人没有到过我们家，竟然还知道我们家地址，可想而知和我姐的关系有多好，我姐估计把啥话都会给她说。”冯乔唠叨着。
<br/>
<br/>        “她是什么时候到的你家？”
<br/>
<br/>        “下午吧，不到吃下午饭的时候，她走不久我妈就将饭做好了。”
<br/>
<br/>        “说了些啥啊。”
<br/>
<br/>        “她说她去找我姐，好几天没有见到我姐了，给我姐打电话，我姐没有接，她就到我们家看看，还说我姐夫找了个女人，我姐心情不好，怕我姐出什么事情。”
<br/>
<br/>        “哦。”
<br/>
<br/>        “嗨，我姐真的是很可悲，自己的亲生儿女连一个干女儿都比不上，人家外人还知道担心她呢，你看看她的儿子，人死了都不知道。”冯乔叹息了一声。
<br/>
<br/>        “其实，我也好久没有见到我姐了，我们拆迁的款我妈一点也不给我姐，伤了我姐的心，她就很少回去，我天天的打游戏，就忘记我姐了。”冯乔说着说着就哭了。
<br/>
<br/>        陈子昂看了一眼坐在副驾驶上的冯乔，这一会的冯乔才是真情流露。
<br/>
<br/>        “我们家就我和我姐两个孩子，我妈重男轻女思想严重，从小到大好东西都给我，之前我们家家庭条件也不好，都是我姐帮衬着，现在日子好过了，她却不在了。”冯乔絮絮叨叨说完靠在座椅上睡着了。
<br/>
<br/>        陈子昂叹口气，冯睿的一生过得是有多么卑微啊。
<br/>
<br/>        到了警局之后，询问了相关的手续和情况，说是司机驾车逃逸，找不到人，但是冯睿身上有过汽车剐蹭，有一点汽车掉落的漆皮，他们会根据这个寻找相关的车辆，已经发了通知给各修理部门，如果看见了喷漆的车一定要上报。另外，也交给有关的信息部门，让在网上公示，让车主及早归案。
<br/>
<br/>        “你们先回去将人安顿好，我们也尽快破案，有问题我们会及时通知你们！”
<br/>
<br/>        陈子昂和冯乔没有得到想要的信息，只能回家，将冯乔送到冯睿家之后，她又开车去见了她父亲和三姐陈子澜。
<br/>
<br/>        陈子澜已经能坐起来了。
<br/>
<br/>        “三姐，回复得不错啊，坐做起来了，么么哒，你真棒！”陈子昂笑着对陈子澜说。
<br/>
<br/>        “淘气，你从二嫂家过来？”陈子澜并没有笑，潘青松出差了，还没有回来，家里的事情也没有人她说，她着急的不行。
<br/>
<br/>        “是的，大家都在忙二嫂的事情。”陈子昂声音疲惫。
<br/>
<br/>        “你一宿没睡吧？”
<br/>
<br/>        “嗯，从北京连夜赶回来的。”
<br/>
<br/>        “你似乎有话想说？”
<br/>
<br/>        “我也不知道，是我的感觉，我总感觉二嫂不是简单的车祸，但是没有证据，还有，二嫂的弟弟冯乔说夏颜去他家里找二嫂了。”
<br/>
<br/>        陈子昂将冯乔说的话对陈子澜说了。
<br/>
<br/>        “你回去将二嫂的手机拿出来看看通话记录。”陈子澜看了一样陈子昂。
<br/>
<br/>        “如果没有他们的通话记录。那就说明夏颜在撒谎。”陈子澜知道陈子昂心里在想什么，但是现在所有的事情讲究的是一个证据。
<br/>
<br/>        “我是在想二嫂是不是知道了夏颜的一些秘密。”陈子昂大着胆子猜测。
<br/>
<br/>        “也不一定。光有你的怀疑是不行的，是要让别人也有怀疑。”
<br/>
<br/>        “那么这种情况冯乔是不是要反映给有关部门？”陈子昂问陈子澜。
<br/>
<br/>        “只是能提供一下线索，说明不了什么问题。如果夏颜给冯睿真的打了电话，那就更不能说明什么问题，反而会打草惊蛇。”
<br/>
<br/>        “姐夫去哪了？”
<br/>
<br/>        “你姐夫去沈城了，算是去帮着查一个案子，干我们这行的，就没有闲着的，这都走了好几天了。”
<br/>
<br/>        陈子昂原本想着见到潘青松可以问问洛城的事情。
<br/>
<br/>        “你准备在家里待几天？”
<br/>
<br/>        “再怎么着也得等二嫂走了再说吧。”
<br/>
<br/>        “决定要去甸城了吗？”
<br/>
<br/>        “是的，行程已经定了。看来得改签。另外，我会将都督和团团送到美洲去，让他们在美洲上学。”
<br/>
<br/>        “子昂你告诉我，你是不是碰到什么困难了？”
<br/>
<br/>        “我没有碰到什么困难呀！我要去甸城，而且也不是经常回来。嘟嘟和团团放在哪都一样。我也可以隔三差五的去看他们。再说了，让他们接受一下外面的教育也是好的，等我忙完这一段时间，我就接他们回来。”
<br/>
<br/>        “孩子们已经大了，你不考虑考虑你自己的个人问题。”
<br/>
<br/>        “我觉得我现在这样挺好的。再说感觉没有合适的，太优秀的人家看不上我。一般的吧我看不上别人。哎！找个合适的人比赚钱难多了。”陈子昂和陈子澜开着玩笑。
<br/>
<br/>        “你和张函还有可能吗？我觉得张函是爱你的。”
<br/>
<br/>        “现在谈爱都是奢侈的。爱我有什么用？能给我什么？就是嘴上说我爱你我爱你吗？我们都已经过了谈爱的年龄。现在我就在想怎么样把生活过的好一些，怎么样能才活出自己来，怎么样才能活的有意义。”
<br/>
<br/>        “你已经过得很有意义了，你还想要怎么样呢？别太贪心了。”
<br/>
<br/>        “我就很贪心呀！我就想我身边的亲人一个个都过的很好。都健健康康平平安安。”陈子昂说这话时声音已经哽咽了。
<br/>
<br/>        陈子澜也流泪了，冯睿的离去让她们感觉人生苦短，说不定那一天就找不到人了。
<br/>
<br/>        活着才是最重要的，活着才是最需要勇气。
<br/>
<br/>        “子昂，你这两天一定是累坏了，躺床上休息会，有事情我叫你。”
<br/>
<br/>        “不，我得过去看看有什么要帮忙的。三姐你好好休息吧！我该回去了。”陈子昂看着坐着的陈子澜。
<br/>
<br/>        “也行，你去帮忙吧，看看二嫂手机通话记录，看能不能发现什么。”
<br/>
<br/>        “行，有问题我再来找你。”陈子昂告别了陈子澜，又急匆匆回到了冯睿家里。
<br/>
<br/>        “二哥，二嫂的手机可以借我看看吗？”
<br/>
<br/>        陈子建将冯睿的手机给了陈子昂，陈子昂仔细的检查了冯睿的手机，发现夏颜和冯睿来往不太密切，这几天都没有和冯睿有过通话记录，那她在冯家说的话是不是在撒谎？
<br/>
<br/>        还有如果她们都没有打电话联系，她们是怎么进行沟通的？或者真的是夏颜很久没有看到冯睿了，很担心冯睿？
<br/>
<br/>        陈子昂决定让冯乔和夏颜联系一下。
