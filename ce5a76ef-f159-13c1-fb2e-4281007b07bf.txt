　　嗖！
<br/>　　苏莫御剑而行，本命灵剑催动到了极致，速度快到极致。
<br/>　　而阎泰五人紧追苏莫不放，五人的速度虽然不比苏莫快，但也并不比苏莫慢，双方的速度在伯仲之间。
<br/>　　“这一次损失大了！”苏莫一边飞行，一边叹了口气。
<br/>　　刚才魔影分身使用的帝级兵器，因为魔影分身的崩灭，也被阎泰等人得了去。
<br/>　　不过，这也无妨，只要他日后灭了黄泉魔宗，再夺回来便是。
<br/>　　咻！
<br/>　　就在此时，刺耳的破空声响起，一道犀利无匹的刀芒，破碎虚空，隔着万里距离，闪电般的向苏莫横斩而来。
<br/>　　是阎罗殿殿主出手了！
<br/>　　这一刀，刀意冲天，蕴含惊天的奥义之力，这股奥义阴寒、冰冷、无情，使之这一刀威力大增。
<br/>　　唰！
<br/>　　苏莫没有硬接这一刀，脚下本命灵剑一闪，便堪堪的避了过去。
<br/>　　“你逃不掉！”阎泰的怒喝声响起，其大手连连拍击，一道道巨大如山的掌印，铺天盖地的向苏莫震杀而来。
<br/>　　三位殿主，亦是和阎泰同时出手了，爪芒、拳茫撕裂无尽虚空，隔空向苏莫轰杀而去。
<br/>　　不过，几人的攻击虽然强大，但因为距离苏莫较远，想击中苏莫都很困难，跟别说打败苏莫了。
<br/>　　“碎！”
<br/>　　苏莫头也不回，手中帝皇剑连连闪烁，一道道剑气向身后斩出，密密麻麻的剑气形成剑气风暴。
<br/>　　而他整个人速度丝毫不减，急速向前方飞逃。
<br/>　　剑气风暴与几人的攻击碰撞，纷纷崩碎，不断的爆炸。
<br/>　　几人的攻击，不仅伤不到苏莫，反而让得他们自身的速度稍稍受阻。
<br/>　　“混蛋！”阎泰几乎气炸了肺，他们五大强者，就算是对上准帝强者，都能勉强斗上一斗了，居然杀不死区区武皇境二重的苏莫！
<br/>　　阴罗殿殿、尸罗殿殿主、鬼罗殿殿主三人，亦是脸色铁青，他们大意了。
<br/>　　若是早知道苏莫有如此实力，他们绝对不会让苏莫如此轻易逃走。
<br/>　　定然会布下绝杀大阵，让苏莫死无葬身之地。
<br/>　　只不过，他们太大意了，虽然都知道苏莫不凡，可能有了武皇境九重的实力，但五人原本以为，凭他们五人之力，击杀苏莫轻而易举。
<br/>　　但现在，因为大意，却造成了这种局面。
<br/>　　本来他们想直接杀进苍穹神宫，但试探了一番苍穹神宫的水幕大阵，发现根本无法击破，便一直在等待苏莫出来。
<br/>　　他们没想到，苏莫出来了，而他们居然杀不死！
<br/>　　“诸位放心，天上地下，都不会再有他容身之地！”阎罗殿殿主，那名面色冷淡的年轻人，面上却是没有丝毫怒色，显得冷静异常。
<br/>　　其他几人闻言，才稍稍压制了心中的怒气，的确，虽然现在他们没有杀死苏莫，但苏莫想逃走也没有可能。
<br/>　　只要他们一直追，总有机会追上苏莫。
<br/>　　虽然他们都有超强的盾术，可以短时间内速度暴增，但盾术不受控制，用来逃命是利器，但用来追杀却是不适合。
<br/>　　“我们通知二魔老吧？让他亲自出手击杀苏莫！”阎泰沉吟了片刻，沉声说道。
<br/>　　“不妥！二魔老一旦出手，怕是会暴露身份，影响日后圣宗的计划！”鬼罗殿殿主摇头，不赞同阎泰的意见。
<br/>　　“管不了这么多了，苏莫不除必然祸患无穷，即便暴露，也在所不惜！”
<br/>　　阎泰眸中杀意深寒，继续道：“宗主和大魔老均在闭死关，唯有二魔老出手，方能彻底击杀苏莫！”
<br/>　　众人闻言，沉默了片刻之后，便俱是点了点头。
<br/>　　的确，阎泰所言不虚，苏莫如此的天赋，如此快的成长速度，又有苍穹神宫的底蕴，若是这次除不掉，真的会祸患无穷。
<br/>　　若是再拖个一年半载，天下间再也没有人能杀得了苏莫了，到那个时候，他们黄泉魔宗必然被动无比。
<br/>　　虽说宗主和大魔老均在冲击武帝之境，但能否成功尚未可知。
<br/>　　征得四位殿主的同意之后，阎泰手掌一翻，掌心中出现了一只拇指头大小的黑色飞虫。
<br/>　　这黑色飞虫长有三对透明的翅膀，身体上覆盖着黑色的甲壳，闪烁着森冷的寒光。
<br/>　　唰！
<br/>　　黑色飞虫乍一出现，一闪之间便盾入了虚空深处，消失无踪。
<br/>　　这是黄泉魔宗的一种特殊的传讯工具，三翅冥虫。
<br/>　　用三翅冥虫传讯，没有任何距离上的限制，哪怕是距离传讯之人万亿里之遥，三翅冥虫也能飞到。
<br/>　　只不过，需要的时间更久而已。
<br/>　　随即，几人也不再出手攻击苏莫，就这样紧紧的吊在苏莫的后方，等待二魔老的到来。
<br/>　　就这样，几人不断的追逃，无休无止，这一追就追了十数日。
<br/>　　飞逃了十数日之后，苏莫也是有些不耐烦了，可他也是无计可施，无法甩开几人。
<br/>　　虽然他还有些其它手段，但这几人实力太强，想彻底甩掉太难了。
<br/>　　当然，他还有大三宝术，但不到生死危机之时，他不想动用此术。
<br/>　　这一日，苏莫的视线中，前方出现了一片一望无际的山脉。
<br/>　　山脉中兽吼震天，巨峰延绵，数百丈高的苍天古树林立，整个山脉中都充满了荒古苍茫的气息。
<br/>　　“蛮荒山脉！”苏莫心中讶然，由于他一直飞逃，也没有仔细辨认方向，没想到居然逃到了蛮荒山脉来了。
<br/>　　“也好，就去东海龙宫！”
<br/>　　略一沉吟，苏莫面上露出了一丝笑容，过了蛮荒山脉便是东洲，跨越了东洲之后便是东海。
<br/>　　只要他逃到东海蛟龙一族，谅阎泰五人实力再强，也不敢追到东海蛟龙一族。
<br/>　　到时候，他便能安全脱身，甚至借助蛟龙一族的力量，反杀五人。
<br/>　　想到此处，苏莫毫不犹豫，极力催动本命灵剑，向蛮荒山脉上空飞去。
<br/>　　咻！
<br/>　　身形如箭，苏莫一路所过之处，空间都被划破，速度如风似电。
<br/>　　阎泰五人同样不慢，紧随苏莫进入了蛮荒山脉上空。
<br/>　　吼！吼！吼！！！
<br/>　　无数的兽吼之声，从下方的山脉中响起，震天动地。
<br/>　　苏莫几人的气息非常强大，惊动了山脉中无数的蛮荒妖兽，让得无数妖兽吓的四散奔逃。
<br/>　　就算是强大的七级妖兽，亦是不敢现身。
<br/>　　蛮荒山脉方圆八百万里，苏莫当初从东洲到中洲之时，跨越蛮荒山脉用了一个多月。
<br/>　　而现如今，他只是几个时辰的时间，便跨越了一小半的距离。
<br/>　　“五、六天的时间，差不多就能到东海龙宫了！”苏莫呢喃自语，转头看向后方的阎泰五人，他心中稍稍有些疑惑。
<br/>　　这段时间，这五人也不言语，也不攻击，就只是吊在他的身后，不知又有什么阴谋！
<br/>　　苏莫皱眉沉思，就在此时，异变陡生，一股冰冷的杀机瞬间笼罩他的心灵。
<br/>　　在苏莫的侧方，原本平静的空间突兀的碎裂，一杆黑色的大枪瞬间飚射而出。
<br/>(本章完)
