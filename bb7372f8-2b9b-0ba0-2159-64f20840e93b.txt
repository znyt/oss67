　　就这样，七七四十九天之后，那数百混沌晶石全部被元始炼化，化为一大团淡蓝色的玄清气。
<br/>　　之后，元始挥手便将九天之上的封印撤去，将这团玄清气送入到大窟窿之中。
<br/>　　当初盘古开天劈地，便是以清浊二气衍化天地，这玄清气，正是补天的良材。
<br/>　　传说，盘古精修玄清气，之后再推理创新，一分为三，分别将玄清气分化为玉清气、太清气与上清气，融入到他的元神之中。而元始三清，正是他那三道融入三清气的元神所化。
<br/>　　只是，是真是假，谁也不知。不过，空穴未必来风，既然有这样的传说，那未必就是假的。
<br/>　　伴随着那玄清气渐渐弥漫，无形的力量将清气衍化，元始的造化法则在不断造化天地，重塑乾坤。渐渐地，玄清气堵住了那道缺口，玄光大盛，光芒四射，光芒散后，天空终于再次恢复了往日的蔚蓝。
<br/>　　元始补完天后，发现乾坤鼎中还有一团玄清气。突然，他诡异一笑，笑得有些莫名其妙。接着，他往里面打如几道灵光，顿时，那道玄清气化作五颗大小不一、颜色各异的晶石。
<br/>　　其中一颗为黑色，释放出摄人的魔性，让人看了有着一股嗜血的冲动。一颗为白色，如雪一般的苍白，更有一个冰冷之气在其中孕育。一颗晶莹剔透，如水一般无色，但其中却透露着惊人的寒意，周围空气都被凝结成冰，仿佛能够冰冻人的灵魂。
<br/>　　一颗为金色，好似没什么变化，只是一颗闪烁着金光的普通神石。但仔细看却能发现，这是一颗大小如意，随心随意可以变幻无穷的神石。最后一颗为晶莹剔透的九彩晶石，与最初的混沌晶石颇为相似。
<br/>　　元始将黑色石头与白色石头往乾坤鼎内一扔，之后祭出神火稍微锻炼，不到一时三刻，便炼制出两件兵器。
<br/>　　乾坤鼎能够返本归源，反后天为先天，元始炼器所用的材料本事就不凡，两件兵器都成了先天中品灵宝。
<br/>　　那是一刀一剑。
<br/>　　黑色的长剑，剑身锋利，透露着嗜血的魔性，这就仿佛是一柄魔剑，上面刻着“绝世好剑”四字。
<br/>　　长刀雪白如露，刀柄弯翘，刀尖如月，刀身更有一道血槽，一看就是不凡。刀柄刻着“血饮刀”三字。
<br/>　　元始想了想，将一刀一剑扔给武祖玄天，众圣虽然诧异、不解，但一样没有多说什么，他们都知道，既然元始之前能够派自己的徒弟去帮助人族，自然就说明他与人族之中存在某种关系，眼前他能够将两件灵宝送给人族，也说得通，这与他们无关。
<br/>　　更何况，这样的兵器对于他们而言，已经没有多大作用了，他们也不是很关心。他们如今关心的是，元始这样做的目的。不过，一时半会他们也想不通。
<br/>　　倒是女娲，她眼神闪烁，看了一眼元始，眼神复杂，再看了一眼人族，却也没有多说什么。
<br/>　　之后，元始将剩下三块神石随手扔了下去，其中那块闪烁着九彩光芒的晶莹神石落在了十洲祖脉之上，那颗金色的神石碑人到西湖里面，沉入湖底，也许哪一天它就出世了，也许它永远不会出世。至于那颗寒冰神石，则是流落到天山之顶。
<br/>　　准提圣人看着那三颗神石，眼神闪烁一下，不知道在想些什么。
<br/>　　忽然，元始看见那断裂的半节不周山，顿时眼前一亮。他伸手将之抓来，扔进乾坤鼎之中，祭出神火一番祭炼，几日之后，炼出一道四方大印。
<br/>　　大印莹白如玉，之上刻着不周山的形状，大大方方，最底部刻有“翻天印”三个篆形大字。
<br/>　　“不错不错，一印拍下，犹如不周山镇压下来，威力巨大，神挡杀神，佛挡杀佛。算得上是一件上品先天灵宝，而且是不是普通的上品先天灵宝，盘古身上的东西，果然不简单！”
<br/>　　元始看着手中的翻天印，眼中闪过一丝满意之色，淡淡道。
<br/>　　一旁的准提与接引听了此话，脸色顿时有些发黑，什么叫“佛挡杀佛”，真当我们不存在吗？不过，他们心底还是隐隐有些佩服元始的，就连他这个平时眼光独到的寻宝狂魔，都一时之间没有注意到这半节不周山，但元始却发现了，这不是眼光独到又是什么。
<br/>　　至于剩下的半截不周山，他们看了看后，却是没有多想。这是因为它与地脉相连，为洪荒最中间，不能乱动，否则会遭巨大因果。而且这半节坐落在地面之上的不周山与其他散落的山石，变成了一座全新的不周山。
<br/>　　只是，如今的不周山已经不同于之前的不周山了。虽然它还叫不周山，但却不是之前那座洪荒第一神山了。
<br/>　　这时，通天从北极之地斩杀了一只巨大的鳌鱼，这是开天辟地后诞生的第一只鳌鱼。他以这鳌鱼的四肢重立四极，代替不周山成了那撑天的四根天柱。
<br/>　　女娲则是收集了大量的芦草，将它们烧成灰，来堵塞洪荒大地上四处流窜的洪流。
<br/>　　此次共工怒触不周山造成的灾祸，留下的祸患，一时间被解决了。只是，这场灾祸留下的痕迹，却是改变不了的。
<br/>　　自此以后，天向西北方向倾斜，地向东南方向倾斜，因此有了太阳、月亮东升西落，大江大河一切都往东方汇流之说。
<br/>　　而且，因为北极鳌鱼被杀，庞大的怨气、死气弥漫了整片北方之地，生灵一旦靠近，便会被死气侵蚀，全身腐烂，久而久之，那里成了一片洪荒生灵都不敢靠近的死寂之地。
<br/>　　元始炼石补天，重塑乾坤，再立四极，天道有感，降下功德，元始一人得六成。其余众圣也因拯救苍生，梳理河道，分别得其余四成。
<br/>　　值得一说的是，那被通天斩杀的鳌鱼的魂魄，都得到了一定的功德，若是他转世重生，一定会是福源深厚之人，修炼起来事半功倍。此刻那道魂魄被通天收起，只等着一个机会，让他转世。
<br/>　　三族大战，最终因为此次不周山倒塌带来的天灾而暂时终结了。
<br/>　　此番大战连续数十年，人族大战巫、妖两族，底蕴尽显，强者倍出，震惊洪荒，一跃而上，成为洪荒除却巫、妖两族之外的第三大种族。
<br/>　　不管是人族，还是巫族、妖族，此次大战都损失极大，人族死伤无数，多少中小部落被无情屠灭，至少失去三分元气。
<br/>　　巫族死去祝融、共工两大祖巫，更是被玄天镇压一位，倒是时间祖巫烛九阴，最后还是被帝江救了出来，看来他命不该绝。大巫、小巫更是死伤无数，这让巫族本来人数就少的情况，此刻变得更加稀少了。
<br/>　　至于妖族，被人族屠杀了数十妖神，数百大罗金仙妖王，无数太乙金仙大妖，无数妖兵要将。甚至因天柱倒塌，天界崩裂，天河之水泛滥纵横，死去更多，可谓是三族中损失最严重的。
<br/>　　可以说，这场大战是三败俱伤，谁也没有占到便宜。
<br/>　　洪荒之中，因为此次天灾人祸，死去的生灵更是多得数不胜数，无数种族被灭族，无数散修高手被弱水淹没。
<br/>　　洪水泛滥，生灵涂炭！哀鸿遍野，尸骨如山。
<br/>　　此次天灾，影响深远，自此以后，洪荒许多势力要重新排名了。
