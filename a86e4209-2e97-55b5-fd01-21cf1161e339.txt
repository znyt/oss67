　　　　“你先变回原形，你这样我怎么跟你解释啊”
<br/>　　这场战斗就这样要结束了？
<br/>　　涂小安有点措手不及，但看着落子烟，他知道也打不下去了。
<br/>　　他现在迫切的想知道到底是怎么回事。
<br/>　　“哼...”
<br/>　　只闻五十米巨大的紫金巨蟒傲娇的冷哼一声，然后原地缩小，画面非常的诡异。
<br/>　　一会儿后，在这破败的庭院内，涂小安变成了清秀的男子，而金蚕蛊蛇也是美妇人的模样，优雅而美丽。
<br/>　　同时，落子烟站在他们的中间。
<br/>　　“姐姐，您很久没来妹妹这里了”美妇人一向开口号称祖奶奶，但是现在的姿态却很低。
<br/>　　落子烟笑道：“是姐姐太忙，你别怪姐姐”
<br/>　　“姐姐瞧你这话说的，折煞妹妹了”
<br/>　　呃...
<br/>　　涂小安一脸的蒙蔽，数次张了张嘴，又合上了，这画风好别扭。
<br/>　　“梨儿，今天是一场误会，给姐姐一个面子，不要跟他计较了”落子烟握住对方的手。
<br/>　　美妇人沉默了，看了看涂小安，看了看落子烟：“姐姐，你为什么要保一条小小蛇精”
<br/>　　“他不是一般的蛇精”落子烟浅浅的说，然后轻轻的俯身到她耳：“他是姐姐...”
<br/>　　美妇人娇躯一震，不知道听到什么非常的震惊，看向涂小安的眼光一下子不一样了。
<br/>　　半晌后，落子烟看向涂小安，用同样柔声的声音道：“安儿，今天是一场误会，你...”
<br/>　　“你给我打住，安你个头”涂小安的脸直接黑了起来，这娘们什么鬼，今天走慈母路线？
<br/>　　这一口安儿，叫的他鸡皮疙瘩都出来了。
<br/>　　其实战斗就这样结束，涂小安心里是开心的，因为他要做的事情都做了，还吞了别人的樱桃树跟十几条金蚕蛊蛇，进化能源暴涨了二万多。
<br/>　　如何都不吃亏，这条蛇妖，可以先留着。
<br/>　　等待日后开启第三阶段进化，只是落子烟跟对方的关系，让涂小安微微的蹙眉。
<br/>　　半小时后，涂小安跟落子烟离开了白家。
<br/>　　“说吧，你跟那条蛇精怎么是姐妹的关系”涂小安走在无人的街道上，那美妇人可是金蚕蛊蛇，这并不是天然的蛇种呀。
<br/>　　难不成，落子烟也是一条金蚕蛊蛇？
<br/>　　顿时间，涂小安看她的眼神非常的怪异起来。
<br/>　　落子烟到没有什么异样，坦然的很，面对涂小安的怪异眼神，也没发作，然后边走边说：“我跟梨花在三百年前相识了”
<br/>　　涂小安心中一凸，这敢情也是一位祖奶奶，或许是祖祖祖祖..奶奶。
<br/>　　“你也是蛇妖对吧”他再次一问。
<br/>　　其实之前落子烟在面馆跟涂小安将一个捕蛇人跟蛇妖的故事，他就怀疑了。
<br/>　　不然莫名其妙跟他将那种故事干嘛。
<br/>　　“我不是”落子烟再次否认：“我的身份，等你证道了，我在告诉你好吗”
<br/>　　涂小安面无表情道：“这跟我证不证道有什么关系？”
<br/>　　敢情我现在没证道，还没资格听不成。
<br/>　　“我跟你说说梨儿的故事吧”落子烟开口。
<br/>　　呵呵...转移话题。
<br/>　　但涂小安对那条妖王级别的金蚕蛊蛇好知欲也强，点了点头。
<br/>　　“梨儿，是被一位邪恶的巫师创造出来的金蚕蛊蛇，利用她做了很多伤天害理的事情”
<br/>　　“当梨儿这条金蚕蛊蛇的灵智越来越高，它也就不愿意被人摆布了”
<br/>　　“有一天，它终于鼓起勇气从那位巫师的手中逃离了，逃到了一座大山之中”
<br/>　　”可逃是逃走了，但它却身受重伤，那片山，也有很多的毒蛇，更有很多当地的捕蛇人，那山叫做佘山”落子烟说着看了涂小安一眼。
<br/>　　涂小安蹙眉，听她继续说。
<br/>　　“金蚕蛊蛇的消息在当地流传开了，一个个捕蛇人疯狂的上山对它进行了追捕，都想捕到这条与众不同的毒蛇，发上一笔横财”
<br/>　　“可惜，觉醒的金蚕蛊蛇可不是一般的捕蛇人能对付，最后造成了很大的伤亡，死了很多的捕蛇人，甚至引来了官府的插手”
<br/>　　落子烟淡淡的说：“突然有一天，金蚕蛊蛇在佘山消失了，消失的无影无踪”
<br/>　　“被人抓走了？”
<br/>　　某一日，在一片山林之外，有一位白姓书生在湖边上晃悠，晃着晃着，他突然发现湖边上一处假山里，蜷着一团什么东西，浑身是血，柔弱可怜。
<br/>　　他探头一看，是一条与众不同的蛇，长的煞是好看，纵然全身是血，也丝毫不影响它的美丽。
<br/>　　书生心善，就算知道这是一条毒蛇，依然还是将他带回了家。
<br/>　　这位书生寒窗苦读十几年书，连个秀才都捞不到，家里仅剩的一点钱都拿去买纸和笔了，他为了救这条蛇，只能上山采药。
<br/>　　治疗它的伤口，给它喂肉吃。
<br/>　　几个月后，白姓书生忽然弃笔从戎，当起一位捕蛇人，成为了当时赫赫有名的捕蛇大师。
<br/>　　没有人知道他捕蛇的手段，只知道，他只要上山，就无往而不利。
<br/>　　“梨儿被白姓书生所救，从此报恩，直至守护白家数百年”落子烟落下了句号。
<br/>　　白家祖奶奶？
<br/>　　原来是因为这样？
<br/>　　落子烟显然不是一个会说故事的人，但是涂小安还是听懂了，简单的说，就是金蚕蛊是从巫师的手中逃走来到了佘山，又被佘山的捕蛇人追捕。
<br/>　　最后被白家的祖先所救，为了报答救命之恩，这条后来成精，成妖的金蚕蛊蛇，一直守护白家，让白家成为了古武隐藏世家。
<br/>　　也就是说，白家是靠着一条蛇而崛起。
<br/>　　这算是秘闻，很少有人知道。
<br/>　　怪不得白君然跟白楚燕都对这位祖奶奶非常的敬畏，想必知道这是一条蛇妖，而且还是一条妖王级别的妖。
<br/>　　“那你又是怎么跟它认识的，还姐妹相称”涂小安对这点更加的感兴趣。
<br/>　　落子烟说自己不是蛇，那她是什么，肯定不是人类。
<br/>　　三百年前？这女人到底是个怎样的怪物。
<br/>　　“我也救过它”她简单一句话。
<br/>　　然后呢？
<br/>　　涂小安脚步一停，看着她。
<br/>　　她嫣然一笑：“我本来当初想收它做宠，有个伴，可惜梨儿感念白家之恩，答应一世守护，我就罢手了”
<br/>　　“呵呵...你好会避重就轻啊”
<br/>　　涂小安突然道：“这次虽然你插手了，但是下回我还会找金蚕蛊蛇的麻烦，我要吞了它提升实力，你会拦着我吗”
<br/>　　“要是梨儿对你有帮忙，你觉得自己可以对付打败它了，就行，我不会管”
<br/>　　此话一出，涂小安猛然诧异，说好的姐妹呢？
<br/>　　这个女人什么立场啊。
<br/>　　涂小安整个人都迷了。
