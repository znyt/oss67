　　三十六楼的走廊里，有马贵将带领着一众搜查官向前推进着，到了这个高度，已经很少能看到喰种的身影了。
<br/>　　但所有人都掉以轻心，即将接近楼顶，敌人肯定会有所埋伏。
<br/>　　“咯噔，咯噔……”
<br/>　　高跟鞋的脚步声从走廊内响起，有马挥了挥手，止住了队伍的前行，戒备的看着脚步声传来的方向。
<br/>　　“呐，松前，敌人的数量很多呢。”走廊的拐角处，一个身穿深褐色西装的男人走了出来，挡在了前进的必经之路上。
<br/>　　与之相随的，是松前娇小的身影，和西装男子脸上阳光的笑容不同的是，松前的表情，是无尽的冰冷。
<br/>　　仿佛一块永远都不会融化掉的冰块一般，让人一眼看上去，心底都会冒起寒意。
<br/>　　“伊丙入，木嶋式，你们两支队伍留下，剩下的人和我继续追踪。”有马贵将微微皱了皱眉，这两只喰种身上的气息并不简单，他怕有意外发生，并不打算出手。
<br/>　　这次行动最终的目标，并不是这些连名字都不知道的喰种，这些臭鱼烂虾，就交给下边的人去应对吧。
<br/>　　有马贵将身边，木嶋式和一名淡粉发色的女子应了一声，带着自己的下属，将两名喰种包围了起来。
<br/>　　“可恶，不会让你过去！”松前面部发寒，召唤出自己的赫子，攻向了有马贵将。
<br/>　　似乎没察觉到身后凌厉的攻击一般，有马贵将继续向前走着，脚步不算太快，但也不慢。
<br/>　　在场的搜查官们心中都是一紧，一些新人，甚至惊呼了出来。
<br/>　　“乒！”
<br/>　　一柄长刀从侧面插入，和松前的赫子交合在了一起，发出一连串刺耳的声音。
<br/>　　“你似乎攻击错人了，你的对手，在这呢。”伊丙入双手握着自己的库克因，站在有马身前，死死抵住了松前的赫子。
<br/>　　感受着赫子上面传来的力道，松前脸色一沉，心中开始着急了起来。
<br/>　　仅仅是被这样稍微阻挡了一下，身后众多搜查官们也都趁势围了上来，双方纠缠到了一起。
<br/>　　对身后的战斗没有表现出丝毫兴趣，有马率领着自己的几名部下，顺着楼梯走了上去。
<br/>　　他不知道目标藏在什么地方，故此只能一层层寻找下去，不过，虽然这种方法比较笨拙，但无疑使最为保险的做法。
<br/>　　距离最顶部，也只有两层了，他肯定会在剩下的两层中，找到月山习的踪迹。
<br/>　　“咯噔，咯噔……”
<br/>　　沉重的脚步声在走廊内回响着，有马带领着下属，搜索着楼层内的每一个房间。
<br/>　　哪怕厕所，也都没有放过。
<br/>　　但是已经搜索完半个楼层了，他还是没找到目标的半点踪迹，甚至，他都不怎么确定目标还在这里了。
<br/>　　当然，这次行动的目标，自然不是楼顶那个还苦苦等待救援的可怜少爷，而是那个在月山家别墅留下了一地尸体的修罗。
<br/>　　事实上，月山习等待的救援，永远不会来了，在行动之前，CCG就封锁了附近所有街道以及天空。
<br/>　　只要有直升机进入这里，瞬间就会被击落下来，哪怕是误伤也没关系。
<br/>　　“吱呀。”
<br/>　　留下两人在外面监察，有马推开了眼前的屋门，看到门后的场景，瞬间警惕了起来。
<br/>　　这好像是美林大厦的监控室，四周都是播放着监控画面的电脑屏幕，里面空无一人，只有三具无头尸体，和挂在天花板上转动的三颗头颅。
<br/>　　这种虐杀方式，和月山别墅中的那些尸体如出一辙，而现在有马也终能确定，修罗，就在这栋大厦里。
<br/>　　带着戒备的神色，有马仔细查探着房间内的每一个角落，希望能找到一些线索，或者，直接将目标找出来。
<br/>　　虽然之前是他亲手将‘修罗’抓捕到了库克利亚，但有马也不敢有丝毫大意，那次战斗，很明显是‘修罗’隐藏了实力，故意被他给抓捕了进去。
<br/>　　况且，借助库克利亚内的监控，他们自然能发现，修罗在库克利亚内做的一切事情。
<br/>　　共喰了那么多尸体，谁也不知道现在的修罗强大到了什么地步。
<br/>　　花费了一些时间，有马依旧没能找到什么线索，目标很聪明，根本没留下什么痕迹。
<br/>　　唯一能证明的，也只是目标来过这里，并且杀掉了这几名喰种而已。
<br/>　　看着地面上的一汪血迹，有马的脸色并不好看，片刻，像是想到了什么，挥了挥手，示意自己的下属离开这里。
<br/>　　虽然现在的情况很被动，但他却不怎么慌张，修罗来这里，必定是有什么计划的，只要自己耐心一些，总会找到破绽。
<br/>　　或许，可以从月山习身上下手，不管目标想要做什么，总不会放任月山习落到他们手中吧。
<br/>　　如果月山习对修罗毫无作用，早在别墅的时候，修罗就能杀掉了他了，也不会放任其逃亡到了这里。
<br/>　　一行三人，很快从监控室走出，突然，有马的身影猛然僵住，瞳目中，带着不可置信的神色。
<br/>　　门外，原本负责监察的两位下属，已经变成了两具干枯的无头尸体。
<br/>　　他们的头颅，被一根粗麻绳串在一起，绑在了窗户边上，像是正在嘲讽他一般，随着夜风，晃晃悠悠的飘动着。
<br/>　　仅仅是不到一刻钟的时间，他的两名下属，竟然就在他眼皮子底下被无声无息的解决掉了，甚至还被做成了这种模样！
<br/>　　要知道，有资格跟随他的，至少也有S级的实力，甚至普通的S级，都不是他们的对手。
<br/>　　他没指望自己用两名手下就能抓住修罗，一开始将两人放在外面监察，也只是想要起到一个预警的作用。
<br/>　　在有马的想法中，假使真碰见了目标，这两名下属怎么也不至于，连警报都发不出来吧。
<br/>　　看着窗边飘荡的两颗头颅，有马眼中升起了一抹沉思，他在想，修罗这样做的目的是什么。
<br/>　　总不可能，仅仅是因为嘲讽，那没意义。
<br/>　　“有马先生，好久不见。”就在几人搜查走廊的时候，一道略带笑意的声音从有马后方传来。
<br/>　　在他们身后，之前搜查过的监控室内，一名带着红色面具的喰种，笑嘻嘻的坐在老板椅上。
