　　第398章
<br/>　　白元飞顺利潜入城主府，时间该是子丑相交，他趴在大殿的屋脊上，刚潜进去，就被城主府的景象吓了一跳。
<br/>　　底下是密密麻麻的巡逻小队，每一刻钟一个班，互相交替，严密得没有任何缝隙。
<br/>　　看他们的动作，都保持着不错的体力，并没有因为夜深，便流露出松懈。
<br/>　　白元飞心情有些无语，这城主府究竟藏着怎样的秘密，需要用的着这样严密的警戒。
<br/>　　一开始只是觉得凉州城有些奇怪，现在看来，这里面肯定藏着一些秘密。
<br/>　　现在以白元飞的本事，也没有自信潜下去不会被发现。更重要的原因是，他目前尚不知道李行哉和福德勒住在哪里。
<br/>　　城主府是极大的，站在高处望过去，有大殿、庭院、回廊、假山、绿水，无数个院子连在一起，也分辨不出哪里住着李行哉。
<br/>　　白元飞的任务是刺探这里的情报，只要随便打探出些消息，平安返回琴川关，他就算完成任务。然而，这并不是白元飞想要的，他的目的是见到李行哉，拿他一件信物回去。如此，这次任务才算得上出色。
<br/>　　现在尚摸不到李行哉的藏身处，白元飞倒也不着急，他悄悄顺着柱子趴下去，潜入了城主府的厨房，然后一翻身，跃到了柱子上。
<br/>　　贼，常常用胆小来形容，但胆小有时候也意味着谨慎。想要偷一件东西，他们从来不缺乏耐心。
<br/>　　他为偷一件东西，曾经在阴沟里躲了三日，靠随身携带的小米充饥，将自己的身体机能降到极限，忍受着污秽和有时会从身上爬过的老鼠，直到等来最后的机会。
<br/>　　白元飞，本就是此道高手。
<br/>　　在黎明时分，城下的卫兵尸体被发现，层层汇报，最后报给宋伯康。宋伯康也是个相当谨慎的人，目前的李行哉可以称得上奇货可居，在未来将发挥出多大价值，没有人想象。因此，宋伯康不得不谨慎，城里城外，都加派了人手，以免李行哉出现任何意外。
<br/>　　当事情发生后，宋伯康第一时间就预感这件事和李行哉有关。有人潜入城，想要救出李行哉，然后，宋伯康就针对全城展开了大搜捕，一时间，凉州城风声鹤唳，草木皆兵。
<br/>　　宋伯康甚至专门见了见李行哉，看到李行哉安稳的出现在眼前，他悬着的心才算放下来。
<br/>　　不得不说，宋伯康对李行哉的招待一直是不错的。凉州城内有名的歌姬，一波一波往这里送，吃穿用度，仆人伺候，只要宋伯康能满足的，绝不会有半分折扣。
<br/>　　宋伯康有自己的心思，未来的李行哉会成为怎样的人，谁也无法预判。说不得就面南座北，成了九五之尊。就算是个傀儡皇帝，可傀儡皇帝也是皇帝。如果自己今日对他有亏待，他把仇恨的种子埋在心里，只等来日报复，这种事情还是尽量避免发生。
<br/>　　同样，在生活待遇上不会有亏待，可在别的事情上，宋伯康则是绝对限制李行哉的自由。比如，绝不允许他和福德勒见面。
<br/>　　李行哉确确实实被软禁起来，对于这样的待遇，他并没有表示出什么情绪。
<br/>　　终日花天酒地，与歌女饮酒作乐，躺在万花丛中，消遣着美人的胭脂脂粉，常常醉得不省人事。
<br/>　　许多见过这一幕的人，连连摇头。还未成为帝皇，李行哉就显露出一个昏君的气质，若有朝一日，他执掌天下，大概会成为和桀纣比肩的存在。
<br/>　　当然，一个昏庸无能的废物更适合成为一名傀儡，也更加容易被控制。
<br/>　　这也是宋伯康，包括宋伯康背后那位崔相乐于看到的。
<br/>　　可很少有人能了解，李行哉内心深处的苦闷。
<br/>　　自从失火之后，他便和福德勒被分开关押。李行哉和福德勒相处的时间很长，二人一起流浪，一起偷鸡摸狗，一起坑们拐骗。这些事情都是二人一起做，早已产生很深的默契。这突然的分开，剩下李行哉一个人，他像是失去一条臂膀一样无助。
<br/>　　一切问题，都得他自己想办法解决。
<br/>　　况且，李行哉现在处于严密的监控下，所住的院子内，仆人、厨师、园丁，包括每日陪他的歌姬，都是宋伯康的人，甚至李行哉每天睡觉的时候，都至少有两双眼睛看着他。
<br/>　　李行哉如果能够肋生双翅，他可以从这里离开，但可惜他没有程大雷身无彩凤双飞翼的本事。
<br/>　　李行哉唯一能做的，也就是花天酒地，借此蒙蔽自己的敌人，从而以期待对方能放松警惕。
<br/>　　他在外人面前演绎的多放纵，内心就有多苦闷。被关在这里，失去了所有自由，也失去了长安城的消息。
<br/>　　现在的长安城，每分每秒都可能发生大事。莫说身为帝国皇子，便是身为人子，也该替父亲分忧。但李行哉能做的，也只是被关在这里。
<br/>　　所以，他纵情声色，一方面有麻痹敌人的用意，另外一方面，未必没有借酒浇愁的心情。
<br/>　　长夜漫漫，酒喝得越多，后来便越不容易醉。李行哉赤身果体躺在床上，盯着高高的屋顶，身边躺着今夜侍寝的女子。
<br/>　　在严密监视下，李行哉便是在外人面前流露出任何情绪都不能。只能一夜一夜，盯着屋顶发呆，等待黎明到来。
<br/>　　他就像被绑在断龙台上的巨龙，虽然体内流淌着高贵的龙血，可以呼风唤雨，翻江倒海，但却被绑在柱子上，一动不能动。
<br/>　　吱呀声响，窗户被推开一条缝，一个竹筒探进来，随着吹入烟雾。
<br/>　　李行哉眼睛已经有些沉重，挣扎了两下，忽感觉一方湿透的手帕盖在自己脸上。等他清醒过来，就看到一个黑影立在自己面前。
<br/>　　浑身污秽，身上散发着令人作呕的异味，他睁大眼睛，好不容易才将白元飞认出来。
<br/>　　白元飞到琴川关的时间不长，和李行哉算不上熟悉，此刻的情况下，彼此看对方都有些陌生。
<br/>　　“是你？”
<br/>　　“好啦，好啦。”白元飞挥挥手：“你能不能先把衣服穿上再说话，这样很不雅观的。”
