　　　　冲出来的牛妖王直接进入战斗状态，浑身黑色妖气缭绕沸腾，一根铁棍握在他的手中，棍影漫天挥洒。
<br/>　　“吃老牛一棍”
<br/>　　铁棍在手，牛妖王势如惊雷，一股旋风骤起，百步之内，飞沙走石，狂暴凌厉的龙卷风汹涌而出，威势慑人。
<br/>　　罗子杰抬头望天，天空似乎要垮塌下来了，旋风席卷，一股窒息感侵袭着他的身心。

<br/>　　“哈哈...这点剑气算什么”
<br/>　　牛妖王棍影大开大合，气芒雄浑凛冽，极尽刚猛霸道，令人望而生畏，将一道道席卷劈来的剑气打散。
<br/>　　旋即，他出现在了罗子杰的头顶，当头棒喝砸下。
<br/>　　此一幕，恐怖如斯，罗子杰那张脸惨白的接近透明，青筋条条可见，他眼睛似闭非闭，整个人好像要进入一种空灵的状态，眼猛然一张开，精光暴射。
<br/>　　“君临天下，人剑合一，杀...”
<br/>　　霎时间，罗子杰全色淡青色光芒闪烁，他翻滚飞身而起，身躯上幻起一层层一片片宛如盛开莲花般的绚烂光彩，紧接着，整个人变成一柄巨剑，迎头而上，剑吟声冲天而起，凌厉无比。
<br/>　　牛妖王微微诧异，但威势慑人，一棍携惊天之力，砸到了巨剑之上。
<br/>　　锵...
<br/>　　别墅大院空间猛然被撕裂，如山洪爆发，异象无比可怕，有天崩地裂之预兆。
<br/>　　不远处的涂小安被一股股气浪席卷，他却眼有神采，看着巨剑幻化下的人影，闪烁不断。
<br/>　　这罗子杰有勉强匹配三阶的实力。
<br/>　　剑修的实力难以预测，一把剑――扫平天下不平事一把剑――杀尽仙魔阻路人一把煞气的剑，仙魔，妖鬼皆可杀。
<br/>　　这便是剑修的剑意。
<br/>　　可惜，牛妖王是早就步入三阶层次多年的妖灵，他的战斗力非常的可怕。
<br/>　　纵然是涂小安，也是用了三种元素混合力量，强行给打败。
<br/>　　牛妖王的铁棍下，黑色能量铺天盖地，巨剑铿锵一声，嚓...
<br/>　　一道剑折断的清脆声响去。
<br/>　　只见巨剑消失，罗子杰身形显化出来，猛然大喷一口血的倒飞而出，他手中的青色长剑一折为二，已经坠落在地。
<br/>　　“哈哈...人类小娃子，能挡的住老牛这一棍，不错不错，但下一棍，你就是肉泥了”
<br/>　　牛妖王看着罗子杰倒飞而出，嘴角噙着一丝残忍的笑，铜铃大眼凶光大亮，骤然跟去，还未等对方倒地，他就出现在了罗子杰倒飞的上空，铁棍蕴含无比可怕的力量，怒砸而下。
<br/>　　已经没有剑的罗子杰，只能当棍下亡魂。
<br/>　　这要是砸中了，他真的成了一团肉泥，绝无例外。
<br/>　　倒飞中的罗子杰心如死灰，完了，这三阶妖灵太强大了，几乎是不可战胜。
<br/>　　弟弟，大哥来陪你了。
<br/>　　罗子杰满心不干的闭上了眼，宣告了死刑，如果他是三阶的实力，谁胜谁负未可知，但现在...
<br/>　　“牛妖王，住手”
<br/>　　忽然，一道冷冽的声音，在罗子杰即将殒命的时候响起，牛妖王闻言，愣了一下，铁棍已然出来，已覆水难收。
<br/>　　他不得不调转一个位置，轰...
<br/>　　铁棍砸到了地面，地面出现一个深坑，而罗子杰险险的躲过一劫，倒飞撞到了院墙之上。
<br/>　　全身跟散架了一样，战斗力全失，身受重伤。
<br/>　　那一棍，他挡下来了，但也是用了半条命一样。
<br/>　　“主人，你为何要喝住老牛”牛妖王转身看下涂小安。
<br/>　　涂小安淡淡的道：“我说让你杀死他了吗”
<br/>　　“啊...好像没有”
<br/>　　“一边呆着去，你的任务完成了”
<br/>　　涂小安朝着瘫痪在地，连连咳血的罗子杰走去，闲庭信步一样。
<br/>　　走到之后，看着罗子杰身受重伤的样子，摇了摇头：“现在你知道自己跟我有多大的差距了吧，你连我的战宠都远远不是对手”
<br/>　　“你说，我是不是应该送你跟你弟弟团聚”
<br/>　　罗子杰脸色苍白的宛如一张白纸，惨然一笑：“我罗家两兄弟，将来都有可能成为三阶生灵级别，没想到却毁在了你这条蛇妖手中，来吧，我技不如人，不能帮弟弟报仇，死便死了”

<br/>　　罗子杰怒声道：“废话少说，要杀就杀”
<br/>　　谁对谁错，他根本不在意，他只在意自己的弟弟被杀了，死了。

<br/>　　呃...
<br/>　　此话一出，罗子杰的眼睛猛然瞪大，像是听到了什么天大的笑话。
<br/>　　“哈哈...”
