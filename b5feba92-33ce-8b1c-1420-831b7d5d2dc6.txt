　　不周之巅，三处战场，大战不止，纷争不休。
<br/>　　一处为元始，他的对手为妖族两大妖皇，只是看样子两大妖皇不够他玩，面对两大妖皇的攻击，元始随意出手，便能化解。但，看样子，他一点也没有迅速结束争斗的想法。
<br/>　　另一处战场为老子与女娲。女娲修炼的是造化法则。造化法则的最大作用在于造化天地，对于战斗却并不擅长，故而，尽管她的修为与老子一样，但并不是老子的对手。
<br/>　　老子可不像元始一样，他面对女娲，一点也没有手下留情的意思。
<br/>　　太极阴阳图一出，阴阳逆乱，神通自现。面对老子的攻击，女娲手里只是一个上品先天灵宝红绣球，并不能发挥出多大的战力。
<br/>　　更何况，老子还头顶天地玄黄玲珑宝塔，玄黄之下，万法不侵，已经先天立于不败之地。女娲的攻击，并不能给老子造成丁点伤害，而老子的攻击，却是往往能够重创女娲。
<br/>　　很快，两人的战斗就结束了。
<br/>　　“女娲道友，得罪了。”
<br/>　　老子轻念一声，一道逆乱阴阳之术，在女娲换气之际，迅速打进她的身体之内，将她重创。女娲瞬间便失去战斗力，跌倒在地。
<br/>　　之后，老子停了下来，将目光投向另外两处战场，却是没有一点儿上前帮忙的想法。那是因为，他相信自己的兄弟能够解决各自的敌人。
<br/>　　至于第三处战场，便是通天与伏羲了。原本是通天对上女娲，但通天并不想与女子争斗，就将她抛给了老子，自己对上伏羲了。
<br/>　　通天的道为截剑道，剑道是以杀伐著称的大道，极其擅长杀伐，攻杀之术非凡。通天的截剑道更是厉害。
<br/>　　他的道取意为截取天道之下的一线生机。
<br/>　　天道五十，天衍四九，遁去其一。五十道中才遁去一道，可见这一丝生机不是那么容易截取的。故而，想要截取这一线生机，也就意味着需要更强的力量。是故，这就造成了通天的剑道杀伐力极强的结果。
<br/>　　通天的剑道强大，但伏羲的道也不俗。他往往能够提前推算出通天的出剑轨迹与破坏力，并且能够还快做出相应的对付手段。再加上，他的伏羲琴威力极大，能够控制人的心灵，造成各种阻碍，使得通天并不是那么的顺利。
<br/>　　两人暂时斗得有些旗鼓相当，不相上下。但伏羲知道，这只是暂时的，因为正所谓久守必失。长时间的防御，总有那么一次失误。
<br/>　　而通天却是抓住了那一次失误，果断出手。
<br/>　　“剑断天涯！”
<br/>　　通天抬手就是迅速一剑，剑光乍现，汇聚成一条银色细线。细线极细，仿佛若有若无，时隐时现。
<br/>　　以细线做锋刃，通天一剑斩向心力略有些枯竭的伏羲妖圣。剑光锋芒，剑刃虽细，但却极为霸道，其中蕴含着一种势不可挡的剑意，仿佛就连苍天都能斩断的决心。
<br/>　　天若阻我，我便断了它！
<br/>　　细线剑刃以一种无可阻挡的剑意斩在了伏羲的心间，再加上之前伏羲弹琴是用的心力过多，有些衰竭的原因，这一剑他已经不可阻挡了。
<br/>　　“铮铮——”
<br/>　　伏羲琴数根琴弦挣断，琴音紊乱，伏羲倒飞数里，这才停了下来。
<br/>　　“噗嗤——”
<br/>　　一口逆血喷出，染红了伏羲那雪白的道袍，伏羲脸色变得苍白无比，这是心力枯竭的表现。
<br/>　　显然，伏羲与元始一样，一样会心道的运用。但他的心道并没有元始的心道强大，心力亦不如元始多。用元始划分的心道一境来说，伏羲最多达到冰心境界。
<br/>　　通天趁伏羲受伤之机，继续持剑，斩向伏羲，显然，他并不打算手下留情。
<br/>　　看着向自己持剑斩来的通天，伏羲脸上并没有出现想象中的慌乱。只见他站立起来，之后迅速将几道玄法印诀打入虚空。顿时，虚空震动。
<br/>　　之后，他手捏印诀，大喝一声：
<br/>　　“先天八卦阵，起！”
<br/>　　顿时，虚空之中，浮现出一个巨大的八卦图形，天地风水，地火雷泽，八色光芒齐齐闪耀，整个天地仿佛是笼罩在一片独特的立场之中。天地，山峰，雷霆，大水，赤火，暴风，沼泽，万事万物，齐齐释放出一股恐怖的威压笼罩在这片虚空，将通天围住。
<br/>　　空间之中各种异象，全部攻向通天，那恐怖的威压，亦在此时降临道他身上。这时，各种风刃，天火、雷霆大河……全部轰向通天。
<br/>　　八卦阵之中，八八六十四道变化，变化中又有变化，让人防不胜防。
<br/>　　“先天八卦大阵？有点意思。”
<br/>　　通天见此，顿觉得自己找到了对手，他自己也是阵法大家，平时除了元始外，并没有碰到过其他的对手，如今有人与他比阵，他也求之不得。
<br/>　　“阵道天眼，给我——开——！”
<br/>　　顿时，通天的额头之上，长出第三只眼睛。此眼青光闪闪，神痕闪耀，各种玄奥，尽显其中，端是神奇无比。
<br/>　　只见他将眼睛看向虚空的大阵，阵中的种种玄妙，顿时清晰地显现在他眼前，不过片刻时间，他便将八卦阵的大致情况，了然于胸。
<br/>　　很快，他便有了想法。
<br/>　　“八卦阵，卦卦相连，一卦破，其余七卦便不足为虑。既然如此，那么——”
<br/>　　“看我如何破你大阵。”
<br/>　　通天选择了八卦中的震门，挥手之间迅速打入几道道法进入，之后，他手捏阵诀，虚空颤动，阵光飞舞。
<br/>　　“爆炸！”
<br/>　　轰隆隆——
<br/>　　随着通天的话音刚落，震门便发生大爆炸，一片虚空被震碎，阵门洞开，而此时，亦露出了伏羲那狼狈的身影。
<br/>　　“嘿嘿。”
<br/>　　通天怪笑一声，化作一道流光迅速冲出阵门，来到虚空之中。
<br/>　　“青莲通天！”
<br/>　　通天手中青萍剑连斩，顿时，虚空之中出现一朵巨大的青莲，青莲玄光闪烁，耀眼无比。
<br/>　　一朵莲花，顶天绽放，两根莲茎，相互交织，三片莲叶摇摇曳曳，氤氲生烟，好像在阐述着道生一，一生二，二生三，三生万物的大道。
<br/>　　莲花绽放，开尽整片虚空，演绎出一副美妙的虚空图像。花落叶残，落花之中，满是惊天剑气，剑光冲天，杀机四伏。
<br/>　　于美好之中杀人，于绝美之中绝杀。
<br/>　　美好的事物，本就存在杀机，不是吗？
<br/>　　结局似乎早已注定，伏羲最终还是输在了通天剑下。
<br/>　　但通天并没有斩杀他，或许是在等待元始的发话，也或许是他不想杀伏羲。总之，因为种种原因，伏羲还是活了下来，虽然他已经被通天打得半残。
<br/>　　“唔，不玩了。”
<br/>(本章完)
