<br/>　　柴管事点了点头，面色威严的道：“他正在接受公会炼器师考核，任何人都不能打断。”

<br/>　　“考核继续！”
<br/>　　柴管事没有理会脸色变幻的巫梵，朗声宣布道。
<br/>　　巫梵没有阻止，因为他阻止不了，现在他只能希望苏莫考核失败了。
<br/>　　他也是今日才知道，苏莫居然是炼器师。

<br/>　　一个苍穹世界的土著，武皇境九重的修为，不足三十岁的年纪，会是帝级炼器师？
<br/>　　这怎么可能？
<br/>　　“小友，你无需担心，拿出你巅峰的炼气水平！“柴管事对苏莫说道。
<br/>　　苏莫点了点头，没有说话，向柴管事抱拳一拜。
<br/>　　而后，他努力屏弃心中的杂念，这一次他只能成功，不能失败。
<br/>　　成功，暂时性命无忧，还有了炼器师公会这个靠山，失败，就要再次生死逃亡。
<br/>　　唰唰唰！！
<br/>　　苏莫手臂一挥，十数种材料便平稳的飞到了炼器熔炉之中。
<br/>　　而后，玄力涌动，火焰在炼器熔炉中沸腾了起来。
<br/>　　所有的人，都仔细的观看着苏莫的炼制，心思各异。

<br/>　　“老夫倒要看看，你是否能炼制出帝级兵器！”巫梵眸中一片冰冷，心中暗道。

<br/>　　因为有了数次的成功经验，苏莫这一次的炼制，过程行云流水，顺畅无比。

<br/>　　仅仅两个多时辰的时间，炼制便结束了。
<br/>　　炉盖打开，炼气熔炉之中，强烈的气息冲天而起，属于帝级兵器的威势，弥漫整个炼器师之中。

<br/>　　所有人都是心中剧震，真的炼制成功了？
<br/>　　巫梵和卢云两人的脸色，非常的难看，这股帝级兵器的威势，傻子都知道炼制成功了。
<br/>　　卢云心中震惊不已，苏莫的炼器水平，居然真的达到了帝级！
<br/>　　而巫梵，比卢云还要震惊，他可是知道苏莫来自苍穹世界，一个苍穹世界的小小土著，都能成为帝级炼器师，这简直不敢想象。

<br/>　　柴管事眼眸闪亮，虽然苏莫炼制的重渊斧品级不高，但他心中非常的满意。
