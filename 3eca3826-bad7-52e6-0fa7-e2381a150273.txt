　　叶少阳缓缓点头，这时候又听见道渊真人说道：“茅山也很重视这次机会，云秋生，还有单传的弟子伏明子也都去了，哦对了，伏明子还带着一个孩子。”
<br/>　　“孩子？什么孩子？”
<br/>　　“可能是他弟子吗，这个就不清楚了，我也只是匆匆看到他们，立刻就回来了。少阳子，你……这次去了，要不要见见他们？”
<br/>　　叶少阳迟疑起来。
<br/>　　按说，现任掌教云秋生，和下一任掌教伏明子，都是自己的祖师——伏明子是青云子的师父。如果可以，自己真的很想拜访一下他们，但问题是，自己没有身份……怎么跟他们相见？告诉他们，自己是从未来过来的，是你茅山的后人？
<br/>　　且不提他们信不信，就算是信了，也会知道自己的名字，知道自己百年后将继承茅山掌教，将来收了青云子之后，万一忍不住告诉他：将来你要收一个徒弟，叫叶少阳，会继承你的位子……
<br/>　　这样会对历史造成什么影响，叶少阳不知道。最重要的是，如果现在茅山的几大法器都在云秋生或伏明子的身上，自己再拿出同样的，又是一种错乱，会产生什么，叶少阳不敢想。
<br/>　　他想到了以前不知从哪里看到过的一篇文章，说穿越时空之后，见到自己，千万不能握手，因为两人中一个是正物质，另外一个就是反物质，正反物质一旦接触，会产生比核裂变还要可怕的事情……叶少阳也不知道这个科学原理，能不能套用在自己遇到的事情上，但是还是尽量避免得好。
<br/>　　“那个，跟你们说个事，等到了桃花山，你们不要说我是茅山的，我也不承认，免得见到云秋生师徒，我不知道怎么解释，我穿越过来这种事，不想让更多人知道。”
<br/>　　两人都表示理解，一口答应。
<br/>　　在前往杭州之前，叶少阳三人顺路去了平南县，去见了翠云，还有那个叶少阳从蛤蟆精那里救下来的云儿，两个都非常开心，叶少阳把自己从古墓里拿的金元宝都交给了翠云，让她先拿着用，多余的收起来。
<br/>　　本来叶少阳打算见她一面，报个平安就走的，但是叶少阳这么多天不回来，翠云当然不肯就这么放他走，硬是拉着他们住了一天。
<br/>　　翠云已经不在客栈住了，在附近租了个农家小院，跟云儿一起过日子，因为是两个姑娘家，还都是漂亮姑娘，有点怕怕的，因此养了一条大黑狗。
<br/>　　包子和美华都隐了身形，人是看不见他们的，但是大黑狗能看见，一直在冲他们叫，包子被叫得烦了，坐在叶少阳肩膀上，两只手勾住嘴角，往两边扯到了一个夸张的地步，那狗看到这一幕，吓得哼了一声，夹着尾巴就跑了。
<br/>　　叶少阳照头上给了包子一巴掌，“不要欺负人家！”
<br/>　　当天，叶少阳三人就在小院里住下了，美华和橙子不用睡觉，出去逛了一圈，就回到阴阳镜里呆着去了，阴阳镜里的环境，对修炼也更有帮助。
<br/>　　当天晚上，翠云下厨，云儿打下手，做了一大桌饭菜，叶少阳一看，很多菜都有花样，不是农家能吃到的，一问才知道这是翠云在县城的饭店里学到的，她现在带着云儿一起，在饭店里帮厨，因为两人名字里都有一个云字，于是结拜成了干姐妹，又都是单身，在一起过日子倒也其乐融融。
<br/>　　云儿很喜欢缠着毛小方，吃饭的时候，一个劲给他夹菜，各种殷勤。毛小方表达得有点害羞，叶少阳看在眼里，觉得很好玩。
<br/>　　晚饭的气氛很轻松，吃完之后，两个姑娘去收拾碗筷，之后一起聊天，云儿缠着毛小方要拜在他门下，毛小方对应付女人的纠缠方面没什么天赋，被云儿缠的团团转。叶少阳把翠云叫到屋里，跟她商量，让她用自己给的钱去开一家饭馆。
<br/>　　“开饭馆倒也不错，不过，姐更想给你当管家啊，等你娶了媳妇成家，姐再考虑这件事。”翠云一句话就把他堵了回去。
<br/>　　叶少阳无语。
<br/>　　“那你先去买一所房子，或者就把这所房子买下来，这样也算有个自己的家，很多事也方便一点。”叶少阳提议。
<br/>　　翠云不想花他的钱，想给他攒着娶媳妇，叶少阳跟她说了半天，才勉强同意。
<br/>　　“对了，少阳，你还记得二龙山土匪寨吗？”翠云突然说道。
<br/>　　叶少阳一怔，立刻就想到了在二龙山上遇到的那个家伙、办了鬼见愁，成了二龙山大当家的三爷。
<br/>　　“二龙山怎么了？”
<br/>　　“我在饭店帮厨，能听到很多新鲜事，最近有一个当兵的去吃饭，跟老板聊起来，说二龙山被官府给剿了，就是前几天的事。”
<br/>　　“好事啊，那个土匪窝。”顿了一下，叶少阳问道，“死了很多人吗？”
<br/>　　“据说山寨被官府放火给烧了，死了不少土匪。”
<br/>　　“那个三爷也烧死了？”叶少阳心想，如果那个三爷也死了，那可真是除了一个未来的祸害。
<br/>　　“三爷没死，我听说，那个三爷，带着师爷和一帮心腹跑了，好像是去投奔刘司令了。”
<br/>　　叶少阳怔住。
<br/>　　刘司令，刘震匀，江西境内最大的土匪头子。叶少阳设想了一下，三爷那种人，心狠手辣，但是做事面面俱到，二龙山没了，去投奔刘震匀，没准对他个人的前途来说，还是好事。
<br/>　　叶少阳相信他一定是能够混出来的，这种人，只要不死，就绝对不会久居人下，不过，叶少阳有点为那个素未谋面的刘震匀担心，以三爷的手段，不会不觊觎他的地位。
<br/>　　不过这些事情，跟自己并没什么关系，叶少阳也懒得去想。
<br/>　　晚上，叶少阳一个人躺在房间里，拿出钱包，找到那张跟芮冷玉的合影，在月光下静静地端详，有那么一刻，他感觉自己就像是一个风烛残年的老人，当一生都尘埃落定之后，捧着昔日恋人的照片，缅怀昔日的美好时光。
<br/>　　世上最美好的东西，都是已经失去的。
