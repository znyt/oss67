　　第167章
<br/>　　“那……就逃吧。”
<br/>　　程大雷高高举起的手无望放下，像溺水者要抓住最后一根稻草，结果却是什么都抓不住。
<br/>　　可随之他就将手举起，道：“逃，也得听我安排！五千弟兄做鸟兽散，只能成为别人的刀下之鬼，又能活下来几个。我带着大家走，五千弟兄一起走，不放下一个老的，也不丢下一个小的。”
<br/>　　“遵大当家吩咐！”
<br/>　　程大雷这话倒是没人反对，戎族大军压境，其实众人都没了主心骨，程大雷此刻肯站出来，大家愿意以他为主。
<br/>　　“林少羽听令！”
<br/>　　“属下在！”
<br/>　　“命你一人爬上牛角峰，带上一月干粮，以大树为信，指明戎族来犯方向！”
<br/>　　牛角峰是青牛山最高峰，在峰顶下望，整座青牛山一览无遗。
<br/>　　“廖甲廖乙接令！”
<br/>　　“属下在！”
<br/>　　“你们二兄弟与我各带一千人，挖河，毁道，以山石大木阻塞道路，为大家撤退争取时间！”
<br/>　　“属下接令！”
<br/>　　程大雷又目视点将台下众人，道：“诸位弟兄，想走大家一起走，五千弟兄一起走，我程大雷不会放下一个弟兄。”
<br/>　　众人都有了主心骨，尤其那些老弱，跑也跑不了多远，没了大部队结果就是死路一条。如今程大雷既然放出这话，大家心里一块石头落了地，山谷内山呼海啸一片。
<br/>　　程大雷又叫徐神机，高飞虎几个留下，商量大家该如何撤退。
<br/>　　“大当家，此事宜急不宜缓，咱们是否该连夜动身？”高飞虎道。
<br/>　　“莫要着急！”程大雷道：“戎族打下黑石城，自然要烧杀劫掠一番，把物资奴隶运走，如此没有十天半个月下不来。我们先被吓得乱了手脚，到时候慌不择路，岂不是自寻死路。”
<br/>　　见程大雷如此镇定，高飞虎也放下心来，嘴上说了一句：“一切听大当家吩咐就是，跟着大当家我们都是放心的。”
<br/>　　“那是，等离开这里，咱们再占一个山头，照样吃香的喝辣的。”熊大熊二道。
<br/>　　程大雷微微一笑：“诸位放心，一切都在我怀中。现在咱们商量一下往那里走？”
<br/>　　几家头目聚在一起，商议撤退的路线。
<br/>　　“要不咱们往并州逃？”熊大道：“那里路好走，一马平川？”
<br/>　　“并州和戎族接壤，逃到那里，可能比这边更惨。”程大雷把熊大的提议否了。
<br/>　　“如今只有往冀州去，在冀州飞龙山我有个朋友，与我是过命的交情。咱们不如去投靠他们吧？”高飞虎道。
<br/>　　“信得过么？”程大雷道。
<br/>　　“倒是和我关系不错，当初他未发迹时我曾经帮过他。”高飞虎。
<br/>　　“如今，似乎也没有别的办法了。”程大雷道：“去那里有多远路程？”
<br/>　　“咱们五千人都过去，大概要走一个多月。”
<br/>　　“时间够久的，那就要带上足够一个月吃的粮食。”程大雷道：“还有武器，也要防身用……诸位一起准备吧，多伐大木，以制马车。”
<br/>　　徐神机道：“大当家，准备这些可不是一天两天的事情，戎族不会杀过来吧。”
<br/>　　“我让少羽上牛角峰干什么？”程大雷白了他一眼：“放心，这些我都有算计，绝不落空。”
<br/>　　……
<br/>　　黑石城，夜幕。
<br/>　　福德勒小心的翻过墙，潜入城主府，一路小心谨慎，不敢被人发现。
<br/>　　紧跟着，他悄悄的潜到城主府的菜窖中。这时代，就算一城之主也得在家里挖个地窖。
<br/>　　“什，什么人？”黑暗中有人小声问。
<br/>　　“别害怕，是我。”福德勒手里提着一个篮子：“快吃吧，我给你带过来的。”
<br/>　　没过多时，黑暗里便响起小声咀嚼的声音，像一只贪吃的狗。
<br/>　　“你别噎着了，我给你带了水喝，以后我怕是不能一直过来，这是三天的食物，你要慢慢吃。”
<br/>　　“谢，谢你把我藏在这里……你挨打了……”
<br/>　　黑暗的地窖中没有点灯，借着模糊的月光，能看清福德勒脸上挂着淤青。
<br/>　　“没什么，我哥哥经常打我，我都习惯了。”福德勒咧嘴笑着，看着绿樱桃狼狈的吃相，他想起曾经自己养过的一只小狗。那时它也像绿樱桃这样贪吃，只是后来被哥哥杀了，煮成了肉。
<br/>　　“是我连累了你……所以谢谢你，你们什么时候走啊？”
<br/>　　“我也不知道什么走，可能就不走了，哥哥说要在这里找一样东西，我也不知道是什么。”
<br/>　　“其实……你是个好人。你和他们不一样的……”绿樱桃吃得狼吞虎咽，忽然问：“你们要是不来打我们就好了，你们为什么要那么坏，来抢我们的东西？”
<br/>　　气氛瞬间安静下来，福德勒的呼吸似乎也停止了。绿樱桃恍然意思到自己面前是一个戎族，自己可能不小心触怒了他。
<br/>　　“对，对不起，我说错了话，你别在意……”
<br/>　　“没什么……其实我们早吃不上饭了。部落去年就遭了灾，大片草原都荒了，牛羊成片成片的饿死，不来抢你们，我们也活不下去。”
<br/>　　声音渐渐低下去，福德勒突然道：“就这样，我先走了，你好好的，我会经常过来看你。”
<br/>　　此后，这件事就牵住了福德勒的心，把绿樱桃藏在地窖，像养了一只小犬一样，总想每天看一眼，瞧瞧有没有长大。他心中的太阳乌力罕，这几日也不怎么想了。
<br/>　　这天旁晚，他又悄悄来到地窖，进入地窖才敢说话：“小桃，小桃，我来看你来了，你饿不饿，我给你带了好吃的。”
<br/>　　没有人说话，待走近最里才看到，小桃已经没了踪影。在草堆上，一个年轻人正在打盹，令人惊奇的，对方竟然有一撮红色的眉毛。
<br/>　　瞬间，福德勒冷汗直冒，身体不自禁的打哆嗦。
<br/>　　“哈！”赤眉打了个哈欠，悠悠醒来：“好弟弟，你带了什么好吃的来，我睡了一觉，刚好饿了。”
<br/>　　“你……杀了她？”福德勒失声问。
<br/>　　“你的眼神……真让人怕啊。”赤眉收起笑容：“像饿极的狼，如果我告诉你，她已经死了，你会杀了我么？”
<br/>　　
<br/>(本章完)
