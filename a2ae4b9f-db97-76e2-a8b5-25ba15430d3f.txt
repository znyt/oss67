　　“是很古怪。”老郭沉吟道，“你回去好好回忆一下——你还是不要回去了，你就在我这呆着吧。下次再梦到这些，争取记住一些细节。”
<br/>　　美华不解地问道：“这是做什么？”
<br/>　　“你先别问，反正你就在这我住着，按我说的去做。”
<br/>　　谢雨晴对美华道：“你可别听他的，这老色狼，你在这跟他一起住，简直就是羊入虎口啊。”
<br/>　　老郭一头的郁闷。“你说的好像我能打得过她似的。”
<br/>　　“哦，说的也是。那我就放心了。”
<br/>　　老郭一听更郁闷了，“敢情我要是打的过她，你还不放心了？我是那种人吗？”
<br/>　　谢雨晴点了点头。“你还真是这种人。”
<br/>　　老郭摆了一下手说道：“我跟你们说正事，还调侃我，不理你们了。”
<br/>　　谢雨晴正了正色，说道：“美华做的梦，怎么了？你为什么这么重视？”
<br/>　　“这个……我只是有一个猜测，也没什么头绪，先不说这个，如果有结果再说，先让美华在我这住下一阵子再说。”
<br/>　　结束这个话题之后，老郭又想起别的事，询问小青他们青冥界之战眼下怎么样了。
<br/>　　“就是那样，战事胶着。尸族这次真的是有备而来，也倾尽全力了。”
<br/>　　老郭喃喃说道：“那当然，尸族蛰伏了那么久，这一次既然发起攻势，自然不死不休，不知道空界守得住吗？”
<br/>　　瓜瓜插了一句：“我看啊，尸族想要完全占领空界，那是不可能的，不过空界诸方势力想要把尸族赶走，也是做不到的了。”
<br/>　　“尸族，要的应该就是这个结果。”老郭喃喃说道。
<br/>　　瓜瓜跳到他椅子扶手上，晃了一下他的肩膀，说道：“你说的那个龙华会，他们想要选出新的第一弟子，你跟苏钦章他们有什么办法阻止啊？”
<br/>　　“有什么办法，小苏同学在会上会据理力争，只有这样，至于我……我可没资格参加龙华会。”
<br/>　　瓜瓜道：“你一定要参加，而且要带我去。”
<br/>　　老郭斜眼看了他一眼，“你去能干啥？”
<br/>　　瓜瓜咬了咬嘴唇，道：“如果他们实在咄咄逼人，要将老大从茅山驱逐出去，我……我或许有办法阻止！”
<br/>　　他严肃的表情，让大伙感觉他不是在胡扯，于是问道：“你能有什么办法？”
<br/>　　瓜瓜叹了口气，道：“我有一个秘密，跟谁都没有说过的……”
<br/>　　老郭愣了一下，猛然想起这茬，说道：“是不是上次，道风逼问你为什么要潜伏到小师弟身边，后来小师弟询问你，你也没说的？”
<br/>　　瓜瓜点点头。
<br/>　　“我不说，是为了老大着想，确切说……这是两个秘密。我一直守口如瓶，只是怕老大知道了多想……”
<br/>　　老郭过去把门关好，在屋里焚烧了三支迷迭香，这是一种特制的迷迭香，烟气会在局部干扰气场，避免被一切生灵偷听到其中的人说话。
<br/>　　这种方法没有叶少阳的秘术好用，瞒不过谛听的耳朵，不过一般生灵绝对是听不见的。
<br/>　　“一个秘密，其实是关系到老大的身世……”
<br/>　　老郭和谢雨晴等人互相看去，都是一脸的怀疑。
<br/>　　“少阳不是叶家村出生的吗，父母双亡，祖上是什么叶法善……还有什么身世一说？”谢雨晴好奇地说道。
<br/>　　瓜瓜犹豫着，对老郭道：“真要现在说吗？”
<br/>　　“废话，你都说到这份上了，突然不说了，你要急死我们啊！小师弟又不在这，如果是对他不好的事，大不了我们一起保密，谁也别告诉他！”
<br/>　　大伙纷纷点头，这帮家伙，为了叶少阳可以抛头颅洒热血，更别说保守一个秘密了。
<br/>　　瓜瓜总算下了决心，虽然环境很安全，还是把大家聚集到一起，低声说道：“叶法善，没死……”
<br/>　　“什么！！”
<br/>　　几个人一起叫起来。
<br/>　　老郭手中的茶杯打了，茶水溅了一身，也顾不得擦拭了，怔怔地看着瓜瓜。
<br/>　　“叶法善没死，当年他的确是进了太阴山，试图击杀无极鬼王，但是失败了……但是他元神还在，仍旧留在太阴山中。”
<br/>　　“这……怎么可能。既然失败了，怎么会不死？”
<br/>　　“叶法善去太阴山之前，已经斩了三尸，证得混元大道，可以说是天下无敌的，不然他怎么能闯进太阴山。他舍弃肉身，在鬼域徘徊了上百年……太阴山在荫水河的尽头，浮屠海中，不过并不是你们想的那样是一座山耸立在那，而是……”
<br/>　　瓜瓜挠了挠头，“怎么说呢，就像很多邪物开辟的洞府一样，只要实力足够，就算是一粒沙，也可以开辟出一个巨大的空间……太阴山的入口，就隐藏在浮屠海中，如果没有线索，想要找到太阴山所在，就像大海捞针一样，基本上不可能成功。不过叶法善在浮屠海中找了上百年，还真让他找到了。
<br/>　　但是太阴山外有强大的禁制，就算找到太阴山的入口，也是不可能进得去的。不过，叶少阳又用了百年时间，居然找到了突破禁制的法子，钻了进去……”
<br/>　　“乖乖，这叶法善，神了。”老郭听到这，忍不住赞了一声。
<br/>　　瓜瓜没理会，接着往下说道：“叶法善在进入太阴山之后，被发现了，与无极鬼王大战一场，不过败给了鬼王，但他是斩却三尸的大罗金仙，没有肉身，也没有魂魄，元神不灭，又用了一种禁制，将自己牢牢钉在中间，就算是无极鬼王，也没有办法真正杀死他，只有将他困在太阴山中……”
<br/>　　老郭缓缓点头，都说仙人是不死的，斩却三尸的人，元神不灭，的确等于是不死的。
<br/>　　“我本是长于地下的十二年蝉，当时叶法善被封印的地方，就在我附近，我也不知道他是怎么通过封印，用神念跟我交流的，可能是……”
<br/>　　话没说完就被小青打断，说道：“仙人的法术，我们怎么能懂，这个不重要，老二你直接说重点。”
<br/>　　瓜瓜横了他一眼，接着说道：“我在地下很多年，身边无人为伴，他说我资质好，传授了我一些玄功，助我修炼。”
<br/>(本章完)
