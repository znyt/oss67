　　　　涂小安也无需在多想，因为一会儿后，他的眼前就出现了一位亭亭玉立的女子。
<br/>　　只见这女子约莫十八九岁年纪，一张典型的瓜子脸，明艳动人，她一头靓丽的秀发微微飞舞，她肤光如雪，双眸犹如辰星般闪亮，珉着一张樱桃小口，杵在哪里，有点不知所措。
<br/>　　“海棠，恭喜你，你现在可以当陈高阳真正的女朋友了”
<br/>　　仅仅一眼后，涂小安满意的点了点头，很美的女子，这才配的上它那柔声细语的声音。
<br/>　　“我...我化为人形了”
<br/>　　海棠不敢置信的摸了摸自己的脸颊，不在毛绒绒，而是泛着淡淡光晕，吹弹可破。
<br/>　　如果不愧是猫精所化，自带美艳，容貌差不到那里去。
<br/>　　如今海棠走在大街上，回头率绝对是非常的高，看的涂小安都不禁有点心动。
<br/>　　这具身体虽然是虚假的形态，但是做做别人的女朋友是没什么大问题，当然，这前提是两个人要止乎于礼，不能做什么儿童不宜的事情。
<br/>　　呆滞了半晌，海棠确定了自己的人形态，激动的难以言语，突然，她直接跪在了涂小安的面前：“谢谢你，谢谢你，我从来没想过自己可以化为人形”
<br/>　　幸福来的太快，让这只猫精宛如梦境。
<br/>　　其实能化为人形，哪怕只有一天的寿命，她都死而无憾了。
<br/>　　唯一的牵绊便是放心不下看不到光的陈高阳。
<br/>　　怕他照顾不好自己。
<br/>　　涂小安连忙将海棠给扶了起来，：“你不用谢我，这是你应得的好处”
<br/>　　“不过我要提醒你一点，你这具身体是虚假的形态，要保护好，如果受到什么强大的外力干扰，那么便会溃散”
<br/>　　要是溃散，那么涂小安就帮不了海棠了，就算再给她购买化形术，也用不了了。
<br/>　　只能等她自己突破到三阶生灵才可化形。
<br/>　　“恩公谢谢你，我会记得你的话，好好珍惜这具身体”海棠再次朝着涂小安跪下。
<br/>　　恩公？
<br/>　　涂小安有点诧异，连忙又将海棠给扶起来。
<br/>　　恩公这两个字，要是涂小安没有得它的蛇王石，或许还受得起，毕竟这等化形，等于让这只猫精重获新生。
<br/>　　但涂小安是先得了她的东西，所以恩公两个字可就算了。
<br/>　　“海棠，你真的不用如此，接下来，你就好好陪着你的高阳哥哥，别的事情交给我”
<br/>　　海棠依然非常激动：“谢谢恩公，海棠要不是守护高阳哥哥，一定为你当牛做马”
<br/>　　此言一出，涂小安顿时无语，这猫精是在人类世界待的太久了呀。
<br/>　　这话说的，你就好好当你的猫就行了。
<br/>　　“走吧，我们应该离开了”
<br/>　　“恩恩”
<br/>　　现在化形的海棠是完全已涂小安马首是瞻，对他的感恩之心，宛如滔滔江水一发不可收拾。
<br/>　　由此可见，生物化形，对它们的意义有多么的重。
<br/>　　不知不觉，夜已深，温城校园不知怎么样，不过涂小安相信，群蛇没了蛇王石的特殊气息吸引，慢慢的就会散去。
<br/>　　这已经不关他的事，蛇不走，警察也会处理。
<br/>　　随后，涂小安带着海棠走出了这座山洞，环视一眼，这矮山静悄悄，明明藏匿着不少觉醒生物，却愣是没有发出一丁点动静。
<br/>　　恐怕是被之前自己的蛇类气息吓得不轻。
<br/>　　“海棠，这古学院有点意思吧”
<br/>　　涂小安在月下行走，随意的开口。
<br/>　　海棠如今走的有点别扭，还在适应这具人形态，不过出尘的容貌，曼妙的身段，让她不管怎么走，都别有一番风味。
<br/>　　“这古学院藏着不少人类的修武者，以前我都不太敢来”
<br/>　　作为一只猫精，虽然实力不凡，但是海棠也不敢招摇过市，毕竟这是属于人类的世界。
<br/>　　任何觉醒生物，都要夹着尾巴，太高调，就会引来一群“正义之师”降妖除魔，说难听点，抓你回去炼丹。
<br/>　　说起觉醒修武者，涂小安忽然想起之前山洞动静那么大，要是有实力强大的修武者，怎么可能没有出现呀。
<br/>　　难不成都没有发现？
<br/>　　走着走着，涂小安又来到了之前的小湖泊面前，一阵凄凉的笛声忽远忽近的游荡，就跟黑暗中的幽灵，漫无目的。
<br/>　　涂小安脚步停止，看着湖泊。
<br/>　　“恩公，你是不是好奇这笛声”海棠开口。
<br/>　　涂小安看着她：“你难道不好奇吗，笛声可是在湖底传出来的，据说这湖底好像是镇压这一头三阶生灵，或许是什么古阵法引起了，不知道属于那种”
<br/>　　“是第一种”海棠笃定。
<br/>　　涂小安连忙道：“你怎么知道”
<br/>　　海棠回忆了下道：“早在二十年前，我就偷偷的溜进了这栋古学院，想找一些人类提升实力的药剂服用，当时我听到笛声也好奇，就潜入了湖底下”
<br/>　　话说着，海棠的脸颊上忽然出现了一种恐惧，涂小安忙问：“你看到了什么”
<br/>　　“我看到了一只似鱼非鱼，似蛇非蛇的水怪，好像在看护什么东西”
<br/>　　“后来我看到湖底下，有一座铁牢笼，笛声就是在里面传出来的，我本想看清楚，但是却被水怪发现，被它攻击”
<br/>　　海棠心有余悸道：“那水怪的实力很可怕，我不是对手，就连忙逃了上来，再也不敢下去了”
<br/>　　“那么玄幻？”
<br/>　　涂小安露出了好奇心，还真的想下去一探究竟。
<br/>　　已他的实力，就算下面镇压着什么三阶妖物，他都不怕，说不定将它吞了，自己又能等级疯狂提升。
<br/>　　当然，要先弄清楚。
<br/>　　犹豫了下，涂小安继续朝前走，就算要下去一探究竟，也不是现在。
<br/>　　随后，涂小安带着海棠翻出了这栋古学院的外墙，悄然离开。

<br/>　　两位老者不断的交流，窃窃私语。
