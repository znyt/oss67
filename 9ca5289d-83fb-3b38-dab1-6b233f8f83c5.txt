        好啊！——朱小溪。
<br/>
<br/>        叫上蓉蓉一起。——连星儿。
<br/>
<br/>        嗯。——朱小溪。
<br/>
<br/>        连星儿还真没想到，读个大学还能遇上高中同学。
<br/>
<br/>        中午的时候，连星儿抛弃了自己的三个室友，专门去了约定的地点见朱小溪和张蓉蓉。
<br/>
<br/>        校外的一家小餐馆里，连星儿赶到的时候，朱小溪和张蓉蓉已经点好了菜。
<br/>
<br/>        饭菜还未上桌时，朱小溪和张蓉蓉两个人见连星儿进来，彼此交换了个眼神。
<br/>
<br/>        “星儿，其实有件事情，我想请你帮我。”朱小溪娇羞不已地说道。
<br/>
<br/>        连星儿一脸莫名其妙地看着朱小溪：“是什么事情？”
<br/>
<br/>        朱小溪拿起一个黑色的背包，从里面拿出了一个包装精致，却只有巴掌这么大的小盒子，递给了连星儿。
<br/>
<br/>        “你可不可以帮我转交给那天泼了我一身冰激凌奶茶的男生？”朱小溪弱弱地问道。
<br/>
<br/>        “你指明司寒啊？”连星儿顿时难以置信地看着朱小溪。
<br/>
<br/>        朱小溪接着说道：“在操场军训的时候，我经常有看到他。”
<br/>
<br/>        “他打球好帅！”张蓉蓉补充道。
<br/>
<br/>        连星儿却忍俊不禁起来：“不会吧！你俩喜欢他啊？”
<br/>
<br/>        “呸呸呸，不是我两，是小溪。小溪对那个泼了她一身冰激凌奶茶的学长念念不忘。没想到，在大学里又能遇上。真是缘分。”张蓉蓉笑道。
<br/>
<br/>        连星儿看向张蓉蓉，问道：“你为什么不替小溪去送？”
<br/>
<br/>        “我胆子小啊！”张蓉蓉顿时笑容渐敛。
<br/>
<br/>        朱小溪又补充道：“我、我怕被拒绝。”
<br/>
<br/>        “难道我去送就不会拒绝吗？”连星儿反问。
<br/>
<br/>        朱小溪和张蓉蓉不约而同地点头。
<br/>
<br/>        “星儿，你就帮我一次啊！看在我们是朋友的份上。”朱小溪苦着脸哀求。
<br/>
<br/>        张蓉蓉也跟着帮腔：“对啊！星儿！你胆子比较大！就算被拒绝了，你也不缺男生追求。”
<br/>
<br/>        “……”连星儿有些语塞。
<br/>
<br/>        在朱小溪和张蓉蓉软磨硬泡下，连星儿终于答应了。
<br/>
<br/>        不过，不巧的是，今天操场上，连星儿并未见着明司寒的身影，这朱小溪的礼物自然也就没有送出去。
<br/>
<br/>        而朱小溪则时不时地给她发简讯，询问她结果。
<br/>
<br/>        连星儿只能回复说没见着人。
<br/>
<br/>        其实，不用她说，朱小溪应该也知道吧！
<br/>
<br/>        今天一整天，平日里会在操场上打球的明司寒，居然“失踪了”。
<br/>
<br/>        第二天也不见人影，第三天也是。
<br/>
<br/>        直到一个星期后……
<br/>
<br/>        军训下操后，连星儿挽着顾千凝的臂弯一起去了球场。
<br/>
<br/>        明司寒和几个男生正在打篮球，当连星儿和顾千凝走过来时，有男生在一旁吹口哨，因此而吸引了明司寒的注意。
<br/>
<br/>        看到连星儿，明司寒先是一怔，随即又装作若无其事的样子，继续打球。
<br/>
<br/>        他轻轻松松地进了一个三分球后，顾千凝忍不住拍手叫好。
<br/>
<br/>        连星儿“咳”了一声，喊道：“明司寒，你过来，我有东西要给你。”
<br/>
<br/>        她这话一落，一旁的男男女女，全部都将目光落在了她的身上。
<br/>
<br/>        明司寒将手中的球扔给了自己的室友，抹了把额头上的汗水，朝连星儿走了过去。
<br/>
<br/>        就在离他们不远处的香樟树下，朱小溪和张蓉蓉则远远地看着他们。
<br/>
<br/>        明司寒走过来后，顾千凝激动不已地摇着连星儿的手臂。
<br/>
<br/>        “星儿，学长真的过来了诶！”
<br/>
<br/>        “我知道。”连星儿拉住顾千凝的手，示意她冷静一下。
<br/>
<br/>        明司寒走到她们俩的跟前停下了脚步，嘴角噙着一抹意味不明的笑意，微微歪头，俯视着面前的这个丫头，问道：“找我什么事？”
<br/>
<br/>        “喏！我朋友，朱小溪，就是那个被你泼了一身冰激凌奶茶的女生。她要我转交给你的。”连星儿直接开门见山道。
<br/>
<br/>        明司寒垂眸看了一眼连星儿递过来的小礼盒，咧了咧嘴，道：“你喜欢我就直说，何必借你朋友的名义。”
<br/>
<br/>        “？！”连星儿一脸错愕。
<br/>
<br/>        顾千凝很认可地点点头。
<br/>
<br/>        “喜欢你个头啊！”连星儿连忙将手中的礼盒往明司寒的手中塞，“拿着吧你！别自恋了！”
<br/>
<br/>        她这话刚落，一旁又有人吹起了口哨。
<br/>
<br/>        顾千凝见大家都用异样的目光打量着她们俩，连忙悄悄地扯了扯连星儿的衣角，凑到她耳边，小声说道：“星儿，我们还是走吧！感觉要引起公愤了！”
<br/>
<br/>        明司寒正饶有兴趣地打量着这小礼盒时，连星儿二话不说，拉着顾千凝就这么转身走掉了。
<br/>
<br/>        回寝室后的第一件事情，其他三个室友都去冲凉澡了，明司寒却拉了条凳子，坐在书桌前，淡定地拆着小礼盒。
