        此时的傅夜沉，也早已经换了身干净的衣服，他跟她一样穿着一件白色的衬衫，就像情侣装一样，正站在开放式厨房的案台前，有些笨拙地切着老姜。
<br/>
<br/>        “大哥，还是我来吧！”童筱颖踱步上前，同时也夺走了傅夜沉手中的菜刀。
<br/>
<br/>        傅夜沉识趣地让到一旁站着，然后看着童筱颖干脆利索地将那块不规则的老姜，切成了薄薄的姜片。
<br/>
<br/>        “有核桃、黑芝麻、腰果、枸杞和红糖吗？”童筱颖接着问道。
<br/>
<br/>        傅夜沉点点头，然后朝冰箱走去，打开冰箱将童筱颖要的食材全部拿了出来。
<br/>
<br/>        童筱颖动作娴熟，非常麻利地处理了这些食材。
<br/>
<br/>        她将干果切成碎粒后，将这些食材往刀身上一抹，放到平底锅上干炒。
<br/>
<br/>        待一旁的奶锅中红糖水已经煮沸了后，她又下入了炒好的干果碎和姜片。
<br/>
<br/>        两碗核桃姜糖茶出锅后，童筱颖端起其中一碗，坐到了餐桌边，自顾自地喝了起来。
<br/>
<br/>        她只是在告诉别人，就算小卫抛弃了她，她也能好好地照顾自己。
<br/>
<br/>        “我的未婚妻悔婚了。”傅夜沉端起案台上那碗核桃姜糖茶，走到餐桌旁，一手拉开了椅子，在童筱颖的对面淡定地坐下。
<br/>
<br/>        童筱颖手肘抵在餐桌上，两手捧着碗，一边轻轻地吹着，一边小口呡着。
<br/>
<br/>        傅夜沉的话并未引起童筱颖任何情绪上的波动。
<br/>
<br/>        她依旧，从容自若地喝着她的核桃姜糖茶。
<br/>
<br/>        “你得跟我结婚。”傅夜沉凝视着童筱颖，深情中又带着一种与生俱来的霸气。
<br/>
<br/>        他严谨而郑重的用辞逗得童筱颖“噗哧”一笑。
<br/>
<br/>        “大哥，你发烧呢？”
<br/>
<br/>        “那天晚上，在云城酒店，我未婚妻撞见，你把我给上了。所以，你得对我负责。”傅夜沉一脸严肃，慢条斯理地说道。
<br/>
<br/>        童筱颖听着他这话，差点将嘴里的红糖水给喷了出来，还好，她及时捂住了嘴巴。
<br/>
<br/>        她把他给上了？！
<br/>
<br/>        那天晚上明明是他差点上了她，就因为他醉酒，在她身上寻寻觅觅找不着入口，才被她有机可乘，一记拳头打中穴位给打晕了过去。
<br/>
<br/>        总而言之，提起那晚的事情，他若是不觉得尴尬，她都觉得尴尬死了。
<br/>
<br/>        “咳咳——大哥，你放心。我可以去跟你未婚妻解释的！那天晚上，我和你什么事也没有。更何况，我是你弟妹……”
<br/>
<br/>        “你和阿景，已经离婚了吧？而且，我的未婚妻也已经悔婚了。现在，你单身，我也单身，要不凑一对，去结个婚。”傅夜沉毫不识趣地打断了童筱颖的话。
<br/>
<br/>        自他昨天从李忆莎那儿得知，那晚在云城酒店里跟他暧昧的人是她童筱颖以后，他就派人帮他调查了她和阿景最近的动向，才得知她和阿景回云城是为了领离婚证。
<br/>
<br/>        如果不是李忆莎跟他坦白，他至今都以为那晚醉酒后所发生的事情，只是他的黄粱一梦。
<br/>
<br/>        毕竟，在梦里跟筱颖做那种亲密的事情，他也不止一次。
<br/>
<br/>        这么一想来，他突然发觉，他在没有她的日子，自己过得很颓废。
<br/>
<br/>        童筱颖表情僵硬，眨巴着眼睛，心情极其复杂地看着傅夜沉。
<br/>
<br/>        “你、你还有苏漫雪啊！”她愣了一会儿，才回过神来，干笑道。
<br/>
<br/>        “她现在是大明星了。”
<br/>
<br/>        “这有冲突吗？”
<br/>
<br/>        “女明星一般不会急着结婚。”
<br/>
<br/>        “所以，你又被抛弃呢？”童筱颖有些难以置信，但静下心来想了想后，又觉得这还真像是苏漫雪的作风。
<br/>
<br/>        对于苏漫雪来说，与其当豪门阔太，还不如去当光鲜艳丽的大明星。
<br/>
<br/>        傅夜沉很淡定地回答：“是！”
<br/>
<br/>        其实，是他心里除了她童筱颖以外，压根就装不下其他女人。
<br/>
<br/>        而现在，他不想跟她有过多的解释，只想跟她来点直接的。
<br/>
<br/>        他爱她爱得太久，现在只想替自己要一个结果。
<br/>
<br/>        哪怕她不爱他，他也无所谓，只要她的名字能出现在他户口本上妻子那一栏便足矣。
<br/>
<br/>        顿时，童筱颖的思绪有些混乱了。
<br/>
<br/>        据她所知，跟他傅夜沉有关系的女人，一个苏漫雪、一个季思妍，还有一个就是那天晚上在云城酒店里撞见的那个女人吧！
<br/>
<br/>        苏漫雪现在是大明星不结婚也很正常，而季思妍已经为傅昼景生了孩子，在酒店里撞见的那个女人又跟他悔婚。
<br/>
<br/>        童筱颖忽然觉得，大哥傅夜沉的感情史怎么比她还悲惨。
<br/>
<br/>        不过，他俩算不算是同命相怜？
<br/>
<br/>        都是被另一半给抛弃了一次又一次……
<br/>
<br/>        “大哥。”良久后，童筱颖忽然意味深长地唤了声。
