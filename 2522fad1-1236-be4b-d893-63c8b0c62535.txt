　　数月之后，苏莫顺利的回到了龙都城。
<br/>　　由于龙都城附近，有很多监视之人，苏莫在进入天龙神朝之后不久，特意给龙腾传讯，然后大长老将他接回了龙都城。
<br/>　　苍穹祖龙府，辽阔的大院之中，苏莫和龙腾、池儿三人均在，司空炎则在闭关之中。
<br/>　　“池儿，这里殿宇众多，你可以随意找一座居住！”苏莫对池儿说道。
<br/>　　“嗯，多谢师尊！”池儿向苏莫躬身，大眼睛打量着四周，显得格外的兴奋，他刚刚被苏莫从空间宝器中放出来，但心中已经猜到，看来这里就是龙族的龙都城了。
<br/>　　“师尊？”龙腾闻言一怔，随即打量了池儿一番，诧异的向苏莫问道：“苏莫，你居然收徒了？”
<br/>　　龙腾有些无语，这都什么时候了，苏莫居然还有心情收徒弟，这心可是真大啊。
<br/>　　“呃！看她以后的表现吧！”苏莫略显尴尬的说道。
<br/>　　“我叫梅歆池，见过这位前辈。”池儿立刻向龙腾行礼，并笑着介绍自己的名讳。
<br/>　　“不错！”
<br/>　　龙腾微微点头，笑着道：“不管你天赋如何，只要拜了苏莫为师，以后就是天骄中的天骄。”
<br/>　　龙腾对眼前的小丫头，也是颇为喜爱，此女外表乖巧可爱，一看就是心思单纯之辈。
<br/>　　“天骄中的天骄？”池儿闻言一怔，有些不明所以，为何拜了苏莫为师，就能成为天骄中的天骄？
<br/>　　龙腾微微一笑，也没过多解释，看向苏莫，道：“苏莫，我还要闭关，就先走了，你也不要耽搁时间。”
<br/>　　他的修为，还是四品高级，和苏莫相同，而距离千选战还有四年出头的时间了，他必须在千选战之前，达到五品的修为。
<br/>　　同时，他也看出来苏莫达到了四品高级，所以叮嘱苏莫不要耽搁了时间。
<br/>　　“嗯！”苏莫点了点头，目视着龙腾离去之后，对池儿道：“我现在也要闭关，没有时间指导你，你自便吧！”
<br/>　　言罢，苏莫便转身向一座宫殿走去。
<br/>　　“这……师尊，你就忍心丢下我一个人？”池儿见此，顿时傻眼了，樱桃小嘴嘟起，一副受了委屈的模样。
<br/>　　“那宫殿里有个人，是你师叔，你无事可以去找他。”苏莫头也不会，一指旁边的一座宫殿，那里是司空炎居住和修行的地方。
<br/>　　苍穹祖龙府中，宫殿不少，他们均是分开居住。
<br/>　　“好吧！”池儿闻言，无奈的点了点头。
<br/>　　苏莫回到宫殿之后，便进入了密室之中，再次开始了闭关修炼，淬炼混沌源力，参悟吞噬法则。
<br/>　　时间一天天的过去，没有人来打扰苏莫，他完全沉浸在了修炼之中。
<br/>　　眨眼间，四年多的岁月过去了，密室之中，苏莫睁开了眼眸。
<br/>　　“时间差不多了！”苏莫心中暗叹一声，距离千选战的日子，已经不远了，而他的修为凝练的时间，还是有些不足。
<br/>　　好在，吞噬法则的参悟，也达到了四阶后期。
<br/>　　“管不了这么多了，必须突破了！”苏莫深深的吸了口气，即便根基不够扎实，即便要浪费无数的资源，他也要突破修为。
<br/>　　哗啦啦！
<br/>　　苏莫大袖一挥，身前光芒闪耀，无数的五行源石出现在身前。
<br/>　　嗡~~
<br/>　　苏莫直接释放出了吞噬元神，开始大规模的吞噬源石，滚滚源气入体，苏莫极力炼化，并开始冲击五品的修为。
<br/>　　轰隆隆！
<br/>　　强大的源气，如同江河咆哮，在苏莫体内浩荡，发出震耳欲聋的声音。
<br/>　　虽然苏莫的修为，已达四品高级的巅峰，但是，境界瓶颈稳如泰山，没有一丝要突破修为的征兆。
<br/>　　但是，苏莫不管不问，拼命的吸收源气，冲击五品境界。
<br/>　　时间缓缓的流逝，一堆有一堆的源石消耗一空，苏莫的修炼，仿佛没有止境，不停的吞噬吸收，不停的冲击境界。
<br/>　　只是短短半个月的时间，他消耗的源石，便已经是天文数字，这些源石的数量，足以让一个普通的虚神境的武者，修炼到真神七八品的地步。
<br/>　　但是，经过半个多月的修炼，苏莫的境界瓶颈，只是有细微的松动，距离突破到五品的境界，差距甚远。
<br/>　　咚咚咚！！
<br/>　　这一日，密室的大门，终于被敲响了。
<br/>　　苏莫睁开了眼眸，暂时结束了修炼，他的眸中闪过一丝无奈之色，看来千选战的日子要到了，而他依旧没有突破。
<br/>　　打开密室的石门，石门之外，正是龙腾。
<br/>　　“苏莫，你……还没有突破？”龙腾打量了苏莫一眼，见苏莫还是四品高级的修为，不由得眉头一皱。
<br/>　　“看来，你比我快了一步。”苏莫叹了口气道，他没有达到五品，龙腾却先他一步达到了，这算是一个好消息。
<br/>　　“我在五日前才突破，主要是祖龙精血的力量还没有消耗殆尽，我又服用了不少龙族珍惜宝物！”龙腾道。
<br/>　　“你达到了五品，算是多了一份保障！”苏莫沉声道，龙腾的天赋和实力，不比他差多少了，有龙腾参加，他也安心不少。
<br/>　　“嗯！我到时候一定尽力！”
<br/>　　龙腾点了点头，继续道：“半个月之后，出发前去混沌神朝，距离千选战的日子，还有三十日，你能否突破？”
<br/>　　“不敢保证！”苏莫深深的吸了口气，摇头说道。
<br/>　　“过两日，族中就要选择参加千选战的十个名额了，我给你留一个！”龙腾道。
<br/>　　“可我修为不一定能突破！”苏莫阴沉着脸，他的修为瓶颈如同一座大山，想要短时间能冲去，几乎没有可能，吞服天才地宝都无用。
<br/>　　“突破不了也给你留着！”龙腾郑重的说道，以他在龙族的地位，这点能力还是有的。
<br/>　　换言之，苏莫就是一个备选之人，若是能在千选战到来之时突破，那就参加，突破不了就临时换人。
<br/>　　“好！”苏莫点了点头，随即再次关闭了密室，继续冲击境界。
<br/>　　还有半个月的时间才出发，距离千选战还有一个月，他还有机会，不能放过。
<br/>　　这一次，苏莫不再吞噬源石，而是吞噬丹药，丹药的药力更加的精纯一些，效力自然也更好。
