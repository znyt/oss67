　　不周之巅，四位妖族大神隐隐将三清围住，气氛顿时变得剑拔弩张。
<br/>　　元始看着眼前四妖，突然觉得自己是不是太过仁慈了，要不然什么人都敢来阻拦他。于是，他决定狠一点，不管是对自己，还是对敌人。
<br/>　　“吾元始多年未曾出手，看来许多人都不知道本座的威名，什么阿猫阿狗都想来欺负一下。两只乌鸦交给本座，至于另外两个，你们自己分配。”
<br/>　　两大妖皇帝俊与太一修为都在大罗金仙顶峰，半只脚已经踏入了大罗圆满，伏羲与他们差不多。至于女娲，资质就比其他三人要好一些，经过紫霄宫听道后，修为已经是大罗金仙圆满了，只差没有斩尸，不然已经可以跨入准圣境界了。
<br/>　　但，女娲修炼的是造化大道与至阴之道，并不擅长战斗，亦没有强大的灵宝，故而，四人中就属女娲的实力最弱。接着便是伏羲，他虽然没有大罗圆满，但他有他的伴生灵宝——伏羲琴。此琴威力巨大，可以控制人的心灵。但是相比于战斗，他还是更擅长推衍一道与阵道。
<br/>　　四妖之中，就属帝俊与太一实力最强，不仅有先天至宝，而且修炼的大道也极其擅长战斗，经过的战斗也有许多，不是一般人可比的。
<br/>　　元始一人对付四妖之中最强的两妖，不得不说，他对自己的实力极为自信，甚至自信到藐视对方的地步，就好像是对方在他眼里只是小狗小猫一样。
<br/>　　其实，对于肉身与元神双双突破混元的元始来说，他们的确就像是小狗小猫。尽管他们有先天至宝在手，能够以大罗金仙巅峰的境界发挥出准圣初期极限的实力，但，先天至宝也不是那么好炼化的。要对付元始，他们依旧不够。
<br/>　　可眼前四妖并不知道元始的想法，在他们看来，元始这是极度的自负，嚣张，完全不把他们放在眼里。这样的行为，让他们觉得自己被对方深深地看低了，于是，他们觉得，今天一定要将对方狠狠地教训一番。
<br/>　　“可恶！”
<br/>　　太一心中早已怒火难抑，当下，拖着一口古老大钟，对着元始，就是奋力一击。
<br/>　　“当——”
<br/>　　但闻一声悠扬的钟声响起，刹那之间，仿佛自九天之上传来，笼罩下方大地，周边空间，像是突然之间凝固起来一般，竟然让人诡异的静止在半空之中。
<br/>　　“哼！”
<br/>　　元始的一身修为可不是吃素的，他身体稍微一动，便挣脱了那钟声的禁锢，而且随着一声冷哼，禁锢空间的庞然伟力，竟然宛如泡沫一般，瞬间破碎，消失的无影无踪，就好像方才那一幕，好似幻觉。
<br/>　　“咚——”
<br/>　　与此同时，悠扬的钟声再次响起，空气宛若被震动的湖水一般，突然之间，泛起阵阵肉眼可见的紫色涟漪，绚丽柔和，仿佛是神灵的恩赐，圣洁的光辉，笼罩大地。
<br/>　　然而，在别人眼中看似柔和的光芒，可在元始眼中，却不是那般柔和，紫色的波纹宛如一道道翻天覆地的巨浪，横空向着元始覆盖而来。
<br/>　　“哼！”
<br/>　　元始冷喝一声，使出法天象地，无上法身瞬间便涨到数十万丈之高，法身周身洁白如玉，头戴圣冠，身披玉袍，脚踏大地，神威震天，威压苍生，就像一尊从混沌之中苏醒过来的魔神一般。
<br/>　　看着那向自己涌来的巨大紫色浪涛，无上法身慢慢伸出左手，对着那浪涛轻轻一掌，便将之完全破碎。
<br/>　　“元始，给我死来！”
<br/>　　与此同时，帝俊手持妖异血红的金色王剑，浑身散发着耀眼的金色光芒，还有那令人臣服的皇者霸气，向着元始劈来。
<br/>　　元始看都不看他一眼，抬起右手，对着向他砍来的帝俊遥遥一指。
<br/>　　这一指并不算快，相反，他看上去十分缓慢，但却给人一种深陷其中的感觉，帝俊对上这一指，他只感觉自己头皮发麻，他想要躲开，但那一指就好像他的影子，不管他躲到哪里，都会出现，给他一种无处可逃的感觉。
<br/>　　“当——”
<br/>　　当断不断，必受其乱。帝俊不愧为妖皇，既然躲不开，他就奋力一击。帝俊手持金色王剑，遥指苍穹，一身实力灌注于剑身之上，霸道的剑力以一种势不可挡的气势斩向元始的手指。
<br/>　　但结果很残酷，帝俊毫无阻挡地被元始一指弹飞，他的奋力一击，给不了元始丝毫的伤害。
<br/>　　“混蛋，这怎么可能？！”
<br/>　　帝俊不可置信，他想不到元始竟会有如此实力，自己的奋力一击，就是一般的准圣初期，他都有把握对付。但，到了元始这里，竟然不能给对方造成丁点伤害，这怎么可能？元始怎会有如此厉害？他想不通。
<br/>　　轰隆隆！
<br/>　　太一见此，极为焦急，他手持混沌大钟，奋力一击，同时，口吐太阳真火，幻化为数十三足金乌，向着元始飞去。
<br/>　　金色的太阳真火带着煌煌之威，焚天裂地，三足金乌一声鸣叫，无数太阳真火向着元始攻去。周边空间，瞬间变得灼热无比。
<br/>　　“吸——”
<br/>　　元始看着向自己飞来的三足金乌，冷笑一声，太阳真火，这是自己早已经吃剩下的了，如今还想给自己造成威胁，这可能吗？
<br/>　　只见他对着那数十只三足金乌一吸。顿时，无数太阳真火瞬间便被他吸到口中，吞入腹中，消失不见，只留下鼻孔中发出的丝丝火苗，而且瞬间之后，就连火星都不剩了。
<br/>　　“味道太差，不好吃。”
<br/>　　元始微微一笑，淡淡点评道。
<br/>　　“这怎么可能？太阳真火都敢吃？！”
<br/>　　太一大吃一惊，这人竟然不怕太阳真火？要知道太阳真火为十大真火之一，而且还是排名靠前三的真火，代表着火之一道的极致，为至阳一道的代表。洪荒之中，也只有太阳星有太阳真火，威力自然不用说。
<br/>　　太阳真火一直是他们两兄弟的招牌，洪荒之中，他还从没有听说过，除了他们两兄弟之外的其他人会太阳真火的。以往他们对敌，只要太阳真火一出，敌人瞬间变得分烧成灰烬。
<br/>　　可如今，元始竟然不怕太阳真火，更可怕的是，他还将真火吃下去，竟一点事儿都没有，这还怎么打？这不是欺负人吗？
<br/>　　元始可不管帝俊与太一两人的心情，面对两人的围攻，元始一直不曾使用出大招，他只是随意一击，便能将两人的攻击化为乌有。但他并没有更进一步，一点也不着急，既然他们想玩，元始自然是奉陪到底。
<br/>　　更何况，另外两处战场，也并没有结束战斗的意思。
