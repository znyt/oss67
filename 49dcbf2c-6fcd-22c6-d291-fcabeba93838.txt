　　第403章
<br/>　　像一点火星溅入油锅，随即火冒三丈，然后铺天盖地熊熊燃烧。
<br/>　　程大雷分明看见，宋游渠几人眼中的火焰都燃烧起来。年轻人，有一眼望不到边的人生，或许明天他们就会人头落地，但他们都愿意相信，他们还有几十年的无聊人生。
<br/>　　如何打发这漫长无聊的人生，金钱、权力、女人……这当然是值得追求的，可更有价值的东西则是信仰。
<br/>　　信仰是个很玄的东西，看不见摸不着。可只要心怀信仰，那么即便一文不值，睡在寒窑的穷小子也比腰缠万贯，枕金穿银的富豪富有。因为，心怀信仰，过去的每一秒都是真实的活着。
<br/>　　一只手、两只手……所有人的手叠在一起，众人重重的点头。
<br/>　　程大雷面露微笑，似乎也被这热情感召。当然，程大雷自己觉得，还是金钱、权力、女人这些更值得追求，至于信仰……
<br/>　　忽悠别人的话，别把自己也骗了。
<br/>　　……
<br/>　　凉州城。
<br/>　　在和珅正筹措谈判交换人质的事情是，城主府内一场谈判也正在进行。
<br/>　　李行哉和宋伯康相对而坐，互相都板着脸，没有任何笑容。
<br/>　　“他有没有被用刑？”
<br/>　　“这个自然。”宋伯康心平气和。
<br/>　　李行哉脸上肌肉绷紧：“人还活着？”
<br/>　　“这个也自然。”
<br/>　　李行哉的目光稍微松弛：“我要见到他。”
<br/>　　宋伯康摇摇头：“这不可能。”
<br/>　　李行哉目光内的火焰燃烧起来，是怒火，他恨不得将眼前这个男人焚为灰烬，却什么都做不了。
<br/>　　这时候，双方都卸去了那些虚伪的伪装，像福德勒期盼的那样，直来直往，干脆利落。
<br/>　　但当时福德勒没有想到，李行哉肯卸去伪装，以自己最真实的模样面对宋伯康，竟然是为了他。
<br/>　　良久，宋伯康在李行哉的逼视下摇摇头，道：“殿下，能否容我说句话。”
<br/>　　李行哉双目仍如刀子般寒冷，直直的盯着宋伯康。
<br/>　　宋伯康没有理睬李行哉的目光，缓缓言道：“人非草木，孰能无情，殿下的心情我并非不可理解。但殿下可知道，他在我府中杀了多少人？”
<br/>　　“十七个。”宋伯康直面李行哉，道：“这些人有男有女，又老又弱，有跟了我二十年的老仆，也有没长大的孩子。殿下是否想过，这些人也有妻儿父母，他们死了，也会有人难过。”
<br/>　　“请问殿下，他们都可以死，为何你的手下不能死？”
<br/>　　李行哉一怔，一时竟有些无言以对。
<br/>　　宋伯康的呼吸放缓，整个人又平静下来。
<br/>　　“殿下，我们不算陌生，在长安城我们便见过面。当时，殿下摆开酒宴，我还去喝过殿下的酒。”
<br/>　　“喔，还有这样的事，我倒是不记得了。”
<br/>　　“殿下素有风流之名，放浪形骸，不拘小节，我这种俗人，殿下不记得也正常。”宋伯康道：“我想说的是，殿下生来富贵，注定高人一等。本不用在意其他人的悲喜，本不用在意那些无关人等的生命。”
<br/>　　李行哉心中一惊，一向他自以为自己还算个仁慈的。但在出事之后，他当真是觉得福德勒的安危毕竟重要，至于那些被福德勒杀掉的人……废话，我连他们名字是谁都不知道，和我又有什么关系。
<br/>　　人本身，原来和自己认为的不同。
<br/>　　“你想说什么？”
<br/>　　“长安城破，必定天下大乱，国不可一日无君，我们需要秩序，愿意奉殿下为君。日后殿下可以继续不在意其他人的悲喜，可以仍旧不在乎别人的性命。但想要做到这些，殿下日后不可以再继续幼稚下去了。”
<br/>　　“幼稚？”李行哉发出一声冷笑。
<br/>　　“一意孤行，优柔寡断，分不清形势。这难道还不是幼稚么？殿下，既然有帝王之志，就该有帝王之才。想要成为一国之君，靠风流倜傥，放浪形骸是不行的。”宋伯康平静道。
<br/>　　李行哉再也笑不出声来，眼睛直勾勾的盯着宋伯康。
<br/>　　“殿下如今只能与我们合作，如今也只有我们能帮助殿下。”
<br/>　　李行哉沉默下来，低头沉思着。
<br/>　　宋伯康站起身，道：“殿下该好好考虑一下，我，并非是殿下的敌人。”
<br/>　　空旷的大殿只剩下李行哉一人，他抬头盯着房顶，各种念头冒了出来。
<br/>　　宋伯康的一些话，的确被他听进了心里。自己现在当真一无所有，空顶着一个帝国皇子的名头而已，现在的他，急需要一股自己的人马。
<br/>　　君子趁势而起，于混乱中汲取力量，将想要的牢牢握在手中，这，才是真正的帝王之道。
<br/>　　李行哉忽然苦笑一声，读万卷书，行万里路，看来自己要学的东西还有很多。
<br/>　　宋伯康踱步出了院子，外面早有人在等待。见到宋伯康进来，陆洛向前踱步道：“大人，琴川关来人了？”
<br/>　　“喔，他们来做什么。”
<br/>　　“想要拿公子与我们交换。”
<br/>　　“哼，想也不要想，我怎能将殿下交给他们。”
<br/>　　“不，不是换殿下，是换那个戎族。”
<br/>　　宋伯康怔了怔：“他们真的愿意和我们交换？”
<br/>　　“人现在已经到了，看来诚意很足。”
<br/>　　宋伯康低头寻思着，脑海中翻来覆去。福德勒在自己手中，自然有好处，可以用来限制李行哉。但宋游渠毕竟是自己的儿子，况且，城内很多大人物的子弟也被困在琴川关。就算自己不同意，他们也会想把人换回来。
<br/>　　况且福德勒在自己手中，作用并不是很大。
<br/>　　眼下还有一个关键问题，宋伯康突然问道：“人还活着么？”
<br/>　　福德勒被擒住后，就被丢进大牢中，手下人出手也没个轻重，现在人是死是活还不清楚。
<br/>　　“还活着。”陆洛点点头：“大人，琴川关的人就在府内，你要不要见见他们？”
<br/>　　宋伯康摇摇头：“不必了，还是让钟伟虎去和他们谈，如果他们真有诚意，就安排交换人质的事情。”
<br/>　　“嗯，我现在就去安排。”陆洛点头答应。
<br/>　　“对了，在谈判期间，千万别把人给弄死了。”宋伯康不放心，又补充了一句。
<br/>
