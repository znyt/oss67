　　第181章
<br/>　　“大当家，他们好像退兵了。”
<br/>　　城头之上，徐神机对程大雷道。
<br/>　　程大雷目光看过去，只见戎族正在拆除帐篷，这的确是拔营的迹象。
<br/>　　“大当家，不若我们冲出去，趁他们没有准备，好好杀他一回。”徐神机。
<br/>　　程大雷摇摇头，他否定的原因倒不仅因为这建议是徐神机提出的。立在城头，目光望着牛角峰上的杨树，杨树不动，程大雷不动。
<br/>　　眼睁睁看着戎族拔营起寨，在视线中消失得无影无踪。
<br/>　　“大当家，下令追吧，再不追他们就逃远了。”徐神机。
<br/>　　“莫急，他们是想引我们出去。”
<br/>　　“诱敌？”
<br/>　　程大雷点点头，视线看着牛角峰上的杨树。
<br/>　　杨树的树梢偏移，像一杆标枪，遥遥指着戎族行军的位置。和程大雷预料的不错，对方的退兵，果然是诱兵之计，一离开视线范围，行军的速度便减慢了。
<br/>　　“下去，把城门打开？”
<br/>　　“大当家要出兵。”
<br/>　　“出什么兵……”程大雷啐了一口：“等他们回来，快，多备弩箭、重石，准备好好招待他们。”
<br/>　　大军向前近，赤眉却随时盯着身后的动静。
<br/>　　“他们追出来没有？”
<br/>　　“还没有。”
<br/>　　“再探！”
<br/>　　赤眉心里打鼓，对方怎么还不追出来，以帝国的好大喜功，见己方退兵，该出城追击溃兵才对。而一旦他们杀出来，野战的领域是属于戎族的。
<br/>　　可他们这次怎么这么沉得住气。
<br/>　　“王子，会不会是他们识破了？”托蛮。
<br/>　　“怎么可能，他们又没有千里眼，怎么识得我们是真退还是假退。”
<br/>　　“那他们怎么还不出来……”
<br/>　　“再等等，即便不出来，等他们麻痹大意，我们干脆杀他们一个回马枪。”
<br/>　　带着大军在山中转圈，最后赤眉寻了一空旷之地扎营，而从始至终，他们现在还没离开青牛山范围。
<br/>　　赤眉一直派人盯着蛤蟆寨的一举一动，目前看来他们已被吓破了胆，就算退兵他们也不敢出来。当然，这也符合帝国人的性格。
<br/>　　现在，赤眉等的不是他们出兵，而是等着他们松懈。
<br/>　　“报！他们似乎在庆祝！”
<br/>　　“报！他们城门上的守兵降到一百人！”
<br/>　　“报！守兵降到五十人了。”
<br/>　　“报！城头只有十几个守军，城门打开，他们在收拾城门外的兵器。”
<br/>　　收到最后一条情报时，赤眉正在啃肉干，等情报过来，他腾地一声站起来。
<br/>　　“出发！与我攻城，杀光他们！”
<br/>　　戎族拥有极强的行动力，这也是帝国难以相比的地方之一。从大军集结到抵达蛤蟆寨城门外，过程没有超过半个时辰。
<br/>　　一切和所得到的情报不错，城门洞开，守军不过十几个，见大军杀过来，他们睁大眼睛，脸上露出惶恐表情。
<br/>　　想不到吧！老子根本没走，老子又杀过来了。
<br/>　　这颗讨厌的钉子，今天终于可以拔掉了。
<br/>　　赤眉手下大将托蛮一马当先，率先冲杀过去，口中狂吼咆哮着。
<br/>　　“杀呀，杀！”
<br/>　　托蛮带着十几骑闯进了城门。
<br/>　　赤眉心中一喜，此城破哉！接下来就是自己熟悉擅长的事情：屠城。
<br/>　　“草原上的勇士们，随我杀呀！”赤眉挥刀大吼。
<br/>　　突然间，有了变故。
<br/>　　敞开的城门突然关上，将托蛮和大部队隔开。城门之上本来只有十几个守军，可在一瞬间人头从城垛下冒了出来，城头上人群黑压压一片。
<br/>　　迎接他们的是弩箭、重石、火药，煮沸的粪便。
<br/>　　不好，中计！
<br/>　　赤眉喊出了这句历久不衰的经典台词，他回忆守军刚才的表情，恍然意识到，他们张大嘴巴，表达的不是惶恐的『你们怎么来了』，而是充满热情与欣喜的『你们可来了』。
<br/>　　而他们早已准备好用什么东西招待贵客。
<br/>　　“退！退！”
<br/>　　赤眉挥刀大吼，可还来得及退么。
<br/>　　身边尸体倒下无数，人仰马翻着砸在地上发出惨叫，马踏破肚皮，血从腔子里涌出，肢体和身躯分离。
<br/>　　城内。
<br/>　　托蛮刚冲入城，便被长矛绊倒马蹄，紧跟着一群人一拥而上，刀剑锋利。
<br/>　　“大当家，要不要留个活口？”徐神机。
<br/>　　“留什么活口。”程大雷用绣帕捂嘴，挥了挥手道：“用弓箭，杀。”
<br/>　　看着程大雷的样子，徐神机忽然意识到，程大雷的重点不在『杀』，而是手中的绣帕。
<br/>　　“大当家，你这手帕绣得很精致啊？”
<br/>　　“有么……”程大雷假装不在意道：“苏樱绣的，我不爱用这些。”
<br/>　　城外，在付出极大的代价后，赤眉带着手下的残兵败将撤离城门范围。
<br/>　　即便是他，肩上也中了一箭，其他人的惨状也就可想而知。
<br/>　　他盯着城头，眼中露出极大的恨意，一张脸只剩下狰狞，而无平日的慵懒。
<br/>　　程大雷款款走上城头，绣帕抿嘴：“来者是客，我这顿饭招待的，你还满意么？”
<br/>　　“狗贼！”赤眉咬着牙道：“我非将你千刀万剐！”
<br/>　　“出口伤人，真是没文化的野蛮人。”程大雷撇撇嘴：“不过呢，我们是礼仪之邦，即便是对待野蛮的客人，也要保持礼仪和风度。这是与你送别礼物，带回去吧。”
<br/>　　嗖……
<br/>　　一物从程大雷手中丢过来，咕噜噜滚到城头下，这是托蛮的人头，他眼睛睁大，牙齿咬紧，即便已死，他还保持着狰狞模样。
<br/>　　“托蛮！”赤眉血红的双眼盯着程大雷：“狗贼，你……”
<br/>　　“住了吧！一个手下败将，有何资格在我面前狺狺狂吠？”程大雷一字一顿，如叩击赤眉的心门：“你已经败了。”
<br/>　　围城将近二十日，以一万铁骑打五千老弱病残，赤眉没有打下蛤蟆寨的一草一木，己方反而死伤惨重，如今所率部幸存者不过四千余人。
<br/>　　“跳梁者虽强必戳，犯我山河虽远必诛，尔等披毛戴角，茹毛饮血的畜类，也敢妄图国器，今日天道不来罚你，吾来罚你！”程大雷义正词严，最后摆摆手：“何况你还不够强。”
<br/>　　噗！
<br/>　　马上赤眉忽然吐血，身体晕倒。
<br/>　　咦，这算雄辩技生效了么。
<br/>
