　　“小弟”
<br/>　　此时，庄园内出现了一道轻柔的声音，白君然看到来人，浑身凌厉的气息顿时一消，和声道：“二姐，你怎么来了”
<br/>　　白家嫡传二女一子。
<br/>　　旁支更是开枝散叶，子弟众多，那位白尘，就是白家的旁支子弟。
<br/>　　“我听说你将勾魂笛弄丢了是吧”白楚艳向前而来，就一副兴师问罪的架势：“当初，我让你将勾魂笛送我，你这个小气鬼还不舍得，现在给外人，却给弄丢了”
<br/>　　白楚艳故作生气，冷艳的面容，到是别有一番风韵，要是此处还有别人，一定看的眼珠子都要掉下来。
<br/>　　可惜，她面对是她亲弟弟。
<br/>　　白君然闻言，亲切的钩住了白楚艳的肩膀：“我的亲姐呀，不是弟弟不舍得给你，而是勾魂笛需要灵魂之力来吹笛，你的灵魂力太弱，拥有勾魂笛对你没好处的”
<br/>　　“那我留着不吹，当首饰不行吗，现在一拍两散，笛子也没了”
<br/>　　“没不了”白君然自信的说：“周秦被人所杀，勾魂笛肯定也被人所夺，迟早会物归原主的”
<br/>　　白楚艳连问：“你知道是谁下的手吗”
<br/>　　“准确的说，应该不是人所为，姐，你跟我来，给你看个有意思的东西”
<br/>　　两姐弟离开庄园，来到了一间气派的大厅中，两人坐在沙发上，白君然从一旁拿出一台笔记本，按了几下键盘，骤然间，笔记本出现了一幅壮观的画面。
<br/>　　画面中，天空澄碧，纤云不染，一条巨大的金光蛇影仰天咆哮，威严不凡，底下一片黑压压的人头，激动的大喊：“蛇王万岁”
<br/>　　“这...”
<br/>　　白楚艳被这副画面看的呆若木鸡。
<br/>　　“不知道是不是我们白家离开白镇实在太久，白镇居然出现了一条蛇精，妖言惑众，蛊惑百姓”白君然此时脸上噙着一抹轻蔑的笑容。
<br/>　　白楚艳非常聪明，举一反三：“小弟，你莫非怀疑周秦是被这条蛇精给杀的”
<br/>　　“八九不离十吧，周秦虽然只是一个蚊虫的掌控者，但他手握我给的勾魂笛，笛子一吹，数以千万计记的蚊虫遮天蔽日，岂是那么好对付的”
<br/>　　“这蛇精能显化如此大的动静，想必起码二阶了，周秦死在这条蛇精手里，到也不冤”
<br/>　　白君然继续幽幽然的说：“多好的一条蛇精，我挺喜欢的，可惜我派人去查的时候，发现这条蛇精既然已经不在白镇了”
<br/>　　说着说着，他幽暗深邃的眸子精光四射，有杀意溢出。
<br/>　　好聪明的蛇精，恐怕知道自己会去找他的麻烦，先行离开。
<br/>　　“啪...”
<br/>　　白楚艳忽然一个爆栗敲过去：“别在我面前露出你这副要吃人的样子”
<br/>　　“姐，我错了”白君然顿时讪讪一笑，他们虽然生在大家族，但是看的出来姐弟的感情还是很深。
<br/>　　白家，白君然可谓一人之下，万人之上，地位很重，而且修为更是恐怖，不过年方二十，就要突破三阶生灵是何等的可怕。
<br/>　　如此一对比，白楚艳虽然也是人类觉醒者，但跟他这个弟弟，就是云泥之别了。
<br/>　　“哦对了，小弟”说起蛇，白楚艳总算想起了自己来的目的：“今日温城大学出了一件大事”
<br/>　　白君然却苦笑：“姐，你在提醒我，过几日要去上学了吗”
<br/>　　“这种事情需要我提醒吗，我跟你说正事”白楚艳翻了翻白眼：“今天校园内忽然群蛇出没，整整有上千多条蛇，将不少学子给吓坏了”
<br/>　　“群蛇出没”
<br/>　　白君然的眼睛猛然一亮，还真是想什么来什么。
<br/>　　“这些蛇出现在校园做了什么诡异的事情吗”
<br/>　　“那倒没有，这些蛇没有伤人之心，就莫名其妙的出现在了校园内”
<br/>　　白君然一下子笑了，笑的非常的开心，眼睛微微的眯了起来，呢喃自语：“猫精，原来你藏匿在温城大学呀”
<br/>　　“小弟，你说什么呀”
<br/>　　白君然很开心：“姐，你可真是帮了弟弟的大忙”
<br/>　　她疑惑，有点不明白。
<br/>　　“姐，你是不是觉得校园内出现群蛇，很奇怪是吧，不过弟弟知道为什么了”
<br/>　　“为什么呀”白楚艳连忙追问。
<br/>　　白君然回答：“弟弟前些日子寻觅了一块蛇王石，却被一头猫精给吃了”
<br/>　　“现在看来，极有可能这只猫精是藏匿在温城大学内？”
<br/>　　白楚艳疑惑：“蛇？猫？这两种有什么关系呀”
<br/>　　“姐，你有所不知，这蛇王石，据说是曾经一条修炼大成的蛇妖所留，这块石子无论在哪，都会吸引很多蛇类汇聚”
<br/>　　“温城校园突然出现那么多条蛇，还不够说明问题吗”
<br/>　　只有吞了蛇王石的猫精出没，才会吸引无数蛇的追随。
<br/>　　白君然笃定：“只要那只吞了蛇王石的猫精继续在温城大学，那么校园中的蛇只会越来越多，恐怕不久，整个学校都有可能成为蛇窟”
<br/>　　蛇王石的来历不凡，那是在一座蛇岛所得。
<br/>　　整座岛全是蛇，起码上百万条，这块蛇王石在一个深不见底的蛇窟中。
<br/>　　白君然为了得到这块蛇王石，可是付出了不笑的代价，觉醒修武者，都死了不少。
<br/>　　他猜测，猫精并不知道那是一块蛇王石，直当是什么天材地宝，所以才贪念一出的给吞了。
<br/>　　任猫精开了灵智在聪明，岂能想到以后自己所到之处，就是群蛇跟随，无处藏匿。
<br/>　　白楚艳听完之后，消化了一会儿，才总算明白弟弟说的意思，疑惑的说：“可校园现在的群蛇已经都离开了，没你说的那么恐怖，还蛇窟呢”
<br/>　　“离开了？”白君然诧异：“莫非猫精发现不对，从温城大学跑了？”
<br/>　　只有猫精离开，群蛇也才会离开。
<br/>　　白楚艳连忙解释：“那倒不是你说的原因，，群蛇的离开是因为一位觉醒修武者，他好像有控蛇的本事”
<br/>　　修武者？
<br/>　　白君然感觉事情越来越有意思了，到点扑朔迷离的味道。
<br/>　　“姐，你说的修武者是什么意思”
<br/>　　白楚艳开口：“小弟，我说的那么修武者不但能控蛇，而且实力还非常的厉害”
<br/>　　“莫非他是蛇类掌控者？”白君然疑惑了下，顿时摇了摇头：“天底下的觉醒能力不会重叠，我记得山野集团的蛇博士陈默便是一位蛇类掌控者”
