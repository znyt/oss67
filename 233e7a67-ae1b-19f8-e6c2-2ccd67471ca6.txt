<br/>				郝仁将蔬菜拉回港区后，凌经理就开始按照高层的交代，先将这些蔬菜用在了港区酒店内。<br/>　　<br/>　　本来，在港区酒店，由于针对的目标覆盖面小，所以酒店里的入住率并不是多高。在晚上的时候，来酒店吃饭的人也不是很多，还没有达到酒店餐厅容纳量的一半。<br/>　　<br/>　　整个酒店餐厅，本来是两层楼的占地，但是包括大厅和包间，客人也只是一少半而已。<br/>　　<br/>　　凌经理直接在酒店菜单后边补充了两页纸，上面是两个菜品。一个蒜蓉青菜，直接100元一盘。一个是韭菜炒鸡蛋，直接就是180元一盘。<br/>　　<br/>　　这个价格，也算是他深思熟虑的结果，虽然价格高，但是只要明码标价，另外服务员也解释一下的话，只要有人点了这个菜品，就会喜欢上，会感觉物超所值。这是凌经理的信心，也是对陈默产出的蔬菜品质的信心。<br/>　　<br/>　　常经理是市里一家大公司的高管，今天来港区酒店，主要是送一个客户，由于是第二天一早的飞机，所以就将酒店定在了港区。今天晚上主要是送风宴，也算是和客户交流感情。<br/>　　<br/>　　他和几个手下，还有几个客户一起在酒店餐厅包了一个大包间。拿过菜单的时候，倒是没有什么好奇的，和市里一些餐厅的东西大差不差，价格上也说不上差距，都还那样。也就随着点了几个，可是翻到最后的时候，却发现两种炒菜。<br/>　　<br/>　　炒菜也就炒菜吧，但是没有想到一个炒青菜，一个韭菜鸡蛋的，既然标价这么贵，倒是令他非常的好奇，虽然上面也写了精品蔬菜，但是也不至于这么贵吧。<br/>　　<br/>　　“服务员，你这两个菜真的是这个价格？”常经理询问道。<br/>　　<br/>　　“是的，先生！这是我店推出的最新菜品，采用有机蔬菜制作，此蔬菜所有检测高于有机蔬菜标准，没有任何农药和重金属残留，属于纯绿色食物。”服务员没有吃过这类蔬菜，而且由于是上面下的通知，也只能这样说了。<br/>　　<br/>　　“哦！真有这么好？”常经理问道。<br/>　　<br/>　　“是的，菜单背面有此类蔬菜的质检报告，你可以看一下。”服务员说道。<br/>　　<br/>　　常经理翻过去一看，是个专业报告，但是其中一些检测项目他还是看的懂得，在一些检测指标上还标注了国标含量，因此也是一目了然。<br/>　　<br/>　　“可以啊，今天我就吃吃这个蔬菜，看看到底有些什么好？”常经理说完，就直接点了这两道菜，反正不差钱，几百块钱而已。<br/>　　<br/>　　又对客户笑着说了下，然后就让服务员上菜。<br/>　　<br/>　　在包间其他地方，也发生类似的情况，也都是发现菜的价格非常的高，所以也就想着尝尝，看看味道究竟如何。<br/>　　<br/>　　不过，还没有等那两盘菜上来的时候，常经理就感觉今天晚上的菜，都比以前要好吃些，味道要香一些。<br/>　　<br/>　　难道餐厅换厨师了？叫过服务员一问，才知道没有换出事，那这是怎么回事？<br/>　　<br/>　　其实餐厅里什么都没有更换，但是在一些菜品烧制的时候，用了从陈默哪里购进的香葱，直接将菜品口味提高了一个等级。<br/>　　<br/>　　等两盘菜上来后，常经理他们才知道，青菜其实也很好吃的，韭菜炒鸡蛋也是非常的爽口。几个人，也没有夹几筷子，就被吃完了。好么，常经理直接又要了两盘，开吃，真的非常的酣畅。<br/>　　<br/>　　今天晚上的聚餐，最后可以说是宾主皆欢，都留下了很深的映像，客户也非常的高兴，非常感谢让他吃到这么好吃的饭菜。<br/>　　<br/>　　等常经理回到家的时候，已经是晚上十点了。看到自己媳妇和孩子等已经睡了，他也就洗漱一下后准备休息，为了一个项目，忙碌了十几天，今天将客户送走以后，总算是能休息一下了。<br/>　　<br/>　　等躺倒床榻上以后，才感觉自己小腹有股火热劲，似乎有些臆想哎！自从人到中年后，其实在夫妻房事上的需求已经有些力不从心了，但是没有想到今天晚上有些意动，就是睡不着觉。<br/>　　<br/>　　于是，他翻身抱住了旁边的妻子，在将其弄醒以后，不明所以的情况下，开始和妻子热烈起来。<br/>　　<br/>　　一时间，浪里浪外的，两个人都投入了激情，啪啪了！<br/>　　<br/>　　等早上起床后，常经理却没有感觉自己有什么疲惫，反而非常的正常，就和平时一样，更不像是昨天晚上劳累了半晚上。<br/>　　<br/>　　妻子也非常的惊讶：“老常，你今天没有不对劲么？”<br/>　　<br/>　　要知道往常，老常如此劳累下，第二天早上还能这么精神的起床上班去，她当然惊讶了。<br/>　　<br/>　　老常看着容光焕发的妻子，自己也是神清气爽的，顿时豪气的说：“没有什么不对，这是身体好！”<br/>　　<br/>　　上班途中，常经理也在时刻回想，自己到底怎么回事，想着想着，就想到昨天晚上吃的韭菜炒鸡蛋。因为最近由于工作忙碌，也根本没有什么时间去保养身体或者吃什么补药。<br/>　　<br/>　　拿起电话，拨通自己下属。<br/>　　<br/>　　“老王，昨天晚上有没有什么不对的感受？”<br/>　　<br/>　　“没有啊，都很好啊！”<br/>　　<br/>　　“不是！我是问你，昨天晚上回来后，你的身体有没有感觉异常？”<br/>　　<br/>　　“没有，一切正……咦？你不问我还没有注意呢，你问了我才想到，昨天晚上我的精力很好呢！早上起床后，感觉身体还不错，好多天都感觉非常疲惫的身体，今天早上似乎要好很多。”<br/>　　<br/>　　“哈哈！你昨天晚上没有去浪？”<br/>　　<br/>　　“嘿嘿！当然有，不知道怎么回事，昨天晚上有些冲动，就去浪了一回！”<br/>　　<br/>　　老王离过婚，已经是单身一枚很久了，不过和常经理关系很好，一直做他的手下在工作中，也能很好的配合好他，所以很多话都能相互聊聊！<br/>　　<br/>　　“哈哈！那就对了！”常经理笑着说道。<br/>　　<br/>　　“什么那就对了？”<br/>　　<br/>　　“呵呵！昨天晚上吃的菜，没有想到功效不错哦！”常经理说道。<br/>　　<br/>　　“呀！你怀疑是昨天晚上的韭菜？”<br/>　　<br/>　　“当然，要不然我们也没有吃其他东西，昨天晚上怎么可能那么冲动呢？”<br/>　　<br/>　　“哦？昨天晚上你也冲动了，可以啊，是和嫂子还是出去浪啊？”<br/>　　<br/>　　“去！我的人品能让你怀疑？我才不会和你一样出去浪呢，我可是顾家的好男人！”<br/>　　<br/>　　两个人都是一阵的欢笑，并且约定了，等过几天再去港区吃饭。<br/>　　<br/>　　港区酒店餐厅推出的两种蔬菜，悄然兴起了一股浪潮，很多人吃了蔬菜以后，不仅对青菜喜欢的很，而且对韭菜鸡蛋更是超喜欢，很多离酒店比较近的人，在晚上的时候又定了餐，还预定了两盘韭菜炒鸡蛋。<br/>　　<br/>　　短短几天时间，港区餐厅就已经开始爆满，不仅是这里的菜品更加的好吃，而且这里的两种有机蔬菜更加的受人欢迎。<br/>　　<br/>　　高恒听到下面人的报告后，对自己的先见之明很是高兴，嘿嘿!这还是酒店餐厅，等果断时间，将蔬菜推向飞机商务舱和头等舱，如果更加的受欢迎，岂不是自己的功劳？<br/>　　<br/>　　凌经理也接到了命令，直接加大进货力度，另外就是经过几天的消耗，一车菜真的不够，这还是就在餐厅用，如果扩大到真个飞机上，那么消耗量则是无比巨大的。<br/>　　<br/>　　所以让陈默增加种植区域，增加品种就成为了凌经理的任务。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
