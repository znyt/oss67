　　第510章
<br/>　　徐神机等人听到这话直撇嘴，心道你刚刚还和百里胜聊得火热，转过头就说自己从未见过。
<br/>　　程大雷瞪他们一眼，转头面向钟伟虎，露出温和笑容。
<br/>　　“不知这百里胜和我有什么关系，还要向钟大人请教？”
<br/>　　“程当家有所不知，这百里胜是将军府派来的，统领虎口、鹿尾、弦阳三关。程当家想一想，凉州是我们的地盘，他这不是将钉子扎在我们眼皮底下么。”钟伟虎气愤道。
<br/>　　程大雷默默无语，心道，这是给你扎钉子，却未必是给我扎钉子。
<br/>　　关于这件事，程大雷也不是一点情报都没得到。
<br/>　　宋伯康封为凉州王，凉州成了他个人的封地。理论上，在凉州的封地中，他拥有统兵权，立法权，税收权，当然，这只是理论上。无论是李乐天还是将军府，都不可能看着宋伯康在凉州做大。
<br/>　　凉州与戎族接壤，帝国在险要地带都设有重兵把守，当然，琴川关原先也是凉州四关之一。李乐天派出百里胜统领凉州三关，很显然是想在宋伯康的后花园扎下钉子，以备后来事。
<br/>　　而为了这件事，相府和将军府已在朝堂上吵得不可开交。
<br/>　　程大雷脑袋转了一圈，手重重拍在桌子上：“啊呀呀，当真是欺人太甚。”
<br/>　　“谁说不是呢。”钟伟虎赶紧附和道：“咱们能看着他们这么办么，绝对不能啊。”
<br/>　　程大雷呵呵笑笑，道：“不知凉州王有何打算？”
<br/>　　钟伟虎抿了口茶润润嗓子，道：“我家王上十分敬重程当家的为人，恨不能与程当家结为弟兄。现在外人要过来，我们绝对不许。如果程当家答应，我们愿与程当家联手，将外人打出去，事成之后，我家王上愿意和程当家共治凉州。”
<br/>　　程大雷眯起了眼睛，钟伟虎虽然绕了一个好大的圈子，但目的和百里胜其实差不多，无非就是双方联手，将另一方办到。
<br/>　　他沉默思索片刻，道：“巧了，凉州王偏偏是和我想到一起了。”
<br/>　　钟伟虎眼睛一亮：“程当家是答应了。”
<br/>　　“容我再想想。”
<br/>　　“呃……”
<br/>　　“钟大人且先回去，容我和兄弟们商量一下，一旦商量出结果，必定第一时间告知钟大人。”
<br/>　　钟伟虎也没有期待第一次谈就能谈出结果，他起身告辞，临走之前，还不忘向程大雷嘱咐。
<br/>　　“程当家，这件事可一定要保密呐。”
<br/>　　“一定，一定。”
<br/>　　当钟伟虎离开后，蛤蟆城的几个人，立刻聚在一起商量。
<br/>　　“诸位兄弟，关于这件事，大家怎么看？”
<br/>　　从一开始，各人便在考虑。程大雷一问，和珅首先开口，道：“依我看，我们还是该和百里胜合作。百里胜毕竟代表将军府，将军府背后是当今皇帝，与他们合作，我们道义上是顺的。真占了凉州，我们行事便方便许多。”
<br/>　　“言之有理。”程大雷轻轻点头。
<br/>　　“在我看来，这件事还得好好考虑。如果和百里胜合作，他怕是帮不了我们太多，最后还是我们的兵上阵打仗。而与之相反，如果我们和凉州合作，兵马粮草上都得得到不少帮助，毕竟我们挨着凉州是近的。”刘悲道。
<br/>　　“也有道理。”程大雷同样没有反对。
<br/>　　程大雷没有给出确定意见，山寨几个人先吵了起来，有人想与朝廷合作，变相等于接受招安。有人想与凉州合作，能得到些眼前利益。
<br/>　　最后吵来吵去，没有吵出个结果。和珅看向程大雷：“大当家，是不是已经有了打算，如此的话，倒也不用我们白费口水。”
<br/>　　程大雷轻咳一声，道：“与百里胜合作不妥，容易被人当枪使，朝廷答应我们的封官进爵，到时候完全可以不算。”
<br/>　　程大雷还颇想去了凉州一地，到时候进可逐鹿天下，退可居安自保。但问题是，程大雷的名声在天下间已经臭了，真占了凉州，到时候说不得会有多少麻烦。
<br/>　　刘悲眼前一亮，道：“那就是要和凉州合作，如果能守望相助，也未必不是一个办法。”
<br/>　　程大雷摇摇头：“宋伯康不过是墓中枯骨，成不了大事。”
<br/>　　“那大当家的意思是？”刘悲奇怪的问。
<br/>　　程大雷略沉吟片刻，道：“最好的办法，是保持目前的局面，三足鼎立，我们才能从中取利。如果把这平衡打破，三足鼎立，也就变成两雄争霸了，到时候会平添许多麻烦。”
<br/>　　对于程大雷的话，其实如刘悲、和珅这些人未必就真心认同。不过，目前的蛤蟆城还是程大雷一人说了算，既然程大雷已经如此决定，大家也只能按照他的想法办。
<br/>　　百里胜如今还留在蛤蟆城，每天度日如年，帝国当然不会抬举一个山贼当诸侯王，除非程大雷已经足够强。但现在，程大雷还不够强。
<br/>　　将军府对程大雷的心意，还是利用的成分居多。偏偏凉州宋伯康也是同样的心意，都想要拿程大雷当枪使。
<br/>　　不过，程大雷一直没有吐口，他究竟怎么打算，百里胜心中也揣测不透。可时间一天天过去，事情赶得也是越来越急，程大雷再不吐口，他便真不知该怎么办是好了。
<br/>　　“侯爷，侯爷。”
<br/>　　程大雷从院外进来，一见程大雷，百里胜如见至亲。他忙迎过去：“贤弟公务那么忙，怎么还有时间过来，当真让为兄不知该如何是好。”
<br/>　　程大雷压低声音：“能否借一步说话。”
<br/>　　百里胜眼睛一亮，将程大雷引到屋中，屏退左右，道：“贤弟此来，为的是什么事？”
<br/>　　程大雷深吸一口气，道：“凉州城那边来人了？”
<br/>　　“啊！”百里胜一惊：“他们是想与你联手对付我。”
<br/>　　“可不是么。”
<br/>　　“哎呀呀，贤弟，你可要做好抉择，万不能走错了路。”百里胜。
<br/>　　“侯爷放心，侯爷放心，如果我要和他们合作，我今天不就不来了么。”程大雷道：“他们是不知道咱们兄弟之间的感情。”
<br/>　　“那是，那是……”百里胜在房间中踱步，道：“贤弟，你可一定要帮帮为兄啊。”
<br/>(本章完)
