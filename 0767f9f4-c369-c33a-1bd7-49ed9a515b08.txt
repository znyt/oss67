　　没等叶尘和冥沧荨开口，金老便将事情的来龙去脉，一五一十的全部讲了出来。
<br/>　　原来，
<br/>　　金老当初被冥沧荨一通训斥，并且还要将他赶出冥家，金老便怀恨在心，在返回冥家的路上，又刚巧遇到了冥沧山。
<br/>　　两人一拍即合，决定将联手诬陷冥沧荨和叶尘，将是非黑白颠倒。
<br/>　　金老原本是冥沧荨的护卫，如今站出来指认自己的主子，冥家众人自然不疑有他，连冥战也不得不相信，是冥沧荨联合外人，杀掉了冥家大长老，想要杀掉自己的亲哥哥，于是才有了之前的那一幕。
<br/>　　原本在金老看来，叶尘的实力虽然有些变态，但毕竟只有元婴境而已，而且这里是冥家的地盘，即便他再厉害，还能和冥家相抗衡？
<br/>　　但是他怎么也没有想到，眼前的这个年轻人，竟然具有如此强大的能量，连三大家主都对他如此的畏惧！
<br/>　　哗！
<br/>　　等到金老将事情的经过，全部说出来之后，冥家众人顿时一片哗然。
<br/>　　“原来这一切，都是大公子的阴谋啊！”
<br/>　　“我就说嘛，大小姐心地善良，怎么可能回做出那种事情？”
<br/>　　“真是该死啊！就因为他，险些连累了我们整个冥家！”
<br/>　　……
<br/>　　一时间，所有人开始对着冥沧山声讨起来。
<br/>　　冥沧山看到这一幕，顿时都快要崩溃了，之后连滚带爬，来到冥战的跟前，苦苦哀求起来，
<br/>　　“父亲大人！我知道错了，我真的知道错了！求求您，救救我，一定要救救我啊！”
<br/>　　冥战顿时一脸不忍之色，但是他似乎也知道，如今这种情势之下，冥沧山的生杀大权，根本不在他的手上，而是取决于那个年轻人。
<br/>　　“砰！”
<br/>　　冥战抬腿就是一脚，直接将冥沧山踢到叶尘和冥沧荨的跟前，冷喝道：
<br/>　　“混账东西！你对不起的人不是我，而是叶前辈和你的妹妹！”
<br/>　　冥战表面上是在痛骂冥沧山，可是明眼人都看得出来，这是在间接提醒自己的儿子，要求饶也应该向叶尘和冥沧荨求饶才对。
<br/>　　冥沧山也不傻，很快就明白了冥战的用意，当即又连滚带爬，来到叶尘跟前，
<br/>　　“叶前辈，我错了！求求您饶了我吧！”
<br/>　　叶尘一脸淡漠，丝毫不为所动，甚至连看都没有看地上的冥沧山一眼。
<br/>　　冥沧山见叶尘完全不鸟他，只好又转而爬向冥沧荨，痛哭流涕的哀求起来，
<br/>　　“沧荨妹妹，都是哥哥的错！哥哥不该和你争夺下任家主的位置，更不该诬陷你和叶前辈，求你看在我们血脉相连的份上，就饶了哥哥这一次吧！！”
<br/>　　冥沧荨到底还这是一个十几岁的小女孩，看到冥沧山的凄惨模样，顿时就有些心软了，不由得扭头望向一旁的叶尘，
<br/>　　“叶大哥……”
<br/>　　叶尘也看出，这丫头又要心软，不由得暗暗叹了一口气，脸色一板，道：
<br/>　　“还记得刚刚我怎么和你说的吗？这么快就忘了！”
<br/>　　冥沧荨听到叶尘这话，不由得脸色一肃，一番纠结之后，最终一咬牙，将俏脸扭到一旁，任由冥沧山磕头求饶，也不再理会。
<br/>　　冥沧山见状，先是一阵绝望，继而恶向胆边生，眼眸中猛然露出一抹浓浓的恨意，
<br/>　　“贱丫头！既然你如此无情，那我就算是死，也要拉一个垫背的！！”
<br/>　　从绝望到满腔恨意的冥沧山，直接一声低吼，抬起双掌，便朝冥沧荨身上，狠狠拍了过去！
<br/>　　轰！
<br/>　　这冥沧山虽然算不上顶级天才，但修为也不弱，只比冥沧荨稍差一些而已，而且他又是突然偷袭之下，即便是冥沧荨一时间也反应不及。
<br/>　　眼看这一掌，就要拍落在冥沧荨的身上，冥沧荨显然也没有想到，冥沧山会突然出手，顿时美眸一瞪，想要阻挡已经有些来不及。
<br/>　　这一切，发生的实在过于突然，即便是一旁的冥战，也完全没有反应过来，此时想要搭救，也来不及了。
<br/>　　“贱丫头，你给我去死吧！”
<br/>　　冥沧山满脸狰狞之色，似乎已经看到冥沧荨死在自己掌下的情景。
<br/>　　然而，
<br/>　　就在这个时候，
<br/>　　一道人影，陡然拦住了他的双掌去路，犹如一座高山，挡在了冥沧荨的跟前。
<br/>　　“砰！”
<br/>　　一声巨响，冥沧山的双掌，狠狠砸落这个突然出现的人影身上。
<br/>　　“叶大哥！！”
<br/>　　冥沧荨反应过来之后，顿时惊呼了一声。
<br/>　　那道挡在她面前的身影，自然正是叶尘。
<br/>　　而冥沧山先是一愣，等他看清楚叶尘的面庞，顿时又露出一抹喜色，
<br/>　　“小子，你这是在找死！！”
<br/>　　原来，叶尘不知道是不是因为仓皇出手的缘故，竟然没有运起护体真元，竟然单纯凭肉身挡住了冥沧山的一击。
<br/>　　轰！
<br/>　　冥沧山恨极了叶尘，将全身的真元，都注入到了自己的双掌之上。
<br/>　　要知道，哪怕是化神天君，在不运起护体真气的情况下，肉身也强不到哪儿去。
<br/>　　更何况，对方只有元婴境？
<br/>　　这一刻，
<br/>　　就连一旁，四大家族的家主们，也纷纷瞪大了双眼。
<br/>　　在他们看来，纵然叶尘再厉害，这一次也有些托大了。
<br/>　　虽然冥沧山这一掌还不足以杀死他，但经脉被震断几根，只怕是避免不了。
<br/>　　然而，
<br/>　　下一刻，
<br/>　　结果却让所有人目瞪口呆。
<br/>　　叶尘用肉身硬接下冥沧山的全力一击，竟然浑若无事，甚至连晃动一下都没有！
<br/>　　“这么会！”
<br/>　　冥沧山感觉自己的双手，犹如打在了世上最坚硬的岩石之上，不但未能撼动对方分毫，双掌反而剧痛无比，手骨都被生生震碎！
<br/>　　“蝼蚁一般的东西！就凭你，也想伤我？”
<br/>　　叶尘淡漠开口，同时身体微微一震，一股毁灭气息，自他的体内涌出，
<br/>　　“彭！”
<br/>　　冥沧山整个人，顿时血肉迸飞，竟然被生生震碎，化成了一团血雾！
<br/>　　“沧山！！”
<br/>　　后方的冥战顿时忍不住大吼了一声，可惜他却根本没有对叶尘出手的勇气，最终喟然长叹了一声，一脸凄然的退回人群之中。
<br/>　　叶尘轻松灭掉了冥沧山，转而又望向那金老，淡淡开口道：
<br/>　　“你，悖主弃义，本不容赦，念在你认错及时，自断一臂，今后不得再踏足南斗星域！”
