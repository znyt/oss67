夏夕绾抱着小圆圆回到了卧室里，小圆圆窝在自己的窝里，很快就睡着了。
<br/>
<br/>        夏夕绾躺在床上，她想着奶奶刚才说的话，翻来覆去的睡不着。
<br/>
<br/>        侧了一个身，她伸出小手缓缓抚上了他睡过的枕头，他现在在干什么？
<br/>
<br/>        夏夕绾拿出手机，给他发了一条微信今晚你真的不回来了？
<br/>
<br/>        “叮”一声，他的回信很快就来了，让人不得不怀疑此刻的他是不是跟她一样拿着手机准确发信息。
<br/>
<br/>        陆寒霆我睡在公司。
<br/>
<br/>        夏夕绾数了一下他回的字，一二三四五…只有五个字，还真是简单。
<br/>
<br/>        夏夕绾又编辑了一条我想跟你聊一聊…
<br/>
<br/>        但是这消息还没有来得及发出去，他的信息又来了我去开会了，你早点睡，晚安。
<br/>
<br/>        他单方面宣布这场聊天结束了。
<br/>
<br/>        夏夕绾生气的将手机丢到了床柜上，然后将他的枕头抱在怀里，脑袋里想象着他那张可恶的俊脸，她捏着拳头就用力的锤了枕头两下。
<br/>
<br/>        很快，她又将枕头枕在了自己的脑袋下面，这枕头上还残留着他身上那股干净清冽的味道，她将小脸埋进去，迷迷糊糊的就睡着了。
<br/>
<br/>        ……
<br/>
<br/>        翌日清晨，夏夕绾早早的来到了枢密研究院，现在还不是上班时间，没看到什么人。
<br/>
<br/>        夏夕绾准备回药房，但是这时她在前方看到了一道年迈的身影，是一个老夫人，老夫人虚弱的身体摇晃了两下，一下子就栽倒在地上了。
<br/>
<br/>        夏夕绾瞳仁一缩，迅速跑上前，“老夫人，你怎么了，不要怕，我是医生，我会救你的。”
<br/>
<br/>        夏夕绾迅速给老夫人把了脉，老夫人患了重症，加上年迈的身体，可谓是数病齐发，相当凶险。
<br/>
<br/>        这时老夫人鼻尖一热，鲜红的血液当即从鼻子里面穿涌而出了。
<br/>
<br/>        老夫人瞳仁有些涣散，她虚弱痛苦的看着夏夕绾，“我这是要死了么，我死之前竟然见到了仙女。”
<br/>
<br/>        被“仙女”的夏夕绾跪在了地上，她迅速用手帕捂住了老夫人的鼻子，鲜红的血液当即染红了她纤白的手指，她另一只手捏起了一根银针，一双澄亮的翦瞳看着老夫人盈盈的笑，“老夫人，我可不是什么仙女，你今天走运，我可是上天派来救你的大罗神仙。”
<br/>
<br/>        “大罗神仙？这么漂亮的大罗神仙我真是第一次见，不要以为能唬住我，你明明就是小仙女。”老夫人看着夏夕绾，苍白的唇角竟然也跟着勾起了一抹笑意。
<br/>
<br/>        这时夏夕绾将手里的银针刺进了老夫人的穴位里，老夫人的鼻血迅速止住了。
<br/>
<br/>        这一次夏夕绾换了一根细长的金针，她摸了一下老夫人头部的穴位，轻声问道，“老夫人，你怕么？”
<br/>
<br/>        老夫人有点抗拒，“可以不拿针扎我么？”
<br/>
<br/>        夏夕绾摇头，“不行。”
<br/>
<br/>        “那小仙女轻点，我怕疼。”
<br/>
<br/>        “好，老夫人，那你把眼睛闭上。”
<br/>
<br/>        夏夕绾柔软的嗓音好像带着一股无法抗拒的魔力，老夫人真的闭上了眼。
<br/>
<br/>        在这么一瞬间，夏夕绾准确无误的将手里的金针刺进了老夫人的大脑穴位里。
