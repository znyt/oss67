　　迷宫城之中，苏莫停了下来，目光望向脚步声传来之处。
<br/>　　少倾，通道的尽头，转弯之处，一名青年的身影显现而出。
<br/>　　此人身材挺拔，穿着一身白袍，面容俊逸，给人一种卓尔不凡之感。
<br/>　　青年看到了苏莫身后的出口，便大步向出口走来，每一步都发出沉默的声响。
<br/>　　“天阳族！”苏莫见此这位白袍青年，顿时认出了其所属的种族。
<br/>　　此人的额头之上，有一个淡淡的太阳标记，身上的气息也是至刚至阳，正是天阳族之人。
<br/>　　苏莫微微有些惊讶，天阳族这么厉害的种族，其族人居然甘愿做龙族的奴仆。
<br/>　　天阳族的白袍青年，仿佛是没有注意到苏莫一般，大步而来，向出口走来。
<br/>　　“留下你身上的宝物，你就可以出去！”苏莫沉声说道，对于他而言，同阶之中，无论是什么种族，都可普通人族一样。
<br/>　　“你很自信？”白袍青年凝视着苏莫，淡淡的问道，他倒是知晓苏莫是龙腾的奴仆。
<br/>　　不过，其并没有停下脚步，依旧是大步向苏莫走来。
<br/>　　“过来！”苏莫没有过多废话，大手一探，直接向对方抓了出国。
<br/>　　浩荡的源力，再次化为了一只大手，向白袍青年抓了过去。
<br/>　　不过，这一次的源力大手，不再是单纯的火焰大手，而是呈暗红之色。
<br/>　　这是因为，源力大手的源力，乃是木火两种属性的彻底融合，而不再是单纯在木属性源力滋养火属性源力。
<br/>　　两种属性源力融合，让得苏莫的实力大增，强大的源力大手，裹挟着滔天的威势，直击白袍青年。
<br/>　　“果然不简单！”白袍青年见此，双眉微挑，他虽然自信，但也没有小觑苏莫。
<br/>　　因为，他知晓苏莫是龙腾的奴仆，作为龙腾这种妖孽的奴仆，肯定不会简单。
<br/>　　嗡~~
<br/>　　光芒一闪，白袍青年的手中，出现了一柄弯刀。
<br/>　　此刀如同一轮弯月，雪亮无比，其上散发出至阳的气息。
<br/>　　唰！
<br/>　　弯刀一挥，一刀炫目的白色刀气，撕破虚空，向源力大手斩击而去。
<br/>　　这道刀气极为强大，刀气之中蕴含着至阳之力，恐怖的刀气如同凝聚了数个太阳的精华，炙热无比。
<br/>　　轰！
<br/>　　一声爆响，刀气重重的斩中了源力大手，源力大手瞬间奔溃，化为滚滚源力浩荡。
<br/>　　而强大的刀气，也蹦碎开来，劲气四射。
<br/>　　炸开的冲击波，冲击在通道的墙壁之上，让得墙壁震动，轰响不断。
<br/>　　不过，墙壁有大量的阵法守护，丝毫不会被损坏。
<br/>　　“战力不错！”苏莫微微一笑，总算是碰到一个战力还算说得过去的人了。
<br/>　　当然，对方也只是勉强说得过去而已，与一般的万古天骄相比，都有很大的差距。
<br/>　　听闻苏莫之言，白袍青年眼眸微眯，眼前之人貌似极度自信，难道还没有动用真实的战力？
<br/>　　两人势均力敌的一击，自然也被外界很多人看在了眼里。
<br/>　　“呵呵，龙腾，我说的不错吧，你这个奴仆遇到了势均力敌的对手！”龙城面带微笑，他很乐意看到这种情况。
<br/>　　若是那天阳族的阳云，能击败龙腾的奴仆，那就再好不过了。
<br/>　　“阳云虽然只有一半天阳族的血统，但天赋也算妖孽级别，胜负不好说了！”龙洌摇头说道。
<br/>　　“错，下一招，阳云就会败了！”龙腾笃定的说道。
<br/>　　众人闻言一怔，这龙腾居然对这个奴仆如此的有信心，这奴仆真的有这么强？
<br/>　　不过，他们也没有多问，因为迷宫之中，两人已经再次交手了。
<br/>　　“杀！”白袍青年阳云怒喝一声，抢先出手了，他知道苏莫实力很强，所以立刻便全力出手了。
<br/>　　只见白袍青年的弯刀高高举起，刺目的白光在弯刀上绽放，几乎是一瞬间，弯刀便化为了一轮烈日。
<br/>　　磨盘般大小的烈日，发出恐怖的至阳之力，如同凶猛的火焰，焚灭虚空。
<br/>　　唰!
<br/>　　弯刀一挥，烈日破空，向苏莫击杀而去。
<br/>　　“你已经错过了机会！”苏莫面色淡淡，刚才他出第一招之时，动用的实力很小，对方如果这般攻击，或许还有逃出去的机会。
<br/>　　但是现在，机会已经消失了。
<br/>　　轰隆隆！
<br/>　　苏莫大手一探，再次幻化出一只源力大手，直接向着烈日抓去。
<br/>　　这一次，源力大手呈现灰色，苏莫动用了水木火三属性源力融合，源力大手的威力暴增一大截。
<br/>　　轰隆隆！
<br/>　　源力大手所向披靡，直接碾碎了烈日，向着白袍青年抓了过去。
<br/>　　“什么？”白袍青年阳云大骇，他完全没有想到，苏莫的实力居然强大到了如此的地步。
<br/>　　眼见大手临身，他脚下一踏，身形急速暴退。
<br/>　　与此同时，他手中弯刀暴斩，企图挡住源力大手的攻势。
<br/>　　但是，一切都是徒劳的，他的实力根本挡不住源力大手。
<br/>　　恐怖的大手，碾碎一起，直接就擒住了白袍青年。
<br/>　　“怎么可能？你怎么可能这么强大？”阳云怒吼，他面带惊恐之色，被源力大手擒住，依旧在不停的挣扎。
<br/>　　其身上源力鼓荡，整个人都好像化为了一轮骄阳，想要震碎源力大手。
<br/>　　但是，这是根本不可能的事情。
<br/>　　“是你太弱了！”苏莫大手用力，顿时将对方的肉身捏成了麻花，肉身都差点捏爆了。
<br/>　　随即，他大手一甩，直接将对方从出口处扔了出去。
<br/>　　源力大手收回，苏莫的手中，出现了一把弯刀和一枚储物戒指。
<br/>　　这弯刀是对方的兵器，既然储物戒都抢了，兵器也就一并抢夺了。
<br/>　　随即，苏莫开始查看储物戒，这储物戒是一枚空间宝器，其内居然有大量的物品。
<br/>　　之前，苏莫强度了很多人，其它人的储物空间之中，均是空无一物，而此人身上居然带了很多东西。
<br/>　　“看来此人很有自信！”苏莫不由的摇了摇头，只有自信的人，才会带着自身的物品进来，而不惧别人抢夺。
<br/>　　不过，这储物戒指之中，源石、兵器、神果、神草玉简各种晶石等应有尽有，不知道有没有迷宫城之中的宝物。
<br/>　　苏莫没有仔细的检查，随意的打量了一眼，便收了起来。
<br/>　　不管是此人自身的宝物，还是对方在迷宫城之中得到的宝物，全部都是他的了，出去之后再好好的查看。
