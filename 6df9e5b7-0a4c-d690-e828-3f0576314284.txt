        &lt;h3&gt;第三卷第一百四十七章  陆玉在甸城&lt;/h3&gt;
<br/>
<br/>        陆玉到甸城已经快一年了，在接手工作的这一段时间里，她没有太多的心思想别的，虽然她的心思在秦少卿的身上，但是工作还是工作。
<br/>
<br/>        她从北京调到地方上来，第一要熟悉当地的环境，第二，要熟悉当地人文，第三，其实也是最重要的，要熟悉当地的人物关系。
<br/>
<br/>        理清上面的这些事情，她才能更好的开展自己的工作。
<br/>
<br/>        爸爸临走之前给她郑重的交代了又交代：
<br/>
<br/>        “玉儿，我们陆家大部分的资金来源都是从这里发出去的，所以你要明白你来这里的目的，还有无论是个人问题还是工作问题，你一定要低调，不可像之前那么张扬，你还有更高的地方去攀登，而不是就在这个地方。”
<br/>
<br/>        “爸......”
<br/>
<br/>        “你听着，在我的心里你比你哥哥还有能力、还能干，你哥哥做事情瞻前顾后，成不了大器。有些事情我根本不让他碰。你明白我的意思吗?”陆战君挥挥手，打断陆玉的话严肃的告诉她。
<br/>
<br/>        “这个地方虽然比较偏远，对我们陆家来说这里确是一块风水宝地，你就是镇守这块风水宝地的人。”
<br/>
<br/>        “我知道了。”
<br/>
<br/>        “用你的智慧和魄力来做一番事情吧。我回去后，只要你想要的资源我都会想办法给你。”
<br/>
<br/>        “好的，爸爸。”
<br/>
<br/>        “秦家和谷家掌握着甸城的大部分资源，尤其是谷家，我听说谷家的企业虽然是谷正娟打理，但实际上是谷强，这个你特意关注下，最好是能将谷强掌握在手中。”
<br/>
<br/>        “爸？”陆玉有点吃惊。
<br/>
<br/>        “至于怎么做那是你的事情，做大事者不拘小节。”陆战君站起身接着说：
<br/>
<br/>        “我走了，你好好工作。”
<br/>
<br/>        她目前和秦少卿负责信息化建设，成立了一个特别小组，秦少卿为正，她为副，按照陆玉的性子原本她是不会为副的，但是想到父亲临走之前的交代，她并没有过多的去争取。
<br/>
<br/>        现在，她必须得站稳脚跟。
<br/>
<br/>        这一段时间以来，她发现，秦家和谷家比他们想象的还要厉害，在甸城，他们说了算，就算是有人反驳，都是起不了一点点儿风浪。
<br/>
<br/>        在这两年里，她仅仅就见过谷强三次，在谷正娟的口中，她了解到谷强对感情特别认真，认准了一个人就不会三心二意，对朋友也是如此。
<br/>
<br/>        第一次见到谷强的时候，她觉得很面熟，似乎在哪里见过，但是她又确信她没有见过谷强。
<br/>
<br/>        “谷总，我们见过吗？”在为迎接她的晚宴上她问谷强。
<br/>
<br/>        她知道谷强是谷家的义子，也知道谷正娟心仪谷强。
<br/>
<br/>        谷强有一双好看的丹凤眼，但是那里面全都是盛满寒冰，没有一丝丝温度。
<br/>
<br/>        长着象征美人的瓜子脸，微微一笑的时候还有酒窝出现，只是那微笑也像是冬天的寒冰，有些冻人。
<br/>
<br/>        “我们从来没有见过。”谷强疏离的就像是和陆玉隔着十万八千里。
<br/>
<br/>        “可是我觉得谷总好面熟。”陆玉是真的觉得在哪里见过谷强。
<br/>
<br/>        “我长着一张大众脸，谁见我都会说面熟。”谷强和陆玉轻轻碰了一下杯：“陆总，以后还需多多关照。”
<br/>
<br/>        “陆玉初来乍到，还希望谷总多多指教呢。”陆玉含笑看着谷强，对人下菜这种事情陆玉已经熟悉得不能再熟悉了。
<br/>
<br/>        “你们聊什么呢？”谷正娟拿着酒杯走过来，站在谷强的身边。
<br/>
<br/>        “陆总说她在哪里见过我。”谷强对谷正娟笑着说。
<br/>
<br/>        “我给你看过他的照片啊。怎么样？帅吧？”谷正娟像显摆一件属于自己的宝贝一样。
<br/>
<br/>        “我好像不知不觉的就吃了一顿狗粮，我得找人也秀一秀。”陆玉举起酒杯冲着谷强和谷正娟笑了一下，转身走了。
<br/>
<br/>        父亲让她将谷强掌握在手中，可是她都见不到谷强，而且每次见谷强的时候，谷正娟都在，她怎么能将谷强掌握在手中？
<br/>
<br/>        况且他们工作上接触的人基本都是谷强的手下，但是工作人员真的就是工作人员，想让她屈尊去向工作人员打听谷强的消息她又很不甘心。
<br/>
<br/>        最近，谷强又忙着成立新的公司，准备做信息化。
<br/>
<br/>        秦寿找到她：“陆总，谷强成立新的公司，你们现在的信息化将来就用那个新公司来做。”
<br/>
<br/>        “少卿不是说从北京找人过来做吗？”
<br/>
<br/>        “北京的人只是谷强用来做障眼法的，这种大项目必须得有人托底。”
<br/>
<br/>        “谷强真的是很能干呢。”陆玉想到了谷强的意思。
<br/>
<br/>        “但是还得你的支持。”狡猾如狐的秦寿怎么能不知道陆家是要掌控他们呢，只是放一个女孩子来，怕是小瞧了他们吧。
<br/>
<br/>        “秦叔，你放心，我会告诉爸爸的。哦，对了，少卿和闫芈最近关系好吗？”
<br/>
<br/>        “啊，应该好着吧？”秦寿不知道陆玉为什么要问秦少卿和闫芈的事情。
<br/>
<br/>        “少卿说他都后悔和闫芈结婚了呢。”
<br/>
<br/>        “啊，有这回事情吗？这孩子有啥事情也不给我们说。”
<br/>
<br/>        “那一次和我聊天说起来的，还说他曾经心仪我，哎，既然当初心仪我，为什么不告诉我呢？”陆玉叹口气。
<br/>
<br/>        “我们怎么能高攀你呢。”秦寿看着陆玉，这怕是看上他家少卿了。
<br/>
<br/>        “叔叔，什么叫高攀不高攀啊，我和少卿也算是青梅竹马了吧？”陆玉故作不高兴的样子。
<br/>
<br/>        “唉，少卿的心思从来都不告诉我，只是你们现在........”秦寿很是为难。
<br/>
<br/>        “现在少卿也说他还是喜欢我，只是中间夹着一个闫芈，让他不知道该怎么办。”陆玉深深的叹了口气。
<br/>
<br/>        “还有一个张函啊。”秦寿也叹了口气，要是少卿和陆玉在一起，也就不那担心了。
<br/>
<br/>        “张函在北京，他怎么能管得上我呢。”陆玉拿着手中的笔转着圈。
<br/>
<br/>        “叔叔，少卿再干几年，我给爸爸再说一下，他年轻能干呢。”陆玉前倾着身子看着对面坐着的秦寿。
<br/>
<br/>        “哈哈哈，那就谢谢玉儿了，少卿对我们之间的事情一无所知，你呀什么都不知道，还有，谷强所作的一切都是为了我们------陆家、秦家和谷家。”
<br/>
<br/>        “叔叔，今天终于是给我交了底，你这样一说我就明白了。”
<br/>
<br/>        “你以后有什么事情就来问我，我知无不言。”
<br/>
<br/>        “叔叔对陆家的心意，玉儿一定会转告给父亲的，叔叔应该将甸城的势态告诉玉儿了吧。”
<br/>
<br/>        “玉儿来了这么久，怎么可能不了解了，你眼中看到的听到的都是真的。”
<br/>
<br/>        “那玉儿在甸城就得靠叔叔提点了。”
<br/>
<br/>        “我们两家本来就是亲人，现在更是亲上加亲了，一家人别说两家话。玉儿，我还有事情我就走了，我们回头再联系。”
<br/>
<br/>        “叔叔，我记得你有个外甥，叫何少华，据说心仪闫芈呢，不知道是真是假?”陆玉笑着问秦寿。
<br/>
<br/>        “唉，家门不幸，让玉儿见笑了。我走了啊。”秦寿站起来就走。
<br/>
<br/>        “叔叔。再见，不送！”
<br/>
<br/>        陆玉盯着秦寿离开的背影，微微一笑：果然不亏他的名字。
