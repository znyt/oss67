　　神域，天星神朝。
<br/>　　天星神朝地域辽阔，拥有数千座城池，数之不尽的人口。
<br/>　　紫星城，是位于天星神朝腹地的一座中等大小的城池，城池之中，便拥有一座飞升大殿。
<br/>　　此飞升大殿，坐落在城北，建筑老旧，显得很是荒凉。
<br/>　　嗡~~
<br/>　　这一日，殿中光芒闪烁，一道年轻的身影突然出现了。
<br/>　　此人，自然便是苏莫，只不过，他现在改变了相貌，隐藏了修为。
<br/>　　“咦！”
<br/>　　一道轻咦之声响起，一位体型消瘦的青年闪身而出，出现在了苏莫的身前。
<br/>　　“这么多年了，终于有下界之人了！”消瘦青年大量了苏莫一番，面带感慨之色。
<br/>　　“这里是天星神朝？”
<br/>　　苏莫看向消瘦青年，微笑着问道，此人貌似是一名妖族之人。
<br/>　　适才，他的神念已经扫荡而出，发现此城之中，人族、妖族、异族等等非常的复杂，不过，倒是有不少星辰族之人。
<br/>　　所以，他有所猜测，这里是天星神朝。
<br/>　　“你居然知晓天星神朝？”消瘦青年闻言，顿时面带诧异之色，下界来的人，怎么会知晓天星神朝？
<br/>　　“当然，天龙神朝之人在下界更换阵基，我们下界很多人都知道神域的情况！”苏莫点了点头。
<br/>　　“原来如此！”
<br/>　　消瘦青年心中恍然，道：“这里的确是天星神朝，你可以选择下来，也可以离开！”
<br/>　　对于不是星辰族之人，他们天星神朝不会有要求，爱去哪里去哪里。
<br/>　　“告辞！”苏莫微微一笑，向对方拱了拱手，便大步走出了大殿。
<br/>　　“奇怪，我居然看不出他的修为！”
<br/>　　消瘦青年望着苏莫离去的背影，心中有些惊疑不定，不过，他也没有阻拦苏莫，这些都与他无关。
<br/>　　苏莫离开了大殿之后，便立刻离开了这座城池，火速向时空界域而去。
<br/>　　修炼的速度太慢了，唯有在时光神殿之中，他才能节省出大量的时间。
<br/>　　虽然他很想去找夕儿和小莫等人，但是，他根本没有太多的精力。
<br/>　　好在，在回神域之前，他已经交代了龙腾，对方会为他寻找。
<br/>　　还有就是，现如今，他在神域的名气如此之大，人尽皆知，夕儿等人肯定都知道他受龙族的庇护。
<br/>　　天星神朝，距离时空界域距离不近，不过，现如今苏莫的速度，也不是以往可以比拟的了。
<br/>　　匆匆半月而过，苏莫来到了时空界域，进入了极空城中。
<br/>　　这半个月的时间，他一路上也打听了一番最近神域的局势，从他下界到如今，神域的确是发生了一些事情。
<br/>　　那就是，五大神朝有变，天星神朝和霸天神朝公开表示，不再插手帝一魂与苏莫、拓跋青云之间的争斗。
<br/>　　而这段时间，幽冥神朝、亘古魔国、不死神朝的强者，也去了一趟龙都城，但是并没有大战，据说是面见了吞虚龙帝，具体谈了什么，不得而知。
<br/>　　不过，面对龙族，三大神朝虽然没有什么大的动作，但是，另一边对地王宫可就不客气了。
<br/>　　大批的三大神朝的强者，以及天王宫之人，杀到了地王宫，想要擒拿拓跋青云。
<br/>　　不过，地王宫的有护宫大阵，虽然无法挡住三大神朝和天王宫的强者，但是，在大阵即将破碎之时，兽王宫和魔王宫、鬼王宫的人，纷纷出现，前去为地王宫助阵。
<br/>　　若只是这三大王宫，自然无法抵挡帝一魂的势力，最主要的是，混沌神朝的强者出现了。
<br/>　　才压制住了这次冲突，最后不欢而散。
<br/>　　据说，这一次战斗，虽然持续的时间不长，但是，由于神域的稳定性因为已经开始更换阵基，而有所下降，差点让半个轮回神朝都崩溃。
<br/>　　从那次以后，这段时间，神域无比平静，没有再掀起任何的波澜。
<br/>　　苏莫走在极空城的街道上，脑中思索着自己打听到的情况，虽然这让他有些意外，但却是最好的情况，局势总算是稳住了。
<br/>　　那么，他也就能静心的修炼，提升修为境界了。
<br/>　　不多时，苏莫来到了时空之城下，等待了四日之后，便进入了时门之中。
<br/>　　他又进入了上次来到的第十三重时光神殿，在接近两万倍的时间流速之下，开始了修炼。
<br/>　　时日匆匆，这里的时间过的很快，十年的时间眨眼而过，苏莫的修为根基足够扎实，他开始吞服丹药，冲击五品高级的境界。
<br/>　　一切都很顺利，苏莫毫无意外的达到了五品高级的地步，下一个目标，便是达到六品。
<br/>　　资源充足，时间也充足，他按部就班的修炼。
<br/>　　第十三重时光神殿中的人不多，虽然苏莫修炼混沌源力，让不少人意外，但并没有人认出他的身份。
<br/>　　……
<br/>　　时空界域之中，一行五人，向极空城的方向急速而行。
<br/>　　这五人，为首之人正是不死神朝的贺兰殇，另外四人，两位是魔族，两位是幽冥族。
<br/>　　五人之中，除了贺兰殇是七品低级的修为之外，其它人均是六品的修为。
<br/>　　“贺兰兄，你说苏莫有可能会去时空之城吗？”一名幽冥族的高大青年问道。
<br/>　　此人名为冥生，也是一位万古天骄，上次参加过千选战。
<br/>　　“很有可能，神域的空间稳定性已经减弱，说明龙族正在更换阵基，以帝一魂之言，苏莫肯定会下界！”
<br/>　　贺兰殇面色凝重，继续道：“根据我族收买的龙族之人所言，苏莫应该是和龙腾一起下界的，而现在龙腾已经回到了龙都城，但并不见苏莫！”
<br/>　　“那你就能肯定，苏莫会去时空之城？”另外一位魔族的青年问道。
<br/>　　“你们有所不知，我以前在时空之城曾经见过苏莫，而据我所知，苏莫不止一次去时空之城，这是因为什么？”
<br/>　　贺兰殇轻笑一声，不待别人回答，他继续道：“苏莫的天赋，毋庸置疑，他缺的就是时间，只有在时空之城，他才能以最快的速度强大起来！”
<br/>　　“说的也是，若是这次能找到苏莫，他别想逃出我们的手掌心！”冥生冷笑一声，当初在千选战之时，他们修为和苏莫同阶，不是苏莫一招之敌。
<br/>　　但是现在，他们任何一人，都足以镇杀苏莫。
<br/>　　“苏莫必死无疑，我们若是能找到他，只是锦上添花而已！”贺兰殇笑道。
<br/>　　现在，他们三大神朝之所以没有全力出手，一是因为神域稳定性大降，担心导致神域大崩溃，二是现在不值得硬拼。
<br/>　　因为，目前的帝一魂，有把握在数十年之内，晋升高级神王。
<br/>　　本来，正常情况之下，帝一魂需要七八百年，甚至是千年以上的时间才能达到高级神王，但是现在，有他们三大神朝的鼎力之助，再加上时间阵法，数十年足矣。
<br/>　　以帝一魂的天赋，一旦达到高级神王，在神域基本上无敌了，就算是混沌元王，也不一定是帝一魂的对手。
<br/>　　到时候，对方拓跋青云和苏莫，将轻而易举。
<br/>　　所以，目前，他们三大神朝，没有理由去硬拼，白白损伤族人。
<br/>　　“放心吧，若是我所料不错，苏莫有极大的可能性会去时空之城！”贺兰殇笑着说道。
<br/>　　“那拓跋青云，不会也去时空之城吧？”有人问道，若是拓跋青云也去时空之城，那他们可完全不是对手。
<br/>　　“这倒是不会！”贺兰殇摇了摇头，继续道：“无论是帝一魂还是拓跋，行踪一直都被监视，他们哪里敢去时空之城。”
<br/>　　“那就好！若是苏莫只身一人在时空之城，我们手到擒来！”
<br/>　　“哈哈！！”
<br/>　　众人大笑，虽然只是来碰碰运气，但是，他们也抱着极大的期望。
