　　茂密的山林中，苏莫立身于一颗古树之下，手中依旧在擒拿着巫森。
<br/>　　“苏莫，你想怎么样？”巫森忐忑的问道，他的脸色不停的变幻，暗思活命之策。
<br/>　　“想怎么样？”
<br/>　　苏莫闻言冷笑一声，道：“你觉得落在我手中，你还有活命的机会吗？”
<br/>　　苏莫对于巫森的杀心极重，就单单是将宏青璇送给巫天御这一点，就是他不能容忍的事情。
<br/>　　“苏莫，你不能杀我，杀了我对你没有任何好处！”
<br/>　　巫森急声道，他眸中精光一闪，继续道：“只要你不杀我，我愿意奉你为主，终身听候差遣！”。
<br/>　　为了活命，巫森已经甘愿为奴为仆了。
<br/>　　“是吗？”
<br/>　　苏莫轻笑一声，问道：“那我问你，巫山和皇甫擎呢？还有黄泉魔宗、阴阳神宫已经北堂世家的所有人呢？”
<br/>　　“这……！”巫森闻言面现迟疑之色，面色不停的变换。
<br/>　　“嗯？”苏莫见此顿时眉头微皱，眸中杀机爆闪。
<br/>　　“在空间宝器之中，全部都在空间宝器之中！”巫森见此，顿时颤声说道。
<br/>　　他不得不说，因为他空间玉镯就在手腕上，想隐瞒也瞒不住。
<br/>　　“拿来！”苏莫手掌一伸，冷声命令道。
<br/>　　巫森面现迟疑之色，但还是颤颤巍巍的摘下了手臂上的玉镯，递给了苏莫，他别无选择。
<br/>　　虽然玉镯中有他的儿子，黄泉魔宗一众高手，阴阳神宫的人、北堂世家的人，但若是以这些人的性命，能救他一命，还会毫不犹豫。
<br/>　　巫森的心中郁闷到了极点，原以为巫族试炼者降临，他将再无忧虑，安安心心跟着巫族试炼者离开就行。
<br/>　　没想到，连巫族试炼者都栽了，栽在了苏莫的手中，这是他从未想过的。
<br/>　　接过空间玉镯，苏莫稍稍检查了一番，这只玉镯呈碧绿色，通体纯净，没有任何杂质。
<br/>　　这的确是空间宝器，其上弥漫着隐晦的空间波动，与他的虚界神图一般无二。
<br/>　　只不过，他不是此玉镯的主人，无法探查其内的情况。
<br/>　　“苏莫，你看……你能不能饶我一次？”巫森颤声问道。
<br/>　　“不能！”苏莫摇了摇头，随即一记手刀猛然砍出，直劈巫森的难道。
<br/>　　犀利的手刀快如闪电，重如泰山，杀机无限。
<br/>　　“不！”巫森惊骇欲绝，顿时狂吼了起来，可是他的神丹和战魂都被封住了，完全没有半点反抗之力。
<br/>　　嘭！
<br/>　　没有任何意外，手刀斩过，巫森的脑袋如同一个气球般炸开了，四分五裂，血液伴随着脑浆迸射四方。
<br/>　　随即苏莫手掌一探，直接插.进了巫森的小腹中，将对方的神丹直接抓了出来。
<br/>　　“不错！”苏莫点了点头，便将神丹收了起来，巫森毕竟是武帝境武者，虽然实力不强，但神丹中的玄力还是不弱的。
<br/>　　看着倒在地上的巫森，血液横流，苏莫立刻取出了十亿魔剑，直接插在了对方的胸膛上。
<br/>　　霎时间，巫森的肉身快速干瘪了下去，十亿魔剑血光爆射，猩红刺目。
<br/>　　十亿魔剑现在的品级，已经超越了一般的帝级上品神兵，到了接近尊器的地步。
<br/>　　虽然现在魔剑吸收血液，还能提升威力，但提升的极为有限了，就算有无尽的血液，能不能提升到尊级，都不一定。
<br/>　　毕竟，魔灵虽然可以提升实力，但魔剑的材质限制了它。
<br/>　　旁边不远处，秦不死三人静静的看着，巫森的死对于他们来说无所谓。
<br/>　　少倾之后，魔剑吸光了巫森的血液，苏莫便收起魔剑，准备炼化空间玉镯。
<br/>　　但此时，他突然一拍脑袋，差点忘了一事，战魂，巫森的战魂何在？
<br/>　　只要巫森战魂不灭，对方就不会死！
<br/>　　“前辈，他的战魂被封在了体内？”苏莫看向秦不死问道。
<br/>　　“嗯！”秦不死点头。
<br/>　　苏莫闻言，面现沉吟之色，他的吞噬武魂可以吞噬他人武魂，那应该也能吞噬战魂吧！
<br/>　　毕竟，战魂是武魂进化而来！
<br/>　　不过，战魂毕竟是高于武魂的存在，现在的他能不能吞噬还需一试才能知晓。
<br/>　　沉吟了少许，苏莫一拳轰出，直接轰在了巫森的尸体上。
<br/>　　轰！
<br/>　　一声爆响，伴随着大地的震动，巫森的尸体化为了飞灰。
<br/>　　嗷！
<br/>　　一声狼嚎陡然响起，便见到一只三头妖狼，身形一闪，就欲逃离此地。
<br/>　　这三头妖狼，正是巫森的战魂，地狱三头狼。
<br/>　　“逃得了吗？”苏莫轻笑一声，他早有准备，怎么可能让对方逃了？
<br/>　　就在地狱三头狼战魂刚一现身之际，苏莫大手一抓，一只巨大的玄力掌印便盖压而下，一把将三头狼战魂擒拿。
<br/>　　嗷嗷嗷！！
<br/>　　地狱三头狼不停的嚎叫，疯狂的挣扎，但根本无济于事，无法挣脱苏莫的玄力大手。
<br/>　　“给我吞！”苏莫试着催动了吞噬武魂，随即，地狱三头狼战魂立刻全身剧震，而后全身颤.抖了起来。
<br/>　　“哈哈！果然可以！”苏莫见此大喜，立刻释放出了吞噬武魂，将吞噬之力催动到了极致。
<br/>　　轰隆隆！
<br/>　　吞噬武魂如同一个巨大的漩涡，漩涡中漆黑如墨，深不见底，恐怖的吞噬之力倾泄而出，将地狱三头狼完全笼罩。
<br/>　　“这……这是怎么回事？”
<br/>　　地狱三头狼口吐人言，眸中尽是惊恐之色，它的身躯如同糠筛一般颤.抖。
<br/>　　若不是其被玄力大手抓着，已经被吸进了漩涡中。
<br/>　　没有肉身玄力的加持，地狱三头狼即便是战魂，也挡不住吞噬之力。
<br/>　　地狱三头狼中，拥有巫森的神魂，也可以说是他的命魂所在。
<br/>　　巫森真的怕了，惊恐欲绝，他已经嗅到了身死魂灭的味道。
<br/>　　“死吧！”苏莫手臂一动，玄力大手猛然一甩，直接将巫森扔进了吞噬武魂的漩涡之中。
<br/>　　唰！
<br/>　　霎时之间，地狱三头狼便消失不见，被完全吞噬了。
<br/>　　苏莫身形伫立不动，心神仔细的感受，地狱三头狼战魂被他吞噬之后，一股绝强的魂力涌入了吞噬武魂之中，被其完全吸收。
<br/>　　可惜，这股魂力虽然超强，但苏莫预想中的武魂晋阶，却并没有出现，他的武魂依旧是天级五阶。
<br/>　　叹了口气，苏莫微微摇了摇头，不能晋阶也无所谓了，他也不强求。
<br/>　　能证明吞噬武魂可以吞噬他人的战魂，便足够了！
<br/>　　之前他斩杀了大量的巫族试炼者，只不过为了追杀巫天御，那些人的武魂他也没来得及吞噬。
<br/>　　不远处，秦不死三人目瞪口呆，满脸的呆滞之色。
<br/>　　他们简直不敢相信自己看到的一幕，巫森的战魂，居然被苏莫的武魂给吞了！
<br/>　　这……苏莫的到底是什么武魂？
<br/>　　居然如此的恐怖，能吞噬他人的武魂！
<br/>　　秦不死三人心中的震撼，久久无法平息。
