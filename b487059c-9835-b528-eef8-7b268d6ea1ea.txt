　　随着奇怪老人经过一间间关押的牢笼，里面各种狰狞凶煞的生物一看到老人，顿时吓得鸦雀无声，浑身瑟瑟发抖，仿佛看到了什么无比可怕的存在。
<br/>　　就像看到了屠夫似的，担心被宰。
<br/>　　老人在监狱中不断的深入，不知不觉走到二阶精怪的区域。
<br/>　　这时，老人停在了一间特殊精怪打造的铁牢房面前，浑浊的眼睛望去。
<br/>　　铁牢房内，关押的是一头威风凛凛，狂猛无匹的老虎。
<br/>　　此老虎高大威猛，目光炯炯冒绿芒，血盆大口一张，露出比宝剑还要犀利的獠牙，能轻而易举的将一个人绞碎。
<br/>　　最为可怕的是这老虎的气息非常恐怖，一股百兽之王的皇者霸气无形之中就散发出来，让人不由自主的心生害怕，无法自控。
<br/>　　这是一头修炼成精的二阶虎精，被关押在妖精监狱中已经有二年多的时间了。
<br/>　　曾经它残忍的将一个三百人口的村庄给吃光了。
<br/>　　罪孽深重，所以惹来了妖精管理局的人，花了整整一个月的时间，才将这只虎精给关押了进去。
<br/>　　“嗷...”
<br/>　　随着老人佝偻的站在铁牢前，一声怒吼声冲天而起，旋即就是一股巨大的气浪排山倒海而出，震的铁牢笼都不禁颤了颤。
<br/>　　气浪席卷到老人的面前，却无声无息的溃散掉了，风轻云淡。
<br/>　　老人盯着这头虎精，发出苍老的声音：“不错，被关押二年，还没抹去你的锐气，正合适”
<br/>　　“该死的人类，放虎爷出来，不然虎爷一口吃了你”
<br/>　　虎精眼发绿芒，无比凶猛，散发出来的气息让空间隐隐波波动涟漪。
<br/>　　这是一头无限接近三阶的精怪，有证道的希望，只可惜被关押了进去，别说证道了，能活着就算不错。
<br/>　　“老头我都半截身子入土了，难为你还想吃我，那么老头就进来让你吃吧，就怕你没那么好的牙口”
<br/>　　说罢，老人一双皮包骨的手拿出一把锁，轻松的打开了铁牢的门。
<br/>　　霎时间，里面的虎精虎眼精光四射，待老人才一脚迈进去，它猛虎出山的咆哮一声，扑了过去，通灵摄厉，无比凶煞。
<br/>　　这等场景，换做一般人，早就吓得三魂不见了七魄。
<br/>　　但老人面对如此猛兽，却泰山崩于前而面不改色，仿佛就是来给虎精送食的。
<br/>　　倏然，虎精猛前，老人浑浊的眼神轻轻一瞥，透着无比阴森，这一瞥，仿佛时间被静止，一股窒息感侵袭，虎精顿时没来由的害怕起来，好像陷入了地狱深渊。
<br/>　　紧接着，一道凄惨的声音从铁牢中传出。
<br/>　　这叫声不是老人发出，而是属于虎精。
<br/>　　只见不知何时，老人皮包骨的手抓着威猛的虎精让它动弹不得，用力一撕，一块虎皮被硬生生的撕下，蕴含生命的鲜血滚滚而流。
<br/>　　“虎血，至阳，蕴含能量，能让老头我恢复一点气血”
<br/>　　恍然间，一人一虎的角色对换，不是虎精残忍的吃掉老头，而是老头残忍的趴在虎精的身上，疯狂的吸食它的血液。
<br/>　　虎精发出凄惨无比的叫声，透着无比绝望，既然毫无反手之力。
<br/>　　要知道这可是一头无比接近证道的虎精。
<br/>　　周围一些牢笼中的精怪，此时全都瑟瑟发抖，一片死灰，死亡的恐惧感侵袭着它们，不敢动弹一分。
<br/>　　不少精怪口吐人言：“这万恶的老头又来了，之前一年才吸食一头精怪的精血维持，现在不到三个月，就来吸食一头精怪，这笨虎既然敢惹他”
<br/>　　“我们不逃出去，迟早也要被这老头给杀了”
<br/>　　“都说我们这些妖怪残忍，这老头比我们残忍一百倍”
<br/>　　老人疯狂的吸食虎精的血液，他浑浊的眼睛开始炯炯有神起来，皮包骨的手丰盈了起来，整个人开始返老孩童，慢慢的年轻，仿佛时间在他身上返回。
<br/>　　他从一个古稀之年的老人，已肉眼可见的速度变成了一位中年男人，高大威猛，让人觉得不可思议，叹为观止。
<br/>　　十分钟后，虎精整整瘦了三圈，变成了皮包骨的样子，奄奄一息的倒在地面，那双眼神充满了恐惧。
<br/>　　变成中年男人后，他嘿嘿一笑：“你的虎血效果真的不错，看在效果好的份上，本座就在养养你”
<br/>　　他转身踏出这间牢笼，一位黑衣男子骤然出现，跪在他的面前。
<br/>　　“别让这头老虎死了，给它吃点好东西补补，或许还能让本座将就的再用一次”
<br/>　　此话落到奄奄一息的虎精耳中，浑身猛然一颤，顿时晕迷了过去。
<br/>　　“是，局长”黑衣男子俯首领命。
<br/>　　局长，局长，没错，这看似用阴邪之术返老孩童的老人，就是这温城妖精管理局的局长，位高权重。
<br/>　　对任何被关押的精怪都拥有生杀大权。
<br/>　　他因为修炼出差错，寿命骤减，气血败退，很容易衰老，必须要喝一些强大生物的精血，才能维持年轻的面貌。
<br/>　　他此时变成中年男子样子后，把笑容挂在脸上，然后开始欣赏一间间牢笼中关押的精怪，那神情惬意的就跟逛动物园似的。
<br/>　　里面的精怪，但凡被这中年男子看上一眼，便诚惶诚恐，害怕下次自己受到虎精一般的凄惨待遇。
<br/>　　局长慢慢看，慢慢走出去，快要离开妖精监狱的时候，此时急急忙忙来了一位男子，男子手中捧着一盏特殊的灯，灯是暗的，没亮。
<br/>　　“局长大事不好，冷霜组长在外殒命”
<br/>　　他捧的是生命灯，每一位妖精管理局的人，都有属于自己的生命灯。
<br/>　　人死如灯灭。
<br/>　　但凡修武者在外面死去，留在管理局的生命灯就会灭掉。
<br/>　　局长看着属于冷霜的生命灯暗了，眼神闪烁不断，森然无比的勃然大怒：“是何方神圣，既然敢杀我妖精管理局的组长，简直找死，给我查，彻查”
<br/>　　“属下明白，立刻下去查”
<br/>　　来人离开后，这局长却忽然笑了起来，笑的无比的开心：“冷家安排进来的女人终于死了，这下，没人在跟本局长作对了”
<br/>　　这是谁做的，简直帮他除了一个眼中钉。
<br/>　　由于他身有暗伤，气血容易衰老，所以他这个局长的位置就被人虎视眈眈。
<br/>　　谁都想坐上妖精管理局局长的位置，位高权重。
<br/>　　这个冷霜，就是其中一位。
<br/>　　冷霜可是天赋异禀的觉醒者，更身负冰霜元素能力，非常的有潜力，既然能死在了外面。
<br/>　　到底是别的势力偷偷动的手，还是死在了什么强大的妖精手中。
<br/>　　局长开始目露沉思，一副老谋深算的样子。
