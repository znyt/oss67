　　第107章
<br/>　　青牛山有五十几家山寨，老弱妇孺都算上，大概有五千多山贼。蛤蟆寨是有几万斤粮食，却是杯水车薪，养活不了这五千多口子人。
<br/>　　苏樱把程大雷引进后院屋子，等火炉烧起来，屋里才渐渐暖和起来。程大雷把袍子脱下来，自己盘腿坐在炕上。
<br/>　　“你怎么想起管这件闲事？”苏樱。
<br/>　　“天天眼前都有人死，若是看不见也就罢了，看见了总也不能装作看不见。”
<br/>　　苏樱盯着程大雷的脸看了一会儿，将面前的茶杯斟满，慢慢说：“落叶城没你要的粮食。”
<br/>　　“没粮？”程大雷睁大眼睛。
<br/>　　“赤旱千里，不仅落叶城没粮，整个幽州城都没你要的粮食。而且如今粮价暴涨，你有钱也买不起。”
<br/>　　“我可以抢的？”
<br/>　　“抢一二十万斤粮食？”
<br/>　　“不太现实哈。”
<br/>　　“呵呵。”
<br/>　　“难道就真的做不到么。”程大雷。
<br/>　　“却也并不是没有法子，只是一般人做不到而已。”
<br/>　　“其实我这个人除了长得俊一点，也挺一般的。”
<br/>　　“……”苏樱。
<br/>　　……
<br/>　　“你山寨能换粮食的只有那些辣椒。落叶城在幽州不过是一个小地方，幽州在帝国也是个小地方，整个帝国是东贫西贵，南富北贱，想要换粮食，就要往南走。”苏樱用手蘸着茶水，在桌上画出一道线，最后停在一点，写下两个字：扬州。
<br/>　　扬州熟，天下足，从古至今，扬州都是天下粮仓。
<br/>　　“从这里到扬州有多远？”
<br/>　　“穿过冀州，兖州，豫州，走上三五个月总是能走到的。”
<br/>　　程大雷无语的看着苏樱，那表情分明再说：你这不是扯淡么，等我回来，人都差不多饿死了。
<br/>　　“陆路就要走这么久，落叶城靠海，可以走水路，顺风五天能到，来回大概要半个月左右。”
<br/>　　“十五天么？”程大雷想了想：“好像也只剩下这一个办法。”
<br/>　　“这是在一切顺利的情况，可大海波涛汹涌，瞬息万变，海盗、河匪、当地豪强，一个不好，便是有去无回。曾经也有不少人走这条路，却是九死一生。”
<br/>　　程大雷直起身，抓起炕上的袍子：“那就这样定了，你帮我备一艘大船，我今晚就要用。”
<br/>　　走出房门，院内的伙计厨子都在，到了冬日，湖底捞的生意也冷清下来，总的说来，他们最近都很闲。
<br/>　　“大当家，这就要走了，不多待一会儿么？”
<br/>　　“走了，走了，歇不了啦。”
<br/>　　程大雷出了落叶城，一路快马回了蛤蟆寨。
<br/>　　“大当家，你去做什么了，这么冷的天？”
<br/>　　蛤蟆寨和和睦睦，因为天太冷的缘故，连日常训练都没有。校场零零散散的人，都是吃饱了饭出来消化食的。
<br/>　　“忙活起来吧弟兄们，要干活喽。”
<br/>　　……
<br/>　　高飞虎受伤之后，一直卧床养伤，最近已有些好转。但他不愿走出屋子，关键是不知道有人向他要粮，他该怎么办。
<br/>　　飞虎寨算是青牛山最大的一家势力，但人多每天吃的粮食也多，今年大旱，抢东西也没什么收获。还亏得从蛤蟆寨换回来几百斤粮食，才撑了一些日子。
<br/>　　但熬到今天，飞虎寨也马上就要断粮了。这个漫漫长冬天，自己该要怎么熬过去。
<br/>　　每到饭点，都有无数双眼睛往蛤蟆寨方向望，看着那里袅袅升起的炊烟流口水。
<br/>　　“蛤蟆寨有几万斤粮食吧。”
<br/>　　“人家才不用饿肚子，要我能活在蛤蟆寨就好了。”
<br/>　　这样的声音，高飞虎总能听到一些，他也没有办法。如果可能的话，他也想加入蛤蟆寨了。
<br/>　　“大当家，我有个主意……”于虬然走进屋子道。
<br/>　　“军师，你有主意！”高飞虎眼前一亮：“你有办法搞到粮食？”
<br/>　　“没有。”
<br/>　　“那什么主意？”
<br/>　　“我派几个手脚利索的兄弟趁天黑潜入蛤蟆寨。”
<br/>　　“偷点粮食出来？”高飞虎眼前又是一亮。
<br/>　　“不，偷估计够呛，我们一把火烧了蛤蟆寨的粮仓，到时候大家要死一起死，再让他们猖狂。”
<br/>　　高飞虎看了于虬然好久：“军师，你可够狠的呐。”
<br/>　　“量小非君子无毒不丈夫，凭什么大家都要饿死，就蛤蟆寨活着。”
<br/>　　“烧人粮仓，这种缺德事，会断子绝孙的……”
<br/>　　『好消息』
<br/>　　『好消息』
<br/>　　正这个时候，锣声从山下传来。这声音一到耳边，高飞虎就感觉头一阵阵疼。
<br/>　　这声音好久没听见过了，怎么又响起来了，程大雷这又搞什么幺蛾子。
<br/>　　“妈的，我亲自去看看，他要再搞鬼，就烧了他的粮仓。”
<br/>　　高飞虎怒发冲冠，带着于虬然亲自赶往蛤蟆寨。等他到了的时候，发现蛤蟆岭下已经聚了太多人。各家各寨的首领都到了，看来大家这些日子吃不饱肚子，每个人都够闲的。
<br/>　　“高寨主来了，高寨主……”
<br/>　　高飞虎走过去：“蛤蟆寨这次又搞什么鬼？”
<br/>　　“我们也是刚到，还没弄明白怎么会事，估计又想出了什么坏主意。”
<br/>　　“高寨主，咱们商量一下，大家联手，打下蛤蟆寨。”
<br/>　　这个时候，蛤蟆寨的大门开了，众人都望过去，只见先出来的是高飞豹和小白狼两拨人。
<br/>　　“兄弟，程大雷没有怎么着你吧？”
<br/>　　“他敢！没看我都胖了么。大哥，你可是瘦了。”
<br/>　　十几个人都是白白胖胖，高飞虎看着这一幕，枉自己还担心他，看他的样子，当初该是自己被抓进去啊。
<br/>　　程大雷这时才走出来，冲着山下的人挥挥手。
<br/>　　“诸位，诸位……”
<br/>　　“程大雷，你又耍什么幺蛾子，葫芦里究竟卖什么药？”
<br/>　　“诸位，告诉大家一个好消息。从今天开始，蛤蟆寨开始放粥。”
<br/>　　山下的众头领还在乱哄哄议论，突然间，所有的声音都安静下来，众人恍然回过神来，几百双眼睛都看着程大雷。
<br/>　　“我没有听错吧？”
<br/>　　“他说得是真的嘛，该不会耍我们吧。”
<br/>　　“他有这么无聊么？”
<br/>　　“有。”
<br/>　　“程大雷，你究竟搞什么，你为什么这么做？”高飞虎盯着程大雷。
<br/>　　“为什么？”程大雷摸摸鼻子，无比严肃道：“因为我是个好人。”
<br/>　　
<br/>(本章完)
