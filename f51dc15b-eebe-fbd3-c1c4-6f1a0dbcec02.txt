　　“哈哈！”
<br/>　　大祭司闻言大笑了起来，摇了摇头道：“从今以后，你便是我族的少司，地位仅次于核心长老，凌驾在普通的长老之上，族中谁敢找你麻烦？”

<br/>　　“我估计不会超过五招！”
<br/>　　“那也不一定啊！巫尘能在考核中得到最好的成绩，天赋亦是超强，说不定战力也很逆天，能撑个十几、二十招呢！”
<br/>　　广场上嘈杂的声音此起彼伏，所有人都是在谈论巫尘和巫人王的战斗，各种猜测的声音不绝于耳。
<br/>　　见大祭司等人的身形突然出现在看台上，所有人顿时精神一震，立刻凝目完全。
<br/>　　嘶！
<br/>　　这一望之下，倒吸冷气的声音此起彼伏，所有人都愣在了当场。
<br/>　　只见巫尘完好无损的站在大祭司的身边，而另一边的巫人王，却是面无人色，眸中神光涣散，若不是被大祭司扶着，都要倒下去了。
<br/>　　众人震惊莫名，心中掀起惊涛骇浪，面上均是露出不可思议之色。
<br/>　　巫人王败了？
<br/>　　巫尘胜了？
<br/>　　这怎么可能？
<br/>　　巫尘是怎么打败巫人王的？
<br/>　　所有人震惊的同时，亦是心中充满了疑惑，巫人王战力无双，叱咤风云多年，同阶之中未曾一败，为何这次会败了？
<br/>　　难道巫尘的战斗力，真的超越了常理，达到了逆天的地步！
<br/>　　巫天御、巫源和巫穹三人，脸色难看了起来，巫尘成为了少司，会不会对付他们？
<br/>　　巫穹还好一些，他毕竟只是言语有些得罪了巫尘，只要道个歉就好。
<br/>　　但巫源和巫天御两人，却是心中微沉，眸中闪过阴霾之色。
<br/>　　“从今日起，巫尘成为族中少司，这次少司选拔就此结束！”大祭司环视众人，朗声说道。
<br/>　　众人闻言唏嘘，这少司之位，最终还是属于巫尘了，巫人王彻底的一败涂地。
<br/>　　“恭喜巫尘少司！”
<br/>　　“恭喜巫尘少司！”
<br/>　　“恭喜巫尘少司！”
<br/>　　霎时之间，呼喝之声四起，广场上所有的人都向苏莫抱拳，面现祝福之色。
<br/>　　无论以前巫尘如何，但现在成为了少司，就彻底崛起了，一跃成为族中高层，身份地位不可同日而语。
<br/>　　所以，所有人均是收起了轻视之心，再也不敢小觑巫尘。
<br/>　　苏莫站在看台上，目光扫视下方无数抱拳的巫族之人，心中亦是豪气干云。
<br/>　　他的豪气，不是要在巫族发展，闯出一番名堂，而是要祸乱整个巫族，搅动腥风血雨。
<br/>　　少司的身份，将让他无往而不利。
<br/>　　当然，这一切都交给分身了，他的本体要尽快撤离出去，免得陷入巫族的泥潭之中。

<br/>　　“是，大祭司！”苏莫点了点头。
<br/>　　“嗯！”
<br/>　　大祭司微微颌首，随即带着巫人王闪身而去。
<br/>　　苏莫见此眸光一闪，大祭司居然带走了巫人王，会不会等巫人王清醒过来之后，询问对方刚才的战斗情况！
