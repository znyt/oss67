　　萧逸回到席位上。
<br/>　　此事，并未再多想。
<br/>　　傲东楼，只是一番好意，且打算与他交好罢了。
<br/>　　在傲东楼看来，‘无上剑诀’的副本，本就纳入了奖励之中。
<br/>　　也就是说，剑诀，终归会泄露出去的。
<br/>　　而易霄，若继续参加比拼，取得第一名，几乎是铁板钉钉之事，终究会得到剑诀。
<br/>　　故他倒不如现今卖个人情，直接将‘无上剑诀’的副本给易霄。
<br/>　　毕竟，在他的猜测中，易霄不缺名额，应该是为了‘无上剑诀’而来的。
<br/>　　只是，他万万没想到，易霄来此参加的原因，很是单纯，仅仅是为了第一名的丰厚奖励罢了。
<br/>　　席位上，萧逸刚要闭目假寐。
<br/>　　一旁，冉琦凌厉的目光直直视来。
<br/>　　“一拳便能败贺幽，将他轰下比武台，赢得干净利落。”
<br/>　　“易霄，你比我想象中的要强。”
<br/>　　萧逸瞥了眼冉琦，“贺幽，不过是接近武道大能的修为。”
<br/>　　“我没看错的话，他才9400道的水准，还差一百条完整武道，方入大能。”
<br/>　　“以你的实力，同样能在数招之内结束战斗。”
<br/>　　十八府天骄，没一个是泛泛之辈。
<br/>　　这般年纪，便有此等修为，换了别人，可是百年乃至数百年都不一定能达到的高度。
<br/>　　拿贺幽来说，必定是从小便受全幻光府倾力培养。
<br/>　　武道方面，有早就踏入绝世多年的老府主，有一众动辄武道大能层次的长老，可以时常为其解疑。
<br/>　　资源方面，更不必多说，根本不可能有欠缺一说。
<br/>　　年幼之时，便能做好完美的武道启蒙，打好武道根基，随后武道之路一路高歌，突破连连。
<br/>　　一年前，不，未足一年，顶多十月左右，血光洞府内，第二次武魂觉醒后，拥有黑色武魂，修为更踏入圣皇境后期。
<br/>　　虽那时被萧逸直接重伤，还损了武魂。
<br/>　　但幻光府全力之下，还是替他修复好了武魂与伤势。
<br/>　　这才不足一年的时间，他便能一路突破至圣皇境巅峰，仅差些许便几乎踏入武道大能层次。
<br/>　　而冉琦，则比他更出色。
<br/>　　冉琦，毫无疑问，是个武痴。
<br/>　　说到武痴，还有性格，萧逸不禁想起了当年在东域时的钟无忧。
<br/>　　不过，对比冉琦，钟无忧却少了一份韧劲，还有狠劲。
<br/>　　对，狠劲。
<br/>　　萧逸不会看错，冉琦此人，对武道，有着远比他人更强的执着。
<br/>　　特别是那一身敛而不发的气势，还有那双凌厉的眸子，无不表明着他这几年必然是千锤百炼，此刻正是锋从磨砺出，随时可一朝爆发。
<br/>　　在萧逸的判断中，此刻的冉琦，甚至远强于顾非凡这个天藏学宫第一天骄。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　整整三天时间，比武台上的战斗，仍旧在继续。
<br/>　　天骄上场，落败，下场，新的天骄上场比拼，周而复始，从未间断。
<br/>　　可即便如此，要想真正比完所有比拼，还需不少时间。
<br/>　　来东离剑宫参加名额争夺的势力，何止数千。
<br/>　　论比拼场数，还有时间，此次的争夺，将远甚于百院之争。
<br/>　　来此的各大势力、家族、宗门，也远比当日的各大学宫数量要多得多。
<br/>　　不过有一点不同。
<br/>　　百院之争时，各大学宫的比拼，终究带着一点考验性。
<br/>　　故百院之争时，有三轮比拼，而且规矩各异。
<br/>　　而东离剑宫这里的名额争夺，则简单得多，名额，强者得知。
<br/>　　这场比拼，会一直进行下去，期间不会有半分中断。
<br/>　　争夺，一直到所有天骄比拼完毕，定下名额归属为止。
<br/>　　萧逸猜测着，应该还要三、四天时间，这场比拼才能完全落下帷幕。
<br/>　　……
<br/>　　两天后。
<br/>　　比武台上，又是一轮比拼落下。
<br/>　　裁判未有半分迟疑，高声宣布下一场比拼，“下一战，血光府，血柏。”
<br/>　　嗖…
<br/>　　一身血衣的血柏，跃上比武台。
<br/>　　衣袂，在空气中划过一道幻影，如若血影飘拂，潇洒间又带着一抹凌厉。
<br/>　　“又轮到血柏上场了。”
<br/>　　“这两天，血柏已上场数次，每次皆是轻松胜出。”
<br/>　　“看来，此番名额争夺，他会是十八府中最有力的争夺者。”
<br/>　　“不过，血柏下手狠辣，与他对战，不死也残。”
<br/>　　“接下来，便要看哪个倒霉蛋碰上他了。”
<br/>　　这两天时间，血柏的表现，无疑异常亮眼。
<br/>　　屡战屡胜，动作潇洒而凌厉，每每赢得干脆利落。
<br/>　　“剑宫裁判。”比武台上，血柏笑笑，“愣着做什么？还不快宣布我的对手？”
<br/>　　“额。”剑宫裁判，手中拿着名单，但刚才，却愣了一阵子。
<br/>　　此刻闻言，反应了过来，高声道，“易霄，上台。”
<br/>　　“易霄？”这次到血柏一愣。
<br/>　　“易霄？”周遭观赛席，亦是同时一愣，鸦雀无声。
<br/>　　但这鸦雀无声，仅仅持续了一秒，随后便是同一时间的惊呼声响彻全场。
<br/>　　“时隔数天，又到易霄上场了。”
<br/>　　“啧啧，看来倒霉的是血柏。”
<br/>　　嗖…一道火光，跃上比武台。
<br/>　　正是萧逸。
<br/>　　这次比拼的规矩，虽然简单，但对战者间，却不是胡乱比配。
<br/>　　东离剑宫，主持此次比拼，故剑宫的长老、执事，会负责对手间的抽签等等。
<br/>　　而这抽签，则是根据东离剑宫长老对中域各大天骄的实力判断，而分出一个个层次。
<br/>　　同一层次间，才会被抽中互相对战。
<br/>　　拿十八府天骄来说，他们属于霸主天骄的层次，他们互相间可能会对上，也可能会对上一流势力的天骄。
<br/>　　但他们绝不会对上那些二流势力的天骄对手。
<br/>　　至于萧逸，能与他作为同层次对手的，也就十八府天骄，还有寥寥几个声名赫赫的中域绝世天骄。
<br/>　　故这数天时间，他几乎都没有上场，也就上一场对战贺幽，之后一直未上过场。
<br/>　　实在是东离剑宫的裁判，在他那一层次中，能抽出对战的对手，太少。
<br/>　　这不，足足数天时间，才再次抽到他。
<br/>　　这也是他自比拼开始以来，第二场战斗。
<br/>　　“是你自己下去，还是我轰你下去？”萧逸只淡漠地吐出一句话。
<br/>　　“你…”血柏之前那一身潇洒的动作再无，取而代之的，是不自觉地后退了两步。
<br/>　　贺幽的前车之鉴摆在面前，他自是心头一惊。
<br/>　　“易霄，我承认你很强。”血柏咬了咬牙。
<br/>　　“但，若你当真如此自信的话，可敢接我一招？”
<br/>　　血柏的目光，忽然变得戏谑，甚至带着些许挑衅意味。
