　　江行泽的想法是极端的，他全然忘记了没有蛇博士陈默，他早就在地狱中排队准备投胎了。
<br/>　　跑？
<br/>　　此时的吸血蛇发狂了，像是失去理智，江行泽感觉一股危机感笼罩全身。
<br/>　　他虽然变成不人不鬼的样子，也同样还是怕死的，狼狈的起身，朝着铁门的位置跑去。
<br/>　　“吼...”
<br/>　　吸血蛇仰天咆哮，倏然，化为一道火影子挡住了江行泽的去路。
<br/>　　二十米的庞大蛇躯，巍峨的犹如一座无法翻越的大山。
<br/>　　吸血蛇血盆大口一阵阵作呕的血腥味滚滚喷发，巨大的蛇头摇曳在江行泽的上空，好像随时要一口活吞了他。
<br/>　　江行泽真的怕了，此时已经不是危机感，还是死亡弥漫。
<br/>　　“你若不想认我为主，就算了，你不能杀我，杀了我，你不但没有自由，连吃人肉的机会都没有了”
<br/>　　江行泽面目可憎的脸，满是惊恐，越发的丑陋不堪。
<br/>　　“吼...”
<br/>　　吸血蛇又是仰天咆哮一声，蛇口猛然冲下，江行泽吓的闭上了眼睛，吸血蛇要杀他，他只能等死。
<br/>　　这座特质的牢笼昏暗潮湿，就像是地狱一般让人压抑，已经有不少人死在这里，死了连一块骨头都没留下。
<br/>　　比死无全尸还要惨。
<br/>　　他的耳边不绝于耳的响起吸血蛇的咆哮，它浑身火红鳞片闪烁着冰冷的幽光。
<br/>　　可...半响...
<br/>　　他还无事，还活着，耳边响起的不在是吸血蛇发狂的咆哮声，而是一道沙哑阴森的声音：“主人...”
<br/>　　嗯？
<br/>　　江行泽猛然掀开眼，只见二十米庞大的吸血蛇已经匍匐在自己的面前，巨大的蛇头就趴在他的脚跟前。
<br/>　　“认我为主了？还是蛇魂发挥的作用”
<br/>　　江行泽怔怔的连兴奋激动的心情都遗忘了，看着吸血蛇，忐忑的说：“你在叫一声”
<br/>　　“主人”
<br/>　　吸血蛇血口一张，口吐人言，那双铜铃般的蛇眸散发异样的光芒。
<br/>　　“哈哈...天助我也”
<br/>　　光线昏暗的牢笼顿时响起江行泽肆无忌惮的笑声，笑的张狂，笑的不可一世。
<br/>　　有了吸血蛇为宠，他什么事情不可做。
<br/>　　什么仇报不了。
<br/>　　大笑后的江行泽仿佛又变成了那个不可一世的男人，目中无人，他放开手脚的直接骑在了吸血蛇的身上。
<br/>　　“给我起”
<br/>　　他就要看看吸血蛇是不是真的臣服于他，认他为主。
<br/>　　吸血蛇动了，直接将江行泽拖到了十几米高的空中，任由他骑在自己的身上。
<br/>　　江行泽狂喜，这吸血蛇要不是真心认他为主，已此蛇的凶煞跟狂傲，是不可能让他骑在自己的身上。
<br/>　　成功了，这条绝世蛇种终于属于他了。
<br/>　　就在此时，铁门外冲进来三个全副武装的保安，看到江行泽骑上二十米庞大的吸血蛇的身上，顿时把嘴张得像箱子口那么大，一下子就愣住了，接着他咽了两三口唾沫。
<br/>　　“江少，你在做什么，博士发话，你不能在进来了，门卫陈超呢”
<br/>　　这牢笼是有门卫守护的，任何人不得进来。
<br/>　　这条吸血蛇何等凶煞，外人谁敢进来，博士的命令就是针对他的徒弟。
<br/>　　“我师父可没有跟我说，我不准进来”江行泽邪魅的笑着，然后看着三名保安，继续道：“你们要是找那门卫，恐怕是找不到了”
<br/>　　保安顿惊：“江少，你什么意思”
<br/>　　“因为门卫陈超已经在吸血蛇的肚子里了，你们要找他，唯有进吸血蛇的胃里找了”
<br/>　　三名保安闻言，面面相觑，相互的眼神对碰一眼，感觉大事不好。
<br/>　　“吼...”
<br/>　　吸血蛇仰天一声咆哮，通灵摄厉的朝着三名保安嘶吼一声，铜铃般的蛇眸满是嗜血贪婪。
<br/>　　被吸血蛇盯着，让三名保安浑身冷颤，从上到下的僵硬起来。
<br/>　　“不好，快走”
<br/>　　三名保安不是傻子，调头就准备离开牢笼。
<br/>　　只见江行泽从吸血蛇的身上跳了下来，乖张暴戾的说：“既然来了，那么就别走了，不是要找陈超了，还是进吸血蛇的肚子里慢慢找吧”
<br/>　　话音还未落，只见吸血蛇化为火影子骤然将三名保安一圈一圈的盘卷起来，困在其中。
<br/>　　“江少，你疯了吗，你想拿我们喂吸血蛇，博士知道，不会饶过你”
<br/>　　江行泽得到吸血蛇，什么都不怕了，嘎嘎一笑：“要怪，就怪你们命不好，偏偏在这个时候闯了进来”
<br/>　　“江少，饶了我们吧，我们可以当做什么都没看到”
<br/>　　死，有很多种死法，但是被一条巨蟒当食物的吃了，也死的太惨了吧。
<br/>　　稍稍一想，就感觉灵魂出窍了。
<br/>　　江行泽露出残忍的笑，像是什么都没听到，冲着吸血蛇柔声的说：“宝贝，尽情的享受你的美餐吧，吃完，我们就要离开这里了”
<br/>　　啊...啊...啊...
<br/>　　倏然，牢笼传出不绝于耳的惨叫和哀嚎，二十米的蟒蛇吃人，画面何等残忍。
<br/>　　昏暗的牢笼，潮湿的环境，更是血腥味弥漫，愈演愈烈。
<br/>　　“畜生尔敢吃人”
<br/>　　一声充满怒火的咆哮声在牢笼中激荡起来。
<br/>　　江行泽的脸色顿变，只见蛇博士陈默已经站在铁门前，他如一头发怒的狮子，瞪起了眼，眉毛一根根竖起来，脸上暴起了一道道青筋，愤怒的看着他。
<br/>　　“师父你来晚了，吸血蛇刚将三名保安将吞进了肚子里去”
<br/>　　看到陈默，江行泽短暂的心慌一下，但旋即就风轻云淡，现在的他还有什么好怕的。
<br/>　　“泽儿，你在做什么，你居然让吸血蛇吃人，你的人性跟良知都被狗吃了吗，我怎么会教出你这样的孽徒，早知如此，我就不该从阎王手里将你抢回来”
<br/>　　陈默声音由低到高，渐渐地咆哮起来，脸色涨红，进而发青，脖子涨得像要爆炸的样子。
<br/>　　他如何能不气，纵然之前有所猜测，可他没想到自己的徒弟能做到如此无法无天，公认让吸血蛇吃人，手段残忍的令人发指。
<br/>　　他呕心沥血创造绝世蛇种出来，是为了给人类做贡献，而不是一个杀人机器。
<br/>　　蛇冷血无情，陈默明白，最让他痛心疾首的是自己的徒弟悄然的已经变成了一个恶魔。
<br/>　　他让吸血蛇吃人，自己却在一边看着，他如何能做到这般的抹灭人性。
<br/>　　畜生，比吸血蛇还要禽兽不如。
<br/>　　看来自己只能大义灭亲了。
