　　“可恶！”
<br/>　　尤天韩闻听苏莫之言，顿时心中怒不可竭，他也是天才，一等一的天才，但苏莫此言却是对他莫大的侮辱。
<br/>　　他若是就这样认输了，以后在神武学府，在太史世家派系，估计就再也抬不起头了。
<br/>　　“苏莫，要战便战！”尤天韩朗声喝道，就算败了他也要一战。
<br/>　　言罢，尤天韩全身玄力鼓荡，他的手中出现了一柄战刀，刀身雪亮，寒光森森。
<br/>　　他之所以敢和苏莫一战，也是能断定苏莫不会下杀手，虽然苏莫恨他们太史世家派系的人，但在战界已经杀了很多人，气已经消了。
<br/>　　他暗中勾结巫人王一事，苏莫并不知晓，现在双方明面上已经和解了。
<br/>　　嗡嗡~~
<br/>　　尤天韩背后虚影升腾，两柄巨大的长刀战魂升腾而起，刀威弥漫八方。
<br/>　　这两柄长刀战魂，一柄是金色长刀，一柄是银色长刀，气势交相呼应，威势倍增。
<br/>　　“接我一剑，希望你不会死！”苏莫冷漠的说了一句，一步踏出，瞬息百里，而后立刻出手了。
<br/>　　他只出一剑，对方若是能侥幸活下来，他不会杀对方，但对方若是挡不住，那就死吧！
<br/>　　唰！
<br/>　　手臂一挥，苏莫一剑斩击而出，浩大无匹的三色剑气直击尤天韩。
<br/>　　这一剑虽然苏莫没有用出巅峰实力，但亦是强大无比，剑威如九天神剑，惊世骇俗。
<br/>　　由于苏莫向前踏出了一步，和尤天韩之间拉近了距离，这一剑几乎是瞬息而至。
<br/>　　强大的剑威瞬间压迫而来，尤天韩怒目圆睁，剑气还未临身，那惊天的锋芒，绝世的剑威，让他全身冰寒，如坠冰窖。
<br/>　　尤天韩知道，这一剑他必须挡住，不然绝对凄惨无比。
<br/>　　来不及多想，尤天韩全身玄力沸腾，手中战刀光芒耀眼，将玄力和战魂全部催动到了极致。
<br/>　　“破！”怒喝声震天，战刀如同一轮弯月，划破空间，裹挟亿万钧之力，狠狠的斩在了剑气之上。
<br/>　　轰！
<br/>　　一声震天的爆响在青色平台上炸开，剑气崩碎，化为万千剑雨，迸射八方。
<br/>　　嗖！
<br/>　　只见尤天韩的身形，如同一道闪电一般，直接倒飞了出去，速度快的不可思议。
<br/>　　轰！
<br/>　　又是一声爆响，尤天韩足足倒飞了一万多里，身形撞在了连接圣门的巨大气罩上，才彻底停了下来。
<br/>　　这巨大气罩正是一开始所形成，将真圣碑与外面隔开，只有通过圣门才能进入其内。
<br/>　　而后，便见到尤天韩仿佛一条死鱼被呼在了墙壁上，全身鲜血犹如泉涌，其身体顺着气罩向下方滑落，气罩上留下触目惊心的血迹。
<br/>　　尤天韩没有死，但身受重创，双臂已经被彻底震碎了，肉身已经到了毁灭的边缘。
<br/>　　下一息，滑落的尤天韩消失不见，被传送出了战界。
<br/>　　众人暗暗心惊，这苏莫是真狠啊，一剑将同门打的半死，看来双方仇怨不小。
<br/>　　一剑之后，苏莫没有多看尤天韩，二话不说便走下了青色平台。
<br/>　　苏莫心中暗叹，看来他还要再挑战一场，才能算是通过第三关的考验。
<br/>　　他目前已经挑战了两次，接受挑战了一次，但并没有什么通过考验的迹象。
<br/>　　看来每个人，都要主动挑战三次，并且三战皆胜才能过关，接受挑战并不算数。
<br/>　　随后，苏莫只能继续等待了起来，青色平台上的挑战继续进行。
<br/>　　战斗一场接着一场，没有任何一人挑战苏莫，像姜神夜、飞长云等顶级天才，也不会选择挑战他。
<br/>　　毕竟，大家都是来真圣界获取真圣之气的，并不是来分高低的。
<br/>　　姜神夜和真圣榜排名前几名的巨摩、飞长云等人，实力当真强大，简直是势不可挡，几乎没有一合之敌。
<br/>　　几人的实力，让苏莫都是有些看不透。
<br/>　　时间不断的流逝，战斗一场接着一场，毫不停歇，第二轮很快便过去了。
<br/>　　经过了两轮挑战，只剩下了一百二十余人，其余的全部被淘汰了。
<br/>　　第三轮又开始了。
<br/>　　到了第三轮，战斗更加的激烈了，因为战斗的双方，一般实力差距都不大，甚至势均力敌。
<br/>　　除了少数的几人之外，几乎每一场战斗都要战半个时辰，甚至是更久。
<br/>　　没过多久，轮到了苏莫的第三次挑战，他随便挑战了一名天才，两剑便将对方击败了。
<br/>　　至此，他三次挑战，三战全胜，算是通过了最后一关考验。
<br/>　　不过，三战全胜之后，也并没有出现什么异像，和其他三战全胜的人一般无二，看来要等到第三轮所有的战斗都结束之后，才会统一获得真圣之气。
<br/>　　战斗还在继续，留下的人越来越少，又过去了十几个时辰之后，还剩下的人已经只有五十余人了。
<br/>　　这个时候，还剩下四个人没有参加第三次挑战，其他的人全部是三战全胜的人。
<br/>　　这四人可以互相挑战，用两场战斗来彻底结束第三关的考验，也可以挑战其他人，用四场战斗来结束。
<br/>　　所以说，其他三次挑战全胜的人，也不算通过了真圣界的三关考验，因为可能还会被别人挑战。
<br/>　　这四人之中，还有一名厉害的天才人物，他就是巨摩。
<br/>　　嗡~~
<br/>　　真圣碑上，巨摩的名字亮了起来，青光大放。
<br/>　　咚！
<br/>　　巨摩一步踏出，高大雄.伟的身躯立刻落在了青色的战台上，发出一声闷雷般的响声。
<br/>　　仅剩下的三名还未参加第三次挑战的人，全部都紧张了起来，他们都担心被巨摩选中。
<br/>　　一旦被巨摩选中，他们就必败无疑了。
<br/>　　他们已经战斗到了这一步，若是最后败给了巨摩，那实在是太倒霉了。
<br/>　　巨摩伫立在青色平台上，灯笼般的眼眸瞥了一眼真圣碑，大嘴一咧，勾起了一丝戏虐般的笑容。
<br/>　　随即，他转头望了望苏莫，又望了望姜神夜，大笑着说道：“哈哈！苏莫、姜神夜，你们两人同样强大，你们说我该挑战谁呢？”
<br/>　　巨摩的目光不怀好意，他向挑战苏莫，但又想挑战姜神夜，一时间也拿不定注意。
<br/>　　众人闻听巨摩之言，纷纷心中一震。
<br/>　　这最后一战，巨摩不挑战实力相对较弱的人，居然将矛头对准了苏莫和姜神夜？
<br/>　　这是想要干什么？
