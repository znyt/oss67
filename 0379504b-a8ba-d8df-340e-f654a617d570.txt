        &lt;h3&gt;第三卷第三百零七章&lt;/h3&gt;
<br/>
<br/>        凌辰和董亚子在陈子昂的公司办公，陈子昂给了他们一间办公室，库房也借给他们用了，他们需要采集数据，这几天需要将数据采集好送回总部和相关的部门，再过两天就是三一五了。
<br/>
<br/>        他的父亲凌思永回山西之后，便没有和宁家联系，凌思永无论是从公从私来说都是心向着自己的儿子的，凌家虽然没落了，在山西也算是大户人家，每一个大的家族都有自己要培养的接班人，而凌辰就是凌思永培养的接班人，凌辰这些年的努力大家都看在眼里，凌辰也有能力自立门户，他可不愿意将自己的儿子逼出家门。
<br/>
<br/>        再加上他也看出来了，他的儿子根本就不喜欢宁玥，宁玥就是剃头担子一头热，自己追不到凌辰，想借助家里的势力来达到自己的目的，这样硬性捆绑的婚姻最后也会是以悲剧为结果，他还没有老糊涂呢。
<br/>
<br/>        宁玥这个女子的心思够深沉，小小年纪不走正途，非要整那么些幺蛾子出来，差点害得他们父子反目。
<br/>
<br/>        凌思永倒是安宁了，可是宁少天不想安宁，第一，宁少天想和凌家合作，跨界合作，第二，宁玥哭哭啼啼说凌辰占了宁玥的便宜，让他心里火大，和他大女儿纠缠不清，现在又占了他小女儿的便宜，不讨个说法，他心里膈应的慌。
<br/>
<br/>        宁少天给凌思永打电话：“凌总，你什么时候再来杭州？”
<br/>
<br/>        “宁总，最近特别忙啊，怕是一时半会儿过不去。”凌思永听了儿子的话，打着哈哈。
<br/>
<br/>        这话听在宁少天的耳朵里就是敷衍和推诿！
<br/>
<br/>        宁少天看着邮箱里的无名文件，心里左思右想，最后还是决定将这些数据提交给有关部门。
<br/>
<br/>        凌辰和董亚子正在库房清点商品的时候，接到了他老爸的电话：“凌辰，宁少天给我打电话了，问我什么时候去杭州和他谈合作的事情，我说我最近很忙，能感觉到他很不高兴。”
<br/>
<br/>        “不高兴就不高兴，难道你还能让所有的人都高兴？”凌辰似乎预料到宁少天会不高兴。
<br/>
<br/>        “你这个孩子，我是提醒你，怕你以后不好相处，你看看你弄的啥事情？”凌思永在电话那头替凌辰着急，再怎么说也是老丈人，难道以后就不见面了？
<br/>
<br/>        话说，他还得出面将这个误会解开，要不然以后怎么相处？
<br/>
<br/>        “我什么事情都没有做！好了不说了，我要工作了。”凌辰嫌弃凌思永唠唠叨叨，而且现在手头上正在看商品的条码。
<br/>
<br/>        凌辰挂了凌思永的电话，微微一笑，宁少天会怎么做呢？
<br/>
<br/>        今天他们的数据就要出来，陈子昂他们拿回来的商品都有发票和批次，有的是打的收据，说是没有发票了，等下个月给补上，像这种事情也会是有的，陈子霖就让他们把手据收好，以后好换发票。
<br/>
<br/>        下午，凌辰和董亚子的工作基本已经干完了，董亚子也准备将资料发回厂家的时候，来了一帮人。
<br/>
<br/>        “你们被举报了，说你们非法囤积烟酒。”有一个看似是领头的人拿出工作证。
<br/>
<br/>        “什么？”凌辰蒙了，他们是厂家的，原本就是来寻找证据的，现在竟然被举报了？
<br/>
<br/>        “我们是厂家的，这个是我们查处的一批有问题的商品，准备回收进行销毁。”凌辰拿出了自己的名片。
<br/>
<br/>        “你给我名片是啥意思？名片就能证明你的身份？”领头的轻蔑的一笑。
<br/>
<br/>        “那我的身份证可以证明我的身份吧？”凌辰准备拿出身份证。
<br/>
<br/>        “你的身份证只能证明你是你，你们查处有问题的商品？谁给你们的权利？”
<br/>
<br/>        “我们真的是厂家的，这些商品都是我们想办法回收的，我们在暗中调查，想知道这些有问题的商品的来源，确认好了自然就会向你们汇报的。”董亚子也站起来说。
<br/>
<br/>        “哈哈哈，真的是什么借口都会找，来人，将这里的所有东西都拉走，一件不留，拿回去检测。”那个领头的一挥手，身后几个人一拥而入，都开始动手拿东西。
<br/>
<br/>        “你们住手，这些东西你们不能拿走！”董亚在也急了，这些东西被他们拿走，岂不是就坐实了陈子霖他们公司违规囤货，而且这些东西还是有问题的。
<br/>
<br/>        “你们的领导是谁？明天让你们公司领导来我们单位。如果你们再阻碍我们执行公务，出了问题你们负责！”
<br/>
<br/>        凌辰和董亚子相视一眼，他们现在似乎什么也不能干了。
<br/>
<br/>        眼巴巴的看着东西被人拉走，他们也无能为力，不知道他们那一步走错了？
<br/>
<br/>        陈子霖赶回来的时候库房里的东西都被拉走了，陈子霖叹了口气，原本他们在地下库，谁会注意到这件事情？
<br/>
<br/>        现在的他们是跳进黄河洗不清，只是后期看怎么处理。
<br/>
<br/>        “凌总，董总，你们也不要着急，我明天去找他们，看看他们怎么说。”陈子霖安慰着凌辰和董亚子，东西在他的库房，必须得他出面解决了，况且凌辰和董亚子对西城人生地不熟的。
<br/>
<br/>        “陈总，很抱歉给你添麻烦了。明天我们和你一起去，先看看情况。”凌辰歉意的说。
<br/>
<br/>        “都是朋友，说那么见外的话干什么？对了，那些发票和收据你们拿好，只有这些东西能证明我这些货的来源。”陈子霖吩咐着。
<br/>
<br/>        “他们将抽屉里所有的东西都拿走了，包括你们的账单。”董亚子无奈的说。
<br/>
<br/>        “应该没事，今天先这样，我找人探听一下消息，看看怎么回事。”陈子昂临走的时候再三叮咛他注意，这几天忙的还是将凌辰和董亚子给疏忽了。
<br/>
<br/>        他们的库房在地下，一般很少有人关注，是谁走漏了风声？难道是他们公司的？陈子霖急匆匆的去公司进行排查，他将库房里的几个人叫到一起仔细询问，看看有没有人发现或者是走漏风声。
<br/>
<br/>        “我们都没有说什么，而且我们也不知道这批货的用途，我们怎么会说出去呢？”几个人委屈的说。
<br/>
<br/>        “那你们有没有发现可疑的人？”陈子霖接着追问。
<br/>
<br/>        “可疑的人倒是没有发现，都是我们附近的人，对了，那个顾总前几天跑我们库房来转了一圈，他问我们是不是有什么事情，拿这么多酒。”
<br/>
<br/>        “顾恩赐？”陈子霖问道。
<br/>
<br/>        “是的。”
<br/>
<br/>        陈子霖二话不说转身去找易倾秋去了。
