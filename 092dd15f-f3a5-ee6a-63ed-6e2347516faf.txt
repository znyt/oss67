　　　　一条条蛇从人的嘴中钻出来，好像对舒瑶太对残忍，她不过是一个柔弱的女人。
<br/>　　但涂小安也别无它法，要是能要药物流掉，何必如此麻烦。
<br/>　　脸色病态般苍白的舒瑶忽然深吸了几口气，坚定道：“只要能将我的蛇胎做掉，就算让我死，我都愿意”
<br/>　　这个蛇胎摧毁了她的幸福，同归于尽又能怎样。
<br/>　　涂小安凝眸看着她，到现在她还想守护自己的幸福，想着做掉蛇胎，在去找他的男朋友解释。
<br/>　　她很坚强，坚强的让涂小安佩服。
<br/>　　“我想你应该是累了，不如睡一觉吧”
<br/>　　嗯？
<br/>　　舒瑶忽然一愣，怎么又让自己睡觉了，紧接着，她脑子一阵晕眩，身子软绵绵的倒了下去。
<br/>　　那种方式，就算成功了，也会成为舒瑶永远挥之不去的噩梦。
<br/>　　涂小安要做的，就是最大程度的减少对她的伤害，希望她醒来，噩梦已经离去。
<br/>　　“意识锁定，给我起”
<br/>　　金银蛇蛇眸精光四射，只见晕迷的舒瑶娇小玲珑的身子缓缓的悬空了起来。
<br/>　　“统治能力，给我开”
<br/>　　王者之气如狂风骤射，在金银蛇身上汹涌澎湃的暴涌出来。
<br/>　　好在这里不是佘山，否则几个呼吸内，起码出现几百条毒蛇汇聚在一起的恐怖画面。
<br/>　　沉睡中的舒瑶恬静冷漠，仿若没有潮水的大海，静静的，没有一丝涟漪，涂小安的蛇种统治气息在她的身上一层层的席卷。
<br/>　　此时，不到一百平方的套房内，温度骤然下降，宛如寒冬腊月。
<br/>　　“还不给我出来”
<br/>　　金银蛇蛇眸闪闪发亮，盯着舒瑶微微隆起的腹部。
<br/>　　旋即，只见微微隆起的腹部颇动了起来，沉睡中的舒瑶睫毛微微的颤抖起来，她原本平静的面容出现一丝痛苦，她好像是在一场冗长的梦境里，经受折磨跟摧残。
<br/>　　涂小安却为之一喜，好像有作用，统治能力的召唤，让舒瑶腹中的蛇有了反应。
<br/>　　要是按系统等级来划分，她腹中的蛇应该非常的弱小，不可能抵挡的住蛇王的召唤。
<br/>　　人是哺乳动物，所以是胎生，但对于蛇类来说，有胎生，也有卵胎生，就看是什么品种。
<br/>　　舒瑶腹中的蛇早就成型，一条条缭绕在一起，密密麻麻，非常的恶心。
<br/>　　涂小安的统治能力全力释放，到了他这种等级，全力释放统治能力是非常恐怖的。
<br/>　　虽然是全力释放，但也仅仅局限在这件套房内，如果不刻意控制，那么方圆五公里内，只要有蛇类的影子，都会诚惶诚恐的冒出来。
<br/>　　这里是镇中心地带，大量的毒蛇冒出来，恐怕会引起恐慌，不知道会照成多少无辜的人受难。
<br/>　　然而，这件套房狂风肆意，所有的家具都像是长了翅膀的乱飞起来，瞬间，一片狼藉。
<br/>　　涂小安对周围的变化根本不在意，他一心观察舒瑶腹部的动静。
<br/>　　颇动愈演愈烈，像是有蛇要直接破肚而出。
<br/>　　沉睡中的舒瑶发出了痛苦的呻吟声，冷汗涔涔而出，脸色没有一丝血色。
<br/>　　“糟糕，不能这样下去”
<br/>　　涂小安俨然起来，他所想的让腹中的蛇从舒瑶的嘴中钻出来，恐怕有点天真了。
<br/>　　这些蛇受到了统治能力的召唤，蠢蠢欲动，但只是在她的腹中四处乱撞，像是找不到发泄的出口。
<br/>　　继续下去，痛都能痛死她。
<br/>　　涂小安不得已将统治能力收回，舒瑶依然还悬浮在半空中，她的腹部渐渐的动静越来越小，不在颇动了。
<br/>　　失败了，涂小安很不愿意接受这个事实，他更加不知道等舒瑶醒了，怎么跟她说。
<br/>　　说蛇胎还在，对她太过残忍，噩梦还在继续纠缠着她。
<br/>　　该死！
<br/>　　涂小安没想到区区一个蛇胎就将他难住了。
<br/>　　这个蛇胎在舒瑶的腹中，就好像打不得，骂不得，让他有力使不出。
<br/>　　“看来要换一种柔和的方法”
<br/>　　统治能力太过霸道，一旦释放，低等级的蛇类都会失去理智，一激动，就在舒瑶的腹中翻江倒海。
<br/>　　召唤不出来，那么能不能引诱出来呢。
<br/>　　涂小安换了一种思维，这些蛇如寄生虫寄居在舒瑶的腹中，舒瑶吃什么，它们吃什么。
<br/>　　可要是舒瑶不吃不喝呢。
<br/>　　当然，仅仅这样还不行，这些蛇要饿坏了，吃她的五脏六腑都不是不可能。
<br/>　　这个度要把握住，在用什么宝贝将它们慢慢的诱导出来。
<br/>　　有了，涂小安蛇眸一亮。
<br/>　　自己的蜕下来的蛇皮应该是个不错的选择。
<br/>　　早在之前，涂小安试过一回，他在佘山召唤了几百条毒蛇，扔出了一片被他切割成小片的蛇皮出去。
<br/>　　霎时间，这些毒蛇都疯狂了，前仆后继，你死我活，拼命了都想吃到蛇皮。

<br/>　　可以这样说，一条成年的毒蛇吃了它的蛇皮，可以瞬间突破桎梏，脱胎换骨，血脉提升。

<br/>　　然而你，也不能直接试，让舒瑶就这样的睡着，睡个一天一夜在说。
<br/>　　在舒瑶蛇胎不会造反的情况下，饿它们几顿。
<br/>　　到时候在用自己的蛇皮引诱，让它们上钩。

<br/>　　就这样，舒瑶恬静的趟在沙发上，涂小安这条金银蛇安静的守护着她。

<br/>　　不能让那些蛇造反。
<br/>　　看来系统任务还真不是那么好完成，平日没什么日常任务，一出手，就让涂小安大感头痛。
<br/>　　要知道在赵宅村，鸡冠蛇一天杀一人，都没有触发系统任务。

<br/>　　麻烦点，涂小安还是没找到，原本以为她被觉醒蛇缠上，现在有待商榷了。
<br/>　　十天，这十天他恐怕都要跟这个女人绑定在一起，除非任务完成。
