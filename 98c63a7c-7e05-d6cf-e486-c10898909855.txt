　　第79章
<br/>　　落叶城，四海楼。
<br/>　　李大头又一次隔着二楼的窗子往对面望。这都好几天了，对面湖底捞天天爆满，反过来四海楼的生意已经很好有人光顾。
<br/>　　“妹子，你看见没有，从早上就开始排队，从中午饭点一直忙到晚上，挣钱一刻也不歇着。咱们这儿，现在是一个客人都没有。”
<br/>　　李如：“那妮子真这么大本事，把生意做得这么好，你就没想过和他们学学，派人过去，把菜谱偷回来。”
<br/>　　“其他是都好学，但湖底捞生意这么火爆，全靠辣椒撑着，那东西，我们可没地儿寻去。”
<br/>　　李如心中也是老大不痛快，其实四海楼生意好坏，对她来说不算什么，苏家终究是有钱的。她的目的是将李如赶走，可看苏樱如今，是要生根发芽的意思。
<br/>　　正这个时候，两人看到楼下有人走进四海楼。李如道：“好不容易来了客人，还不赶快去接接。”
<br/>　　李大头小跑着下楼，一见这客人还认识，正是以前经常光顾的张员外。
<br/>　　“张员外，您可是好久没来了，今天吃点什么，我让厨房给您来到三笋炖鱼，那可是本店的拿手菜。”
<br/>　　“不吃啦。”张员外一摆手：“我们在对面挂上号了，现在正排着队呢，没地方坐了，我来你们店坐坐。”
<br/>　　“……”李大头。
<br/>　　张员外手里还挥舞着一个小木牌，在李大头面前晃悠。
<br/>　　“大头，我和几个朋友在你这人坐会儿，你没意见吧。”
<br/>　　“这个……哈哈……怎么可能，几位想坐多久坐多久……”
<br/>　　对面湖底捞，生意依旧火爆，隔着门飘散出刚烤好面包的麦香。
<br/>　　生意渐好，苏樱又新雇了一个掌柜。算账她是不陌生的，不过一个女人总是不适合抛头露面，需要一个掌柜支应着迎来送往。好在杜茂的事情后，也没人再敢给湖底捞使绊子。
<br/>　　如今有掌柜支应着湖底捞，她又在城里买了一套宅子，平素她就和小蝶住在那里，偶尔过来查一下帐。
<br/>　　面包每天的销量也很好，客人需要等很长时间才能排到位置。苏樱打算在东城再开间分店，面包的销量也不错，只是没有一开始好，程大雷来信说，可以开间店，专门卖面包。
<br/>　　落叶城靠海，港口贸易还算繁荣，如今许多外地的客商来到落叶城的第一件事，就是在湖底捞吃顿饭。
<br/>　　一座客人吃得大块朵颐，话里话外边说到这酒楼的老板娘。
<br/>　　“这老板娘一直用黑纱遮着脸，也不知道什么模样。”
<br/>　　“看身条肯定不错，如果俺老朱有福分，能……”说着，眼神便猥琐起来。
<br/>　　“呔，住口，你在这里说这话，不想活了么！”一个客人急忙喝止。
<br/>　　“只是说一说又能怎样，至于这么小心么……”姓朱的客人道。
<br/>　　“嘿，你难道不知道这酒楼背后的人是谁，那么杜老大的事情你总知道吧……”
<br/>　　说着说着，这人的声音低下来，听到最后，姓朱的客人冷汗哗哗直落，只盼望自己的话没有被别人听见。
<br/>　　一个成功男人必定有无数女人，可一个成功女人背后有多少男人，也就不好说了。目前，程大雷已经成了苏樱背后的男人，有程大雷恶名在外，也就没人敢在酒楼生事。
<br/>　　到下午，酒楼的客人才算稀疏下来，这时候，掌柜从外面引着几个人来到后院。
<br/>　　为首是那张员外，其他几个苏樱倒是不认识。苏樱忙挂上黑纱，和他们在大堂谈话。
<br/>　　一番客套之后，张员外直奔主题：“苏老板，这位是我在青州的朋友，专门运皮货的。这次来是想向苏老板收购一批辣椒。”
<br/>　　辣椒都是程大雷种的，具体情况苏樱不清楚，只是答应他们考虑考虑。可看他们的样子，是肯拿出很高价格的。
<br/>　　正好孙九准备回山寨，苏樱便写了一封信，让他把信捎给程大雷。
<br/>　　……
<br/>　　蛤蟆寨，地牢。
<br/>　　杜茂躺在稻草上，周围放着几个空酒坛，下巴乱糟糟的络腮胡子。他如今身上已没了麻绳，吃得也算不错，只是想走出去是万万不能的。
<br/>　　有脚步声靠近，杜茂喊道：“给爷爷取酒来，敢晚片刻，打爆你的脑袋。”
<br/>　　随着扭过头来，看到牢外站着程大雷、于虬然、高飞豹三人。
<br/>　　高飞豹和杜茂目光对视，眼睛都瞪大了。
<br/>　　杜茂是有些尴尬，他毕竟也是一号人物，如今却身陷地牢，还被熟人看见。
<br/>　　高飞豹是不敢相信，响当当的杜茂竟然真的被程大雷抓来了。如果说这是假的吧，杜茂就活生生在自己眼前。可若说这是真的，程大雷究竟是怎么做到的。
<br/>　　“杜老大，是你么？”
<br/>　　杜茂冷哼一声：“豹子，需要拿这话羞辱我么！”
<br/>　　高飞豹叹口气：“我回去和我哥说说，让他求情，请程当家放了你。”
<br/>　　“你问问姓程的，他敢放我么！”
<br/>　　程大雷在一旁笑着，他是真不敢。杜茂论名声有名声，论本事有本事，这也罢了，关键是他还有脑子。这要是放出去，招来一帮恶汉，肯定第一个与自己算账。程大雷可没本事再抓他一次。
<br/>　　高飞豹和于虬然的目光看向程大雷，眼神中除了难以置信，还有畏惧。
<br/>　　本来还想高高在上，等看程大雷笑话，最后在施舍般地帮他一下，结果人家已经把人抓回来了。无论他如何做到的，都值得令人恐惧。
<br/>　　嘟，收到恐惧值19……
<br/>　　嘟，收到恐惧值21……
<br/>　　程大雷眼睛一亮，这下恐惧值够两万了，可以直接抽两次。
<br/>　　走出地牢的时候，高飞豹仍旧失魂落魄，最后忍不住还是问了出来。
<br/>　　“程当家，你是怎么做到的？”
<br/>　　程大雷也是无语，他知道这时代的信息闭塞，但没想到竟闭塞到这种程度，对于身边发生的事情却也一无所知。望着于虬然和高飞豹求知心切的眼神，程大雷心里想着，应该如何告诉你们呢。
<br/>　　最后，程大雷拍了拍高飞豹的肩膀，说出了一串话。
<br/>　　“多读书，多看报，少喝冷水，多洗澡。”
<br/>　　
<br/>(本章完)
