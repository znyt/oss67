        &lt;h3&gt;第三卷第一百零三章  家有梧桐树，不怕凤凰来（感谢凌老板盟主的打赏）&lt;/h3&gt;
<br/>
<br/>        凌辰看着镜中帅气的自己，板寸，浓眉，脸上永远都是一副冷冷的拒人千里之外的表情。
<br/>
<br/>        身上的休闲西装永远都是平平整整的，衬衣最上面的两粒扣子潇洒的敞开着，炫耀着主人那健康的麦肤色，黑色休闲运动鞋永远都是干净清爽，银色的行李箱站在他的身后，他冲着镜子中的自己咧嘴一笑：“凌辰，好样的，不问去处，只争朝夕，继续前行！”
<br/>
<br/>        虽然女神拒绝了和他私下见面，但是终究是看到了，每个人都有自己的选择，每个人都有自己心目中的牵绊。
<br/>
<br/>        只要宁雅未婚，他还是照样有机会。
<br/>
<br/>        追妻路漫漫，不是一天两天的事情，这是一场持久的战斗，要想抱得美人归，还需要自己有能力。
<br/>
<br/>        努力的提升自身价值才是正道，他家本身也是家道雄厚，只是几经跌更，浪潮翻滚，留下来的不是很多而已，但是相对于中等世家，他们也是不错的，每个家族都有自己雄厚的文化底蕴，他们遗留下来的不仅仅是钱财，还有一种不屈不挠的精神，有些人传承了自己家族的精神，努力奋斗，有些人跟着古老的气息腐朽，最后只是时代更迭下的牺牲品。
<br/>
<br/>        凌辰不想当时代的牺牲品，在他的字典里没有失败二字，家族的事业虽然踏步不前，保持着原样，但是他目前所做的事业是红红火火，只有改变才有出路，适时的转型寻找新的出路比吃喝等死好。
<br/>
<br/>        他做到了！
<br/>
<br/>        匆匆而来，在北京呆了一天，他要去贵州。
<br/>
<br/>        最近厂家准备开经销商大会，他这个全国总代理是必须要到位的，第一是快要过年了，年底有些事情是必须要走动的，第二也是要看看明年的动态，他现在的这个行业也算是此消彼长，没有人是能一成不变的稳坐老大的位置，要想保住他全国总代的位置，第一是财力，第二还是要靠关系。
<br/>
<br/>        说到关系，大家也别笑，中国五千年的历史文化，仁义礼智信，哪一个字不是围绕着关系二字在进行了。
<br/>
<br/>        做生意靠关系，交朋友靠关系，娶老婆靠关系，没关系老婆会跟你过？孩子上学也得靠关系，那么他这个总代理不去拉拉关系，套套近乎，人家认识你是二两毛线？
<br/>
<br/>        别看凌辰长着一张冷死人的俊脸，生意场上那也是当仁不让，对上不亢不卑，对下至仁至义，要不然能掌握全国各地的经销商？
<br/>
<br/>        最主要的是他还比较年轻。在众多的经销商中，轮年龄，他还是一个毛孩子。
<br/>
<br/>        准备提起行李箱，出发贵州。
<br/>
<br/>        电话响起来了，心里一喜，想着是不是宁雅的，可是看到电话发现是陈子昂的，心里一凉。
<br/>
<br/>        “你好，陈总。”凌辰对陈子昂是不冷不热，疏离有加，虽然陈子昂很能干，他也出手相助，但是那完全是因为有他的雅儿在。
<br/>
<br/>        “凌总，对不起了，昨天比较忙，都没有顾上和你说话。”陈子昂在电话那头抱歉的说。
<br/>
<br/>        他们昨天确实比较忙，中午就在公司吃了顿简餐，便各自都忙开了。
<br/>
<br/>        对于寒伯安他们，陈子昂算是已经很熟悉了，就算是不招呼他们也能谅解，但是凌辰却不一样。
<br/>
<br/>        第一，凌辰不是她的股东。
<br/>
<br/>        第二，凌辰和她的合作并不是很大，没有太多的利益关系，但是凌辰却在她困难的时候出手相助，这份心怀和大气是陈子昂没有料到的。
<br/>
<br/>        陈子昂和凌辰认识还是和江俞轩一起去山西的时候认识的，当时因为要请客，需要带一些东西，陈子昂说随便买一些，江俞轩说山西的项目时间也很长，吃饭这等事件也不是一次两次，不如找个人批发一些，能省则省。
<br/>
<br/>        于是找了一个部门的老总，要到了凌辰的电话，而知道了凌辰就是总代，他们在全国都有业务，一来二去就经常联系凌辰，他们也就熟悉了。
<br/>
<br/>        凌辰到了他们公司，和她说话的时候就一直看着宁雅，当然宁雅是很漂亮的，每个人见了她都会不由自主的看她一眼，大多的眼神是惊艳。
<br/>
<br/>        凌辰看宁雅的眼神就不一样了，是爱恋，是款款深情，是那种一看就不想离开的，似乎他的眼神里藏了几世的爱，浓得化不开，却又有着一种深深的无奈。
<br/>
<br/>        而宁雅看到了凌辰是惊喜，是一种深深的眷恋，还有一种欲语还休的期盼，那种眼神似乎是见到了自己很久不见的爱人，却又碍于有旁人，万语千言只一眼。
<br/>
<br/>        陈子昂自从知道了江俞轩的身份之后，她再也不像以前那样粗线条了，对于身边的人也会观察和发现，说不定身边还卧着另外一个王俞轩。
<br/>
<br/>        果然，见到了凌辰和宁雅用眼神在交流后，于晚上，她在人事上要来了宁雅的资料。
<br/>
<br/>        虽然简单，但是让她发现了一个问题，南方一个大企业老总就是姓宁，其长女也叫宁雅，原来宁雅在公司担任着要职，只是因为弟弟开始接手家里的事情，她已经淡出了商圈，下落不明。
<br/>
<br/>        下落不明？陈子昂笑了，她陈子昂还真是一棵梧桐树，各路凤凰都飞来了，不知道她的公司还有多少个这样的人？
<br/>
<br/>        宁雅和凌辰所受的教育相似，都是在同一所大学，难怪凌辰会出手相助。
<br/>
<br/>        “哈哈哈，看到了陈总的忙，都是朋友，没有必要那么客气。”凌辰笑着说，虽然是笑着，那张脸上的表情也没有太多的变化。
<br/>
<br/>        “如果，凌总今天不忙的话，能否请凌总坐一坐，表达子昂的感激之情？”陈子昂看着对面总裁主任办公室里忙碌的宁雅笑着问。
<br/>
<br/>        “哦，真是抱歉，我已经买好了去贵州的机票，开始新的旅程了。”凌辰一只手翻看手中另一部手机工作群里的消息，一边随意的说到。
<br/>
<br/>        “哎呀，那真是太遗憾呢，那祝凌总一路顺风，下次来北京了一定要一起聚一聚。”原本陈子昂想叫上宁雅一起请凌辰吃顿饭，但是凌辰要走也就作罢。
<br/>
<br/>        凌辰挂了电话，直奔机场，生活容不得你拖拖拉拉，每一分每一秒都在奋斗的路上。
<br/>
<br/>        陈子昂打内线电话叫来了宁雅，笑着对宁雅说：“我美丽的小雅，这一次很感谢你呢。”
<br/>
<br/>        “陈总，你感谢我什么呢？”宁雅装作什么都不知道。
<br/>
<br/>        “你的老同学出手相助难道不是你的原因吗？我不应该谢谢你？”陈子昂看着宁雅那美丽的大眼睛，瞬间宁雅的眼睛里蒙上一层淡淡的忧伤。
<br/>
<br/>        “你都知道啦！”
<br/>
<br/>        “嗯，昨天晚上才知道，没有想到我这个小庙真的是藏龙卧虎，都是大佬。”陈子昂惬意的靠在椅背上。
<br/>
<br/>        “什么大佬啊，我也就是一个不能传宗接代的女儿罢了。”宁雅叹了口气。
<br/>
<br/>        “怎么能这么说呢？生活是在你的手中，你愿意过什么样的生活是你自己选择的，愿意做什么样的人也是你自己选择，女人不仅仅是传宗接代哦！”陈子昂盯着宁雅看。
<br/>
<br/>        “就像陈总这样？”宁雅问。
<br/>
<br/>        “不，做你自己！”陈子昂笑着站起来“你有更好的路要走，加油，宁雅！”
