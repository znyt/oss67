　　第295章
<br/>　　赤檑看着眼前这个落魄大汉，一时倒有些摸不准对方来路。
<br/>　　只见其身高七尺有余，穿一件灰布袍子，头发打结，胡子拉碴，看上去狼狈得紧。可纵然陷在木笼内，却有几分出尘味道，让人不得轻视。
<br/>　　刚才一番乱战，倒是没有注意到此人。
<br/>　　“这是哪个？”赤檑选择问阴万良。
<br/>　　“喔，他是我们擒来的肉票，正准备要赎金。”
<br/>　　我擦，原来是肉票呐，差点以为他是什么世外高人。赤檑气得大叫，这分明是将我当白痴么。
<br/>　　“我要杀了他。”赤檑拔出弯刀。
<br/>　　程大雷望着头顶天空，悠悠道：“我劝你乖乖将人留下，我若心情好的话，可以选择饶你一命。”
<br/>　　赤檑真是不知这肉票吃了什么雄心豹子胆，说话这么大口气，顿时磨刀霍霍，想要一刀宰了他。
<br/>　　“不牢赤檑大人动手，我现在就送他归天。”
<br/>　　阴万良大步过来，连看程大雷一眼都不肯，双手握枪，照着程大雷胸膛扎来，势大力沉，铁枪化作一条黑龙，探入木笼之中。
<br/>　　程大雷一低头，黑发遮盖的眸子里闪过一丝恨意。
<br/>　　所有人都以为程大雷这下必死无疑，木笼内闪避空间并不大，而阴万良这一枪又来势汹汹，虽然一个回合被樊梨花击败，但这并不代表阴万良是个弱手。
<br/>　　等众人再回过神来时，吃惊的发现，铁枪的枪头牢牢捏在程大雷手中。虽然只有两指，可却能挡得住阴万良双臂之力。
<br/>　　阴万良额头冒汗，双臂肌肉有明显的凸起，可浑身力气却像泥牛入海般，挣不脱程大雷的束缚。
<br/>　　“喂，给我吧。”
<br/>　　程大雷忽然发力，阴万良手臂像通电般颤抖，双手下意识松开，铁枪已经落到程大雷手中。
<br/>　　程大雷夺枪的第一时间，就做了一件所有人都没想到的事情，他开始用铁枪撬木笼的栅栏，姿势狼狈，毫无高人风范。
<br/>　　阴万良一招就被人夺枪，尴尬的无地自容，这等本事，未来何以统领一寨。他恼羞成怒，口中大吼：“来人，给我杀，杀了他。”
<br/>　　嘎吱嘎吱响，木笼开始摇晃，七八个人围在木笼周围。铁枪、大刀、长矛几乎同时向木笼内扎来。
<br/>　　空气静了几息，任谁想来，程大雷也该被铁器穿身。可任是谁也没有想到，众人的兵器，竟都被程大雷夺下，摆在他身边。
<br/>　　刚才那一瞬，程大雷接刀夺刀，一气呵成，连续七次出手，次次快若闪电。
<br/>　　此刻程大雷握着一柄弯刀，双手握刀，刀背缠腰，全身力气以腰眼为中心，靠旋转发力，猛地释放出来。
<br/>　　“给俺开，开，开啊！”
<br/>　　“来人，杀了他，快！”阴万良已经意识到将有不好的事情发生。
<br/>　　“开啦！”
<br/>　　一刀将木笼斩断，众人以木笼为中心围杀过来，轰的一声响，程大雷身子扶摇飞起，落在人群之外。
<br/>　　所有人都看到了他高飞的身姿，如苍天白鹤。
<br/>　　程大雷双足落地，心中长出一口气，被关了三四天，浑身上下都快长毛了。他晃了晃身躯，浑身骨节噼啪做响。
<br/>　　之后，他将目光放在赤檑身上。
<br/>　　有那么一瞬，赤檑有种被草原上野豹注视的感觉，你不要动，你一动，他就会扑过来撕烂你。
<br/>　　“杀死他！”
<br/>　　“杀！”
<br/>　　阴万良和赤檑同时下令，两拨人马围向程大雷，弯刀长枪狼牙棒各种兵器都有。
<br/>　　程大雷脚尖一点地，身子从他们头顶飞过，众人一身力气使不上，只能抬头看着程大雷遥遥飞过。
<br/>　　程大雷落在樊梨花身边，一刀将押着樊梨花那人的手臂斩断，微微移动身子，将要跌倒的樊梨花扶进怀里。
<br/>　　“樊家姑娘，莫怕，我在这里。”
<br/>　　刀贴着樊梨花的胸口，切断她身上的绳索，将其背在身上。
<br/>　　一柄长矛向程大雷扎来，程大雷退步夺过，刀顺着长矛滑过去，隔断对方喉咙。
<br/>　　紧跟着，程大雷一刀将长矛切成两段，弃刀换矛，以矛做剑，斩开一条生路。
<br/>　　半截矛杆为剑，程大雷速度快到极致，一个刺字干净利落，进攻进攻再次进攻，直来直往接阿飞快剑，寻找到机会就使出爆烟花，双脚所踏过之处血流成河。
<br/>　　众人皆是心生寒意，能在保护樊梨花的前提下，还有如此凌厉的身手，若没有樊梨花的束缚，他该快到何等程度。
<br/>　　程大雷跃出重围，挟着樊梨花遁入聚义厅，咚的一声，聚义厅的大门在其身后关上。
<br/>　　“剑，我的剑呢？”程大雷。
<br/>　　樊梨花此刻已十分虚弱，本已绝境的时候，没想到程大雷会跳出来，给事情带来一丝微弱的转机。
<br/>　　“剑，剑在我房间……”
<br/>　　“你的房间在哪儿？”程大雷手中的短矛已不堪再用。
<br/>　　“在外面。”
<br/>　　“……”程大雷道：“剑，这里有没有别的剑？”
<br/>　　樊梨花指向程大雷身后，程大雷看见在聚义厅虎椅背后，挂着一柄剑。
<br/>　　程大雷飞身取了下来，这剑是装饰用的，幸好已开了刃，只不过剑身的质地和匹夫剑根本不能相提并论。
<br/>　　勉强能使吧。
<br/>　　程大雷扯下一块衣襟，将剑绑在掌心，过程中看了樊梨花一眼，道：“莫怕，我在这里。”
<br/>　　咚！
<br/>　　大门被撞开，一人闯了进来，程大雷出剑收剑一气呵成，对方变成一具尸体倒地。
<br/>　　樊梨花倒抽一口凉气，她没有想到，世间竟有如此快剑，竟有如此干净的杀戮。
<br/>　　“你没事吧？”程大雷不忘关心一下樊梨花的身体状况。
<br/>　　“没事，还撑得住。”
<br/>　　程大雷看到她脸色苍白，汗如雨下，双手一直捧着小腹。
<br/>　　“我也明白，女人大姨妈来了肯定很痛苦的。”
<br/>　　“……”樊梨花额头浮起黑线：这都什么跟什么啊。
<br/>　　“所以你不要做什么剧烈运动，有我在这里。”程大雷擦掉剑上鲜血，全身贯注盯着门口。
<br/>　　“守，守得住么？”樊梨花还是有些虚弱。
<br/>　　“不好说。”程大雷握剑在掌心：“不过我没有守不住的时候。”
<br/>　　连续两人闯进聚义厅，程大雷问：“可杀否？”
<br/>　　“可！”
<br/>　　程大雷轻取二人性命。
<br/>(本章完)
