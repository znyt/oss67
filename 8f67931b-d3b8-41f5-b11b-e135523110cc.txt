　　“愣着做什么，还不快去？”
<br/>　　剑姬前辈冷喝一声，言语中，尽是冰冷和不满。
<br/>　　“是。”萧逸点点头。
<br/>　　手中两块令牌取出，光芒一闪，凭空消失于手中，随后稳稳出现在剑姬前辈面前。
<br/>　　“剑姬前辈。”萧逸沉声道，“这是小子在风刹、修罗两殿的令牌。”
<br/>　　“里头有大量任务点。”
<br/>　　“剑姬前辈凭此，可在两殿换取需要的东西。”
<br/>　　剑姬前辈倒是没有客气，直接收下，点点头，“行了，滚吧。”
<br/>　　萧逸点点头，转过身，撇撇嘴。
<br/>　　现在走了，等下自己还不是要回来。
<br/>　　不过，剑姬前辈虽是前辈，但说到底，是个女子。
<br/>　　与生气中的女子解释，显然是个不智的行为。
<br/>　　故萧逸也没有多作解释，看了眼承风殿主，拱了拱手，御空飞离。
<br/>　　……
<br/>　　远方，萧逸蓦地叹了口气。
<br/>　　想来也是，剑姬前辈历尽艰辛才带着东域众人前来。
<br/>　　在这危机重重的中域，又要事事小心谨慎。
<br/>　　好不容易遇到了自己，自己又这般‘见外’，着实让这位前辈难过了。
<br/>　　想起当年在东域的种种照拂，萧逸还是一阵感动。
<br/>　　“稍后还是谨言慎行吧。”萧逸摇了摇头。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸一路飞行了过百万里，寻了个深山野林，这才落下。
<br/>　　在周遭快速打下禁制屏障，又快速脱下了一身公子服，换回一身劲装，戴上了幽魂面具。
<br/>　　做好一切，萧逸散去周遭禁制屏障。
<br/>　　不过，他并未急着继续赶路。
<br/>　　而是在原地盘膝坐下。
<br/>　　足足等了大半天时间。
<br/>　　嘭…
<br/>　　紫炎火翼一振，瞬间振翅飞离。
<br/>　　紫炎火翼全力爆发下，其飞行速度，甚至超过了脚踏紫电神剑之时的御空速度。
<br/>　　……
<br/>　　另一边，巨船之上。
<br/>　　白冰雪、叶铭、林劲等人，迟疑着看向剑姬前辈。
<br/>　　“前辈，萧逸那家伙的性格你知道，他不是那种这般不近人情的人。”
<br/>　　“师尊，萧逸师弟他，或许只是无意冒犯…”
<br/>　　剑姬前辈未回答，而是在自顾书写着什么。
<br/>　　“师尊，你在写什么？”白冰雪疑惑问道。
<br/>　　“没什么。”剑姬前辈狡黠一笑。
<br/>　　那张绝美的面庞上，闪过一丝得意。
<br/>　　嘭…忽然，远方，一道破空爆响声传来。
<br/>　　一道火焰身影，转身便至。
<br/>　　来人，自是萧逸。
<br/>　　百万里路，对他而言，不过是半个时辰左右边能横跨。
<br/>　　巨船之上，半空中，萧逸紫炎火翼一消，稳稳落在了船上。
<br/>　　“剑姬前辈。”萧逸微微行了一礼。
<br/>　　声音，已然变得低沉。
<br/>　　对于他这等实力的强者来说，略微改变一下声音，并非难事。
<br/>　　“哦？来得这么快？”剑姬前辈微微惊讶。
<br/>　　“嗯。”萧逸深沉地点了点头，“我恰好在极东之地那边结束完比拼。”
<br/>　　“收到萧逸小子的传信，知晓前辈你与众人来了，便急急赶来。”
<br/>　　数天前，易霄还在极东之地参加各大势力的比拼，这是人所皆知的事。
<br/>　　而这里，距离极东之地并不算太远。
<br/>　　萧逸之前本来就是离开极东之地后，来到这附近的地域，随后再赶来这边，遇到剑姬前辈等人的。
<br/>　　高空中。
<br/>　　“紫…紫炎易霄…”周遭主殿主不自觉地打了个冷颤。
<br/>　　事实上，早在数秒前，当他们看到远方一片火光袭来，那惊人的紫色火焰滔天而至时，已然确定了自己之前心头的想法。
<br/>　　大半天前，当他们听到‘易霄小子’四个字时，事实上也已有所猜测。
<br/>　　如今，偌大个中域，提起易霄二字，怕是所有人都会第一时间想到紫炎易霄。
<br/>　　只是，众人万万没想到，自己心头的猜测，竟是真的。
<br/>　　当这道火焰身影从天而降时，他们心头已然一震，忍不住咽了口口水。
<br/>　　“嗯？”萧逸微微转过身，扫视了一眼高空中的一众主殿主。
<br/>　　“你们倒是识趣，一个都没跑。”
<br/>　　“你知晓这里的事了？”剑姬前辈问道。
<br/>　　“是。”萧逸拱拱手，道，“来之前，萧逸那小子已告诉我一切。”
<br/>　　“他说，若我来到时，这些个主殿主全跑了，那便交给我去解决。”
<br/>　　“呵呵。”萧逸冷漠一笑，“我还以为我得追杀一番…”
<br/>　　高空中，一众主殿主听到这带着杀意的话语，再次身躯一震。
<br/>　　“易霄副殿主。”江流火、许鹤颜等人，连连拱手。
<br/>　　“哼。”萧逸冷哼一声。
<br/>　　“剑姬前辈。”萧逸看回剑姬前辈，道，“我先处理完这里的事。”
<br/>　　“去吧。”剑姬前辈点了点头。
<br/>　　嗖…
<br/>　　萧逸身影一闪，来到承风殿主与横天殿主身前。
<br/>　　横天殿主，原本去了这附近的地域查阅卷宗。
<br/>　　但萧逸消失了大半天，他自然早就回来了。
<br/>　　“这两位，想必就是承风殿主和横天殿主了。”萧逸拱拱手。
<br/>　　“紫炎易霄。”承风殿主与横天殿主，几乎是一瞬间，便眯起了眼睛。
<br/>　　事实上，他们二人这是第一次直面看着易霄。
<br/>　　而果不其然的，以他们的眼力，也一瞬间觉得易霄与萧逸，极为相似。
<br/>　　虽看不清易霄的面容，也感知不到易霄的气息。
<br/>　　但二人心头还是不由得产生一股熟悉之感，以及一股怀疑之感。
<br/>　　两人，早有怀疑，此刻，这股怀疑，不由得更为沉重。
<br/>　　只是，二人看了眼下方的剑姬前辈，却又紧紧地皱起了眉头。
<br/>　　“横天殿主。”
<br/>　　这时，萧逸低沉的声音，打断了两人的思索。
<br/>　　“我听萧逸说，你去查了你风刹殿的卷宗，如何？”
<br/>　　横天殿主虽怀疑着，但还是点点头，手中取出一叠卷宗。
<br/>　　“这是我传令回总殿后，总殿在各主殿收集的卷宗情报。”
<br/>　　“通缉令，本就不是什么机密情报，故总殿那边，很容易就能搜集并传到我手上。”
<br/>　　萧逸点点头，接过这叠卷宗。
<br/>　　上古八殿，无论哪一殿，麾下的大大小小各总殿，都有特殊的传令亚圣器。
<br/>　　足以极快地传送各种卷宗和情报。
<br/>　　萧逸翻开一份份卷宗，看了几眼，眼眸，愈发冰冷。
<br/>　　“掠夺药尊殿天材地宝，一路所过，处处血洗，视若邪修。”
<br/>　　“呵。”萧逸冷眼看向一众主殿主，“六份通缉令，其通缉理由，皆是一致。”
<br/>　　江流火连忙拱手道，“易霄副殿主，这群家伙，我们有理由怀疑他们是邪修。”
<br/>　　“那位萧逸副殿主，更是处处包庇…”
<br/>　　萧逸冷声打断道，“换言之，你打算教易某如何分辨邪修？”
<br/>　　江流火闻言，脸色一变，连声道，“不敢。”
