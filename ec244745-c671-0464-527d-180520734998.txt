　　天生纯阴之体！
<br/>　　苏莫闻言眸光大亮，青璇就是纯阴之体，这说的云悠悠是青璇吗？
<br/>　　这很有可能，青璇既然被人抹除了记忆，就很可能也改了名字！
<br/>　　苏莫激动了起来，终于探听到有用的消息了，本来他也没有抱希望很快探查到消息，毕竟太阴星这么大，找一个人无异于大海捞针。
<br/>　　没想到他运气这么好，居然真的发现了有用的消息。
<br/>　　苏莫静气凝神，继续聆听。
<br/>　　这时，便听到那三名年轻的武者继续开口了。
<br/>　　“纯阴之体、玄阴之体、阴煞之体，这三种特殊体质，每一样都是太阴族道统最适合的传入，如今太阴族已经有了纯阴之体的天才和阴煞之体的天才，不知道以后会不会出现玄阴之体的天才。”
<br/>　　“不好说，这种特殊体质千年罕见，云悠悠和冷邪两人，据说都是在别的星河找到的！”
<br/>　　“你们说这纯阴之体和阴煞之体，那种更为厉害一些！”
<br/>　　“应该是阴煞之体吧，太古异体榜上的排名，都要比纯阴之体高几名！”
<br/>　　“几名的差距而已，其实都差不多，差距可以忽略不计！”
<br/>　　三名年轻武者随意的闲聊，聊着聊着，边聊到了各种体质的强弱之上。
<br/>　　苏莫面色凝重，从这三人的话中，他听出了那云悠悠不是明虚星河之人，而是从其它星河找来的。
<br/>　　这与青璇的情况，倒是很一致。
<br/>　　苏莫继续聆听，不过，接下来那三名年轻武者，却是跳过了这个话题，聊起了各自的修炼情况。
<br/>　　略一沉吟，苏莫拿起美酒，向三人走了过去。
<br/>　　“三位兄台，我可以坐吗？”苏莫来到三人的桌前，指着四方桌空余的一个位置，微笑着问道。
<br/>　　三人闻言均是微微皱眉，他们都看到了苏莫之前坐在窗边，现在居然要和他们坐在一起。
<br/>　　“阁下自便吧！”其中一名壮硕的青年说道，他们三人都是武皇境武者，能看出苏莫不简单，所以没有拒绝。
<br/>　　苏莫点了点头，便自顾自的坐了下来。
<br/>　　不过，他就坐之后，那三名武者却不再闲聊，只是各自饮酒。
<br/>　　苏莫略作沉吟，微笑着问道：“三位，刚才听你们说云悠悠，对方真的是纯阴之体？”
<br/>　　“这是当然，整个太阴星的人都知道！”其中的一名白袍青年说道。
<br/>　　“那请问这个云悠悠是什么来历，何时来的太阴星？”苏莫继续问道。
<br/>　　三名年轻武者听闻苏莫此言，相视一眼，均是会心一笑，笑容让人感觉颇为古怪。
<br/>　　“阁下，我劝你还是死了这条心吧！”另一名黑衣青年摇头说道。
<br/>　　“死心？”
<br/>　　苏莫闻言一怔，满脸不解的问道：“什么意思？”
<br/>　　黑衣青年轻笑一声，仿佛知道苏莫内心所想一般，道：“阁下的想法很正常，这是太阴星乃至整个明虚星河所有男人都有的想法，但太不切实际了！”
<br/>　　“你这说的什么玩意？我怎么听不明白？”
<br/>　　苏莫的脸色沉了下来，什么所有男人的想法，他有什么想法了？对方的言语让他摸不着头脑。
<br/>　　黑衣青年一怔，诧异的问道：“阁下打听云悠悠，难道不是心仪对方的体质和容貌？”
<br/>　　三名青年均是有些惊异，云悠悠容颜倾城，又是极品炉鼎的纯阴之体，这可是让无数男人觊觎的对象。
<br/>　　只不过，云悠悠身份非凡，根本不是普通人能接近的存在。
<br/>　　面前的这个家伙，来打听云悠悠的情况，难道不是心有觊觎？
<br/>　　苏莫闻言无语，原来这三个家伙居然是这么想的，也太龌蹉了吧！
<br/>　　“三位兄台，在下只是好奇，随便问问而已，不是你们想的这般！”苏莫摇了摇头说道。
<br/>　　“是吗？”三人笑了笑，面上均是露出不置可否之色，显然不太相信。
<br/>　　“三位，不知你们见过那云悠悠吗？”苏莫见三人不信他的话，也懒得辩解，沉声问道。
<br/>　　“没有，人家是太阴族的天之骄女，我们哪有资格一见！”白袍青年摇头说道。
<br/>　　“刚才听你们说，云悠悠不是明虚星河之人，她是来自哪里？什么时候来的？”苏莫继续问道。
<br/>　　三名青年闻言疑惑，不知道苏莫为何如此探听云悠悠的事情，不过这也不是秘密，三人并没有隐瞒。
<br/>　　“她好像是四年前来的太阴星吧！具体来自哪个星河，没有人知道！”黑衣青年不确定的说道。
<br/>　　“四年前！”苏莫呢喃自语，心中算了算时间，好像也差不多。
<br/>　　虽然他不知道青璇被人从天冥星巫族带走的时间，具体是什么时候，但他离开苍穹世界已经五年多了，这个时间点倒是还算吻合。
<br/>　　不过，到底是不是青璇，目前还不能下定论，毕竟宇宙之大，纯阴之体也不可能是青璇独有。
<br/>　　“三位，听你们的意思，那云悠悠容颜很美？”苏莫微笑问道。
<br/>　　“是啊！听说容颜如画，倾国倾城，风姿能让百花失色！”黑衣青年感叹道。
<br/>　　苏莫闻言点了点头，眸中闪过沉吟之色，道：“听你们这么一说，我倒是很好奇，不知哪里能见到她的画像？”
<br/>　　“不清楚！”三人同时摇了摇头，若是有云悠悠的画像，他们也去瞻仰一番了。
<br/>　　“三位，多谢了！在下告辞！”苏莫站了起来，向三位青年微微抱拳，随即转身离开了酒楼。
<br/>　　他要出去想办法确认，确认那叫云悠悠的人，是不是宏青璇？
<br/>　　“这个家伙，还说不是觊觎云悠悠？”
<br/>　　见苏莫离开，黑衣青年不屑的摇了摇头，在他看来觊觎云悠悠不算什么，毕竟无数年轻武者都是如此，但不敢承认就显得太没有气度了。
<br/>　　“呵呵！他也就只能在心中觊觎一下而已，哪有资格去追求！”另一名白袍青年笑道。
<br/>　　三人均是没有太过在意，云悠悠自从几年前来到太阴族，便名传明虚星河。
<br/>　　作为太阴族虚神境大能的亲传弟子，又是纯阴之体这种极品炉鼎的体质，年轻的男性武者，哪个不想追求？
<br/>　　这种天之骄女，若是能成为双.修道侣，无论是对于武道修为、还是对于自身的前途，都是有天大的好处。
<br/>　　但在整个明虚星河，能有资格去追求此女的人，实在是少之又少。
