等大家都下班了，整个电台关了灯，夏夕绾拿着那把钥匙，来到了电台的后院，站在了那扇暗红色的大门前。
<br/>
<br/>        她现在心跳有点剧烈，因为她已经感觉到里面有一股可怕而神秘的力量正在召唤她了。
<br/>
<br/>        夏夕绾将钥匙插在了门锁里轻轻一转，伴随着一道清脆的咔嚓声，门锁开了。
<br/>
<br/>        夏夕绾推门而入，走了进去。
<br/>
<br/>        这是一片森冷，里面是无边无际的黑暗。
<br/>
<br/>        什么都看不清，在这里就连世界都万籁无声了，什么声音都没有，静到令人心里发慌。
<br/>
<br/>        夏夕绾走进去，双脚踏在松软的泥土上，她整个大脑神经都紧绷着，高度的警觉，防备着这里随时可能会发生的危险。
<br/>
<br/>        可是走了一路，什么都没有。
<br/>
<br/>        夏夕绾继续走，走啊走，这条路好像根本就没有尽头。
<br/>
<br/>        她要走到哪里去？
<br/>
<br/>        妈咪的第二个宝箱究竟放在了哪里？
<br/>
<br/>        这时夏夕绾的脚步倏然一顿，因为她在前方那棵大树底下看到了一样东西，是一个紫檀木的小宝箱！
<br/>
<br/>        她找到了！
<br/>
<br/>        她找到了妈咪留下来的第二个宝箱！
<br/>
<br/>        夏夕绾迅速狂奔过去，眼见着自己离那个宝箱近了，又近了一点，她伸手，去拿那个宝箱。
<br/>
<br/>        就在这个时候，后面突然伸出了一只骨节分明的大手，一把拽住了她纤细的皓腕，耳畔传来了一道低沉的怒斥声，“夏夕绾，你在干什么？”
<br/>
<br/>        夏夕绾一顿，当即回头，陆寒霆那张俊脸已经在她的视线里收缩放大了。
<br/>
<br/>        陆寒霆来了！
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，有点茫然，“你怎么来了？”
<br/>
<br/>        陆寒霆抿着薄唇，整个人覆着一层阴戾的寒霜，他掌面青筋暴跳，狠狠的盯着她，“如果我不来的话，你现在就跳下悬崖摔死了！”
<br/>
<br/>        什么悬崖？
<br/>
<br/>        夏夕绾抬眸一看，刚才看到的那棵大树和宝箱全部不见了，她前面就是深不见底的万丈悬崖。
<br/>
<br/>        现在她已经站在了悬崖边上，她脚下的泥土松动了，当即塌了一块。
<br/>
<br/>        啊！
<br/>
<br/>        夏夕绾惊呼了一声，整个人出了一身的冷汗，她几乎是跳起来钻到了陆寒霆的怀里，伸出两只小手紧紧的拽住了他精硕的腰身，“好可怕啊，好怕。”
<br/>
<br/>        陆寒霆一张俊脸隐在这片黑暗里，说不出的阴沉，“现在知道怕了，那你还来这片禁地干什么？刚才我就看着你向这个万丈悬崖狂奔而来，如果不是我拉着你，你就摔下去了！”
<br/>
<br/>        夏夕绾惊魂未定，“可是刚才我没有看到悬崖，我明明看到一棵大树，还有我妈咪留下来的第二个宝箱。”
<br/>
<br/>        “那宝箱现在在哪里呢？”
<br/>
<br/>        “我…我不知道。”夏夕绾将小脑袋埋在他精硕而温暖的胸膛里，害怕的不肯抬头。
<br/>
<br/>        刚才她明明看的真切，难道是…镜花水月？
<br/>
<br/>        夏夕绾觉得这件事没那么简单，难道这里有什么玄妙的针法，但凡进入这里的人，都会看到自己内心最渴望的东西，露出贪婪的一面，然后炙烈疯狂的像飞蛾扑火，跳进万丈悬崖？
<br/>
<br/>        夏夕绾一时间心乱如麻，这里处处透着诡异，此行相当凶险。
