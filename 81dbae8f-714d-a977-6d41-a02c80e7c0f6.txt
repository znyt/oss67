　　叶少阳吃惊地看着橙子，半晌说道：“你是怎么想到这个的？”
<br/>　　“我经常在阴司啊，对他比你们都了解，徐公是个很讲义气的人，你帮过他不少，让他设计害你，他做不到的。”
<br/>　　叶少阳听她这么说，心情倒是好了很多，为自己当时意气用事、没听他的直接走人感到有些过意不去，喃喃道：“但愿他不会因为这个受罚。”
<br/>　　橙子道：“那不会的，他虽然没有官职，只是个师爷，但地位还是很高的，而且他是转轮王的亲信，他既然敢这么做，那一定是不怕的。只不过……”
<br/>　　“不过什么？”
<br/>　　橙子歪着脑袋想了一会，道：“我在想，这次行动，究竟是不是转轮王下令的。假如是他坚持要抓你，那不会只派那些人去，给你逃走的机会，更不会派徐公去——倒不是为了你，你想啊老大，轮回司有那么多人，转轮王为啥偏偏派徐公这个跟你关系好的去，难道是故意让徐公难做不成？”
<br/>　　叶少阳心中一惊，顺着她的思路一想，立刻想到了某种可能，没等开口，小青率先问道：“难不成，转轮王并不想真的抓老大？那他为什么还要下令？”
<br/>　　橙子道：“转轮王跟老大没仇没怨，抓他做什么？他下令……可能是不得不下令吧。”
<br/>　　小青立刻说道：“这不可能，阴司之中，能下令让转轮王做事的，只有大帝。”
<br/>　　“没准就是大帝呢？”
<br/>　　叶少阳听到这里，心中咯噔了一下，仔细一想，真是细思极恐，这时候小白说道：“能给转轮王下令的，不光是大帝……老大，你上次不是说，地藏菩萨派人去捉拿你吗，没准这件事是他主使的呢？”
<br/>　　“地藏菩萨？”
<br/>　　橙子点点头，“转轮王是佛门中人，虽然在阴司任职，但他真正的老大是地藏菩萨，对于大帝，他也是听调不听宣的，如果真是大帝想捉拿你，根本不用找转轮王，随便跟那几个大佬一说，老大你根本逃不出酆都城……但地藏菩萨的势力，更多是在几大地狱，在酆都城做事，还是有所顾忌的。”
<br/>　　叶少阳仔细琢磨了一番，如果真的是地藏菩萨，那一切都清楚了——虽然他不清楚地藏菩萨为啥要处心积虑地对付自己，但毕竟是有前科的。
<br/>　　“我以前怎么没发现你这么聪明。”叶少阳伸手捏了捏橙子的脸蛋，突然想起萧逸云好像是走了吧？
<br/>　　橙子吐了吐舌头，卖萌说道：“也不是我聪明啦，只是人家一向在天子殿做事，对阴司这些大佬的关系和行事风格比较了解，这事情也不复杂，大致可以想到的。不过老大，你往后可真多加小心，地藏菩萨不是好惹的……”
<br/>　　“我当然知道不好惹。”叶少阳苦笑。
<br/>　　虽然地藏菩萨一向在地狱和无量界法诀寺带着，在酆都城没什么存在感，但毕竟是跟酆都大帝一个级别的超级大佬，得罪他自然是很麻烦，最委屈的是……自己到现在也不知道自己哪里得罪这老人家了，说是因为山海印，但他总觉得这理由有点牵强。
<br/>　　地藏菩萨是顶级大佬，又是佛门中人，按说犯不着为了个山海印跟自己死磕吧？
<br/>　　不过既然想不明白，干脆也就不去想了。
<br/>　　橙子表示回头可以找崔府君打听下，看他是否知道原因。
<br/>　　“老大，你跟徐福聊什么了，知道山海印的下落了吗？”瓜瓜好奇地问道，大伙也都瞪大了眼睛看着叶少阳，等着听。
<br/>　　“这个……他倒是说了点什么，我还没想明白，只能回去慢慢想。”
<br/>　　隔墙有耳，叶少阳担心万一被人听到，只好这么说，同时用眼神给他们暗示，大伙会意，虽然还是好奇，但是不再问了。
<br/>　　叶少阳跟大伙聊了一会，话锋一转，压低声音说道：“你们都听好了，我感觉，我跟阴司的关系就这样了，只会进一步恶化，万一真的闹崩了，你们在阴司也不好待下去，小青，到时候你记得带大伙一起离开这里，去空界找军师，至少也有个落脚的地方。哦对了，橙子除外。”
<br/>　　“凭什么，你们要走的话，我当然也跟着。”
<br/>　　橙子不服。
<br/>　　“你有府君做靠山，就算是地藏菩萨也不敢动你的，你当然要留下，真到那一步，我还指望你给我通风报信呢。”
<br/>　　叶少阳在她鼻子上刮了一下。
<br/>　　橙子抿嘴一笑，抱住叶少阳的胳膊，用力摇着，嗲声说道：“你想让我答应啊，那除非你告诉我，你最宠的是我。”
<br/>　　“真没脸啊你！”小白故意做呕吐状。
<br/>　　橙子上去踢她。
<br/>　　“你有脸，是谁没事去缠着我倾述对道风的思念之情啊……”
<br/>　　“你闭嘴！”小白冲上去捂她的嘴，橙子赶紧躲到叶少阳身后，两人绕着他打闹起来。
<br/>　　叶少阳赶紧逃开，跟七宝和炭头那几个小家伙聊了一会，问他们修行地怎么样，帮他们解答了一些疑惑，又鼓舞了他们一番，看着他们自信满满、对未来充满憧憬的样子，叶少阳也是倍感欣慰。
<br/>　　他们这一批，应该算是捉鬼联盟的二代成员了，只要没有意外，他们将来肯定会成长起来，叶少阳现在唯一的担心，就是他们将来会因为自己，而卷入可怕的事件之中。
<br/>　　好在现在林三生在空界站住脚了，如果阴阳司这边待不下去，至少还可以去空界发展。
<br/>　　叶少阳如此这般地谋划了一般，总算放下心来，又跟他们亲近了一番，然后带着瓜瓜回到了人间。
<br/>　　躺在床上休息了一会，叶少阳感觉有点渴，出来倒水喝，发现那三个家伙居然还趴在沙发上看电视。
<br/>　　碧清趴在沙发上，包子坐在她后背上，正在卖力地给她按摩，眼睛却盯着屏幕。陈悦坐在一边。
<br/>　　碧清一脸惬意。
<br/>　　“这是什么情况。”叶少阳好奇。
<br/>　　“不给按摩她就不让我看。”包子一脸委屈。
<br/>　　“见到徐福了吗？”碧清问道，目光还停留在电脑屏幕上。对她来说，就没有什么事是比电视剧还重要的。
