        &lt;h3&gt;第三卷第二十一章  陈子建的心思&lt;/h3&gt;
<br/>
<br/>        江俞琪咱西城项目又重新开工了。
<br/>
<br/>        陈子昂一走，陈子建没有多久就去了江家的工地，他心里充满了愧疚。
<br/>
<br/>        而且，令他想不到的是，冯睿竟然都开始干涉他的工作了。
<br/>
<br/>        陈子昂在他办公室的时候，小陈支支吾吾的，陈子昂走后，他就问了小陈是怎么回事。
<br/>
<br/>        “嫂子前几天找到我，说您说的，江家的那块地皮有问题，估计需要重新勘查，问我这几天有没有发出通知。
<br/>
<br/>        当时我说没有，嫂子就说，你看看你们这些人是怎么工作的，领导的心思都不知道？还要领导亲自给你们说？要你们不是为领导分忧的吗？如果那个地皮有问题，你们谁都得跟着领导一起遭殃。于是我们就发了个通知。“
<br/>
<br/>        “后来呢？”
<br/>
<br/>        “后来他们倒是来找了两次，嫂子让我们多凉他们几天，原本他们就应该早些时间来找我们的。”小陈低声说。
<br/>
<br/>        “真是荒唐，你们是听我的，还是听你嫂子的？去安排，我们去工地！”既然事情已经发生了，那就赶紧补救。
<br/>
<br/>        陈子建带着人去装模作样的勘查了一番，这个当时人家已经勘查过了，而且他和广州的那个专家其实是同学，相互之间还沟通了一下，他当然是知道这个情况的。
<br/>
<br/>        江俞琪亲自陪着陈子建去了工地，她知道强龙压不过地头蛇，他们原本是勘查过了，但是人家不认你也没有办法，如果不是陈子昂，他们怕是要费一番心思了。
<br/>
<br/>        刚好，就此机会认识一下陈子建，后期还有很多事情怕是要和他打交道了，而且他们最近看了好几块地皮。
<br/>
<br/>        勘查完之后，江俞琪和陈子建相互留了电话。
<br/>
<br/>        “陈总，非常感谢你能亲自来我们工地勘查，你一出面，我们就放心很多了，前期我们的工作没有做好，给你们添麻烦了，后期一定改正，及早去你单位报到。”
<br/>
<br/>        “其实，谁勘查都一样，那个牡总是我同学呢，前期我们还沟通过，只是下面的工作人员没有找到你们的档案。”陈子建解释了一下，江家大少和妹妹是朋友，说不定他们以后经常见面，到时候不能让亲家的人认为他是故意为难他们的。
<br/>
<br/>        陈子建自然而然的从心里将妹妹归纳到江家去了，江家是生意人，妹妹也是生意人，没道理还会和张家一样嫌弃他们的出身，而且，妹妹能来找他为江家说事情，这个关系就更不一般了。
<br/>
<br/>        “陈总，如果有时间的话，下午一起吃顿饭？”江俞琪问陈子建。
<br/>
<br/>        陈子建心里转了一个圈，他也想了解一下江家公子的情况，也了解一下江家的态度。妹妹年龄大了，这几年忙着做生意，个人问题都没有时间去考虑，如果江家大少很好的话，就催着妹妹早点解决个人问题，过了这个村可没了那个店了。
<br/>
<br/>        “那就麻烦江总了。是这样的我先回单位安排一下，下午我们联系如何？”
<br/>
<br/>        “好，那我们下午见。”江俞琪也听人说起过陈子建，不爱应酬，一般人是叫不出来的，经常回家陪着老婆吃饭，爱妻如命，唯妻命是从。
<br/>
<br/>        “陈总，那下班之后带着夫人吧，听说夫人是你的贤内助。”江俞琪问陈子建。
<br/>
<br/>        “我老婆去北京了呢，她回来还得几天，要不等她回来了我们一起吃饭？”
<br/>
<br/>        “不，不，不，今天怎么样都得感谢一下陈总，等夫人从北京回来之后，我们再为夫人接风。”
<br/>
<br/>        “好，那就这样，我们回见。”陈子建也不客气，反正日后是要见面的。
<br/>
<br/>        晚宴，陈子建就一个人来了，也没有带单位的同事，他这顿饭主要是想了解江少的情况，人多了他不好说话。
<br/>
<br/>        江俞琪和单位的部门经理和她的老公一起来了，带着部门经理，后期会和陈子建对接，带着她老公，也想着让陈子建认为这个是私人聚会，和工作没有太多的关系。
<br/>
<br/>        况且，陈子昂和他们也熟悉，算是朋友了吧。
<br/>
<br/>        经过江俞琪的介绍，陈子建很满意今天的这顿饭，看来江家已经将他当做自己人了，没有那么人作陪，也没有那么复杂的流程，很好，很好，他最不喜欢应酬了。
<br/>
<br/>        “陈总，今天真是特别的感谢你能亲自来，要不然我这工地还得停几天呢。”江俞琪说。
<br/>
<br/>        他们在某海鲜酒楼订了一个小包间，刚好就他们几个人，说话也方便。
<br/>
<br/>        五个人，他们点了一个鲍鱼烧肉、蒜蓉粉丝蒸加拿大龙虾、辣酒煮小花螺、油盐焗台山咸鱼、葱油罗马生菜、锦绣素拼、酸爽脆黄瓜、蔬菜水果沙拉，喝的某台酒。
<br/>
<br/>        “怪我工作失误，要不是子昂今天找我，我还不知道这件事情。”陈子建抱歉的说。
<br/>
<br/>        “原本不想麻烦陈总的，只是子昂昨天回来，到我这里来了，聊起来，才知道你原来就是子昂的二哥。”江俞琪说的也是实话，她并不知道陈子昂和陈子建的关系。
<br/>
<br/>        “子昂今天早上可是很生气的，还一定要我在个心，将你们这事情处理了，我的那个小妹呀，还从来都没有为什么事情找过我。”陈子建有些感慨，陈子昂在上学的时候从来都不曾问他张口要过钱。
<br/>
<br/>        “子昂和我们俞轩是朋友，他们昨天一起过来的，在这里看了一套房，怕是准备以后要住西城呢。”
<br/>
<br/>        “哦，他们两个一起回来的啊，我都没有见到俞轩。”陈子建从江俞琪的口中知道了江家大少叫江俞轩。
<br/>
<br/>        “听说他们要去下面的一个县城跑业务呢，估计忙的没有见上你。”
<br/>
<br/>        “俞轩，今年多大了啊？”
<br/>
<br/>        “二十七八了。”
<br/>
<br/>        “哎哟，和子昂差不多啊。”
<br/>
<br/>        “是啊，他们年龄相仿，家里最近在准备俞轩结婚的事情呢。”
<br/>
<br/>        “该结婚了，该结婚了。”陈子建大喜，妹妹这是有着落了。
<br/>
<br/>        看着江俞琪的模样，江俞轩肯定也是很帅很有涵养的人吧！
<br/>
<br/>        “那个，江总啊，以后就是一家人了，有什么事情尽管来找我，虽然我只是一个文化人，但是在我们的这个圈子里还是说话算话的，你们要是想开发哪个地方，提前给我招呼一声，我帮你们先过过关，别买了地皮不能用。”终于能为妹妹做点事情了。
<br/>
<br/>        “好，太好了，有了陈总给我们把关，我这颗心就回归原位了。”
<br/>
<br/>        陈子建这顿饭吃得很值，最起码知道妹妹快要结婚了。
<br/>
<br/>        吃完饭，陈子建回到家里，因为心情高兴，将冯睿插手他工作事情的不快也赶跑了，他给冯睿打了一个电话
<br/>
<br/>        “冯睿，告诉你一个好消息和坏消息，你想听哪个？”
<br/>
<br/>        “我当然是想听好消息，坏消息就不要告诉我了，你自己解决。”
<br/>
<br/>        “子昂要结婚了。”
<br/>
<br/>        “啊，子昂要结婚了啊，这个确实是好消息。”这几年她都提心吊胆的。
<br/>
<br/>        “是啊，高兴吧？”
<br/>
<br/>        “高兴，男方是谁啊？”
<br/>
<br/>        “江家大少。”
<br/>
<br/>        “江家大少？搞房地产的哪个？”
<br/>
<br/>        “是啊，下午我和江总在一起吃饭，说是他们在准备结婚的事情。”
<br/>
<br/>        “啊，那太好了，我们可得好好嫁一嫁子昂。对了，我这里先接个电话，等会给你打过去。”冯睿一听，她前几天听了弟弟的话，让人去为难了江家，现在子昂要和江家大少结婚了，要是让子昂知道她为难江家，怕是以后更会恨她这个二嫂了。
<br/>
<br/>        她主动的忽略了陈子建说的坏消息！
<br/>
<br/>        冯睿挂了电话，将电话打给小陈，小陈说事情已经解决了，她心里才安顿下来，不过她没有问事情是怎么解决的。
