　　“叶天师。”小云恭恭敬敬地对叶少阳行礼，叶少阳是他救命恩人，这情分一直不会忘的。
<br/>　　叶少阳看着二人，笑道：“我当时怎么说来着，你们是跑不掉的。”
<br/>　　秦风也笑，“如今我来了，要打一架吗？”
<br/>　　“你可未必是对手。”叶少阳也跟着笑，转头去看小马，故意朝小云那边努了努嘴。
<br/>　　小马当年是暗恋过小云的，钱包里还常年放着她的照片，这件事叶少阳还记的清楚。
<br/>　　“你看我干啥！”小马瞪眼，“我现在也是名花有主，我媳妇你不是也见过！”
<br/>　　“你媳妇？”四宝皱眉。
<br/>　　“对啊。你们都见过啊！玉面罗刹啊。”
<br/>　　四宝噗嗤一笑：“我看你是想抢人家做媳妇吧？”
<br/>　　“这个你们就不知道了。”小马得意地眨了眨眼，随后四下张望，见都是自己人在，这才说道：“实话跟你们说，我俩早就好上了，不骗你们，真的。”
<br/>　　“得了吧，上次在海眼，人家不是还跟你打来着。”
<br/>　　“那是演给别人看的，不骗你们，我以前俘虏过她，她被我男性魅力征服，做了我老婆，然后我又把她放了，让她回到太阴山阵营，给我们做内应，这还是宫梓的主意，不信你们问她。”
<br/>　　面对大家询问的目光，杨宫梓点了点头。
<br/>　　大家都知道杨宫梓是轻易不开玩笑的，见她点头，一个个大惊，再三询问小马，得知了一些细节，原来玉面罗刹早就不想在太阴山混了，具体原因不清楚，想来太阴山这么多人，出一两个叛徒倒也正常，就算是阴司，也不可能没有一个叛徒。
<br/>　　不过，对于她是怎么喜欢上小马的，大家表示匪夷所思。
<br/>　　目前玉面罗刹还在太阴山大军之中，算是风之谷在太阴山中间安排的一枚棋子，早晚会派上用场。
<br/>　　捉鬼联盟和风之谷的骨干人员都到齐了，很多都是许久没见，互相热情地打着招呼。
<br/>　　孙映月毕恭毕敬地对小九行大礼，小九坦然受着。
<br/>　　叶少阳看过去，自己这边，可能也只差雪琪了。她如今在陪谢雨晴，任何行动，叶少阳都没叫她了。
<br/>　　“这次能来的，全都来了是吧？”
<br/>　　叶少阳环顾左右，问道。
<br/>　　“还有十二门徒，在道风袖子里。”小马补充道。
<br/>　　“这下差不多了吧？”叶少阳看向道风。
<br/>　　道风从他手中拿过写着对手的名单，继续安排：之前已经定好的，四宝对付白泽，叶少阳对付清长风，道风对付星月奴。
<br/>　　岳恒主动提出要对付毕方，他本尊是朱雀，跟毕方一样都是上古异兽，而且五行都属火，岳恒期盼跟他来一场火星撞地球的战斗。
<br/>　　道风想了一下，觉得岳恒一个人未必是毕方的对手，于是让孙映月跟着一起，反正他俩一向都是不分开的。
<br/>　　杨宫梓加上建文帝，一起对付司命星君。虽然他们俩也都足够强，但司命星君是个登峰造极的人物，就算是他们两个一起，也未必是对手。道风便提出让小马也加入。
<br/>　　小马不同意，吵着要对付彩云仙子，并且不许别人跟他一起。
<br/>　　秦风加小云，对付灵感仙。
<br/>　　至于太平教主，南宫影要跟他单挑，也是不许任何人跟他一起。
<br/>　　“我呢，我！”吴嘉伟见安排完了，竟然没自己的事，十分不满。
<br/>　　“你不负责具体的敌人，你就在战场里游走，负责补刀收割，这可是最重要的任务。”叶少阳道。
<br/>　　吴嘉伟接受了自己的使命。
<br/>　　如此一来，敌人都被分配完了，但还有几个没有明确对手的，都充作机动，到时候哪里需要帮忙就补上去。
<br/>　　“也不是每个人都要找自己的对手，主要还是相机行动。”道风关照着。
<br/>　　叶少阳把大伙的名字都写在了纸上，一一对应着具体的敌人，然后冲道风说道：“这下子你看我们有几成希望打赢这一架？”
<br/>　　道风也扫了眼名单，道：“两成，最多三成。”
<br/>　　“不会吧，咱们捉鬼联盟和风之谷几乎来齐了，就算二对一，还多出好几个人，这才三成？你是不是说反了？”
<br/>　　“我跟星月奴一样，都是刚证道不久，算是平分秋色，你若是超常发挥，想赢下清长风也有机会，其余他们……”
<br/>　　他没有一个个说下去，“司命星君和太平教主证道不知道多久了，实力深不可测……是真的深不可测，还有那个灵感仙，也毕竟不是善类……不过也只能这样了，看结果吧。”
<br/>　　说完，道风目光缓缓地从大伙脸上扫过，“我只说一句，这一战是有关人间的生死之战，对他们所有人，格杀勿论！”
<br/>　　这番话，点燃了大伙心中的热血，不为别的，自从有轩辕山以来，那里一直被法术界视为仙界，是修行的乐土，比人间好，比青冥界也好。
<br/>　　虽然法术界一直都在抵抗法术公会的入侵，但那是一种软性的抵抗，没撕破脸皮，也不敢撕破脸皮（主要还是实力不行，知道人间不是轩辕山的对手），随着道风这一席话，大家都明白，自己这是被推到风口浪尖上来了。
<br/>　　他们是人间的最后一股力量，已经没有退路，就算对面是轩辕山，是三界之内仅有的几个超一流强者之一的轩辕上帝，也只能放手大干一场了。
<br/>　　当天晚上，叶少阳带着大伙开始布阵——倒不是布置什么阵法，几天的时间，根本来不及布置什么阵法，而且茅山自有风水大阵，什么阵法也没这个厉害。
<br/>　　他们要布置一个口袋阵，利用地理环境，给他们来个包圆，让他们钻进口袋，然后尽数消灭。这就是道风的战略，不打什么防御战，要来就来个歼灭战，一个活的不留。
<br/>　　布置了一个晚上，大伙都找到了自己的位置，然后叶少阳开始了茅山的风水大阵，一旦有外来的邪物入侵，钟声就会响，大家这才放心地在住下来——叶少阳把他们都安顿在了藏经阁的宝塔中，等候战争来临。
<br/>　　这些事情，苏钦章大部分插不了手，只能帮着做一些小事，早上，他安排弟子蒸了十笼包子，一大锅稀饭，送到叶少阳等几个法师的房间里。
