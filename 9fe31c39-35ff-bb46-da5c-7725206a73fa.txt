　　比武台结束，散场后，校园小道上，生物掌控者林凡跟东方袁两个人朝着住所走去，脸上都是满满的震惊跟兴奋。
<br/>　　他们在人群中，亲眼见证了最强新生的恐怖之处，一代大神白君然战败，成为了古学院最大的新闻事件。
<br/>　　“没想到我大舅哥那么强，这回我在古学院真的可以横着走了”东方袁满脸的肥肉抖动着，很高兴。
<br/>　　蛇博士太靠谱了，找了一位这样牛逼的保镖给自己。
<br/>　　林发之前就见识过涂小安的实力，以为他是个冰霜元素能力者，没想到还能掌控雷电，让他惊的下巴就要掉了。
<br/>　　连白家少主，超等天才班前五的白君然都败在了他的手中，太不可思议了。
<br/>　　如今他才知道当初工地上，涂小安对他是多么的客气了，绝对是轻虐。
<br/>　　还好，还好他现在跟涂小安已经算是朋友了，还能搬进他的单独住所一起相处。
<br/>　　绝对是抱上超级大腿。
<br/>　　两人来到住所前，忽然，林凡顿时蹙眉，大惊：“里面有打斗声，快上去看看”
<br/>　　“打斗？”东方袁疑惑：“我大舅哥跟着副院长去了他的办公室，房子里只有那只漂亮而变态的仙灵孔雀呀”
<br/>　　大胖儿子早就被青枭一翅膀呼到了一楼，现在全身还跟散架似的。
<br/>　　“大神不在，只有孔雀在，难道有人对那只仙灵孔雀下手？”
<br/>　　超等生物，绝对会让很多人觊觎，林凡想到此，整个人朝着住所内冲了进去。
<br/>　　二楼中，七八个古学院的学员围着一只九彩色的孔雀围攻，为首的一位男子笑呵呵的道：“不愧是超等仙灵孔雀，果然是厉害，今天我罗子雄就拿你当礼物，跟我的女神约会去”
<br/>　　“哥几个，下手狠点没事，速战速决，别等那个最强新生回来”
<br/>　　罗子雄脸上挂着笑容，并没有亲自出手，而是看着七八个小弟围着仙灵孔雀激战。
<br/>　　只见这只仙灵孔雀翅膀一展，便是道道旋风席卷，全身的九彩色羽毛发光发亮，美轮美奂，明明在跟修武者打架，却感觉它在翩翩起舞，让人眼花缭乱。
<br/>　　七八个学员都有兵器在手，顿时间刀光剑影，但就是近不了孔雀的身。
<br/>　　“尔等卑微的人类，竟敢来抓本孔雀大人，咯咯...打死你们哦”
<br/>　　青枭发出清脆悦耳的声音，并不惊慌，它实力提升到二阶精怪级别，让它即使面对七八名修武者高手，也游刃有余，不时还发出嘲笑，傲娇的很。
<br/>　　“哥几个都给我用上全力，一只孔雀，嚣张什么”
<br/>　　“本孔雀大人可是超凡脱俗，岂能是你们能抓的，笨蛋，一群大笨蛋”
<br/>　　青枭的羽毛灿灿生辉，翅膀一挥，引动空气传来一阵阵轰鸣声。
<br/>　　“哼，看来只有我亲自出手了”
<br/>　　罗子雄脸上阴沉下来，体内元气轰然爆出，掀起一阵阵狂风四溢，他眼睛收缩，找时机对着孔雀下手。
<br/>　　但就在这时，一道吃喝声音出现：“罗子雄你干什么”
<br/>　　林凡冲了上来，看到一伙人围攻仙灵孔雀，顿时大怒。
<br/>　　为首的人，他认识，甚至很熟，是他上等班同班的学员，二阶三段的修武者，而且也是一位生物掌控者。
<br/>　　实力却比他强的多。
<br/>　　“林凡，你小子怎么来了，我听说你跟最强新生认识，怎么你想管我的闲事？”罗子雄看到来人，微微的蹙眉，但旋即不屑一顾。
<br/>　　东方袁也上来了，不识好歹道：“你们什么人，敢对我大舅哥的宠物下手”
<br/>　　罗子雄憋了这个胖子一眼，瞬间无视，一个连身体都未觉醒的废物，不配跟他说话。
<br/>　　“罗子雄，你别乱来，你应该看到最强新生的实力了，连白君然都不是他对手，你敢得罪他？”
<br/>　　周雄冷冷的道：“哼，牡丹花下死做鬼也风流，大小姐发话了，谁要是将这只仙灵孔雀抓来送给她，她就跟谁约会一场”
<br/>　　大小姐？
<br/>　　林凡大惊，大小姐邪七七，可是古学院的风云人物，比白君然还难缠，听说这位邪七七已经是三阶生灵级别，另外，这位邪七七风华绝代，在古学院粉丝众多。
<br/>　　很多天才学员都为她疯狂。
<br/>　　只要她一声令下，对于那些粉丝来说，简直就是圣旨，做事不计后果。
<br/>　　罗子雄就是她的头号粉丝，衣服胸口还绣着一个“七”。
<br/>　　林凡表情难看起来，这个罗子雄能那么的肆无忌惮，主要还有一个原因，就是他哥，他哥是天才般前五的大神学员，罗子杰，跟白君然并列其区。
<br/>　　“罗子雄，看在同班同学的份上，我劝你立刻走，不然等正主回来，你想走都走不了”林凡道。
<br/>　　罗子雄傲然冷笑：“最强新生算个屁，林凡，你好歹也是上届上等班的学员，怎么当起了那小子的狗腿子”
<br/>　　“白痴，你不听劝，非要作死，我也不拦着你，但我林凡现在在这里，你就不能将孔雀抓走”
<br/>　　林凡已经下定决心跟涂小安搞好关系，所以绝对不会坐视不管。
<br/>　　“那你是要跟我作对？”罗子雄脸上挂着冷笑：“林凡，你可要想清楚，你区区一个鼠类掌控者，凭什么阻拦我”
<br/>　　林凡俨然起来，一只灵性的小白鼠一溜烟的站到了他的肩膀上，他大喝一声：“包子合体”
<br/>　　倏然，这只灵性的小白鼠直接钻进了林凡的嘴巴中，林凡整个人气息暴涨，上半身衣服爆裂，体型就地拔高，变成了三米高的妖人，身后一条长长的尾巴触目惊心。
<br/>　　“还变身了”
<br/>　　罗子雄表情阴沉下来，对方这是用行动告诉他自己的决定。
<br/>　　“好，好呀，既然你想找死，那么劳资就只能成全你了”罗子雄气息也暴涨，嘴巴念念有词，他的身旁神奇的出现了一头巨狼。
<br/>　　银月狼王，狼类一脉的王族，高等生物，通体银白毛发高贵不凡，二阶精怪级别。
<br/>　　“我就不变身了，因为你还不配”罗子雄眼中露出了轻蔑，拍了拍银月狼王的头，道：“宝贝，去吧，咬死他”
<br/>　　“嗷..”
<br/>　　银月巨狼一声咆哮，眼神凶光乍现，瞬间双眸中隐隐血光闪烁，同时它全身银白毛发瞬间灌输奇异能量似的，从通体银色瞬间变成血红色，一匹巨狼变得耀眼夺目，威势赫赫，散发王者之威。
<br/>　　银月巨狼，变成了血月巨狼，战斗力攀升。
<br/>　　嗷...
<br/>　　血月巨狼凶煞无比的朝着妖人林凡扑了过去，比猛虎出山还要恐怖。
<br/>　　“我的乖乖，林凡加油”
<br/>　　东方袁赶紧躲到了一边，他一个还没有觉醒的普通人，可什么都做不了，估计只适合当啦啦队。
