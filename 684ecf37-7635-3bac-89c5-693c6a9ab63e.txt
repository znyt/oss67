　　第157章
<br/>　　薛半川盘腿坐在案前，案上摆着文房四宝，信笺摊开，才落笔一句：
<br/>　　『罪臣半川百拜……』
<br/>　　案下还摆着几个纸团，看来薛半川已经遣词造句良久，却还没找到妥善的用词。
<br/>　　大半年光景不见，黄三元发现薛半川整个人老了很多，形容枯槁，脸无生气。
<br/>　　“薛城主，您的身体不要紧吧？”
<br/>　　“咳咳，还撑得住。”薛半川轻咳一声，似乎突然发现了什么：“你来想和我联合起来对付幽州王，是否，我把你抓起来献给幽州王，他就会不怪罪我了。”
<br/>　　黄三元一头黑线。
<br/>　　这还是自己认识的薛半川么，曾经的薛军师喜怒不形于色，无论发生多大事，都是一副智珠在握的模样。可今天的样子，看上去实在有些吓人。
<br/>　　在去年，薛半川谋划良久，架空韩虎距，培植自己的亲信，又以雷霆手段除掉韩虎距，成功坐上城主的位置。
<br/>　　手段不可谓不高。
<br/>　　但坐上这个位置后，薛半川才算明白韩虎距的难处。黑石城在帝国边境，出城就是戎族地界。戎族三天两头过来扫荡，打却也的确打不过。想要发展黑石城，建一支精兵，又是缺财缺物，难以着手。
<br/>　　薛半川此时都有些佩服韩虎距了，韩虎距纵情酒色，可他什么都不做，黑石城却也没出过乱子。而这份能力，薛半川则是没有的。
<br/>　　简单来说，薛半川是一个顶级谋士，却未必是一个顶级城主。
<br/>　　薛半川是个有野心的人，本打算以黑石城为根基，向外扩张，日后未必不能成一方枭雄，会天下英雄猎鹿。可操持黑石城这段时间，他筋疲力尽，疲惫不堪，那野心便也就遇雨打风吹去了。
<br/>　　“薛城主，怕幽州王未必会接受你的投降？”
<br/>　　如今杨龙停喊出的口号是：别个不问，唯诛薛四。他铁了心为韩虎距报仇。薛半川莫说写一封降书，便是写千封万封怕也不会令杨龙停改变主意。
<br/>　　“我知道啊。”薛半川道：“但万一呢，毕竟谋事在人，成事在天，我想试一试，尽自己努力去写降书，说不定幽州王就放过我了。”
<br/>　　“那个……你有写降书的时间，就不能用这份心情去挡一挡杨龙停么？”
<br/>　　“挡，我拿什么挡！”薛半川苦笑道：“你以为这城里是我做主么，你可知城中有多少大户，军中府中都有他们的人，我只不过是个替他们看家的账房先生而已。杨龙停还没过来，已经有人开始给他报告消息了，只等着大军杀来，便开门纳降。”
<br/>　　黄三元此刻竟不知该说些什么，他擅长算账，不擅长领兵，更不懂得如何管理一座城池，平衡各方面的关系。或许，薛半川的确有薛半川的难处，不是真的束手无策，不会令曾经的薛半川变成如此模样。
<br/>　　他沉默一会儿，只好道：“大当家让我给城主带句话？”
<br/>　　“姓程的说什么？”
<br/>　　“低头必死，抬头或许能活。”黄三元道：“大当家的话就是这些，若薛城主要拿我们送给杨龙停，请便。”
<br/>　　“低头，抬头……”薛半川嘴里来回琢磨着这两个词，半晌挥挥手道：“来人，给黄总管搬个椅子。”
<br/>　　……
<br/>　　蛤蟆寨，程大雷也正在头疼。
<br/>　　“关二哥，最近刀法练得这么样，有没有突然顿悟什么刀招？”
<br/>　　“张三爷，蛇矛使得还顺手吧，哈哈，有没有感觉自己神力附体，可以以一挡万？”
<br/>　　“刘……”
<br/>　　“少羽，怎么没去练枪，你可是立志成为大侠的男人。”
<br/>　　程大雷满山寨转悠，升到三级，山寨还差个顶级人物。有潜力升到顶级的人，他都看了一遍，结果谁也无法在短时间内升级。目前就剩下自己了……
<br/>　　可自己虽然得到阿飞快剑，实力却依旧是优秀，本来建设山寨就是为了升级自己的实力。可现在问题又绕了回来。
<br/>　　目前看来，只剩下一个法子了。就是积攒恐惧值，达到一百万后，向系统兑换一个顶级人物。
<br/>　　程大雷查看自己的恐惧值，这些日子过来，自己又攒了些恐惧值，目前有十万出头，距离百万还有很大差距。
<br/>　　“大当家……”徐神机此时跑过来：“大当家，出事了。”
<br/>　　“出什么事了？”程大雷一惊，如今的山寨等于在刀尖上舞蹈，怎么能允许出事。
<br/>　　“幽州的运粮队，和咱们的人闹了起来。”
<br/>　　杨龙停这次攻打黑石城，号称统兵五万。而这五万人，也不完全是上阵的战士，还包括辎重营，粮队，伙头军等等。
<br/>　　大股部队已经开过去了，源源不断，还有几百人几百人一拨的运粮小队从青牛山经过。而当他们路过蛤蟆寨时，蛤蟆寨自然要安排一些草料、酒水供他们歇息。
<br/>　　某种程度上，蛤蟆寨充当了驿站的工作。而现在程大雷还没有实力和幽州王撕破脸，他自然也很乐意做这些事，反正也不用去送死。
<br/>　　罗铁就在这支运粮队中。
<br/>　　伏虎山吃了败仗，罗铁被一撸到底，现在完全是个小卒。因为这件事，罗铁心里自然极其不痛快，对周围的同袍非打即骂，众人满腹怨言，却也无人敢管。
<br/>　　纵然罗铁成了小卒，但谁人不知道这是幽州王刻意锻炼他。你现在给他脸色看，日后他再抖起来，还不是第一个拿你开刀。
<br/>　　所以在这次队伍中，罗铁不是长官却胜似长官。
<br/>　　队伍开到青牛山，程大雷早已安排好人接待，酒水草料都准备得整整齐齐。罗铁坐在后，便开始骂骂咧咧，口中说的是：“一伙山贼真是蹬鼻子上脸，还和我们平起平坐，这特么算个什么事。”
<br/>　　“罗将，小声些，军师吩咐，别和他们闹翻了。”
<br/>　　“军师……哈，你们把军师的话当回事，在我眼里，他就是个屁！”
<br/>　　罗铁有自己的主意，这次被一撸到底，他心里很不痛快。可若自己能平掉蛤蟆寨这伙山贼，替义父去个心头大患，岂不是自己就能重新站起来。
<br/>　　所以，他一边虚张声势，一边也在思考怎么做。
<br/>　　哗！
<br/>　　罗铁把酒碗重重砸在地上，道：“这是酒么，连马尿都不如，程大雷就是这么招待我们的，让他给我滚出来！”
<br/>　　
<br/>(本章完)
