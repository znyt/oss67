　　此时，黄袍老者丝毫不敢怠慢，能将斗宗级别的魔兽当作宠物，此青年的背景该是何等恐怖！
<br/>　　回过神来，黄袍老者露出一抹和煦的微笑，道：“恭迎各位大驾天涯城。”
<br/>　　“天涯城的空间虫洞在何处？”
<br/>　　慕清霄瞥了眼黄袍老者，二星斗皇，在周围各大帝国也算是一名强者，但也仅仅如此而已。
<br/>　　闻言，黄袍老者似乎也察觉到后者语气中的不耐烦，恭敬的说道：“各位，空间虫洞就在天涯城中央位置……”
<br/>　　“仙儿，先进城看一看吧。”
<br/>　　说罢，慕清霄就跃过老者，身影直接落在了城中，小医仙则紧随其后，俏脸上挂满了兴奋。
<br/>　　望着慕清霄与小医仙的背影，黄袍老者一叹，道：“似乎来了不得了的人物，希望别出现任何岔子才好啊。”
<br/>　　天涯城不愧是方圆千里内最繁华的城市，街道上商店林立，来往行人笼络不绝。
<br/>　　慕清霄轻轻拉起小医仙的小手，道：“暂时不急着前往中州，今日就在城中住一宿，明日在去空间虫洞吧。”
<br/>　　小医仙满口同意，至于彩鳞则无所谓。
<br/>　　随后，两人就手牵着手在城中逛了起来，虽说是逛街，可都是由小医仙拉着慕清霄到处乱窜。
<br/>　　慕清霄苦笑一声，逛街是女人的天性，别说小医仙，就连王语嫣她们都一逛就是一下午，而且完全停不下来。
<br/>　　随着时间的推移，夕阳西下。
<br/>　　慕清霄与小医仙则暂时在天涯城的西侧找了一家高档酒楼落脚，准备明日在去城市的中心位置。
<br/>　　夜以至深，卧室中春光满园。
<br/>　　……
<br/>　　……
<br/>　　数日清晨，天空渐渐泛白，慕清霄睁开双眼，望着怀中的两位佳人，眼中满是温柔。
<br/>　　片刻后，彩鳞睁开美眸，微微颔首，望着近在咫尺的俊逸面孔，瞥了眼右侧的小医仙，昨夜荒唐当下如幻灯片一样呈现在她的脑海中。
<br/>　　顿时，绝美冷艳的俏脸上浮现一抹若有若无的红晕。
<br/>　　良久，待小医仙醒过来，简单的洗漱一番，吃过早餐后，三人就出门直奔天涯城的中央位置。
<br/>　　街道上人流不止，慕清霄与小医仙缓步其中，由于城市建造在山林中，周围空气清爽，更是充斥着一股山林中特有的湿气。
<br/>　　青石铺砌的街道两旁商店林立，其中物品玲琅满目，摊位上时不时传来各种吆喝声，诸多行人穿梭其中，整个天涯城都显得忙碌的气氛。
<br/>　　“快给本姑娘把路让开，否则后果概不负责……”
<br/>　　就在这时，伴随着行人的惊呼声与响亮的蹄声，一道身穿红裙女子疾掠而来，身下是一头二阶的赤火兽。
<br/>　　女子驾驭者赤火兽，容貌俏丽，但眉宇间却充斥着一股狂野与蛮横，得意的笑声回荡在街头。
<br/>　　身下赤火兽狂奔下，沿途撞翻许多摊位，甚至掀翻众多行人，但行人们却敢怒不敢言。
<br/>　　因为，女子在天涯城相当的有地位。
<br/>　　赤火兽如同发狂的魔兽，在女子的驾驭下，不停的狂奔。
<br/>　　街上行人纷纷退避，慕清霄与小医仙则抬头望去，正好有一道身影直接撞了过来，丝毫没有停下的趋势。
<br/>　　“难道耳朵聋了不成，给本小姐滚开啊……”
<br/>　　望着挡在路中央的慕清霄与小医仙，特别是看到小医仙绝美的容颜后，女子黛眉顿时一簇，面色不善的呵斥道。
<br/>　　甚至，腰间皮鞭一抽，让赤火兽奔腾的更加凶猛迅速。
<br/>　　周围行人皆是露出同情的目光，他们在天涯城也不是一天两天了，对于这样的事情也早已见怪不怪。
<br/>　　哪怕女子将路上的行人给碾死，估计也就挨一顿训斥，根本不会有任何事情，这就是弱者的悲哀。
<br/>　　慕清霄双眼一眯，他怎会看不出，前面的女子横行霸道，不但没有减速，甚至故意刺激座下魔兽，让它故意加速狂奔。
<br/>　　小医仙瞥了眼红裙女子，美眸中闪过一抹不喜。
<br/>　　无视狂奔而来的赤火兽，莲藕一样的玉臂轻轻搂着慕清霄的胳膊，对后者的呵斥充耳不闻。
<br/>　　“混蛋，既然找死，就别怪本小姐。”
<br/>　　被两人无视，红裙女子脸色一冷，眼底闪过一抹杀意。
<br/>　　随后手掌直接抽出皮鞭，狠狠的一挥，伴随着一道清脆的破风声，皮鞭暴掠而出，直接对着小医仙的俏脸划去。
<br/>　　望着皮鞭的轨迹，慕清霄眼底杀意狂涌，屈指一弹，一道剑气疾射而出。
<br/>　　“咻！”
<br/>　　锐利的破风声响起，一道寒芒闪过，当所有人顺着寒芒的方向望去，脸上的同情顿时化作惊愕。
<br/>　　不知何时，赤火兽已经停住蹄子，身躯匍匐在地上瑟瑟发抖，皮鞭也被切成两断，切口平滑。
<br/>　　最重要的是，赤火兽上的红裙女子脸上的冷意凝固，不知何时，额头上已经出现一个红点，猩红的鲜血自红点中汨汨流淌而下。
<br/>　　“砰……”
<br/>　　在众人目瞪口呆的视线中，红裙女子自赤火兽上坠落，全无声息，似乎已经身亡。
<br/>　　街道上的画面仿佛凝固，周围静若寒蝉，针落可闻，慕清霄与小医仙则旁若无人，缓步向城中央位置走去，脚步清晰。
<br/>　　可就在下一秒，两道声音破空而来。
<br/>　　来者是两名面容苍老的身影，两人背后斗气羽翼振动，望着倒在街道上全无声息的红裙女子，脸色狂变。
<br/>　　“小姐！”
<br/>　　其中一名老者掠下，探查了下红裙女子的气息，脸色阴沉如水，目光落在了慕清霄的身上，杀意四射道：“你居然杀了我家小姐！”
<br/>　　闻言，慕清霄语气平静，道：“蝼蚁而已，杀了又如何？”
<br/>　　此刻，街道四周已经聚集了大量的行人，甚至商店屋顶都有行人汇聚过来，似乎都嗅到有大事发生，一个个占着位置准备看戏。
<br/>　　当众人听到慕清霄平静的语气后，无不倒吸凉气，此子简直太狂了，但心里却无比畅快。
<br/>　　“畜生，居然敢杀老夫的孙女，老夫要你们陪葬！”
<br/>　　顷刻间，一道灰色光芒，猛的自城中心位置爆掠而来，旋即出现在街道上空。
<br/>　　磅礴的气势狂涌而出，铺天盖地的笼罩而下，阴冷中蕴含狂暴杀意的语气在街道上回荡。
