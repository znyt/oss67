　　江左看着态度强硬的鲁初雪，皱起眉小小让了一步：“那这样，平时不是有各基地送来的物资么，加一项特殊丧尸的晶体不就好了？这样一来也不用你们另外派人到处乱找，数量更多也集中。”
<br/>　　听着江左理所应当的话，鲁初雪呆了半响，简直要被气笑：“你在开什么玩笑？那些基地送来物资又不是天上掉下来的，不是你想要就能要，更不是我想提要求就能提的，这几个月的无限供应还不是因为南部边境的尸潮，这是全人类的大事，特殊情况懂不懂？！”
<br/>　　江左点头，一脸认真：“我这里也是特殊情况。”
<br/>　　鲁初雪只觉得一口气卡在喉咙口，咽不下去也吐不出来，她被江左的固执思维给气的浑身发抖，声音也不住的拔高了起来：“你这里算什么特殊情况，物资的供应是给上战场的战士们的，你在基地内呆的好好的还提这么多要求？重点是还非要在这个物资紧缺的节骨眼上提，你以为你们这些研究部研究的东西真有那么重要，能影响战局不成？！”
<br/>　　江左继续点头：“能啊，至少能占一成吧，丧尸引诱剂，新人类觉醒剂，等阶提升剂，这些都是。”
<br/>　　鲁初雪猛然噎住，嘴唇哆嗦着愣是被堵的一句话都说不出，江左的话直接说到了点子上！
<br/>　　眼下南署基地的这场战役能进展到这个地步，与研究部日夜加赶药剂脱不开关系，那些新兵在训练时更快的变强就是靠着这些药剂。
<br/>　　江左继续道：“而且两个月前我们要求提供丧尸晶体的时候，也没这么麻烦啊！为什么现在不行？”
<br/>　　鲁初雪抿了抿嘴，道：“两个月前是楚涵上将拟定，普通的丧尸晶体并不稀缺，供应起来也就不困难，更别说那会儿已经向各大基地的首领知会了这批晶体的用途，那些药剂实在逆天，他们当然会无条件支持。”
<br/>　　江左眼神一亮，笑道：“那这回更应该无条件支持啊，新发明更逆天。”
<br/>　　鲁初雪有些郁闷：“求你别再为难我了，这世上最有用的三种药剂都已经发明出来，你们的新发明再如何逆天如果和战事无关，也只能等到战后才能继续。”
<br/>　　又一次的拒绝让江左一愣，他情绪有些烦躁的开口：“特殊丧尸的晶体用途，一直由我们长官的母亲袁熙叶院长亲自研究，长官有过命令，此发明一旦成功，就要立即大批赶制。直到今天上午我们研究部才收到来自狼牙的消息，并且拿到了最新发明的全部资料，你现在却跟我说不能继续？”
<br/>　　鲁初雪勾起嘴角：“这么说来还真与战事无关？那也应当与民生救济关系不大吧？在这种全人类都抵抗丧尸的大战时期，你们的发明当然只能靠后排。”
<br/>　　“谁说与民生救济无关？”江左靠近了一步低声道：“虽然长官严令封锁消息，但新发明已经问世，我便透露一丝也无妨。”
<br/>　　鲁初雪心脏一顿，忍不住好奇心起。
<br/>　　要知道狼牙的这几个部门都异常神秘，一丁点的消息都不会往外放，就像科技部的那些新武器，段氏基地的高层也是后知后觉。
<br/>　　江左继续开口：“我只能告诉你一个人，新发明是解毒剂。”
<br/>　　鲁初雪呆住，下意识反问：“什么解毒剂？”
<br/>　　这种感觉很失望，好似一个藏宝图终于走到终点的时候，却发现藏在那里的宝藏只是最普通的玩意儿。
<br/>　　就像当下的狼牙研究部新发明，江左与鲁初雪纠缠了半天讨要资源，最终却拿出了这么一个结果。
<br/>　　鲁初雪不仅失望透顶，还有些愤怒交加！
<br/>　　她猛然声音提高了一个八度：“解毒剂，就这东西？！这时候谁还需要解毒剂，解什么毒？”
<br/>　　当鲁初雪这众多的情绪在刹那间一闪而过的时候，江左却已经出声，他推了推眼镜，反射着亮光的镜片后方，是一双科研人员固有的木讷双眼，只是在此时的一瞬间，那木讷之中却透露着一丝犀利。
<br/>　　“丧尸病毒。”江左如是说。
<br/>　　空气刹那间冷凝，并不宽敞的屋内一下子变的极其安静，之前争辩不断的噪杂声无影无踪，只剩下了好似静止了的时空。
<br/>　　鲁初雪半张着嘴，面容上还带有前一秒未来得及收回的生气情绪，她呆呆的看着江左，好似听到了比当初末世爆发更令她震惊的事。
<br/>　　江左没有太多反应，只是回望鲁初雪，眼神里的坚定与一个小时前踏入这里时丝毫未变。
<br/>　　狼牙研究部的新发明，就是丧尸病毒解毒剂！
<br/>　　原材料，就是特殊丧尸的晶体！
<br/>　　这是一项创世纪的发明，更是一项将不可能变成可能的发明，在这个末世三年所有人都对这一想法绝望的时候，狼牙的研究部却成功研究了出来。
<br/>　　昏迷的楚涵并不知道这件事，如果知道一定会对自己的老妈袁熙叶彻底另眼相看，甚至崇拜，因为这一发明是哪怕在上一世末世十年，都未曾研究出的逆天成果！
<br/>　　丧尸病毒的感染是不可逆转的，这是楚涵所在上一世整个末世纪元十年的一项真理。
<br/>　　被感染的人类甚至新人类，都只有一个结果，那就是变成丧尸，不可能救的回来，不可能逆转。
<br/>　　这是楚涵在哪怕一年前都坚信的事实，哪怕上一世他就知道曹氏研究所在对特殊晶体进行最后阶段的研究，更有机密情报透露一直没有被定位的特殊晶体，就是与丧尸病毒的解毒有关。
<br/>　　可是丧尸病毒感染的是基因，已经改变的基因如何逆转？
<br/>　　所以楚涵一直将其视为空谈，更没有放在心上，可知道楚涵自己受到感染，并被回炉系统清除了体内病毒，因此而损失了大笔回炉积分之后。
<br/>　　他的想法便一下子改变！
<br/>　　感染的逆转不是不可能，既然他可以从感染的状态里回归正常，那其他人也一定可以。
<br/>　　于是被楚涵放在一边的那些特殊晶体，就此重见天日并全部转移到了狼牙研究部。
