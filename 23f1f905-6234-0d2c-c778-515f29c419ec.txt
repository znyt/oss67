　　“四长老，你怎么来了？”萧逸忍着伤势，疑惑问道。
<br/>　　四长老道，“萧景发信号回矿脉，知道你去追击铁刀和慕容千军，我怕你有事便赶过来了。”
<br/>　　“他们怎么样了？”萧逸追问道。
<br/>　　“放心。”四长老道，“已经有其他族人接他们回矿脉了。”
<br/>　　“那就好。”萧逸放下心来。
<br/>　　这时，不远处一道道人影倏忽而来，竟是附近的猎妖师和独行武者。
<br/>　　这里可不是隐蔽的密林，而是数十里之外，不时会有猎妖师和武者经过。
<br/>　　刚才萧逸与慕容千军使用玄阶武技战斗，威势太大，再加上慕容家长老的一声大喝，自然把他们吸引了过来。
<br/>　　“咦，那不是慕容家的慕容蝎长老，还有我们紫云城有名的天才慕容千军公子吗？”
<br/>　　“他们对面的好像是萧家的四长老吧，那个满身是血的少年又是谁？”
<br/>　　闻声赶来的武者们议论纷纷，但他们都在远处看着，并没有太过靠近。
<br/>　　“我好像听到萧家四长老称那少年为萧逸，难道就是那个‘闻名遐迩’的萧家废物？”
<br/>　　“哈哈哈哈。”周围的武者不禁笑了出来。
<br/>　　“闭上你们的嘴巴吧。”忽然有些猎妖师沉声提醒道，“这大半个月来，萧逸公子可谓名声大噪，连萧若狂那等风云天才都败于他手上，乃是萧家第一天才，也是你们敢随意腹诽编排的？”
<br/>　　很多猎妖师和武者在陨星山脉中行走，一般都会待上数月，待得猎杀够妖兽才离开，自然不知道紫云城内最近的事情。
<br/>　　“什么，那个小废物...哦，不，萧逸公子竟然连萧若狂都打败了？我听说那萧若狂可是玄火门最出色的弟子阿。”
<br/>　　猎妖师们纷纷露出了惊讶之色。
<br/>　　附近的猎妖师和武者越来越多，他们终日在危险中渡过，难得遇到热闹，都围了过来。
<br/>　　“呵。”一中年武者忽然笑了笑，道，“恐怕，从今天开始，萧逸公子就不再是萧家第一天才，而是紫云城第一天才了。”
<br/>　　那武者面露得意之色，道，“刚才我远远地就在看，萧逸公子一下子就杀了铁刀和他的队员，连慕容千军公子都败在了他手上。你们看，慕容千军公子现在身受重伤，要不是慕容蝎长老来，恐怕他危矣。”
<br/>　　“什么？连慕容千军公子都败了。还有铁刀猎妖队的人，怎么回事，萧逸公子怎么会对付他们？”
<br/>　　“还能有什么，前不久慕容千军公子散播谣言，毁了萧逸公子的声誉。啧啧，那等谣言，本来就没什么人会信，慕容千军公子还借此对付萧家子弟，萧逸公子当然要来报复了。”
<br/>　　.....
<br/>　　各个猎妖师们议论纷纷，现场变得越来越热闹。
<br/>　　慕容千军听着这些话，气得七窍生烟。
<br/>　　慕容蝎直接大喝一声，“都给我闭嘴。”
<br/>　　一声怒喝，周遭的猎妖师们顿时噤声。他们只是来看热闹的，可不想得罪慕容家。
<br/>　　慕容蝎看向四长老和萧逸，怒声道，“萧家的，你们伤我慕容家子弟，这笔账怎么算？”
<br/>　　四长老冷声道，“那你想怎么算？”
<br/>　　慕容蝎怒道，“萧逸打伤千军，犯我慕容家威严，自当跪地叩头谢罪，而后自废双手。”
<br/>　　“慕容蝎，我想你误会我的意思了。”四长老的声音变得异常冰冷。
<br/>　　“嗯？”慕容蝎皱起了眉头。
<br/>　　四长老冷冷道，“我是问你，你刚才远远一掌打伤萧逸，这该怎么算。”
<br/>　　“哼。”慕容蝎不屑道，“老夫乃是先天武者，不过是教训了一个小辈，你还有意见了不成？”
<br/>　　四长老冷声道，“我萧家子弟，何时轮到你来教训了。”
<br/>　　“废话少说。”慕容蝎道，“让萧逸把千军的乾坤袋还来，否则，此事没完。”
<br/>　　“乾坤袋？”四长老冷声道，“这乾坤袋就当作你刚才一掌打伤萧逸的赔礼了。”
<br/>　　“赔礼？你让我堂堂一个先天武者给一个小辈赔礼？萧家长老，你不要太过分了。”慕容蝎顿时大怒，大有一番大打出手的姿态。
<br/>　　“怎么？”四长老怡然不惧，道，“想打架的话，我奉陪。”
<br/>　　两人都是先天武者，而且修为差不多，真要打起来互相谁也讨不了好。
<br/>　　慕容蝎也明白这一点，他原本就没想真的打，毕竟他和萧家四长老实力差不多，但慕容千军现在却是重伤，要是给萧逸趁机杀了慕容千军，他无法向家族交代。
<br/>　　“哼，萧家的，你们蛮横霸道，老夫懒得与你们争辩计较。但，终有一天，我会让你们全还回来的。”慕容蝎放了一声狠话，抱起慕容千军，准备离开。
<br/>　　“慕容蝎是吧？”
<br/>　　这时，一直沉默的萧逸忽然说话了。
<br/>　　“嗯？”慕容蝎疑惑地回过了头。
<br/>　　萧逸冷冷道，“我萧逸要杀的人，从来没有机会活命。今日你将慕容千军救走，算他大命。但你记住了，最好让他躲起来，别再出现在我面前，下一次，他不会有这样的运气。”
<br/>　　“小子，你有种，胆敢威胁我们慕容家最出色的天才？”慕容蝎冷冷地喝道。
<br/>　　萧逸没有再说半句话，转过身，对四长老道，“四长老，我们走吧。”
<br/>　　嗖、嗖，两人离开，其他猎妖师们见没戏看了，也纷纷离开。慕容蝎抱着慕容千军，心怀愤怒，但也只能愤愤地一甩袖子，离开了。
<br/>　　.......
<br/>　　萧家矿脉，矿脉旁有一亭子，亭边树木葱翠，是族人们平时歇息的好地方。
<br/>　　萧逸静静地坐着，脸色淡漠，目光有些失神，似在想什么东西。
<br/>　　距离上次与慕容千军一战，已经过去了五天。
<br/>　　忽然，几个少女盈盈走来，甜甜地叫了一声，“少家主。”
<br/>　　萧逸收回了思绪，认出了面前的几个少女，都是家族里颇有名气的天才，父母皆是家族里的执事。
<br/>　　“舒月，找我有事？”萧逸淡淡地问道。
<br/>　　几个少女中，萧逸唯独和其中一个比较熟悉。萧舒月，今年16岁，已是凡境八重武者，武道天赋倒是不错。
<br/>　　当然，萧逸认得她是因为她父亲是萧百炼，萧逸曾经在炼药堂见过她。
<br/>　　舒月柔声说道，“少家主，我们准备到小型矿脉那边巡视，要一起前行吗？”
<br/>　　“不了。”萧逸淡淡地回答了一句。
<br/>　　“哦。”舒月脸上明显掠过一丝失望，而后转身和另外几人离去。
<br/>　　这时，四六长老刚好经过，看了眼离开的舒月等人，而后走向萧逸。
<br/>　　“你小子，怎么回事，一副欠揍的样子。”六长老打趣道。
<br/>　　“就是，舒月可是家族里最漂亮的子弟，美人相邀，其他子弟求都求不来，你倒是不理不睬的。”四长老也打趣着。
<br/>　　六长老挪揄道，“怕是这小子心里只有依依吧。”
<br/>　　“诶。”四长老摆摆手，道，“萧逸你是少家主，终究要快点成家立室。男人嘛，三妻四妾没什么的，要我帮你把舒月叫回来吗？”
<br/>　　萧逸白了他们一眼，道，“你们这算是为老不尊吧，要不要我回家族去大长老那告你们一状。”
<br/>　　“再说了，我这个人人都挂在嘴边的废物，身上还背着那种谣言，那个女子会看得上我呢。”萧逸自嘲说着。
<br/>　　“得了，尽说些气话。”六长老道，“现在谁不知道你萧逸的名字，紫云城内都传开了，紫云城第一天才，谁还敢说你一声废物。”
<br/>　　“至于那种谣言嘛，慕容家的人出了名的卑鄙，从头到尾就没有人相信过他们。”
<br/>　　“现在紫云城内，谈论起你，都是一片赞赏之声，三长老前些天传信回来时都乐坏了。”
<br/>　　六长老滔滔不绝地说着。
<br/>　　四长老摆了摆手，示意六长老闭嘴，而后沉声道，“萧逸，你尽说些气话，是还在想萧壮和萧子木两人的事吧。”
<br/>　　萧逸点点头。
<br/>　　四长老叹了口气，道，“这其实也与你无关...”
<br/>　　萧逸打断道，“我不想欠别人人情。”
<br/>　　“哎，这都过去五天了，他们还未醒来吗？”六长老问道。
<br/>　　萧逸回答道，“他们丹田被毁，伤势比其他人更重，自然更迟醒来。不过幸好他们没有生命之虞，再过两天必然能醒来。”
<br/>　　“那就好，但我只怕他们醒来时，发现在丹田被毁，日后再也无法修炼，会不会承受不住打击...”六长老话还未说完。
<br/>　　四长老立马瞪了他一眼，“笨蛋，你是哪壶不开提哪壶是吗？”
<br/>　　六长老撇撇嘴，不再说话。
<br/>　　这时，四长老从怀中拿出一封信，道，“萧逸，萧壮和萧子木的事，我派人送信回家族问过大长老。这是大长老给你的解答，好像是有那个可能恢复丹田的。”
<br/>　　四长老将信丢给萧逸，道，“那些炼药的知识我也看不懂，你自己慢慢看吧。”
<br/>　　萧逸顿时眼前一亮，迫不及待地接过了信。
