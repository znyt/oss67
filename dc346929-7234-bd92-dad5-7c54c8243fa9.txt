
<br/>　　<br/>　　    没有一点光线的树林里，王清雪靠着棵树，徐闯顶着她的身子，双手从腰上环抱过去，浓重的呼吸着，撅着两片嘴唇就要印上去。<br/>　　<br/>　　    王清雪伸手推了他一把挣扎了两下，不愿意的说道：“不是说好的要来散步的么？你搞什么啊，万一有人过来怎么办？”<br/>　　<br/>　　    徐闯急头白脸的说道：“谁那么不懂事啊，咱俩一钻树林，他们就知道这是要干啥了，没人会脑子有病的闯进来的，来吧来吧，你看这月光多惬意，多有情调，此情此景不干点什么就浪费了”<br/>　　<br/>　　    “呜呜，不要，太丢人了……哎呀，你顶到我了”王清雪脸色绯红的说道。<br/>　　<br/>　　    王清雪刚把脑袋转过去，徐闯一把拉过她，咽了口唾沫后说道：“我这都是野战军出身的，弹药已经上膛，啥也不说了，来吧……憋啥样了都”<br/>　　<br/>　　    和徐闯，王清雪相隔不远，另外一边的树林里，赵文亮看着羞涩的小雯，一本正经的说道：“雯啊，咱们既然已经有点郎情妾意了，我觉得今天应该就把关系给定了，从今以后我就想着，理工大学的校园里你我可以肆无忌惮的携手漫步了，是不是这么个道理？”<br/>　　<br/>　　    “森马关系啊？”小雯眨着天真的大眼睛说道。<br/>　　<br/>　　    “你中有我，我中有你，处对象的关系呗”赵文亮伸手拉着对方的胳膊，深情款款的说道：“你是风儿我是沙，咱俩永远缠绵到天涯，我宣你，小雯”<br/>　　<br/>　　    “舌头，舌头捋清了和我说”<br/>　　<br/>　　    “我喜欢你，小雯”赵文亮半低着脑袋，瞅着小雯的两片红唇，用手指抬着她的下巴说道：“值此良辰美景，你我一吻定情吧”<br/>　　<br/>　　    小雯闭着眼睛轻声的“嗯”了一下，赵文亮跟个狼狗似的伸着舌头就凑了过去，想要来个法式的湿吻，这时候忽然间树林下的草丛里，传出一阵“悉悉索索”的动静，他俩顿时被吓了一跳，小雯哎呀了一声，说道：“是不是有蛇啊？”<br/>　　<br/>　　    赵文亮的反应还算挺快的，他连忙从口袋里拿出手机调出手电筒就朝着地上照了过去，来回晃悠了几下后，就看见两人脚下不远处一条黄色的大耗子，身长大概为四五十公分左右，屁股后面耷拉着一条长长的尾巴，四肢着地，尖嘴有胡须，两只眼睛很贼的提溜乱转着，但看起来状态好像很虚弱，略微张开着嘴巴一直在喘着气，眼神黯淡无光，一看就是有气无力的。<br/>　　<br/>　　    “老鼠？”女人的天性就是怕这类玩意儿，小雯连忙抓紧赵文亮的胳膊，但赵文亮手中的光似乎把这大耗子给吓到了一样，它慌忙掉头就走又蹿进了草丛里。<br/>　　<br/>　　    但是紧接着，另一边的树林里，就听见有人喊了一声：“哎呀我去，什么玩意儿啊”<br/>　　<br/>　　    “过去看看，好像是徐闯他俩那边，去看下别让这耗子给他们两给咬了”赵文亮拉着小雯就走了过去，片刻后就看见徐闯怒气冲冲，王清雪慌忙整理着衣服，他俩身前那只大号的耗子正虎视眈眈的盯着，呲着牙，在吹胡子瞪眼。<br/>　　<br/>　　    “怎么回事？”赵文亮问道。<br/>　　<br/>　　    徐闯咬牙说道：“这他么的黄毛畜生突然就冒了出来，把我俩给吓了一跳，跑到我脚边后我一脚就给它踢了”<br/>　　<br/>　　    赵文亮这才看见那大耗子的嘴角上好像带着一点血丝，眼里的目光很人性化的带着一点怒气，大耗子低声呜咽了两下，忽然缓慢的迈着前蹄就朝徐闯那边走了两步。<br/>　　<br/>　　    似乎被徐闯踢了一脚，让它略微不太满意，这玩意儿好像还挺记仇的。<br/>　　<br/>　　    “唉？这家伙还不怕人，我踢了它一脚居然还不跑，怎么的呢，你还有报仇雪恨的意思啊？”徐闯不屑的撇了撇嘴，直接抬起脚又朝着那耗子踢了过去。<br/>　　<br/>　　    “唰”对方往后退了两步躲开了，但仍然呲着牙没走，赵文亮见状低头拿着手机在地上找了一会后，就看见一棵树下有两根挺粗的树枝，捡起来后扔给徐闯一根，说道：“别用脚了，小心咬你一口还得打狂犬疫苗，用棍子给它赶走得了”<br/>　　<br/>　　    徐闯接过树枝子就围了过去，赵文亮手里的手机就朝着那耗子照了一下，摄像头的亮光顿时就落在了它的脑袋上，黄毛大耗子的眼睛被强光一透，顿时就出现了短暂的盲视，紧接着徐闯手里的树枝朝前一捅就打在了对方的脑袋上。<br/>　　<br/>　　    “嘭”赵文亮随即一扬手里的棍子，奔着大耗子的身上就砸了过去，一下求砸在了它的腰上。<br/>　　<br/>　　    “嗷呜”耗子吃痛，咧着嘴叫唤了一声，然后掉头就要跑，他俩谁也没看见，它脑袋上眼睛里全是血，刚才正好被徐闯的树枝子给捅到了眼眶上。<br/>　　<br/>　　    赵文亮跟徐闯两下打准了以后，也没收手，仍旧追着对方又敲了好几下，似乎可能是受伤了的原因，黄毛耗子的动作不太利索，撒开腿跑的都没有多块，身上一连被两根树枝子给砸中了好几下。<br/>　　<br/>　　    “嘭”<br/>　　<br/>　　    “嘭，嘭”<br/>　　<br/>　　    “你快吓死你爹了，还他么的把我的好事给搅合了，我打不死你……”<br/>　　<br/>　　    王清雪在后面跺着脚喊道：“你俩别追了，赶跑了就行了，还真想给打死了啊，太残忍了”<br/>　　<br/>　　    “嗖”徐闯气鼓鼓的把手里的树枝子朝着走远了的黄毛耗子砸了过去，回来后说道：“不长眼的黄毛畜生，就坏我好事，一看就是山里成了精的耗子，打死了也是为民除害”<br/>　　<br/>　　    “我看着也不像耗子呢，好像在哪里见过，有点想不起来了”赵文亮拍了拍手，想了半天也没想起来刚才那黄毛尖嘴的动物叫什么，只觉得似乎在小的时候见到过。<br/>　　<br/>　　    一个小插曲过后，赵文亮和徐闯对视了两眼，小声问道：“到哪一步了？”<br/>　　<br/>　　    徐闯磨着牙，恨恨的说道：“刚要上马，就被那只牲口给搅合了，眼看着就快提枪了”<br/>　　<br/>　　    赵文亮叹了口气，说道：“你比我强多了，我是刚牵上手，准备来个一吻定情的时候，节奏就被打乱了”<br/>　　<br/>　　    两个泛着荷尔蒙的男淫，长吁短叹了几声，大好的夜晚全被这只黄毛的耗子给搅合了，节奏已乱，现在再往下走一步，明显不太可能了。
