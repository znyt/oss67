        “是吗？但那不是我生的孩子，是季思妍生的孩子……”童筱颖黯然神伤地说道。
<br/>
<br/>        也是小卫的孩子……
<br/>
<br/>        傅夜沉凝视着童筱颖那被热水泡得氤氲生晕的粉颜，浮泛着一丝忧伤与沮丧，忽然间心疼起她来。
<br/>
<br/>        三年前，他不该放手的。
<br/>
<br/>        “不过，你的心可真大！”要不是手得抓着覆在胸上的毛巾，她还真想给他鼓掌，像他这么优秀的男人，居然愿意娶她这么一个二婚女人。
<br/>
<br/>        “你呢？”顿了下，他反问，“筱颖喜欢孩子吗？”
<br/>
<br/>        “不知道。”童筱颖不假思索地说，回答得简洁有力，毫不拖泥带水。
<br/>
<br/>        其实，她有想过这个问题。
<br/>
<br/>        她想过要给小卫生个孩子……
<br/>
<br/>        可结果……
<br/>
<br/>        哪怕是她脱光了，小卫也不要她。
<br/>
<br/>        “讨厌吗？”
<br/>
<br/>        “也不会。”童筱颖想了下，“但是，我想，要是自己的孩子，没道理不爱。”
<br/>
<br/>        “我们会有属于自己的孩子。”他浅浅地扬了扬嘴角。
<br/>
<br/>        “嗯？”她没听清楚。
<br/>
<br/>        童筱颖眨眨眼装可爱，发现他的视线根本没落在她身上，只是半垂着眼，像是在沉思什么。
<br/>
<br/>        又静默下来了。
<br/>
<br/>        是她搭错话了吗？刚刚的气氛明明还不错的啊，她抛球给他，他都会打回来的，为什么下一刻又停住了？
<br/>
<br/>        她心里别扭着，决定再辟战场。
<br/>
<br/>        “那个……大哥，苏漫雪、季思妍、还有你的前任未婚妻，你……到底喜欢谁？”童筱颖忽然觉得自己是不是有点儿八卦了，但是，总得公平点吧！
<br/>
<br/>        他都知道她的感情史了，她也得多多少少地知道一些有关于他的吧！
<br/>
<br/>        “都不喜欢。”傅夜沉微挑起剑眉，他将挡在长睫前的刘海拨到脑后，顿了下，似有若无地勾起唇，“苏漫雪。”
<br/>
<br/>        他为什么突然改口说苏漫雪，她或许不知道其中的原因，但他心里却十分清楚。
<br/>
<br/>        因为她跟筱颖长得一模一样，他爱的是筱颖啊！
<br/>
<br/>        苏漫雪？！
<br/>
<br/>        童筱颖心头强震了下，因为他慵懒的姿态，尤其他一扬笑，脸上刚硬的线条瞬间柔和。
<br/>
<br/>        那柔软的线条像是一束光线，打落枝上的叶，叶落进她无波的心湖，震起圈圈涟漪。
<br/>
<br/>        她转不开眼，心里又莫名其妙地萌生了坏念头。
<br/>
<br/>        果然……
<br/>
<br/>        他一直把她当成了苏漫雪的替身。
<br/>
<br/>        这样也好，两个互不相爱的人结婚，彼此就不会伤害太深。
<br/>
<br/>        以前，她在小卫面前，总是很卑微的样子，那是因为爱。
<br/>
<br/>        现在，她不需要再卑微了，因为已经无所谓了。
<br/>
<br/>        “大哥。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “如果哪天，苏漫雪对你回心转意了，想当你的傅大少奶奶了。你放心，我会主动在离婚协议书上签字的！不过，你得给我一百万作为补偿，一百万对大哥来说，应该不算什么吧！我只要一百万就行了，足够我过下半辈子了！到了你这儿，我是二婚了，再离婚，就是三婚了。肯定没男人要了！”童筱颖慢条斯理地说，而且还把话说得一本正经。
<br/>
<br/>        “你还想三婚？”傅夜沉哭笑不得地扬挑起眉梢，微勾的唇角有著一抹不解的笑意。
<br/>
<br/>        他的傻筱颖啊！真的是傻得有些可爱！
<br/>
<br/>        “对啊！如果能再遇见一个自己爱的同时又爱自己的男人，我肯定会考虑‘三婚’的。我不是那种死心眼的女人，不会在小卫这一棵树上吊死的！”突地意会自己失言，童筱颖凝住的笑容变得很尴尬，很想再把脸埋进水里，把自己淹死算了。
<br/>
<br/>        然而，尴尬过后，她又心痛了。
<br/>
<br/>        瞧！
<br/>
<br/>        她可以把话说得多漂亮啊！
<br/>
<br/>        不会在小卫那一棵树上吊死！
<br/>
<br/>        而事实上呢？
<br/>
<br/>        她已经快吊得半死不活了吧！
<br/>
<br/>        “不错。”她若不会在一棵树上吊死，那么就代表着他确实还有机会，“哈哈——”

<br/>        童筱颖猛地抬眼，水亮的眸子里映着他笑得放肆的神采。
<br/>
<br/>        他此刻的表情像个大男孩，而不是那个高高在上的傅大总裁。

<br/>        察觉到自己的放肆，傅夜沉敛起笑，轻咳两声：“水有点凉了，要起来了吗？”
<br/>
<br/>        “你先上去，我等一下就上去。”拜托，她跟他还没到这么“坦诚相见”的程度好吗？

<br/>        虽然，在这之前，他确实对她做过一些比较少儿不宜的事情。

<br/>        但是，那几次，都是意外……
<br/>
<br/>        他一直都把她当成是苏漫雪了吧！

<br/>        傅夜沉没说什么，赤裸裸起身，淡定从容，一点都不扭捏。

<br/>        童筱颖连忙闭上眼睛，告诉自己非礼勿视。

<br/>        但是不知道为什么，眼睛像是有了自己的主张，很失控地探向他的背影。
