　　周静茹笑道：“这是我家投资的酒店啊，拥有大部分股份的，告诉他们一声就知道了。”
<br/>　　二十分钟没到，外面有人敲门。
<br/>　　周静茹过去开门，叶少阳也伸头往外面看，居然是个熟人，很快想到了是那个叫陈涛的，周静茹家在这边的分公司的总经理，之前从局子里帮忙捞过自己一次。
<br/>　　面对周静茹，陈涛完全没有一点领导的架子，脸上带着谦卑而夸张的笑，双手捧着一个托盘，上面放着两瓶红酒，冲周静茹点头哈腰：“周经理，饭做好了。”
<br/>　　周静茹看着他手里的酒，道：“我没要酒。”
<br/>　　“这个啊，这个是拉菲庄82年的酒，是一个法国客人前年送给我的礼物，我这个粗人，不懂红酒，想着送去给大小姐……不不，送去给周经理，一直没这个机会，今天也是天赐良机，我就借花献佛了，请周经理跟叶先生享用……”
<br/>　　周静茹这才收下了。
<br/>　　陈涛让开，后面跟着几个女服务员，每个人都端着托盘，上面用金色的罩子罩住，往屋里餐桌上送。
<br/>　　“这是本店大厨特意做的几个招牌菜，我担心两位饿了，赶紧让他做出来几个，还有些燕窝什么的，需要慢慢炖，等炖好了立刻端上来……”
<br/>　　“辛苦了。”周静茹冲他笑了笑。
<br/>　　“不辛苦不辛苦，那个周经理，叶先生，还有什么需要的吗？”
<br/>　　周静茹道：“警方那边怎么说了。”
<br/>　　“哦，已经打过招呼，明天叶先生去录个口供就行，很省事的，主要是签个名，走个程序。”
<br/>　　“知道了，你们去吧。”周静茹道，“叶先生需要休息，如果有警察过来，无论有什么事，都推到明天，今天不许上来。”
<br/>　　“一定一定。”陈涛拍胸脯保证，带着几个服务员离开。
<br/>　　“有钱有地位，真是好啊。”叶少阳双手枕着脑袋，躺在床上，冲周静茹笑道。
<br/>　　“别取笑我了，快来吃饭。”周静茹吐了吐舌头，招呼叶少阳到客厅去吃饭，亲自把金属罩一个个掀开。
<br/>　　叶少阳看了下，菜简直丰盛得可怕，而且都是自己没见过的花样，唯一认识的只有龙虾，不是小龙虾，是整个盘子那么长的大家伙。果然还是资本家牛逼啊。
<br/>　　周静茹让他先吃，自己倒酒，然后给他夹能管饱的菜，让他先垫肚子，然后才把红酒给他，跟他碰杯喝了一口。
<br/>　　“少阳哥，给我讲讲你这次的经历吧。”周静茹坐下来，跟他一起吃。
<br/>　　“这个……一时半会说不完啊。”
<br/>　　“那你就慢慢吃，一边说给我听。”
<br/>　　叶少阳只好根据回忆，把事情从头讲起来，周静茹自己不怎么吃，一直在帮他挑虾肉投食给他，津津有味地听着。
<br/>　　叶少阳讲得也很嗨，时间过得很快，两人不知不觉把一瓶红酒都喝完了。
<br/>　　这酒喝着时候没什么感觉，喝完了后劲居然不小，叶少阳有点犯困——本来这几天在那虚幻空间里也没睡好过，躺下没一会就睡着了。
<br/>　　醒来的时候，房间是黑的，叶少阳想要坐起来，发现自己身上搭着一只手，转头一看，是周静茹，躺在自己身边，一只手搭在自己胸前，脑袋也贴着自己。
<br/>　　叶少阳立刻想到了电视剧里经常演的那一幕，吓的赶紧伸手往自己身上摸，还好还好，衬衣还穿着呢。
<br/>　　转头看周静茹，借着月光，能看到她正睁眼望着自己。这个样子的她，显得更加妩媚。
<br/>　　“你要起床吗？”周静茹幽幽问道。
<br/>　　“我……”
<br/>　　“你就在这里睡吧，放心，我不碰你的。”周静茹笑道。
<br/>　　叶少阳只好躺下了。
<br/>　　周静茹又把脑袋往他怀里拱了拱，黏糊了一会，在他耳边说道：“你心跳得好快。”
<br/>　　“呃，这种局面，我要是不激动就不正常了。”
<br/>　　周静茹轻笑，道：“我怎么没事。”
<br/>　　“我不信。”
<br/>　　“不信你试试？”
<br/>　　叶少阳尴尬了。这算套路吗？
<br/>　　周静茹用双臂环绕住他，嘴巴凑上来，她身上带有一种让人难以抗拒的幽香，叶少阳感觉自己心跳地更快了，呼吸也急促起来。
<br/>　　周静茹的呼吸也在急促。
<br/>　　然后也不知道谁先开始的，两人的嘴唇贴在了一起，叶少阳的手也不老实起来……
<br/>　　叶少阳深吸一口气，强行压住邪念，坐了起来。
<br/>　　周静茹从后面抱住他，把脑袋贴在他后背上，轻轻叹了口气。
<br/>　　“怎么？”叶少阳回身问道。
<br/>　　“你躺下吧。”
<br/>　　叶少阳听话躺下，周静茹把脑袋趴在她胸前，叶少阳没有拒绝。
<br/>　　“像不像偷情？”周静茹问道。
<br/>　　“这……”叶少阳不知道怎么回答。
<br/>　　“少阳哥，你还记得，当时雨晴姐跟我说过的吗，我当时想，可以一辈子这样，但是……我现在很担心自己做不到……”
<br/>　　周静茹苦笑起来，道：“我从小早熟，喜欢看言情小说，我也幻想过自己的爱情……可能是家庭的原因，我从来没想过，我将来的老公要多有钱，我希望他是一个英雄，他能代我去一个我从来没去过的世界，结果我等啊等，直到你出现了……”
<br/>　　她的眼泪不断流出来，落在叶少阳胸口，叶少阳感觉自己的心也在滴血。
<br/>　　“我猜到了开头，却没猜到结尾。你说，我们是不是认识得太早了？”
<br/>　　叶少阳感觉嗓子发干，深吸一口气，道：“静茹，你应该找一个合适的……”
<br/>　　“上哪里去找你这样的了？”
<br/>　　“我……我从来没觉得自己很优秀。”
<br/>　　“但你在我心中，就是最好的那一个，我很清楚，我不可能再对别人动心了。我的事，由我自己决定，你不用过问。”
<br/>　　她抹了一把眼泪，冲叶少阳微微一笑，“你记着，如果有一天我走了，你不要去找我，你记住了，我要走了，就不会再回来……”
<br/>　　叶少阳抱紧她。
<br/>　　一夜就这样过去了。
<br/>　　叶少阳不知道自己是什么时候睡着的，清晨醒来，周静茹已经在洗漱了，叶少阳起床穿好衣服，周静茹听见动静，转头看着他，甜甜一笑。“早安。你是在房间里吃早餐，还是去楼下餐厅？”
<br/>　　（后面的故事，会在开心和热血之余，带一点淡淡的哀伤，毕竟有笑有泪，才是我们的生活啊。）
