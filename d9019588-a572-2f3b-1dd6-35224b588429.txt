　　妙心问叶少阳：“那这天干地支的搭配，明明就是错的……”
<br/>　　“这不叫错，在后天八卦里，确实推不出这种组合，但是先天八卦就不一样了，不过这里只有两柱，说明只用了先天八卦中的一个易数，先天八卦一共有八个易数，布置这阵法的人能参透其中一组易数，也算是不容易了。”
<br/>　　“一派无言！”陈晓宇道，“先天八卦就算有，世人怎么可能参透！你既然大言不惭，那你告诉我，这八组易数是什么？”
<br/>　　“你当我跟你一样猪头啊！我又不是你爹，我凭什么告诉你？”
<br/>　　“你！”陈晓宇突然拔剑，指着叶少阳，“你敢骂我！”
<br/>　　“我骂的就是你！”相比于实力而言，叶少阳的脾气要差得多，这逼，从认识开始，就一直跟自己作对，各种怼自己，不管自己说什么，是对的还是错的，这家伙都要反对，并且嘲讽自己一番，这一次他虽然没说什么重话，不过一路上积累下来，叶少阳实在有点忍无可忍的感觉，忍不住怼起来。
<br/>　　“呵……”
<br/>　　陈晓宇的妖仆、那只鬼面猴立刻跳上陈晓宇的肩膀，喉咙里发出暴怒的声音，对叶少阳龇牙咧嘴，看这模样，只要陈晓宇一声令下，他会毫不犹豫对叶少阳发动进攻。
<br/>　　“来啊！敢动一下，我立刻让你魂飞魄散！”
<br/>　　叶少阳一只手按着七星龙泉剑，表情跟那鬼面猴一样恐怖。
<br/>　　一时之间，大家都被他的表现吓呆了。
<br/>　　从认识开始，叶少阳给人的感觉就是一副不温不火的，就算是接下陈晓宇的赌约，也被人视作是普通人的意气用事，但是此时此刻，他表现出的盛怒的一面，其中透露出的这种霸气，却是与他天师的身份完全不相称。
<br/>　　鬼面猴一时间也被他盛气凌人的气势怔住了，陈晓宇也怔住。
<br/>　　“小孽畜，你动我一下试试！”叶少阳声音阴冷得让人心悸。
<br/>　　那鬼面猴狂吼一声，朝叶少阳扑过去。
<br/>　　“别这样！”妙心和毛小方想要阻止，已经来不及了，心中闪过一个直觉：叶少阳一定会伤在这鬼面猴的手中了。
<br/>　　方才不管怎么盛气凌人，毕竟只是气势上的，他的实力，大家都是知道的，鬼面猴至少是个妖灵，被它这么贴面进攻，以他的实力，只要挨了一击，不死也得重伤。
<br/>　　就在这时，叶少阳猛然举起了右手，口中飞快地念起了两句咒语：“天通降临，万物皆惊！破！”
<br/>　　随着大喝一声，手掌之中，猛然射出了一道灵光，飞射出去。那鬼面猴正在朝他扑过来，猝不及防，被射了个正着，顿时惨叫一声，向后翻过了陈晓宇，坠落在地上。
<br/>　　叶少阳右手一抖，也是被震退了两步，右手抽搐起来，疼得深吸了一口气，暗自感慨自己的实力不济，要是以往自己法力巅峰的时候，用阴阳镜全力拍这么一只孽畜，不说碾死一只蚂蚁吧，至少也是根本就不用费力，轻松就能拍死它。
<br/>　　“真尼玛不中用了！”叶少阳恨恨地说道。不过，他这一手，对于身边那些人的震撼却是了不得，连同本来想要施以援手的妙心和毛小方，也是怔怔地站住，一脸不可思议地望着叶少阳。
<br/>　　至于陈晓宇，早就惊呆了。
<br/>　　鬼面猴在地上翻滚着，早就没有了之前那股气势，一边翻滚还一边哼哼着，显然受伤不轻。
<br/>　　谁也没想到，叶少阳居然一击就将至少身为妖灵的鬼面猴拍成了重伤。
<br/>　　“啊，你找死！”陈晓宇回过神来，挺剑直上，要来刺叶少阳。
<br/>　　这时候妙心和毛小方都来得及做出反应，上前拦住了陈晓宇。
<br/>　　“晓宇，不可如此！”妙心喝道。
<br/>　　毛小方一只手按着陈晓宇的肩膀，说道：“陈师弟，是你的妖仆先对他动手，人家只是反击……既然没什么事，就算了吧。”
<br/>　　陈晓宇不依，但也不好直接对他们两个动手，尤其是妙心，只能恨恨地望着叶少阳，喝道：“你不是我家猴王的对手，你手里有什么！”
<br/>　　“你说这个啊！”叶少阳笑起来，冲他晃了晃手中的阴阳镜。
<br/>　　之前盛怒的表现，并不是他虚张声势。
<br/>　　瘦死的骆驼比马大。他虽然只有真人的法力，但是身上有很多九段光的法器，对付陈晓宇这个法师或许没什么用，但是对于邪物，有非常强大的杀伤。因此，他有自信，甚至潜意识里想要激怒那鬼面猴，给他个教训。受气受了一路，他实在不想再忍下去。
<br/>　　虽然不到彻底撕破脸皮的时候，但至少也得给他个下马威，免得一路上再说些怪话给自己添堵。
<br/>　　陈晓宇虽然不知道叶少阳手中是什么，但也清楚不是一般法器，冷冷说道：“你不过是仗着法器之利，趁人不备，突袭伤人，算什么东西，你这样我更看不起你！”
<br/>　　叶少阳耸了耸肩道：“你尽管看不起啊，反正我出气了，受伤得又不是我。”
<br/>　　“你——”陈晓宇被气得吐血，非要冲过去弄他。
<br/>　　“闹够了没有！”梧桐冷冷喝了一声，声音不大，但是不怒而威，一下子惊住了陈晓宇。
<br/>　　梧桐走到陈晓宇面前，看着他，冷冷说道：“你想干什么？”
<br/>　　“他出口伤人，我妖仆气不过，想要教训他一下，有何不可，梧桐你我是师兄妹，就算你平时不喜欢我，可总不要胳膊肘往外拐！”
<br/>　　“哦，”梧桐冷笑一声，道，“你是要跟他讲道理吗？”
<br/>　　“有何不可？”
<br/>　　“只有弱者才讲道理。”梧桐淡淡说道，“讲这么多道理做什么，你妖仆实力不济，主动挑衅，被人打了，还有什么好说的吗？”
<br/>　　陈晓宇张口结舌，他明白了梧桐的意思：居然道理说不通，两边都觉得自己有理，那么就是靠实力来分个高低……被人打了，还这么磨磨唧唧，这种表现，更加像是弱者。
<br/>　　陈晓宇咬了咬牙，强行克制住仇恨的欲望，立刻又摆出了强者的姿态，冲叶少阳点了点头。
