　　但是自从军部那边的科研部门研究出来一系列的新材料新武器后，猪皮就成为了热门货色。
<br/>　　这玩意可以做成皮甲，可以抵挡三阶丧尸的抓咬，非常的实用。
<br/>　　当然，因为猪皮，诸如裁缝这样的职业，也一下子变成了人上人，哪怕不是黑铁战士都没有任何问题，军部和各大组织都是高福利高待遇的对待这些人，林峰的战士会也是在大力度的招揽这类特殊技艺人才。
<br/>　　也多亏了吴三敏锐的嗅到了商机，提前就让林峰去留意这些。
<br/>　　猪皮，只是其中一个例子，除了猪皮之外，一些其余的变异怪物身上的材料也同样无比的受欢迎。
<br/>　　甚至有些变异怪物身上的肉质还可以食用，味道还特别的不错，比如说变异鹿的鹿肉。
<br/>　　当然了，诸如变异鹿这样可以食用的这还是少数，绝大部分怪物的肉不能吃，其中具备了剧毒，吃了反而死得更快。
<br/>　　林峰看不过几日时间，许多新东西就出现了，好在他也有先见之明，之前那个变异鳄鱼的尸体他可没有丢。
<br/>　　如今拿出来直接当做镇店之宝，足足七阶的存在，足够的震撼。
<br/>　　开业当天，就吸引了一大群人驻足围观。
<br/>　　吴三这家店的名字。
<br/>　　就叫。
<br/>　　新开的店。
<br/>　　非常牛逼的名字，经营范围也是各式各样的，什么都卖。
<br/>　　店主自然是吴三，幕后大老板就是林峰，为了买下这个店面，林峰也是花了不少晶石的。
<br/>　　当天。
<br/>　　也不知道，怎么得到林峰这里有七阶变异鳄鱼尸体消息的，军部科研部的人就来了。
<br/>　　直接花高价格买下了变异鳄鱼的尸体，足足一千五百颗三阶晶石的高价。
<br/>　　主要是因为这变异鳄鱼等级够高，其次就是尸体保存的较为完整。非常具有研究价值。
<br/>　　“我要见你们的老板。”
<br/>　　科研部的研究人员推了推眼镜对着吴三道。
<br/>　　“你怎么知道我不是老板。”吴三愣住了，说气质，他绝对有一种富商的气息，毕竟他末世之前也是小有身家的人。
<br/>　　研究人员笑呵呵道：“你实力不过才四阶，这变异鳄鱼尸体保存的如此完整，除了嘴巴里烂了点之外，你的实力不可能做到，所以你背后肯定有老板。”
<br/>　　吴三目光落在了林峰身上。
<br/>　　“我就是他的老板。”林峰站起身道。
<br/>　　“哦？幸会，我是军部科研部的研究人员，这是我的名片。”
<br/>　　研究人员立即递给了林峰一张名片。
<br/>　　“以后若是你这里还有这种怪物的尸体，当然前提是保持的较为完整的，我军部绝对第一时间高价收！”
<br/>　　研究人员道。
<br/>　　“嗯，一定。”林峰点了点头。
<br/>　　“多谢。”
<br/>　　研究人员说完就挥了挥手，一群人立即开着装甲车离去了。
<br/>　　“一千五百颗三阶晶石，这军部真有钱啊。”吴三喃喃道。
<br/>　　“对了峰哥，这是他买走变异鳄鱼尸体的晶石。”
<br/>　　林峰接了过来，拿出了五百颗给吴三。
<br/>　　“这些钱就当是活动资金了。”
<br/>　　林峰拍了拍吴三肩膀然后就离去了。
<br/>　　回到小区内。
<br/>　　林峰二话不说就把之前那颗七阶的变异鳄鱼晶石给用了。
<br/>　　然而依然还是七阶。
<br/>　　“我去，这么难突破？”
<br/>　　林峰皱起了眉头。一刻七阶的下去居然毫无任何反应，这让林峰都有点难以接受了，七阶晶石，可不是很好搞的东西。
<br/>　　随之又用了足足五百颗三阶晶石，才突破到八阶。
<br/>　　“恐怖！怪不得那个周华寒鸦他们这么久了还是九阶。”
<br/>　　林峰觉得太恐怖了，这可是基本上相当于吃了一两套房子进去了。
<br/>　　“代价太大了。不过没办法，房子都是浮云，有一套可以住一住就可以了，自己的实力才是根本。”
<br/>　　林峰握了握拳头，八阶基础力量三万，而林峰的力量是三万八千斤。九阶的力量也不过才四万斤罢了。
<br/>　　翌日。
<br/>　　叮！
<br/>　　在宿主附近检测到两个C级空投！空投存在时间十二小时！超过时间就会消失！请速度前往捡取！
<br/>　　“我去，直接刷新两个C级空投了！”
<br/>　　林峰立即从睡梦之中醒来。
<br/>　　在她旁边，睡着小妲己。
<br/>　　这几天下来，小妲己每天晚上帮他按摩踩背，特别的舒服，当然屁股啥的该摸的自然没放过。
<br/>　　小妲己迷糊的睁开眼睛，“主人，怎么了。”
<br/>　　“有好东西了。”
<br/>　　林峰把他给收回物品栏，然后就直接离开基地。
<br/>　　他目前有足足二十八个C级空投。
<br/>　　还差两个就可以开个B级空投。
<br/>　　林峰想想都激动。
<br/>　　然而这一次C级空投掉落的距离居然还不近。
<br/>　　林峰一路狂奔，有不长眼的丧尸直接一刀斩杀。
<br/>　　跑了足足一个小时才是抵达一个小镇。
<br/>　　这里依然不是终点。
<br/>　　林峰看着前面的景象皱起了眉头。
<br/>　　眼前，出现了一排的巨大茂密丛林，这里的树林一个个都是参天大树，三十米长的高度，简直就是高耸入云。
<br/>　　在这丛林之中，数不尽的丧尸。
<br/>　　林峰震惊了，他怎没记得江城附近有这么茂密的森林。林峰拿出了一份地图，这地图是侦察队朱民给他的。
<br/>　　可以说这份地图是用不少战士的鲜血换来的。
<br/>　　林峰打开地图，“巨树之森？”
<br/>　　林峰看到这个地图上清楚标明了这个地方，显然朱民他们早就发现了这个地方。
<br/>　　旁边还有关于这个地方的批注。
<br/>　　巨树之森：极其危险，大量丧尸大量怪物以及未知生物，为了探索巨树之森，我们已经死了不下于三十七人，然而只是推进了不到一百米的距离，不得不放弃。此地已经被军部列位第一个S禁地。
<br/>　　禁地是军部的叫法，意思就是危险的地方，生人勿进。
<br/>　　最高级别就是S级，在曙光基地附近，禁地不少。其中大部分都是C级的。属于一般威胁的。
<br/>　　林峰之前做过任务的明月湖就是个B级的禁地，但是因为林峰提交的记录仪发现了这个明月湖内可能存在某种巨大强大变异怪物，所以从原本的B级提升到了A级。
<br/>　　已经算是严重威胁的存在了。
<br/>　　即使如此，也不如这个巨树之森，他是目前这个地图上唯一的一个S级禁地，属于极度威胁。
<br/>　　“S级？还真是够危险的。”
<br/>　　林峰看了眼一咬牙还是冲了过去，没办法，这C级空投太调皮了就掉在这里面。
<br/>　　林峰拿出斩灵刀就是冲了进去。
