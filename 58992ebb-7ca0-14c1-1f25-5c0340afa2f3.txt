        连星儿不知道自己盯着明司寒看了多久，直到明司寒睫毛微颤，肩膀抖动着似乎是要睡醒了，这才重新躺下去，假装自己还在睡。
<br/>
<br/>        明司寒撑了个懒腰，捂着肩头活动着有些发麻的关节，然后从椅子上站了起来。
<br/>
<br/>        看到病床边的吊瓶已经被护士拆走了，他又凑上前去，本能地摸着连星儿的额头时，另一只手跟着抚上自己的额头进行对比。
<br/>
<br/>        这下她的高烧是真的已经退了。
<br/>
<br/>        明司寒松了口气后，替连星儿盖好被子，随后离开了房间。
<br/>
<br/>        连星儿觉得自己这是再也装不下去了，索性坐起身来，晕乎乎地揉了揉太阳穴。
<br/>
<br/>        护士端着药篮子走进来后，将口服的要放置在了床头柜上，叮嘱了一番后，又忍不住夸赞起明司寒来：“你男朋友对你真的好体贴，一整晚不是给你擦汗，就是给你掖被子，还替你盯着点滴。这会儿，估计是给你买早餐去了吧！”
<br/>
<br/>        “他……”连星儿欲言又止，顿了顿后，微笑着对护士说了声，“谢谢。”
<br/>
<br/>        其实，她是想说，他不是她的男朋友，可是，虽不是男朋友但也是她的未婚夫。
<br/>
<br/>        所以，话到嘴边又收了。
<br/>
<br/>        明司寒回来的时候不仅给她买了早餐，而且还给她买了新的衣服和鞋子。
<br/>
<br/>        连星儿在卫生间里换了衣服洗漱完后，看着正在茶几上替她盛粥的明司寒，犹豫了老半天才说了声：“谢谢。”
<br/>
<br/>        至于昨天那尴尬的画面，明司寒只字未提，只是应了声：“应该的。”
<br/>
<br/>        说完这句话后，明司寒又后悔了。
<br/>
<br/>        什么叫应该的？
<br/>
<br/>        说他是她的未婚夫，所以是“应该的”？
<br/>
<br/>        这话指不定又被她连星儿指着鼻子说他不遵守承诺。
<br/>
<br/>        她要他去跟爷爷说取消婚约这事情，他心里可是一直记着的。
<br/>
<br/>        “昨天，我是不是……”连星儿点到为止。
<br/>
<br/>        明司寒大脑神经一抽，条件反射似的地接着说道：“昨天你被绑架了，是被EA战队的幕后老板绑架的。”
<br/>
<br/>        “用来威胁MS战队？”连星儿反问。
<br/>
<br/>        明司寒微微点了点头。
<br/>
<br/>        本来他还想着，自己可能要跟她解释很久，才能解释清楚，哪知道她居然一点就通。
<br/>
<br/>        “你是MS战队的幕后大老板，对吧？”连星儿一边说着，一边凑过去，挨着明司寒的身子，在他身旁坐下。
<br/>
<br/>        明司寒下意识地远离，讷讷地问道：“你怎么知道的？”
<br/>
<br/>        “你爷爷给我挑未婚夫的时候，你的资料上不是写着嘛！”连星儿会心一笑。
<br/>
<br/>        明司寒尴尬地笑了笑，应了声：“是。”
<br/>
<br/>        “那‘C’，就是cold，余寒犹厉，他就是你的手下了，对不对？”连星儿眨巴着眼睛，问道。
<br/>
<br/>        “我……”明司寒刚准备理直气壮地承认自己就是“C”，但是，一看到连星儿这不明所以然的眼神，他又犹豫了。
<br/>
<br/>        因为，他不知道这丫头的葫芦里到底是卖的什么药，万一……
<br/>
<br/>        “当然是我的手下！”明司寒顿了一下，才将这句话回应完。
<br/>
<br/>        连星儿顿时眼前一亮，满脸期待地看着明司寒，突然间撒起了娇来：“你命令他跟我合个影，好不好？”
<br/>
<br/>        “为什么？”明司寒有些受宠若惊地看着连星儿。
<br/>
<br/>        连星儿会心一笑，附在明司寒的耳边，悄悄地说道：“因为，我是他的徒弟。”
<br/>
<br/>        “就这样？”明司寒有些失落地反问。
<br/>
<br/>        连星儿点了点头，回答道：“对啊！就这样！”
<br/>
<br/>        紧接着，连星儿微微嘟着唇，大眼睛可怜兮兮地眨了眨，继续撒娇：“好不好嘛？”
<br/>
<br/>        “C他很冷酷的，不喜欢跟别人合照，就算我是他老板，这种私事，我也是没法左右的。所以，你想都别想了。”明司寒顿时很没好气地说。
<br/>
<br/>        连星儿不禁瘪着嘴，一脸不开心。
<br/>
<br/>        明司寒看着连星儿，含沙射影地问：“在你心里，是你师父重要一些，还是洛羽重要一些？”
<br/>
<br/>        “这可没什么好比较的，两个人是不同世界的人。”连星儿不假思索地回答。
<br/>
<br/>        明司寒不解，不依不饶地追问：“为什么不能比？”
<br/>
<br/>        “我师父对于我来说，是存在在网络世界里的人。洛羽，可是现实世界里的人。所以没法比喽！”连星儿双手摊了摊。
<br/>
<br/>        “那你师父和洛羽一起掉水里，你先救谁？”明司寒有些不甘心地继续问道。
<br/>
<br/>        连星儿看着明司寒，只觉他此刻的表现十分莫名其妙：“对于你来说，你应该问你和洛羽掉水里了，我先救谁吧！你怎么老是拿我师父跟洛羽比啊？”
