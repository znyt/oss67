　　听闻苏莫的话语，扬天成眼眸微眯，转头看向苏莫，冷冷的道：“真以为你打败了幽狼，就武帝境无敌了吗？”
<br/>　　“不是自以为，是已经无敌了！”苏莫沉声说道。
<br/>　　“狂妄！”扬天成闻言，顿时厉喝一声，这个苏莫未免太狂了，居然认为自己武帝境无敌了，还公然的说了出来。
<br/>　　众人闻言，亦是暗暗惊讶，这个苏莫还真是自信，居然公然说自己无敌了！

<br/>　　没有实力叫自大，有实力就是自信。
<br/>　　但是，苏莫之言，却是让同为妖孽的其他几人暗暗皱眉。

<br/>　　扬天成的脸色略显凝重，显然他也丝毫不敢小看苏莫，少倾，他出手了。
<br/>　　铮！
<br/>　　屈指一弹琴弦，顿时响起一声嘹亮的琴声，琴声似宝剑出鞘，又似天地雷音，充满无尽的杀伐之气。

<br/>　　大剑洞穿空间，裹挟无尽的杀机，瞬间便临近了苏莫。
<br/>　　“有点本事！”苏莫冷笑一声，随手一剑斩出，剑光一闪，瞬间便将袭来的音波大剑斩碎。
<br/>　　这音波大剑或许威力强大，但对于他来说，挥手可破。
<br/>　　而此时，扬天成却是毫不停歇，双指连连弹动，激昂而又充满热血的音符不断的响起，形成了美妙的旋律。
<br/>　　这的确是一首琴曲，让人沉浸在那热血激昂的旋律之中。
<br/>　　但是，美妙的音符之下，却是无限的杀伐之气，旋律形成的音波，化为了一柄又一柄虚幻的长刀、利剑。
<br/>　　数以百计的长刀利剑，铺天盖地、劈头盖脸的向苏莫袭杀而去。
<br/>　　剑雨威势恐怖，凌厉的杀机迷漫八方，瞬间便将苏莫笼罩。
<br/>　　“灭！”
<br/>　　苏莫轻喝一声，不退反进，身形如电光般冲了出去，剑光挥洒，前方的一切尽皆破灭。
<br/>　　强大音波形成的长刀利剑，在苏莫面前简直不堪一击，苏莫如入无人之境，眨眼间便临近了扬天成。
<br/>　　“什么？”扬天成面色大变，真正的对上苏莫，他才能感受到后者的强大，任他如何攻击，后者都能轻易破灭。
<br/>　　唰！
<br/>　　毫不犹豫，扬天成立刻身形飘退，与此同时，他手指用力一弹，一根琴弦立刻崩断。
<br/>　　咻！
<br/>　　崩断的琴弦，如同金色的利剑，又如同一根微小的发丝，一闪间便消失不见。
<br/>　　“嗯！”苏莫眼眸一凝，立刻微微放慢了速度，感知力迷漫而出，他能感觉到那根琴弦就在他四周虚空中游走。
<br/>　　咻！
<br/>　　下一息，琴弦猛然出现，直击苏莫的胸膛，速度快到极致，让人无法察觉。
<br/>　　但是，苏莫的精神感知力何等强大，就算现如今精神力还没有完全恢复过来，也比同阶的武者强大数倍。
<br/>　　琴弦乍一出现，他便出手了。
<br/>　　唰！
<br/>　　剑光一闪，苏莫便将琴弦斩成了两半，从半空中跌落而下。
<br/>　　咻咻咻！！
<br/>　　这时，便见扬天成双手连弹，再次有数根金色的琴弦，飚射而来，一闪间便再次消失不见。
<br/>　　“雕虫小技！”苏莫轻笑一声，不想再继续浪费时间，一闪间他的身形便也消失不见了。
<br/>　　“这是……！”扬天成见此，顿时一愣，不明所以，苏莫的人呢？
<br/>　　不过很快，他便心中一凛，猜测苏莫定然是有某种隐藏身形的方法，不然不会如此。

<br/>　　就在扬天成准备动用大面积音波攻击之时，他的后方苏莫突然出现了，还伴随着一阵紫雾涌出。
<br/>　　淡紫色的雾气，弥漫而出，瞬间便笼罩了一片空间，扬天成也毫无意外的被紫雾笼罩。

<br/>　　幽暗的房间之中，巨大的刑架之上，扬天成全身上下都绑满了铁锁，整个人被束缚在刑架之上。
