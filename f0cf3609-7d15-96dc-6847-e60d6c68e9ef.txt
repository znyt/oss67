　　第二城之中。
<br/>　　“诸位，我们既然和第三城结盟，现在就要出去支援苏莫！”龙羽遥望着外界的情况，朗声说道。
<br/>　　“我不同意，还是先看看情况再说吧！”贺兰殇说道，他知道有很多人要杀苏莫，他自然不能让第二城之人前去帮助苏莫。
<br/>　　“阎兄，你觉得呢？”龙羽皱眉看向一直淡然无比的阎公子。
<br/>　　“我赞同前去支援！”阎魔鬼圣道。
<br/>　　“贺兰兄，少数服从多数！”龙羽沉声说道，第二城由他们三人联合统领，现在他和阎公子的意见，都是出去支援。
<br/>　　“阎兄，苏莫的实力你应该知道，若是苏莫不败，我们最后都不一定有机会！“贺兰殇面色凝重的说道。
<br/>　　“贺兰殇，我们和第三城结盟，难道出尔反尔吗？”龙羽怒喝道，他不能让苏莫陷入危机，不然，凭他很难得到最后的胜利。
<br/>　　“呵呵，想要获胜就要审时度势，哪有真正的盟友！”贺兰殇冷笑着说道。
<br/>　　“我赞成出去相助！”阎魔鬼圣面无表情，淡淡的说道。
<br/>　　“阎兄你……！”贺兰殇脸色难看，龙羽是龙族之人帮助苏莫就算了，怎么阎公子也会想着帮助苏莫？
<br/>　　“别浪费时间了，我们出动一半的人手就行了！”龙羽急声说道，现如今，外面已经交起了手，当然，苏莫的人手和叶凌天的人手，并没有落入下风。
<br/>　　目前，第五城和第七城、第九城都没有什么动静，所以，他们还要留一半的人手镇守在城中，以防有变。
<br/>　　“既然你执意如此，那由我带人出去吧！”贺兰殇道。
<br/>　　“你出去？”龙羽闻言一怔，他是准备自己出去的，没想到对方居然主动要出去。
<br/>　　“嗯！”贺兰殇点了点头，随即，立刻开始挑选人手，很快便挑选了四十余人，不由分说的带人离开了第二城。
<br/>　　龙羽见此，并未阻拦，既然贺兰殇愿意出手，也是一样。
<br/>　　此时此刻，外界，正在经历着最大的混战。
<br/>　　第一城和第三城，对战第六成和第八城，两方的人数相差不大。
<br/>　　战况极为激烈，叶凌天和古魔太子交手，已经打出了真火。
<br/>　　古魔太子的实力，明显要比叶凌天更强两分，但是，短时间内也难以击败叶凌天。
<br/>　　“给我滚！”
<br/>　　一声黑色披风的古魔太子，怒喝震天，狠狠的向叶凌天轰出了一拳。
<br/>　　这一拳威力极大，一拳出，滚滚魔气如同一条长河，向叶凌天重重的袭杀而去。
<br/>　　魔气形成的拳茫之中，仿佛有亿万魔物在嘶吼，震人心魄。
<br/>　　古魔太子心中有些焦急，他被叶凌天缠住了，无法出手去对付苏莫，而现在，贺兰殇等第二城的人也出来了。
<br/>　　“破！”叶凌天大喝，同样出拳。
<br/>　　他的身后浮现了一尊巨大的人形虚影，这是一尊巨大的金身，光芒百丈，身高百丈，头顶天脚踏地，如同力拔山河的巨人。
<br/>　　这是天地法身，乃是天地法相元神，太古武魂榜排名第九。
<br/>　　轰！
<br/>　　金色的拳茫，贯穿长空，立刻和强大的魔气相击在了一起，震动整个神昆小世界，虚空崩塌，日月无光。
<br/>　　另一边，苏莫几乎是难逢敌手，基本上他每出一剑，都能重创一人，当真是所向无敌。
<br/>　　不过，他很快就面临了围攻，围攻他的人有霸天族之人，有星辰族之人，有天王宫之人、有雷焱神王殿之人等等。
<br/>　　当然，还有楚天墟。
<br/>　　楚天墟本来已经准备逃走，但是看到古魔太子来援之后，立刻又杀了回来。
<br/>　　“杀！”
<br/>　　楚天墟的手中，拿着一杆银色的大枪，疯狂暴刺，化出漫天枪影，劈头盖脸的向苏莫袭杀而去。
<br/>　　而另外十数人，同时是疯狂出手，个个动用了全力，各种攻击袭向苏莫。
<br/>　　甚至，有的人连精神攻击都动用了，企图击伤苏莫的神魂。
<br/>　　“找死！”苏莫眸中精光闪烁，他感觉到楚天墟等人那凌厉的杀机，立刻就知道这些人动了杀心。
<br/>　　既然这些人想杀他，他自然也不会客气，绝对的杀无赦。
<br/>　　“楚天墟，你是自寻死路！”
<br/>　　苏莫朗喝一声，手中终极神剑高高举起，磅礴的混沌源力暴冲而起。
<br/>　　他的身上，混沌源力沸腾，元神之力和血脉之力，都催动到了极致。
<br/>　　唰唰唰！！
<br/>　　随即，苏莫手中长剑不断的挥击，剑气飚射，如同道道道极光，绚丽无比。
<br/>　　轰隆隆！
<br/>　　在一阵震耳欲聋的爆响声之中，十数道袭来的攻击纷纷被斩爆，恐怖的爆炸冲击波将苏莫淹没。
<br/>　　楚天墟等人，立刻纷纷后退，避开十数道攻击形成的猛烈冲击波。
<br/>　　楚天墟眼眸冷漠，他们十几人联手，这种攻击形成的冲击波，足以灭杀一般的二品中级武者了，不知道苏莫有没有受伤。
<br/>　　咻！
<br/>　　就在此刻，一道恐怖的混沌剑气，如同天外之剑，从浩大的冲击波中陡然刺出。
<br/>　　这一剑恐怖绝伦，洞穿了空间，洞穿了恐怖的冲击波，裹挟滔天之势，瞬间临近了楚天墟。
<br/>　　“终极剑道！”楚天墟见此，顿时面色剧变，感受到这一剑的威势，他简直是心胆俱寒，全身汗毛瞬间炸开。
<br/>　　咻！
<br/>　　剑气的速度无与伦比，之时电光火石之间，便临近了楚天墟的身躯，恐怖的杀机如同死神降临。
<br/>　　在这生死一刻，楚天墟没有还击，而是立刻取出了一块硕大的盾牌，全身源力浩荡，企图挡住攻击。
<br/>　　轰！
<br/>　　一声震天动地的爆响，混沌剑气重重的击在了盾牌之上，天地齐震，万界轰鸣。
<br/>　　这盾牌乃是极品神器，坚固无比，自然不会被斩碎，混沌剑气立刻炸开。
<br/>　　但是，盾牌虽然没有破碎，剑气那巨大的威力倾泄而出，根本不是楚天墟能抵挡的。
<br/>　　轰！
<br/>　　楚天墟的肉身，几乎是瞬间便崩溃开来，直接被震成了一片血雾。
<br/>　　唰！
<br/>　　肉身乍一崩溃，一道虚影立刻急速逃窜，这是楚天墟的元神，他的肉身虽然崩溃，但是元神却是丝毫无伤。
<br/>　　楚天墟的元神，乃是一棵古树，高约一丈有余，笔直无比，没有任何的枝干。
<br/>　　这是通天神树元神，太古武魂榜排名第二十，还在天梦元神之上。
