　　另一边追着叶家人出来的叶子博，心中焦急万分，他很清楚当下自己已经被张玄背叛，而张玄也一定用最快的速度带着叶永阳去往直升机的停放地。
<br/>　　再不赶紧过去，就来不及了！
<br/>　　前方的叶家年轻人和高原基地的那批搜寻队伍正在快步往前赶，他们延着道路一直朝前走，焦虑之中也没有注意后方的叶子博身影。
<br/>　　叶子博的紧张情绪，随着越发靠近那处转折点，就越来越强烈。
<br/>　　张玄必然比他们先一步，也一定会路径那处转折点，如果这批搜寻队伍的能力很强，则就很有可能发现旁边有行走过的痕迹，这样一来不但叶子博不可能截住叶永阳，还有可能让这批搜寻队伍霸占优势。
<br/>　　如此一来他千辛万苦来到此处的计划，就全部泡汤！
<br/>　　“别发现！千万别发现！”他在心中疯狂的叨念着。
<br/>　　终于在叶子博的心跳速度到达高峰的时候，前方的那批人就已经走到了那个转折处，再往前走就是下山，往旁边的植被地而去，就是直升机的停放地！
<br/>　　而那批搜寻队伍也异常配合的直径朝前赶，丝毫没有发现旁边的植被有细微痕迹，显然他们已经在焦急之中忽略了很多关键。
<br/>　　见到这一幕的叶子博大喜过望，他静静的等待着那批队伍走远，然后毫不犹豫的朝着旁边植被地带狂奔而去，他用上了自从被白忧看中以来最快的奔跑速度，跑的气喘吁吁汗流浃背。
<br/>　　同时一股畅快的感觉更油然而生，终于甩掉了叶家那批人，接下来他需要应付的就只有张玄一个。
<br/>　　至于张玄的背叛，以叶子博对此人的了解，只要将任务的价格无限提高，视财如命的张玄不可能经得住这种诱·惑。
<br/>　　思考之中，很快叶子博便跑到了那处平地上，在他的眼前直升机的机身外已经没有了伪装，那些植物藤蔓已经被完全清理，甚至叶子博还运气很好，因为那架直升机还未起飞，虽然其螺旋桨正在飞速旋转。
<br/>　　只要没走，就还有机会！
<br/>　　惊喜之中，叶子博再次提高速度，朝着直升机冲刺中更是大喊出声：“张玄！等等！你带我走，我给你另外一笔费用！我只要你带我走！”
<br/>　　叶子博难得机灵一次，没有一上来就提出叶永阳的所属权问题，而是发布了另一则逃脱任务，他很清楚这样顺路的买卖，张玄不可能错过，更何况在任务价格上，他叶子博可从来没有亏待过张玄。
<br/>　　就在叶子博心中弩定，眼前那即将起飞的直升机，一定会停留片刻等他之时。
<br/>　　呼啦啦！
<br/>　　忽的，一阵狂风大作，直升机上的螺旋桨越转越快，机身更是一下子离开地面，毫不犹豫的朝着高空飞去，没有给叶子博丁点机会。
<br/>　　叶子博愕然的顿住脚步，呆立在原地，难以理解为何张玄的作风会忽然反常。
<br/>　　而就在他停驻不前在原地发呆愣神的时候，身后一阵噪杂的声音由远及近，错乱的脚步声就响起在身后。
<br/>　　叶子博大惊，他瞬间回头一望，当即如同被冷水浇了全身般的透心凉。
<br/>　　是那批搜寻小队！
<br/>　　他们怎么又折回来了？
<br/>　　“飞走了！我们晚了一步！”一名叶家年轻人愤恨的喊出声，望着高空那越来越远的直升机，满脸的懊恼之色。
<br/>　　“怪我。”身旁的另一人也极为后悔，神色复杂的开口道：“我们走过那里之后，我才反应过来有问题，要是能早点……”
<br/>　　“别这么说。”旁边的叶家人打断他，安慰道：“是我们都疏忽了，对付有备而来，必定不可能徒步跋涉，我们是跟着爷爷一路走过来的，这些年都差点忘了世上还有直升机这种东西。”
<br/>　　“眼下怎么办？”在一片懊悔声之中，一人问道：“现在目标跟丢，我们的线索也没了。”
<br/>　　就在这些人的讨论声中，忽然一个人声音提高一个八度，指着不远处大喊：“叶子博！那是叶子博！和那个张玄一起来的！”
<br/>　　此话一出，所有人都一下子望向了他所指的方向，顿时隐藏在一堆杂草中的叶子博身影，便被轻而易举的发现。
<br/>　　“果然是叶子博！”一名叶家年轻人眼中满是怒火和杀意，大喊道：“就是他背叛叶家，害的爷爷逃离到这里了还不够，还要带着人来捣乱！”
<br/>　　“这条白家狗，杀了他！”另一名叶家人更是控制不住情绪。
<br/>　　因为爷爷被掳走而受了刺激的叶家人一下子冲上前，手中武器拼命朝着叶子博身上砸，叶子博本身随是一名新人类，但却由于过早的失去一条手臂，并且长期没有再提升战力，一直养尊处优的他此时就像过街老鼠，被这些杀心四起的叶家年轻人打的毫无招架之力。
<br/>　　从高原基地内出动的那批军队一开始还没反应过来，等到他们在一片混乱中上前后，叶子博早已被打的浑身是血，胳膊和两条腿都被卸了下来，胸口更是被一柄锋利小刀划出了无数极深的伤口。
<br/>　　他口吐鲜血，想不通自己在末世里风光无限了这么久，怎么会在这个偏僻的地方落得如此下场。
<br/>　　对自己下狠手的这些人，一个个都曾是他的远亲，此时却因为他想要得到叶永阳守护的那个秘密，而被打到重伤垂死！
<br/>　　连胸口都中了无数刀，叶子博必定不可能再救活，估计不出一分钟就会死在这里。
<br/>　　一名高原基地出来的人忍不住皱眉：“你们太冲动了，走了一个我们追不上，眼下就剩下了这个人，你们还要把他打死，这下线索不就全断了么？你们的爷爷到底是什么人，怎么会有那种高手绕了这么一大圈过来抢？”
<br/>　　带头动手的叶家人望着地上垂死的叶子博，目光冷漠：“就是因为看到叶子博，我们才确定了到底是谁派人来，也能定下目的地。”
<br/>　　听到此话，不知情的人都面面相觑。
<br/>　　同样听到此话的叶子博奋力的抬起眼，死到临头他都没能明白，那个秘密到底是什么？
<br/>　　而眼前的这些叶家人，似乎已经知道了什么……
<br/>　　正当他的思维渐渐模糊意识不清醒时，忽的，他的目光透过这些人的身旁，看到了远处一片茂密的草丛。
