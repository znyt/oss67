　　“合道！”
<br/>　　紫霄宫中，只见鸿钧的身体越发的淡然，一股莫名的波动，独立于天地之外，却又似乎与天地合而为一，这是两种极端的感觉，如今却同时出现在道祖鸿钧身上，却让人看不出丝毫的矛盾，仿佛这一切就该如此。
<br/>　　“天地混沌如鸡子，盘古生其中。万八千岁，天地开辟，阳清为天，阴浊为地。盘古在其中，一日九变，神于天，圣于地。天日高一丈，地日厚一丈，盘古日长一丈，如此万八千岁。天数极高，地数极深，盘古极长。数起于一，立于三，成于五，盛于七，处于九，故天去地九万里……”
<br/>　　“高卧九重云，蒲团了道真。
<br/>　　天地玄黄外，吾当掌教尊。
<br/>　　盘古生太极，两仪四象循。
<br/>　　一道传三友，二教阐截分。
<br/>　　玄门都领袖，一气化鸿钧。
<br/>　　今日身合于天，来日吾当掌天。”
<br/>　　随着鸿钧老祖那淡淡的声音响起，他的身体渐渐消失不见。元始只感觉此时的天道赤裸裸的完全显现在他面前，他来不及多想，赶紧用鸿蒙元始印将其记载刻录，能刻录多少便刻录多少，毕竟这种就会异常少见。同时，他觉得自己在瞬息之间，似乎就明悟了许多，可又有更多不明白。不知不觉中，他的元神，乃至心境、道行都有了极大的进步。
<br/>　　道行法力的再度进步，让元始心中不由得为之暗喜，这一番感悟，又省去了他好多时间，他对于突破混元境界，已有了十足的把握。
<br/>　　紫霄宫中，元始渐渐从天道感悟中醒来。
<br/>　　醒过来的第一眼，元始发现其他人都还沉浸在大道的感悟之中，显然并不是只有元始一人有所领悟。但其中要说感悟最多的，还是要数元始。因为这些人中，他的修为最高，对道的理解也最为深刻。故而他能够领悟的也就越多。
<br/>　　此刻鸿钧的身影早已消失不见了，他第一时间传音叫醒老子与通天，之后带着四个徒弟，直奔紫霄宫外的分宝岩。
<br/>　　来到分宝岩之前，只见百丈大小的分宝岩上，宝光闪动，五光十色，分外耀眼，让人心动不已。
<br/>　　元始不再犹豫，看到分宝岩后，一道混沌色的神光向着分宝岩刷去。这神通元始之前用过，正是之前争夺先天葫芦之时所用的。此乃混沌道君所创的混沌神光，练至极巅之处，混沌之下，无物不刷。这可比什么七宝妙树、先天五色神光高级多了。当然，元始如今还没有练至绝巅。
<br/>　　但，就算如此，刷去一些先天灵宝，还是可以的。而且它一点也不比用灵宝来套灵宝慢。但就算如此，元始还是嫌慢，这时，从他的身体之中，走出一道身穿红色道袍，手持混沌钟的道人。
<br/>　　这时元始的执尸，他一出现，便带着混沌钟向分宝岩上套去，这一套，便套去无数灵宝。同时，元始的本尊也没有闲着，混沌神光在分宝岩上不断刷来刷去，神光所到之处，灵宝尽皆消失，却是被元始镇压在体内世界去了。
<br/>　　老子与通天见此，也是目瞪口呆，还可以这样？于是，他们也学着元始，取出自己所斩之尸，与本尊一同配合，用两包向着分宝岩上套去。
<br/>　　同时，他的四个徒弟也没有闲着，同样依靠自身灵宝去套去分宝岩上的灵宝。
<br/>　　“不好！”
<br/>　　紫霄宫中，待女娲等人醒过来后，发现三清早已不见，便知道他们已经去取宝了，顿时大惊，纷纷起身，向着紫霄宫外飞去。他们的动静闹醒了其他人，顿时其他人也纷纷起身，飞向分宝岩。
<br/>　　等他们到来之时，元始等人早已站在分宝岩旁边等着了。只见分宝岩上，已经少了三分之一的灵宝。众修士顿时不顾其他，赶紧纷纷争上前去，收取灵宝。
<br/>　　至于元始三人为何不继续收取灵宝，却是他们发现，当自己等人收取到一定程度的灵宝之后，便不能在动了，不论他们怎样收取，都动不了。于是，他们知道，其它的灵宝与自身无缘，或者是说，是天道在作怪，不能让他们全部收取。
<br/>　　你能够拿走的，那便证明此宝与你有缘，你不能取走的，那么，对不起，此宝与你无缘，你也不必再多费心机了，无论如何，你也取不走它。
<br/>　　元始此刻心中颇为欢喜，仅仅他自己就取走八九十件的灵宝，虽然说大多数灵宝只是下品先天灵宝，或者是不入流动先天灵宝，但还是有些中品，或者上品先天灵宝、极品先天灵宝。比如，中央戊己杏黄旗、二十四颗定海神珠、玉虚琉璃灯、阴阳镜、尊神刀、指天剑等。最让元始意外的是，他得到了一件特殊的先天灵宝，那是崆峒印。
<br/>　　崆峒印，为日后人族至宝，能镇压一族的气运。它的功能可达可小。说它功能大，它那是因为它能够镇压一族之气运，坚硬无比，不可摧毁。而且最重要的是，持有此印者，可废立人皇。其中更蕴含着浓郁的生命力，不管受到多大的伤，都可瞬间恢复伤势。说它小吧，那是因为它是应运而生的灵宝，天地间只有人族可以使用它，别的种族，不管谁得到了，都不能使用，只能当一个摆设。
<br/>　　能够得到它，元始估计那恐怕是因为自己前世为人族的原因吧。
<br/>　　看着众修士收起取灵宝，一道道法力撞击巨石之声不断响起，分宝岩上，灵宝众多，元始等人总共才收走一百多件，这才仅仅是三分之一。之后，女娲一人收走十数件，准提接引两人合力收走数十件，帝俊太一等人也得到不少，正是众人各有收获。
<br/>　　之后，还剩下数十件灵宝，飞散到洪荒大地各处，等待有缘人。
<br/>　　“三位师兄恐怕收获不少吧！”就在此时，准提对着元始三人问道，明眼人都知道，三清是最先来到分宝岩前的人，他们的收获不比众人多才怪。
<br/>　　“不多，也仅仅只是比你们多一点而已。”元始淡淡了瞄了他一眼，便不再关注他，而是抬步走到分宝岩前，大手一挥，便将分宝岩收起。
<br/>　　“呃？我之前怎么没有想到啊？”有人见此，后悔道。
<br/>　　“啊？这东西能用？一个放宝的，它除了放宝，还能干什么？”也有人不解。
<br/>　　“笨啊，能够蕴养如此多的先天灵宝的分宝岩，又岂会是废物？”众修士见此，纷纷大喊，后悔不已。暗道自己之前又怎么会想不起来呢。但这世上没有后悔药吃，已经被元始收走的东西，他们也没胆再要回来。
<br/>　　“走！”元始不理他人，之后带着老子、通天，还有自己的四个徒弟，便一刻不停地向着洪荒大地飞去。
<br/>　　众人见此，也纷纷告辞离开，他们此次听道收获巨大，需要闭关参悟，更何况，收获如此多灵宝，还等着他们回去炼化呢。
