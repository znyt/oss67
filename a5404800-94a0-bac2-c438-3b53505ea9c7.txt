　　“不是吗，我听说有些法师养仆人就是为了这个。”
<br/>　　“别人不知道，反而我没干过，你放心吧。百年之后的你我也相处了很久，我从来没碰过你，也没碰过身边任何人。”
<br/>　　美华长出了一口气，有点不好意思地笑了笑，说道：“那就是我想多了，呼……我以为还要伺候你呢。”
<br/>　　“我又不是五通神，还要找妹子伺候。”叶少阳咧嘴笑了笑，“话说，那如果我说需要伺候，你难道还真从了我不成？”
<br/>　　美华听他这么说，低下头，嗫嚅道：“那如果真的需要，作为你的鬼仆，我也没有办法拒绝吧……”
<br/>　　叶少阳差点喷了，顺便瞄了一眼她旗袍下面玲珑的身段，努力咽了一下口水。
<br/>　　两人开始聊起正事，美华说道：“老大，你说徐福让我来找你，这说明什么？”
<br/>　　“好事。”叶少阳道，“这件事，我昨晚上想了很久，如果真的是徐福让你来的，那至少说明，他是站在我这边的，所以想让你来帮我。”
<br/>　　“可是……那他为什么不自己来，至少应该露面跟你说一下，为什么要把你送到这个时代来吧？”
<br/>　　“可能，他有他自己的目的吧，这个就不好猜了。不过至少我是知道，他把我送到这里来，不是来坑我的。”
<br/>　　美华点点头，“那我们现在怎么办，怎么对付三足金蟾？”
<br/>　　叶少阳道：“力拼是不行的，再不济人家也可以跑路。要出手，就要一击必杀。”想到这，叶少阳问她：“你现在是什么修为？”
<br/>　　美华想了一下道：“在人间就算是二等鬼首吧。”
<br/>　　“二等鬼首……如果加上三位天师，与五通神能有一战吗？”
<br/>　　美华说道：“那自然是可以的，就是一定不能让五通神逃走，不然他引来另外四神，我们根本不可能斗得过。”
<br/>　　叶少阳道：“也就是说，我们只有一次出手的机会。”
<br/>　　美华点了点头。
<br/>　　两人开始商量具体的计划，既然不能硬拼，那就只能想办法偷袭。
<br/>　　“对了，那三足金蟾有什么弱点？”
<br/>　　美华想了一想，说道：“我去给你抓一个人来问问。”
<br/>　　“抓个人？”
<br/>　　“那三足金蟾也有很多手下，我去找一个来问问。”美华想起什么，说道，“我们对付三足金蟾的时候，也要避开他的那些手下，这群小喽啰也很讨厌。”
<br/>　　叶少阳道：“你去找谁问？”
<br/>　　“他跟我一样，手下有很多都是水族，我有一个贴身侍女，跟他手下一个水鬼关系很好，我可以让她去打听。”
<br/>　　“不会被告密吧？”
<br/>　　“那绝对不会。”美华神秘一笑，“那个水鬼一向喜欢我的侍女，迷得昏天暗地，早就想投靠于我，我因为怕三足金蟾来找麻烦，特意不许。”
<br/>　　叶少阳无奈一笑，看来为了美女着迷，不光是为了人类的专利，连邪物也有这样的。
<br/>　　美华道：“你在我这里好好睡一觉，我去安排，等问出情况再来告诉你。”
<br/>　　“在你这睡？这……合适吗？”叶少阳看着床上的一袭粉红色的锦被和绣花枕头，还散发着一股醉人的香味，这样的一张充满了香艳气息的床，叶少阳实在有点睡不下去的感觉。
<br/>　　“这怎么不合适，你是我主人，有什么拘束的。”美华笑了笑，起身往外走，到门口又回过头来，冲叶少阳眨了眨眼睛，“真不需要我找人陪你？”
<br/>　　“走你的吧！”叶少阳一脸的尴尬。
<br/>　　美华觉得很好玩，忍俊不止地走了出去。
<br/>　　叶少阳脱了鞋上床躺下，但是睡不着，于是坐在床上吐纳起来，大约吐纳了两个周天，美华也回来了，告诉叶少阳，已经找到那个水鬼打听了情况，别说，还真找到了有关三足金蟾的一个弱点。
<br/>　　“三足金蟾的弱点，就是他的尾巴。”
<br/>　　“尾巴？”叶少阳感到很诧异，“他不是蛤蟆精吗，蛤蟆有尾巴？”
<br/>　　美华说道：“所谓三足金蟾，其实并没有三条腿，第三条腿，就是他的尾巴。一般蛤蟆，在长成蛤蟆之后，尾巴必然就掉落了，但这三足金蟾的尾巴，却是神迹。”
<br/>　　叶少阳一听就明白了，这里所说的“神迹”，是一些邪物在成精之后，身上会长出一些原本没有的器官，例如头生角，脊生尾，便是神迹显现，这个多出来的器官，往往会有神奇的功能，自然也成了妖物的命门所在。
<br/>　　“三足金蟾一身大半修为都在他的尾巴里，如果能斩下他的尾巴，想要擒杀他就容易了，否则硬拼的话，很是不容易对付。”
<br/>　　叶少阳沉吟了一会儿，说道：“我觉得，我们应该找个好机会，被他选中的姑娘，是一个可以利用的机会。”
<br/>　　美华眼珠一转，一副欲言又止的样子，叶少阳忙问她想说什么，美华凑到他耳边，小声说了几句，叶少阳顿时叫起来：“卧槽，你是怎么想起来这个办法来着……”
<br/>　　美华的办法，简直让叶少阳感到相当无语：想利用三足金蟾侵犯那个少女的时候，在GC的时候，不管是人是妖，都是完全放松的状态，注意力也是最为分散，在那个时候如果动手，成功的几率一定非常高。
<br/>　　“不是吗，我这个计划，我觉得很有可能成功。”
<br/>　　“计划是不错，不过要牺牲那个姑娘的……童贞。这就使不得了。”
<br/>　　“你若想斩神，这是最好的机会。再说在她之前，也有很多姑娘被糟蹋了，她不是第一个。”
<br/>　　叶少阳道：“以前的我管不着，反正我遇到了，就不能然他再祸害人，不然我斩神还有什么意义。”
<br/>　　美华听他这么说，也就不好说什么了。
<br/>　　叶少阳道：“不过你这办法，有一部分是可行的，嗯，可以找这个姑娘帮忙……”叶少阳把自己的想法跟美华说出来，美华想了一想也觉得可行，于是两人商量具体的计划，制定好了之后，叶少阳跟美华一起离开洞府，去镇上雇了一匹马，前往最近的一个县城。
