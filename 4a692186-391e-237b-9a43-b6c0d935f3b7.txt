　　苏莫和龙腾离开天龙大殿之后，便和大长老等一众长老同行。
<br/>　　“龙腾，恭喜！”
<br/>　　“龙腾，你将来说不定能达到洪荒祖龙那般成就！”
<br/>　　“现在你是我龙族第一天才！”
<br/>　　不少长老人物纷纷向龙腾开口，仿佛是与同层次的强者交流一般。
<br/>　　“诸位过奖了！”龙腾微笑着向众人说道，这些都是龙族的长老，他必须跟这些人搞好关系。
<br/>　　“龙腾，稍后到老夫的府上！”大长老走在龙腾的身边，轻声说道。
<br/>　　“嗯！”龙腾点了点头，随即对苏莫道：“屠霄，我们回府！”
<br/>　　随后，龙腾和众人告别，带着苏莫向自己的府邸而去。
<br/>　　府邸中，司空炎早已在等待了，见到两人回来，立刻上前询问。
<br/>　　“怎么样？”司空炎迫不及待的问道。
<br/>　　“一切顺利。”龙腾微微一笑，随即将事情简短的向司空炎讲述了一番。
<br/>　　“这龙帝不得不防啊！”司空炎也不傻，听闻大概情况之后，立刻面色凝重了下来。
<br/>　　“无妨，在龙族之中，他还不至于向我动手！”龙腾微笑着说道，这是洪荒祖龙的意志，任何人都不能违背，即便是龙帝都不行。
<br/>　　“龙腾，大长老这个人很关键，或许是你日后制衡龙帝的中坚力量。”苏莫沉声告诫道，很明显，大长老非常遵从洪荒祖龙的意志，龙腾应该极力拉拢此人。
<br/>　　“我明白！等会我去和他单独见面，会探探他的口风！”龙腾面色凝重的道。
<br/>　　随后，三人又闲谈了一番，龙腾离开了府邸，向大长老的府邸而去。
<br/>　　苏莫和司空炎，也没有回自己的住处，就在龙腾的府邸中等待了起来。
<br/>　　苏莫有些无奈，本来，他是打算暂时离开龙族的，毕竟，人德宫主那边，貌似不那么保险。
<br/>　　他也不好给人德宫主传讯，询问情况，毕竟，传讯意味着自己所在之地，可能会让对方知晓。
<br/>　　不过，现在的情况，他不宜离开龙族了，待在龙族反而更加的安全，因为有了龙腾这层保障。
<br/>　　对于现在的苏莫而言，自身所处的局势，实在太过复杂了，他也只能走一步看一步了。
<br/>　　苏莫和司空炎两人，静静的等待龙腾归来，两人并未等太久，约莫半个时辰之后，龙腾回来了。
<br/>　　苏莫和司空炎两人，立刻迎了上去。
<br/>　　“大长老此人应该靠谱，他答应我，会全力助我日后掌控龙族！”龙腾面色郑重的说道，这一次单独面见大长老，他了解了很多事情。
<br/>　　比如，龙帝其实并非洪荒祖龙之子，其次，龙族高层之中，哪些人是龙帝的心腹等等。
<br/>　　“那就好！”苏莫和司空炎闻言，均是点了点头，大长老也是高级神王，而且是龙族之中，实力仅次于龙帝的存在，有大长老帮衬，日后无忧了。
<br/>　　“不过，也有条件！”
<br/>　　龙腾微微一笑，道：“条件也很简单，那就是我要足够证明自己，有接近洪荒祖龙的天赋，这个倒是不急，日后他自然能见分晓。”
<br/>　　“这是小事，有苏莫帮衬，将来超越洪荒祖龙，不是没有可能！”司空炎不在意的说道，苏莫能为龙腾融合一条龙魂，日后就有可能融合第二条龙魂。
<br/>　　“的确！”
<br/>　　龙腾点了点头，继续道：“大长老安排的祖龙大典，在半年之后，这半年之内，会通知各大神朝，各大顶级势力，到时候定然有无数的强者来到龙都城，苏莫，你要不要避一避？”
<br/>　　“我也想过暂离天龙神朝，不过，我担心龙帝会有什么手段。”苏莫叹了口气说道，他虽然没有暴露身份，但是展露的天赋已经足够骇人，龙帝必然对他心存敌意，毕竟，他是龙腾的人。
<br/>　　所以，他若是离开龙都城，真的不是很保险。
<br/>　　龙腾闻言，沉吟了少许，道：“龙帝心机深沉，无法揣测，为了安全起见，你还是待在龙都城吧，待到祖龙大典之时，你不要现身，躲藏在空间宝器之中。”
<br/>　　“嗯！”
<br/>　　苏莫点了点头，目前而言，只能这样了。
<br/>　　“还有，大长老重新为我安排了一处恢弘的府邸，你们都住过去吧，以后没事就不要出来了！”龙腾又道。
<br/>　　“好！”苏莫和司空炎点头，随后，便跟着龙腾，向新的府邸而去。
<br/>　　新的府邸，坐落在龙都城的北部，距离龙帝的吞虚龙府很远，占地数千丈，共有五座宫殿，十数座阁楼和院落，极为宽敞。
<br/>　　而这座府邸，已经改名成了祖龙府。
<br/>　　苏莫和司空炎住进了祖龙府之中，便再次开始了闭关修炼，增强修为实力。
<br/>　　苏莫和司空炎比较清闲，龙腾则是接下来的日子就比较繁忙了，毕竟，他成为了新的祖龙，要筹备祖龙大典，还要经常接见一些前来拜访的龙族高层。
<br/>　　龙腾成为新的祖龙一事，在龙都城闹的沸沸扬扬，可谓是无人不知无人不晓，祖龙府邸也是宾客不绝，热闹非凡。
<br/>　　为了拉拢到更多的强者，龙腾来者不拒，只要是来拜访的强者，他都亲自接见。
<br/>　　一时之间，龙腾在龙族的名头和威望如日中天，大有取代龙帝之势。
<br/>　　只不过，龙腾修为太低，大多数的人，都只是明面上和龙腾交好而已。
<br/>　　所有人都知道，龙腾没有足够的实力之前，是不可能取代龙帝的地位的。
<br/>　　而龙族出现新的祖龙，将要举办祖龙大典一事，在龙族的昭告之下，也以无与伦比的速度，传遍了神域九大神朝。
<br/>　　一时之间，整个神域皆惊，传的沸沸扬扬。
<br/>　　无数的势力在猜测，新的祖龙是何许人也？是如何成为祖龙的？
<br/>　　龙族出现新的祖龙，此事简直惊天动地，毕竟，这意味着将来很可能又要出现一位至强者。
<br/>　　对于祖龙大典，龙族向天星神朝、亘古魔国、不死神朝、混沌神朝、霸天神朝、幽冥神朝等六大神朝，以及轮回神朝六大王宫，和天池神朝八十一个界域的霸主级势力，全部发出了邀请函，邀请这些势力的强者，前来观礼。
<br/>　　对于龙族的邀请，自然不会有哪个势力拒绝，纷纷应允，而后开始安排人手，前去天龙神朝龙都城。
