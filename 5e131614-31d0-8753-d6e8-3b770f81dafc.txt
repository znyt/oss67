　　有副院长亲自解释，没人会去怀疑什么。
<br/>　　所谓有图有真相，那么大一条巨蟒都摆在你面前了，不容你不信。
<br/>　　“老头，戏演的不错，这事情不就圆过去了吗”涂小安暗暗好笑的给副院长传音。
<br/>　　副院长冷哼：“你若不救我孙女，这笔账，老夫跟你没完”
<br/>　　堂堂古学院的副院长，德高望重，地位崇高，却为了这条蛇精，欺骗了整个古学院的学员。
<br/>　　心中的窝火，只有他自己知道。
<br/>　　“弟弟...”
<br/>　　忽然，现场响起了一道冲天般的悲鸣声音，只见一道身影浮光掠影，瞬间来到了罗子雄的尸体面前。
<br/>　　众人一看，大惊，：是罗子杰，天才班的大神学员。
<br/>　　此时大家才想起来，罗子雄可是罗子杰的亲弟弟。
<br/>　　涂小安表情有点难看了起来，别人好糊弄，但这位亲哥哥，想必没那么容易糊弄过去。
<br/>　　罗子杰一到来，看到早就没有气息的弟弟，顿时悲痛欲绝，难以相信，一张俊逸的脸瞬间扭曲起来。
<br/>　　从看弟弟全身的伤势，简直触目惊心，生前是经过了残忍的折磨。
<br/>　　须根之间，罗子杰身上爆发出一股强大的气息，一股气浪自他体外激荡出来，将一众学员给直接掀翻出去。
<br/>　　“我弟弟，到底是怎么死的”
<br/>　　罗子杰一声怒吼，整个人瞬间迸发出令人感到强烈杀意与深渊般愤怒的气场。
<br/>　　大神学员的愤怒，可是能让古学院都颤抖起来。
<br/>　　“子杰，节哀顺变，你弟弟的死，老夫也感到很难过，让蛇妖混进来，是学院的失职”副院长开口。
<br/>　　“蛇妖”
<br/>　　罗子杰猛然抬头看向庞然大物的巨蟒，这巨蟒同样没了生命气息。
<br/>　　“该死，还我弟弟来”
<br/>　　罗子杰周身元力暴涨，只见他一掌朝着巨蟒尸体拍去，犹如平地惊雷，轰鸣声不绝而与，一个十数丈的手掌凭空出现，蕴含强大力量。
<br/>　　轰...
<br/>　　足足将近二十米的巨蟒顷刻间爆炸了，血肉漫天挥洒，空气中充满了血腥味。
<br/>　　“好强...”
<br/>　　众人心惊胆战的看着罗子杰，很显然，弟弟的死，让他陷入了疯狂。
<br/>　　“这人既然不比白君然弱”涂小安看到，暗暗的想。
<br/>　　你可不要怪我，是你弟弟自己先来惹我。
<br/>　　涂小安明白亲人去世的那种悲痛，不过他并不感到愧疚，大家立场不同。
<br/>　　罗子杰一掌将巨蟒轰的支离破碎后，顿时蹙眉起来，愤怒的心情稍稍恢复了一点冷静。
<br/>　　不对，这蛇妖可谓是三阶妖灵，它的肉体怎么可能那么的弱，就算是死了，但是能修炼到三阶妖灵，肉体都是非常的恐怖，早就无坚不摧了。
<br/>　　尤其还是蛇类一族，有蛇鳞片护体。
<br/>　　罗子杰知道自己愤怒的一击，威力非同小可，可也没夸张到将一条三阶妖灵的尸体给轰的支离破碎。
<br/>　　这肉身太弱了。
<br/>　　隐晦间，罗子杰觉得自己弟弟的死有蹊跷。
<br/>　　“院长大人，我弟弟真的是被那条蛇妖所杀吗”罗子杰沉声一问。
<br/>　　副院长看着他那张愤怒而扭曲的脸，镇定道：“这是自然，做不了假，你难道在你弟弟的身上感受不到蛇妖的气息吗”
<br/>　　罗子杰微微一愣，弟弟冰凉的身躯上，却是有蛇妖的气息残留，而且非常的明显。
<br/>　　但...
<br/>　　这蛇妖为什么肉体那么弱，这解释不同。
<br/>　　罗子杰陷入了沉默，忽然，看向了涂小安：“你来说，你当时在场，我弟弟真的是被刚才那条蛇妖说杀”
<br/>　　涂小安不言不语，只是点了点头。
<br/>　　罗子杰死死的握着拳头，指甲都插入了肉内，心中觉得事情有蹊跷，可就是找不到问题所在。
<br/>　　“子杰，你弟弟为救人而死，学院会发抚恤金下来，另外你有什么要求，也可以亲自跟老夫说，能办到的，老夫都答应你”
<br/>　　副院长看着涂小安道：“至于涂小安学员杀蛇妖有功，学院也自有嘉奖”
<br/>　　做戏要做全套，这点副院长自然明白。
<br/>　　涂小安恬不知耻道：“院长，我能获得什么嘉奖”
<br/>　　老头闻言差点没一口血喷出来，他那样说，不过是给众人一个交代而已。
<br/>　　又不是真的将对涂小安这个罪魁祸首嘉奖什么，你一个杀人凶手，自己心里没点逼数吗。
<br/>　　涂小安当着众人所问，偏偏副院长又不能不回答，故作笑呵呵的道：“奖十，不二十块灵石”
<br/>　　他本想说十块灵石，但是杀蛇妖就奖励十块，有点难以服众，只能改口二十块。
<br/>　　“多谢副院长，有妖精闯入学院，这些都是学生应该做的”说着，涂小安朝着副院长伸出了手。
<br/>　　呃...副院长傻眼，真想一掌拍死他，这蛇精故意当众问他要，这是怕他人后反悔。
<br/>　　“很好，很好”副院长摸了摸胡须，压着心中的气，从身上拿出了二十块晶莹剔透，散发浓郁能量的灵石，交到了涂小安的身上。
<br/>　　“古学院果然善罚分明，学生心悦诚服”涂小安朝着副院长深深的鞠躬，隐晦间还挤眉弄眼。
<br/>　　副院长脸色青白一阵，简直就是哑巴吃黄连，有苦说不出，这蛇精太气人了。
<br/>　　分明是在讽刺。
<br/>　　“全都散了”副院长朝着外观的众人挥了挥手，其他的事情可以交给别人去办了。
<br/>　　“副院长，学生还有事情请教你”涂小安看着老头要走，也找了一个借口跟上。
<br/>　　这场蛇妖风波，算是这样告一段落，并没有引气什么人怀疑。
<br/>　　只有罗子杰看着自己弟弟冰凉的尸体，悲痛欲绝，旋即，他眼中一抹坚定：这事情一定没有那么简单，一定没有那么简单。
<br/>　　他目光锁定在涂小安离去的背影上，一阵阴森阴郁。
<br/>　　“大哥，弟弟今年有希望进入天才班，到时候我们两兄弟就都是古学院的传奇学员了”
<br/>　　在数天前，弟弟罗子雄还笑呵呵的跟他这样说。
<br/>　　但转眼，弟弟跟他天人永隔。
<br/>　　可恶...可恶...
<br/>　　弟弟，你的死，大哥一定会帮你调查清楚，不会让你如此不明不白的死。
<br/>　　有一点罗子杰几乎可以肯定，弟弟的死，绝不是刚才那条脆弱肉体的巨蟒所杀。
